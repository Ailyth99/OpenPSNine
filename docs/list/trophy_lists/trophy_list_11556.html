    然而有和神海4截然不同的光照，非常电影风格而且很取巧，仿佛自带PS一样不论视觉效果还是拍照效果都很好，可以用美轮美奂来形容，除了水有点胶质的感觉不太自然，个人比起神海的真实系更喜欢这种风格，单单到这里的话，画面给个突破上限的3.5分都是没问题的。
    不过一开始就说过，缺点也特明显。僵硬的表情，Aloy以外僵硬的动作，怕不是10年前的水准，导致演出十分出戏…这是怎么做到的？有意思的是这游戏所有剧情对话全部都是近距离特写Aloy和npc的脸，谁开口谁就特写全程，基本没有搭配动作或者就点点头挥挥小手，配合各种仿佛迷之微笑的表情(如果是微笑的话)……多来几次会让人产生入戏不能的debuff，偏偏导演还特喜欢用，满屏幕的这种演出完全无法忽视，是个很大的扣分点。
    天气系统很棒，但除了晴天画面全都模糊得半死，感觉效果有点过了，再扣一点分。
    另外建筑的格局大多比较小，除了主线走的几个旧世建筑，大地图的场景设计都比较零碎，这个虽然不至于扣分但比较遗憾吧。

    + 精致饱满的画面
    + 美轮美奂的光照
    - 胶水
    - 僵硬的表情和动作演出
    - 许多天气画面模糊


    系统（4/3）

    让人欲罢不能的最主要原因就是战斗好玩，实在太好玩了，直接突破天际的多给1分没毛病！
    强大的战斗设计独立成一个ACT都没有问题，以至于这一个优点可以盖过其他所有系统做得都不咋地的缺点。从开放世界的角度而言，地平线的系统做得很肤浅，毫无打磨可言，瑕疵一大把，根本不如别家，这是事实。但主轴的战斗确实有足够的特色，挑不出硬伤，何况还有超好玩的副本一样的熔铸厂，可惜有点少，而且后期萎了…

    像这类游戏，很多人对武器丰富的定义大概是，各式各样且动作模组有明显区别的近战武器，对应需要了解敌人各种出招的动作套路，见招拆招并进行输出，远程往往是用来辅助/耍赖风筝的。
    地平线系统的出彩就在于拓展了一个全新的，完全对应远程的整套战斗思路，绝不是变着花样的纠结上弹和后座力的fps，完全不同。游戏提供的不是丰富的近战武器，而是各式各样的远程武器，虽然都是弓弩，但很好的做出了功能和手感的区别；对应需要了解的是敌人每个结构、部位的性质，不仅要打对部位、还要用对武器，才能造成大量的伤害迅速结束战斗。本来一个普通人类面对实力和体型碾压自己的敌人，就应该是用远程武器保证躲避时灵活的输出，哪怕是胶着的近战，也只有远程武器能随时对巨型敌人身上各处的弱点进行精准的打击。这才是还原真正的狩猎，弱小的自己在强大凶悍的敌人追赶和猛攻中求生，灵活切换功能、手感各不相同地武器，配合绳索、绊线陷阱，最终赢得胜利，熟悉以后甚至轻松碾压。游戏没有锁定，全程靠手瞄，敌人是精确到零件的判定，一块一块地打下敌人特定的零件和装甲，引燃油管、超载电池，甚至还能把敌人身上的重型武器打下来自己用，战斗的临场感不要太高，肾上腺素狂飙。拿着各种刀剑斧追着怪跟人家过招虽然好玩，但这种真实系的狩猎感也是一种绝赞的体验。何况这是全新的体验本来就有加分，而系统素质本身已经非常完整而优秀了。
    人各有所好，但以只有远程武器、近战边缘化来指责地平线战斗单调，怕不是游戏史上最大的冤案。
    唯一遗憾的，应该是没有锦上添花的攀爬型boss了。boss战全是强一点的大型精英怪配上其他怪来群殴，有点可惜。

    + 全新的整套远程战斗思路
    + 完备的武器设计
    + 复杂的零件击破
    + 部分副本式地图设计
    + 灵敏的手感
    - 没有设计boss
    - 不是核心所以无伤大雅，但真的有点小多的各种缺点    (空洞的技能树，单调的强化系统，蛋疼的背包扩容，做来不知道干嘛的德智体对话选项，经常跳偏的攀爬，绝对不会被发现的潜行处决，选数量难于上青天的交易系统…等等……)


    剧情（2/3）

    只有2分。
    但游戏的剧本并不差，尴尬就尴尬在这里，让我一腔的悲愤。
    一定要提的是，游戏有非常不错的设定，说优秀也不为过。不能说它大方向有什么创意，无异于人类玩脱、拯救世界这些快被用烂套路，但小方向上确实很有想法，不仅切入点新颖，而且峰回路转，能有不少意料之外的东西……大纲甩出来感觉真的是个怎么讲都讲不烂，随便讲讲都能讲很好的剧本……本该是这样的。

    但这游戏硬生生地就是把这么个剧本讲得糟心至极，仿佛觉得设定很帅就拼命地拿设定糊你的脸，角色塑造基本为零，全靠空口大白话地告诉你啊谁谁谁是一个怎样的人。我已经不想说什么展开了，每一个本该用许多事件串起来最终触发的剧情关键的截点，都被随便找了个理由马上强行触发，简直像背诵大纲一样接二连三地甩在你的脸上，搞得天大的事都跟家家酒一样干瘪至极，仿佛停留在一个人脑子里有个想法想写成故事但是一点事件都编不出来的水平，再配合画面里提过的僵硬至极的演出……导致现世里的一切角色、文化空洞如纸片人，更不用提让玩家对之建立什么感情了。
    反倒是旧世的角色，因为大量设定集中在旧世的历史，加上旧世本来就是靠阅读资料加以想象，可以说其塑造直接建立在设定之上，相比现世丰满得多，至少玩家能代入感情进去。然而旧世角色再怎么重要，这也是一件本末倒置的事情。故事最大的魅力本该在这个经历浩劫又重生、玩家正在游玩的现世世界上的，它和旧世的交织和对比，也本该是为了升华自身，顺便给玩家一点思考……设定本可以轻松出这个效果，结果却成了这样。

    更可气的是，现世塑造的失败并不是没有话题可讲，毕竟本身现世也是刚经历完战争，很多支线也都能看出不少角色的性格、经历、身世背景设定大有故事可挖，只是叙事叙得差……不，是压根就没试图去叙述。
    说实在的，设定原本比地平线简单粗暴数倍的Portal，有了优秀的叙事都能把地平线的剧情按在地上摩擦。主线讲得烂，支线更烂，游戏设定与叙事水准的落差之大可以说让人瞠目结舌，简直可惜得憋屈，只想把导演拖出来暴打，这也算是我玩地平线最大的遗憾了。

    + 新颖的切入点
    + 峰回路转的设定
    - 干瘪的事件安排
    - 翔一般的叙事

    总的来说没有什么洗的余地和必要，地平线的缺点真的很多、很明显，实在给不出9分以上的分数。但只要能忍干瘪的演出，它绝对是一个优点的光芒足矣掩盖这些缺点，能让人玩得十分开心爽快的游戏，设定也可以让人津津乐道好一段时间。
