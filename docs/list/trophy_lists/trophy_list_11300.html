游戏评论

  • person-of-peculi评分 4
    游戏很无聊 要白金 需要用PC远程遥控 通过脚本来挂机 得到毁灭25W建筑的奖杯

    具体流程
    1 通过游戏的内置编辑器 创建地图 地形选简单的 在比较开阔的平原地带 首先放一个出生点 然后在出生点旁放基础建筑1 放194个 在同一个点放 堆叠在一起 最后一个建筑放在比较远的地方 一共是195 在主菜单选择多人游戏 选刚开始创建的地图 进行游戏 最好是一进入游戏 就毁坏了 194个建筑 这样的话因为 另一个建筑 在很远离的位置 未通关 直接在选单里 选重新开始游戏 反复的过程 一直到你得到毁灭25W建筑的奖杯 挂机方法看下面

    2 通过 这个链接 https://www.autohotkey.com/ 下载autohotkey 软件

    3 PC远程遥控游玩 窗口一定要是默认大小
    新建一个记事本 把下面代码复制进去 原脚本来自 https://forum.psnprofiles.com/topic/62974-250000-afk-script/

    DetectHiddenWindows, On

    buildings := 0
    Gui, Add, Button, default, Start
    Gui, Add, Edit, vTEST w250, %buildings%
    Gui, Show,w420 h145, Space Overlords Grinding
    return

    ButtonStart:
    SetKeyDelay, 300
    WinGet, remotePlay_id, ID, PS4遥控操作
    WinGetClass, remotePlay_class, ahk_id %remotePlay_id%
    WinMove, PS4遥控操作,, 0, 0,
    ControlFocus,, ahk_class %remotePlay_class%
    WinActivate, PS4遥控操作

    loop {
    Sleep, 1000
    MouseMove, 100, 100
    MouseMove, -20, -30, 50, R
    Sleep 500
    ControlClick, x648 y530, PS4遥控操作,,,, D
    Sleep 300
    ControlClick, x648 y530, PS4遥控操作,,,, U
    Sleep 1000
    ControlSend,, {Down Down}, ahk_id %remotePlay_id%
    ControlSend,, {Down Up}, ahk_id %remotePlay_id%
    ControlSend,, {Down Down}, ahk_id %remotePlay_id%
    ControlSend,, {Down Up}, ahk_id %remotePlay_id%
    ControlSend,, {Esc Down}, ahk_id %remotePlay_id%
    ControlSend,, {Esc Up}, ahk_id %remotePlay_id%
    buildings += 194
    GuiControl,, TEST, %buildings%
    Sleep 28000
    }
    return

    GuiClose:
    ^k::ExitApp

    打开刚才下载的软件 里面有一个 Script to Exe 的程序 运行 通过此程序 把这个记事本转换为 可执行的脚本 看不见记事本的话 右下角选所有文件

    4 通过多人模式进入游戏 选自建的地图 进入游戏之后 运行脚本 看能否自动重复运行脚本 能的话就可以挂机了 如果不能用的话 看下你在游戏里是按X还是按O确定 如果是按O确定 把脚本下方的ESC改成Enter 再试试 这样应该就可以了 还不可以的话我就不懂了

    5 能自动运行脚本的话 因为是要毁灭25W建筑的 这其实很多 一次游戏流程 就算死的不少到通关也就毁坏了3W左右 这游戏有一个机制 游戏右下角的能量是可以继承的
    所以打故事模式有个简单点的办法 右下角能量挂机到99999 再去打故事流程要轻松的多 如果能量用完了 你 还未通过 再回去挂机攒能量就行了

    6 打故事模式 四个角色选择金黄色的 个人认为比较好用 你要觉得别的好用也可以 这无所谓 记得每次重新进入游戏时一定要选Continue 可千万别选New Game 教程第二次直接跳过就是了

    7 第2 4 5 6星系的最后一个星球 都有一个方尖碑 每个角色都要去打一次 别管对面的霸主 一定要把方间碑打掉 打掉之后通常是直接胜利 这关系到2个奖杯 INPATI HUNTER 和 ULTIMATE GATHERING

    8 基本上通过故事之后 挂机25W的奖杯之后 应该没什么剩余的奖杯了 可能会剩着毁坏XX1000的奖杯 要是在流程里打的不够 直接通过自建地图创建对应的建筑物 去打就可以了 每个1000 很少 就是电站 通讯站 大城市

    9 还有几个关于自建地图的奖杯 这个比较简单 首先自建8个星球可以，然后CHALLENGE CREATOR 自建地图每个模式都要有 这个虽然看似麻烦 其实很简单 在保存自建地图的最后一步 最右边的是更改游戏模式 每更改一次 保存一下 每个选一下就行了

    10 这样最后 恭喜白金 说真的 游戏很无聊 好像我也很无聊。。。
