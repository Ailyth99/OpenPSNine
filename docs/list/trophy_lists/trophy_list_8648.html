顺便说一下，有人人说save不了？。。。。。那可能是你联网了，而且网络较差，，，
网络正常的话 存档是cross save，，，，，没有网络的话 存档都是本地local save。。。。

玩一半网络不正常的话，，，存档会显示失败，，然后你选择存到 save offline 就存到本地了，，，，

读档分几种情况（psv）
1.网络一直很好，默认 读档存档都是cross save
2.刚开始网络好，后来网络不好了。，，，，，cross save 不能读取，但是能读取本地save。。。。
3.网络变差之后，重启游戏，这时网络正常的话，可以读取之前的cross save，也可以读取local save。。
但是这种情况下，网络变好了，如果存档的话，就会全部覆盖为cross save。。


总之，，网络不太好的，就关了wifi玩吧。。。。。。。

ps。。就算网络好，SL一定次数之后也会出现不能读档问题，，，psv和ps4都出现，，，只能退出重进。。。

但是psv的可以 local save，，而且非常快，几乎没有读取时间。。。。ps4的就没有local save，而且可能是我网络一般，SL都很慢，，，没有psv的快。。。
