评分7

先打一下第一印象分好了（不谈游戏难度）。

+ 游戏类型为SLG。
+ Live2D
+ 可以S/L（最多存5个存档）
+ 系统方面有所革新（仔细看的话相比页游版，革新还不少）
+ 报错娘有语音了（虽然只存在于教程之中）
+ 建造公式和页游版大体相同（1小时（页游）= 4天（本游戏））。
+ 扩船位、扩修理槽、买戒指、买家具职人、买资源、买补给舰...等等，都不需要实际课金。
+ 无处不在的妖精。

- 就算是玩过页游版，也需要一定时间上手。
- UI...不谈了
- 战斗中舰娘是不会动的（战斗中没有Live2D）
- SE（Sound Effect）也有些...low
- BUG（如开始的“Press Any Key”按下去后可能无反应，需要多按几下）
- 掉帧。
- 还是脸游。

PS：存完档之后需要手动退出存档界面。
PS2：由于运输船系统，资源是不会随着时间自然增长的。
PS3：战斗指令为默认的情况下可以参考各种页游版的攻略。周回继承一定数量的船 = 变相锁船
PS4：游戏存在Game Over。

SLG游戏爱好者、

勇者。
