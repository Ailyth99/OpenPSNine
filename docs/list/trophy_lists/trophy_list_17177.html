    目测这游戏白金难度应该很高了，不建议白金党入手，白金可能极其痛苦

    优点:
    1.情怀
    2.习惯操作以后其实还蛮好玩的，有离线或在线友人局，fifa和实况太古板了，这个来着更有乐趣

    缺点:
    1.内容太少了，球队也太少了，虽然还有dlc，但是感觉依然不耐玩
    2.报错有点多
    3.剧情不能调整游戏难度，对于不擅长足球游戏的玩家其实有门槛
    4. 自动存档导致有些不小心手残选错了都回不去


    一些tip:
    1.剧情的游戏难度机制暂不明确，试了很多场，大概是你比赛评价高会加星，评价低会减星。第一次失败后重复挑战会降低难度，但是有些场次设定最低难度是2星，所以还是很折磨人(个人感觉就是意大利和德国，巴西和美国还没解锁暂时保留看法)
    2. 多周目的时候最好不要设置跳过已读文本，这样会跳过一些剧情杀，比如踢意大利和阿根廷白送的一个进球
    3. 关于射门，个人感觉火焰射门最舒服。
    4. 关于盘带，个人感觉卡尔洛斯回旋最舒服，因为回旋动作很慢而且期间是无敌的，可以利用这段时间按方块蓄力射门槽
    5. 三衫这个球员很有意思，他的必杀射门有时候会无视距离和守门员槽直接得分，所以可以用来打德国穆勒和意大利齐诺，但是我不清楚这个是概率出现还是有什么隐藏出现条件。目前猜测的条件是下半场替补出场和发动zone
    6. 个人感觉需要优先刷友情的角色: 小翼、日向、施耐德、卡尔洛斯、克鲁伊夫这几个，前四个不说了，克鲁伊夫有破坏者这个技能，效果是对手门将所有能力大幅度减少
