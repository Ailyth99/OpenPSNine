趁着PS4版奖杯列表的出现，咱顺带补一下这款游戏的个人评价。

+++ 游戏基本是djmax系列原班人马制作。
+++ 作曲家也基本是djmax系列的常客。
++ 有一款音游该有的难度。
+ 上手后玩下来蛮爽的，没事就会点开游戏玩几曲。
+ 部分BGA（背景动画）会随着乐曲的内容和风格加上一点小元素。（如：钢琴、骷髅头之类的）
+ DLC曲包值得一玩。

= 判定较djmax系列更松，更适合新人上手。

--- 固定key音。
-- BGA整体是固定的Windows Media Player风格。（Arc那几个除外）
-- 没有CG鉴赏和BGM鉴赏模式（Gallery）。
- 触摸操作除非你是大触不然基本不用碰。
- 摇杆杀手。
- 部分BGA的屏幕闪烁效果太过频繁，不宜长时间游玩（尤其是1000型）。

咱这8分情怀加成还是蛮厉害的，不知道什么时候Nurijoy才会出新音游...
