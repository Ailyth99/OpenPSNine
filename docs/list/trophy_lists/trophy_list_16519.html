    简单测评：
    注意事项：游戏退出或者关机后再打开的话之前所得到的credits会复位变成04...所以要白金的话建议一次性打完OmakeMode再退出吧~~增加credits的办法猜测为一次性游戏时间达到一定时限
    游戏难度：(易)Novice < Original < Maniac < Omake Mode(难)
    接关次数：根据游戏推进，会增加接关次数，目前我已经升到09次了(可见手太残了)
    疑似BUG：个人感觉好像《BOSS 6》奖杯有点问题，没有跳杯。
    如果Omake难度通关完还是没跳的话那应该是奖杯Bug了~不知道会不会有补丁
    难点奖杯：
    《1CC》1币通关看似难，但是打通过几遍后再回到NoviceMode会发现容易一点，毕竟只有五关。
    《Complete Omake Mode》我可能需要更多的币才能通关...有空再试一试吧
    《Defeat the True Last Boss》本以为是1币通关时遇到的最终Boss，但并不是，
    目前还不知道这个奖杯怎么解开，估计是和每关的secrets的发现有关吧~

    2018年11月15日更新了1.01版本，修复了bug~有空的时候试一下
