    轮盘小游戏在让你选择继续转的时候按住不放，指针肯定会停在相邻的下一个位置。先用这个办法把指针停在E
    然后这是手柄转接器Cronus Max Pro用的脚本

    main {

    if(event_press(PS4_R2)){combo_run(hit);}

    }

    combo hit {
    set_val(PS4_CIRCLE,100);
    wait(100);
    set_val(PS4_CIRCLE,0);

    wait(3000);

    set_val(PS4_CIRCLE,100);
    wait(100);
    set_val(PS4_CIRCLE,0);
    }

    指针停在E，让你选择继续时，按下启动脚本。会自动按下，等待3000毫秒之后再次自动按下，这时指针正好停在N(两倍)，很快就能赚够10000分。PS4串流用按键精灵按这个时间设置就可以。其实等待时间可以短点，少转两圈，但懒得试了，上来猜了一个3秒就正好停在N

    如果在要求转之前，一直按住确定键（ps5港版确定键是叉叉，不是圆圈，别搞错了），一定能到顺时针的下一个位置
    也就是说如果是e, 那么下一个就是s，如果是n,下一个一定e

    如果当前是是w,n就可以用这个办法拿分

    那么如果转到e,怎么办？ 开始以后，心中默数大概一秒多一点，就可以转到，w,n,e三个位置，基本很难到s
    这个办法几分钟就可以拿到奖杯了
