  • aikikalaskamaido
    这个是本游戏最刷的奖杯，需要每个角色刷满10万通缉点数。
    目前来说最效率的方法是一个人用两台机器+四个手柄开刷，大号+3小号开搞，手柄不够可以拿PSV或者手机/pc串流代替。
    这个过程大概需要15~18小时。

    第一步：把小号升级为最高通缉等级“Death Bringer” （约5~6小时）
    1.要升级的小号一定要选择角色‘Deserter’，就是胡子壮汉。为啥要选择这个后面会说。
    2.其他三个号全部配合升级小号刷级，模式选择Domination 2Vs2，记得打开本地分屏(GAME MODE -> SPLIT SCR. -> ON)，这样一个机器有两个号，两个机器就凑齐2v2比赛。
    3.2v2是哪个队杀够5人就会取胜，所以杀到四人就换一边杀，一直到本局游戏时间耗尽。这么做的目的是让升级小号尽可能多的造成伤害，因为结算分数很大一部分来自本场的累计伤害值，造成的伤害越多，得到的分数越多。
    4.基本上一局能拿到2000+的点数，一直重复刷到10W点数，升级成Death Bringer。顺便大号杀一次满级小号，可以拿到一个击败Death Bringer的奖杯。


    第二步 开刷5人满级杯（约10小时）
    1.大号把之前的角色删了（outlaw菜单里头按L2），哪怕已经拿到了几万的点数也要删，建一个角色的白板号，一场都没玩过那种。
    2.这游戏有个机制，就是下克上（underdog），由于白板号对满级号造成伤害，可以拿到最高的10倍分数加成。每一局1v1，如果我们累计造成1W以上的伤害，结算时×10，就可以拿到10W分，一场直接满级。
    3.每回合造成的伤害，在400多点，所以需要一局打满25回合，最后再连赢两局拿下比赛，就能打够1w伤害。1v1比赛只要双方每回合结束时都是满血，那么就会平局进入下回合，正常比赛是先赢下两局的获胜，但只要一直平局，比赛就不会结束。
    4..去联机玩1v1模式，对手用满级小号的Deserter。之所以选择Deserter，是因为他能够给自己加血（按△切换技能，再按R2加血，冷却后可以继续加）。
    5.每一回合，不要把小号打死，打了加血，加了血再打，控制好攻击量，保证对小号造成足够多的伤害值，小号又能通过几次加血，在结束时仍是满血，保证回合一直平局。
    6.如果25回合之内不小心赢了一次，那就用小号也打死自己一次，让得分维持在1-1，如果一直放任1-0，过几回合比赛会提前结束。
    7.每回合4分钟，升满一个角色在100分钟多一点。
