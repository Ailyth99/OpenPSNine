    アイラ編Ｂルート（True End）

    このルートに入るには、早くてもアイラ編３周目以降になると思います。
    初めてアイラ編をプレイするという人は、まずスケジュールモードのシナリオを片っ端から読み進めましょう。

    Ｂルートに入る条件は、

    萌葱博士に関連したストーリーを全て見る
    デート24「ふたりだけの結婚式」を見る
    アイラの好感度をMAXにする

    の３点だと思われます。（多分…）
    以下は実際にＢエンドを確認したルートです。


    ストーリー04「親友についての想い出」（読んだよ）
    ストーリー02「アイラが生まれた日のこと」
    ストーリー03「ギフティアの寿命について」
    デート24「ふたりだけの結婚式」（ずっと支えてあげる）
    仕事15「本社にお使い」（大丈夫だと言う）
    デート17「サキコとの再会」（アイラの前に立って守る）
    デート23「両親にご挨拶」（母さんはもう親のつもりだと言う → アイラの意見を聞く）
    休憩15「指輪の話」（どうして指輪なのか聞く → ここに指輪はないと言う）
    デート03「ガチガチのディナークルーズ」（ウェイターに事情を話す）
    デート16「ロマンチックなデート」（イメージで答える）
    デート21「夏祭りをもう一度」（よく似合ってると言う）
    仕事12「サラを訪問」（アイラの言葉に驚いたと言う → アイラのもとへ行く）
    仕事13「サラの回収」（大丈夫だと答える → 家族になっていると思う）
    デート15「アレキサンダーのお散歩」（一緒にリードを持とうと言う → このままにしておく）
    休憩21「萌葱博士からの贈り物」（案外普通のものだと言う → 動物みたいなポーズ）
    デート09「フリーマーケットでお買い物」（アイラが好きそうなお店まで案内する → アイラに確認しよう）
    休憩09「一緒にバスタイム」（ちょっと待って！）
    休憩22「ツカサの誕生日」（言う通りにする）※18日限定
    デート08「腕を組んでお買い物」（腕を組んでみたい）
    休憩16「未来予想図」（どうして子供の話になったのか聞く → 笑顔が似合う気がする → 勉強が得意な子）
    デート12「水族館を満喫できる歩き方」（順路通りに見ていこう）
    デート13「植物園でのんびりと」（世間話をする → やっぱり、いつも通りでいい）
    休憩18「ミチル・ザックとお茶会」（お茶会をしたらいいと思う）


    これでＢルートに行けない場合は、他キャラクターの攻略が必要かもしれません。
