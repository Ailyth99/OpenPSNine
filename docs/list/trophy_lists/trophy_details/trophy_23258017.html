  • chenzhechn
    比 PS3 版加长了一倍，每条路线分成了两半。打完原有的一半和以前一样得黄星，打完后一半得紫星。两个星都拿到，才算完成路线。

    路线的后一半比前一半难很多，所以不要用后一半路线练级。举个例子，第一条路线（NORMAL / 大澡堂 / 대욕탕）原版是 50 层，现在是 100 层。前 50 层敌人经验值是 100，等级是 1~20，后 50 层的敌人经验值也是 100，等级全都是 99。打完 50 层后会自动进入 51 层，但是没必要死磕，早早投降然后去第二条路线（HARD / VOID / 보이드）练级比较划算。

    同理，第二层也是，原版是 100 层，现在是 200 层。打完 100 层后虽然也会自动进入 101 层，但是同样没必要死磕，赶紧去打第三条路线（RISKY / 改造研究所 / 개조 연구실）才是王道。

    第四条路线不用打。嫌第三条路线太长（400 层）可以练到 99 级后去打第五条路线（EVENT / 期间限定 / 이벤트），目前期间限定路线是「Ladies Dungeon / 淑女迷宫 / 레이디 던전」只有 50 层。

    我个人是打完第二条路线的前半段就去把第一条路线的后半段打完了，但是等三条路线的前半段都打完，再回去打各自的后半段可能会更轻松更省时间。

    奖杯图标有误导性，可能是因为直接沿用了 PS3 版的图标……

  • satsukiye_
    建议选个自己喜欢的角色，或者番长、暗影拉比莉斯、完二这种强角，可以全程自动玩，技能刷齐了过不过只是运气问题。伊丽莎白虽然可以靠无限连发C无压力度过前期。但由于AI实在太蠢最后终究是要自己亲自上的，后期用C会被必杀技确反直接吃大伤害，很吃瘪。

    路线选择建议
    Normal 50层→Hard 100层→Normal 100层→Risky 100层→Hard 200层→Risky 165层→期间限定迷宫50层。

    援护带【天城 雪子】，可以主动把血回满一次，很扎实，如果是挂机玩法的话就带【鸣上 悠】。

    技能【彻底分析】【迪卡加】【诸神庇佑】【玛卡拉康】。玛卡拉康一局能反弹1-2管血的伤害，虽然最后10层都是6 7管血的情况下不如神之审判效率，但神之审判送对手满气一次失误能直接把你满血送走，保险起见还是推荐不带。

    如果玩伊丽莎白的话注意【巽 完二】【岳羽 由加莉】【皆月 翔】【阶月 祥】【足立 透】【鸣上 悠】这几个要特殊对策，不能无脑出C。

    打完二在他有50气以后需要出2C用无敌接他的板凳必杀避免自己过早persona break，没塔纳托斯的话拉不开距离，被摸到一套就残了。

    由加莉很白给，只是她可以稳定打塔纳托斯出persona break，碰到以后直接走脸无脑A连死就好。

    中间三个神仙只能在他们没50气的时候出C磨尽可能多的血，一旦有气就只能用A磨血了，绝对不能再出C，等他们自己把气用掉可以接着用A连段把对手冰冻以后再接着C。

    番长我只能说强无敌，你出C他没气可以用波打persona break，有气直接反手电你20000+，贴身他拳脚性能比伊丽莎白优秀得多，可以用中段和下段疯狂择你，是最考验技术的对手了，打不过只能多刷刷属性。
