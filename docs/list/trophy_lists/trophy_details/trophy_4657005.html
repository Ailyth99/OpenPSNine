The Heat is On

Set a valid lap time in One-Shot Qualifying.

奖杯心得 (TIPS)

  • ltiyuki
    推荐和“It Only Takes One Shot”一起拿。
    检查你的比赛设置，确保排位赛(qualify。。。)设置为”One-Shot Qualifying"，然后排位赛就变成一圈定胜负。然后去跑吧，确保这一圈跑完不要有任何的违规，切弯撞车之类的，然后就可以获得奖杯。排位名次与奖杯无关。
    这个可以在任意比赛模式中拿到，包括生涯（CAREER),赛季挑战（SEASON CHALLENGE)，合作（CO-OP),大奖赛（GRAND PRIX)或者在线比赛（ONLINE)。
