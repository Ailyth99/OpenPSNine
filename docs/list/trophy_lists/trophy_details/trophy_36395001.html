  • w_h_hunter
    港澳台服开服玩家，懊悔当初没选亚服，无法秒白……刚开了个亚服号打算零氪打个白金，列一些开荒思路吧（还真别说，零氪这游戏就变成了高难度 jrpg 了）。以下针对零氪、大小月卡玩家，重氪玩家随便 6 + 5 一个就能平推一切了（6 + 5 单角色通常需要 1 万元左右）
    更新于 1.6
    氪金

    1. 优先级：小月卡（列车补给凭证） > 低价大月卡（无名客的荣勋）> 双倍首充 > 高价大月卡（无名客的奖章） > 其他
    2. 氪金优势：
    (1)缩短 5 星获取周期：零氪平均每 3-4 个卡池可以出一个五星，小月卡平均 2-3 个卡池，小月卡&大月卡平均 2 个卡池
    (2)降低角色培养难度：大月卡给的资源基本可以保证缩短一半以上的角色培养时间，并且大月卡送的光锥算是四星里非常好用的了
    3. 大月卡光锥选择：优先「镂月裁云之意」（所有辅助都能用），其他的按照需要换就行了
    抽卡

    零氪玩家一定要做好抽卡规划，大小月卡玩家可以相对不那么严格，如果肯为心仪的角色氪 0 + 1 甚至高命的玩家不用看了，抽就完了
    1. 限定角色抽取优先度：
    (1)队伍不齐全时
    - 强力输出（特指不需要别的角色配合就很强力的输出，如镜流、饮月、刃、希儿） > 生存（存护或丰饶，一个限定角色就可以保证一队的生存了）> 普通输出（需要其他角色配合才能发挥应有的实力，如托帕、卡芙卡）> 辅助（同谐和部分虚无，本质上所有辅助都值得抽，毕竟铁打的辅助流水的 C，但只为了白金或自己不想氪金的话，那么四星辅助就暂时够用了）
    注：如果有布洛妮娅的话，可以降低辅助抽取优先级；如果有白露或杰帕德，可以降低生存位抽取优先级；如果已经有 2 个输出，可以降低输出的抽取优先级
    (2)队伍基本成型后（可以凑齐两队了，即至少拥有 2 个主 C + 2 个生存位）
    - 辅助 > 未拥有属性的输出 > 生存 > 其他输出
    2. 星琼仅用于买专票，通票一概只用送的，不要寄希望于普池抽到想要的角色，除非你是重氪玩家
    3. 对于专武（光锥），限定光锥通常非常强力，有能力的话推荐抽角色就把专武抽出来，即大家说的 0 + 1
    商店

    1. 星芒：通常换专票即可，换光锥的话布洛妮娅的专武是辅助通解
    2. 余烬：专票 > 命运的足迹 > 通票 > 优惠的信用点 > 其他 > 非优惠信用点
    3. 黑塔协议：重氪才换（即每天体力都要用星琼氪满的人）
    4. 光锥呈现（忘却之庭）：基本都用处不大，除非没得可用，不然换出来也用不上
    5. 黑塔商店（模拟宇宙）：基本精 5 的光锥才有正常的五星光锥强度，建议优先把需要用的光锥换满，没必要在这里换通票
    配队

    1. 推剧情配队
    - 无限定生存位角色，通常需要需要两个生存位：1 存护 + 1 丰饶 + 1 辅助 / 副 C + 1 主 C
    - 有限定生存位角色（罗刹/符玄/霍霍），可以把一个生存位换成辅助，即 1 生存位 + 1 辅助 + 1 辅助 / 副 C + 1 主 C
    2. 模拟宇宙配队，前期角色少、练度低，就用剧情队就行了，毕竟模拟宇宙里谁都能很强，只要 buff 选对基本平推了；寰宇狂灾的话要求一些练度了，但道理差不多，最高难度可以等后期练度上去了再打
    3. 深渊（忘却之庭 - 混沌回忆）配队，如果已经到了打深渊的进度了，也不需要看这个攻略了，思路差不多，两队每队基本是 1 生存 + 2 辅助 + 1 输出
    养成

    1. 角色培养优先级：
    不氪大月卡的话资源很紧，只拉一队就行了，一队养好了再针对深渊酌情养第二队
    (1)限定五星：无脑养就对了
    (2)常驻五星：
    - 布洛妮娅：无脑养
    - 白露、杰帕德：无限定五星替代的话，无脑养
    - 克拉拉：推荐养，她的机制确保她基本可以一直用
    - 瓦尔特：没有好用的 C 可以养，即便有好用的 C 他也可以当副 C，缺点是对遗器要求较高，即难培养
    - 姬子：可以养，开荒没 C 的话也只能用，即便后期也可以用来打经验本，并且虚构叙事活动比较适配
    - 彦青：不要养，纯属浪费资源
    (3)四星
    ① 输出（开荒期无限定 C 的话不得不养，但不必养太多，资源谨慎使用，下面只列一些确实值得养的，不过开荒期本质是有啥用啥）：
    - 素裳（比较强力的 4 星，一命就可以养了，突出一个省点儿）
    - 雪衣（抽到可以养，每命都能有不少提升，可以当主 C 也可以当副 C，主打一个击破）
    - 桑博（dot 队可以上场，倍率和伤害都可以）
    - 青雀（高命质变，后期打模拟宇宙繁育命途不得不养）
    ② 辅助（还是那句话，铁打的辅助流水的 C，所有辅助都值得养）：
    - 停云（到手就无脑养，0 命就能用）
    - 艾丝妲（前期就有，没别的辅助就养吧，火弱环境是神，其他环境高命质变）
    - 寒鸦（对于高耗点队都可以上，值得养）
    - 驭空（优先级比较低，因为基本要求手动战斗才能发挥作用，并且非满命的话需要精准配速）
    - 佩拉（高命质变，既可以当副 C 也可以当辅助）
    ③ 生存（不得不养，和辅助同理，养了总归没坏处）：
    - 三月七（开荒时不得不养，如果抽到了克拉拉就可以一直养，否则等到主角转存护了改养主角就行了）
    - 娜塔莎（剧情直接送，直接养就行了，毕竟前期没别的奶了）
    - 玲可（群体驱散很有用，直接养就行了）
    (4)主角：物主一般，解锁火主了直接转就行了，然后把三月七换下来就好
    2. 等级&行迹
    无大月卡：主 C 可以拉满，辅助和生存养到 70 级并突破，不用升 80，中间 4 技能养到 6/8/8/8，其他技能正常都点了就行了，再往上就比较烧资源了，后期 2 队都养起来之后可以酌情拉满
    有大月卡：可以把主队酌情拉满，然后慢慢拉第二队即可
    3. 光锥
    开荒基本有啥用啥，后期每个角色查攻略培养就行了，五星角色就用专武就行了，四星图省事儿也可以用专武即可，同理不氪大月卡就拉到 70 级并突破
    4. 遗器
    (1)前期
    体力不够就全都快枪手四件套即可，两件套用每周给的币随缘刷模拟宇宙即可，只要主属性对的紫色遗器就能凑合用，具体用哪个主属性针对每个角色自行查攻略即可，懒得查或者只为了打白金的话下面给一些通解
    - 主 C：暴击/暴伤衣、速度/攻击鞋、对应属性攻击球、攻击绳
    - 存护：生命/防御衣、生命/速度鞋、生命/防御球、充能绳
    - 丰饶：生命衣、生命/速度鞋、生命球、充能绳
    (2)中后期
    都到中后期了基本都有自己的游戏理解了，直接查攻略或者按照自己的理解刷就行，所有遗器都换成金色的主属性对的就行了，然后副词条就慢慢刷吧，没有尽头的
