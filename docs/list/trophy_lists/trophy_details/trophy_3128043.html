Complete Stealth

Clear the game without entering alert mode (not including events where alert mode is mandatory)

奖杯心得 (TIPS)

正常完成这个奖杯需要熟悉游戏的关卡并熟练操作，还要有一点点运气，也需要在游戏中多存档。不过这项奖杯并不限制特殊道具的运用，也不限制难度，所以如果获得了隐形迷彩的话，这个金杯就显得简单很多了。剧情中几次强制进入警报不计入这项奖杯的条件，如Tanker篇的杂兵战、Plant篇的Emma狙击战、天狗战等。Plant篇中如果不确定是否被发现，可以将游戏存入另外一个存档中(不覆盖之前进度)，再故意自杀，接关画面左下方会显示到目前为止触发过多少次警报 (ALRTNUM—项），在进入护送Emma的狙击战之前，这个数字应该为0。后期剧情中固定的警报也会计算入这个数字中，就不一定能完全作为参考了。
