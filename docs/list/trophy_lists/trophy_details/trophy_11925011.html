    根據實踐和閱讀帖子分析，我的結論是全收集類型的獎盃必須在收集到新的姆依姆依時才會跳。

    大家都知道，如果某一關已經收齊了，那麼過這關時就不會出現姆依姆依收集畫面，也就無法解鎖全收集類的獎盃。我一開始收集時沒有作詳細記錄，後來中途開始記錄了。臨近結束時，根據這個猜想，保留了最後一關沒有全收集姆依姆依的存檔。第一次打完最後一關時沒有跳盃，按記憶補完了之前一關，再重打最後一關，就跳了。

    對還在奮戰的人來說，可以保留一個沒有收集全姆依姆依的最簡單的關在最後，方便補完。

    如果所有姆依姆依收集完後沒有跳盃，並且覆蓋了存檔，那麼這個檔大概就廢了。

    The wording on this one is somewhat confusing so time to clear it up. You don't need to earn the "Perfect!" rank for LocoRoco, Pickories, and hidden MuiMui on every stage all at once on each level. You can get the "Perfect!" rank for LocoRoco in one run in a single level, then get the "Perfect!" rank for Pickories in another run on the same level, and finally get the "Perfect!" rank for hidden MuiMui in the last run on the same level as your two previous runs.

    Getting the "Perfect!" rank for all these collectibles can be pretty tricky at times. If you fail according to the level's end results screen or notice you didn't get a Pickory, a MuiMui, or a LocoRoco mid-level, just restart the level and try again, you'll get it eventually.

    Note:​ ​​​Pickories aren't tracked by the game whereas LocoRocos and hidden MuiMui are, so get a sheet of paper and check each level you get all the Pickories on.
