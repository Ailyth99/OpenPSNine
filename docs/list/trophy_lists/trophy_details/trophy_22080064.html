  • troilite
    根据@xiutime8149，再结合其他人的方法，扩充细化下给小白们（比如曾经的我）看。
    我是<无可触碰> <最后的英雄><恪守原则>3个BUG杯里缺个<无可触碰>，在普通或更高难度的SAM'S STORY章节中，遭遇巨翼蝠时不死亡。重开二周目，通过打蝙蝠前上传云存档，打蝙蝠死了就退游戏下载云存档的方法去速通，结果没跳杯，确定是BUG了。

    下面就开始补充楼上的这个PS5步骤：
    1. 确保所有奖杯已同步并将存档上传至云端。(存档就是第一段我文字描述里二周目通关没跳<无可触碰>的存档)
    2. 切换到第二个帐户。
    3. 设置 -> 用户和帐户 -> 删除用户 -> 删除主帐户(要跳杯的这个账户)
    4.切换用户->添加用户->左边是用户添加到这台PS5，右边是以临时访客身份游玩->选择临时访客的登录->手动登录主账号(要跳杯的账号)。
    5.下载云存档。
    6. 设置 -> 用户和帐户 -> 其他 ->修复凭证
    7. 设置 -> 网络 -> 设置 -> 禁用互联网
    8. 进游戏加载通关之前的存档，这个存档在决定游艇炸不炸再前面一丢丢的开打。
    9. 打到决定潜艇炸不炸，就会立马跳杯，我这边是跳<无可触碰><最后的英雄>。
    10.设置 -> 网络 -> 设置 -> 重新联网
    11.PS回到主界面 -> 右上角“设定”右边的自己账号头像 -> 奖杯 -> 地铁奖杯处option -> 出来几个选项，最后一个“同步到PlayStation Network”。
    12.检查奖杯是否100%
