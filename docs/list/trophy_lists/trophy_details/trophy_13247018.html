Boss Run: World Champion

Beat All Boss Run mode under 25 minutes

奖杯心得 (TIPS)

  • shanshan823
    这个奖杯有个BUG，在打通Boss run之后，游戏会闪退，即使你的游戏时间在25分钟以内，你也拿不到奖杯。截止到评论发布的时间，制作组仍然没有解决这个问题。

    感谢外网的玩家提供的解决办法
    1. 正常通关Boss Run，最后的南瓜头Boss不用打，直接撞上去自杀就行了，亲测没问题。
    2. 游戏闪退之后，重新进入游戏。在我打的时候，即使游戏没有闪退，也不会跳杯。这个时候一定要手动退出游戏，不要Retry，一定不要！！除非你想再打一轮。
    3. 重新打开游戏后，开一轮Boss Run，然后在最开始的大鱼BOSS里，故意死在那里。第一关是蜗牛也没关系，过关蜗牛，然后去打鱼，亲测可行。死在鱼Boss手里之后，点击Back，游戏会再次闪退，但是这时候就跳杯了。

    我就是这么解的杯，不用这个方法的话，我打了4次25分钟以下，都没跳杯。

    给打这个杯的玩家几个游戏建议
    1. 一定要购买快速开火的升级，非常有用。
    2. 多买补血道具，在最终BOSS之前会进一次休息屋，把所有的钱都花光
    3. Cherry + Smakmoffel（第一排第一个）可以合成一个25秒内无敌的道具，很有用
