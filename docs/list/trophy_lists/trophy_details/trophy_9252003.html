MGFマスター

全ての楽曲でウラモードを除く全ての難易度(ＥＡＳＹ～ＥＸＴＲＥＭＥ)をクリアした(ＤＬＣの楽曲を除く)

奖杯心得 (TIPS)

22首歌tv size的easy+nomarl+hard+extreme难度通关和full size的hard+extreme难度过关
tv size里另外还有4个里模式不需要打
full size的2个难度解锁方式是完成该歌曲的Bingo Challenge
另外在tour模式时打完列表的歌曲后会有encore一首full size的hard难度歌曲，打完也计算过关。
所以如果在tour模式直接放入tv size的extreme难度歌曲，打完encore会直接出full size的extreme难度歌曲，打完也计算过关。

luxxsg2016-01-16 00:45
辛苦psn id：yoxi-reborn 等聚聚的开荒。理清了本奖杯的条件：22首歌tv size的easy+nomarl+hard+extreme难度通关和full size的hard+extreme难度通关。

以下是详细解说：
1.开歌。打完TOUR MODE后，你会发现只有12首歌的tv size，通过BACK STAGE——MIRACLE COIN——MUSIC 进去，用MIRACLE币购买余下10首歌的tv size。

2.开难度并完成。
第一种方法：通过完成每一首tv size 歌曲的Bingo Challenge，解锁该首歌的所有难度，并逐一完成tv size的easy、nomarl、hard、extreme难度，full size的hard、extreme难度。
第二种方法：通过完成每一首tv size 歌曲的其中一部分Bingo Challenge，解锁该首tv size歌的extreme难度；再在TOUR MODE打该首tv size歌的extreme难度，触发encore，完成该首歌对应的full size的extreme难度。从而避开完成全部Bingo Challenge。

（1）先看第一种方法：
ONE-MAN LIVE进去后，移动到想完成的tv size的歌曲，拨动摇杆可预览Bingo Challenge界面，按start键即可切换到Bingo Challenge详细界面。如图：


Bingo Challenge的九宫格，大致要求如下列列表：
1.歌曲结束时右下角的槽值为MAX2.在三个不同的场地演出过3.槽值百分比总量累计达到1000%
4.达成100连击（COMBO）（应援道具列表第一个可以降低难度，效果是safe判定不中断连击）5.NORMAL难度达成GREAT以上评价6.累计打中50个星形音符（MIRACLE FEVER TIME期间会出现星形音符）
7.NORMAL难度下所有音符的判定都是safe以上（含）（并没有比较理想的应援道具，只能练习了）8.HARD难度达成EXCELLENT以上评价9.累计获得10000枚MIRACLE币

1宫格简单，不说。
2宫格在ONE-MAN LIVE刷歌时留意换场地即可。
3、6、9宫格都是一个字，刷。不停刷歌即可。
5、7宫格都要求是在NORMAL难度下达成。遇到比较难的歌曲，建议分别达成。
8宫格要求是HARD难度下达成，右下角槽值打到MAX（200%），基本都能保证EXCELLENT以上评价了。

推荐打通TOUR MODE再刷。打通TOUR MODE后，可以选择其中一天，塞三首歌选择NORMAL难度一起打，主要留意打4、5、7、8宫格。因为选择的是NORMAL难度，所以出encore的话，encore歌曲是HARD难度，encore歌曲达成EXCELLENT以上评价，则能顺带解开8宫格。部分歌曲的4、7宫格不好刷的话，建议回到ONE-MAN LIVE单独刷。

一般来说，用TOUR MODE一起刷3首歌，最后会剩下2宫格没达成。此时回到ONE-MAN LIVE 刷，既更换场地也更换难度（easy、nomarl、hard、extreme），完成tv size的4个难度。（更换难度后，看到该难度有显示成绩，则证明该难度打过了，可更换下一个难度）（EXPAND-PLAY HARD、EXPAND-PLAY EXTREME、URA-PLAY HARD、URA-PLAY EXTREME 不用打）此时，完成该首tv size歌的4个难度和Bingo Challenge。该首歌对应的full size 可以留到最后统一刷full size的时候刷。

（2）第二种方法：
依照（1）达成1、2、3、4、5、6宫格，解锁tv size歌的extreme难度和full size歌的hard难度。此时，通过ONE-MAN LIVE 刷tv size歌的4个难度和full size歌的hard难度。剩余full size歌的extreme难度。

full size歌的extreme难度游玩，通过TOUR MODE的ENCORE来完成。
TOUR MODE的ENCORE有如下特性：

1）返场表演（ENCORE）的出现条件是：TOUR MODE任意一天打的所有曲目都发动了MIRACLE FEVER TIME。
MIRACLE FEVER TIME的发动条件是：打该曲目时，在曲目演唱到高潮的起点时（一般为右下角时间指针指到粉色区间的开始位置），下方的进度条完成率超过100%，此时，下方进度条会显示MIRACLE FEVER TIME，进入MIRACLE FEVER时间。当然，有些曲目有多次高潮，也就存在多个高潮的起点。
<全E必出ENCORE，但有一个Great未必出ENCORE的原因就是这里>

2）ENCORE的选曲不以评分高低为判断标准。例如一首歌MIRACLE评价，另一首歌EXCELLENT评价，可能ENCORE的歌是EXCELLENT评价的那首歌。

3）ENCORE的选曲，系统默认首选当天的曲目中，没有发布过special message的歌。

4）当天的曲目若是easy、nomarl、hard难度，ENCORE的是相应full size歌的hard难度；当天的曲目若是extreme难度，ENCORE的是相应full size歌的extreme难度。

所以，简单而言，full size歌的extreme难度游玩，为：TOUR MODE，选择需要打的歌的tv size 的extreme难度，并保证发动了MIRACLE FEVER TIME （保证EXCELLENT评价肯定出ENCORE）出ENCORE，从而游玩过full size歌的extreme难度。
