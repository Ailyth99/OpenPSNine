  • gmremix
    四个结局达成条件：
    结局1：最终成绩排名3以下，正常打完流程即可：大金人摔下一句金句然后走掉；
    结局2：直接选第六章开始游戏，难度调最低，在不续关的情况下打完第六章就跳杯：大金人眼睛变红了；
    结局3：各关卡达成S / A / A / A / A / A：神秘人登场；
    结局4：个关卡达成A / A / A / A / A / S：三代G登场。
    结局四有传闻是除了第一关外，任意一个关卡达成S，其他关卡达成A即可。欢迎尝试……但第六关其实很简单、流程也最短，所以建议还是第六关达成S就能触发最后一个结局了
