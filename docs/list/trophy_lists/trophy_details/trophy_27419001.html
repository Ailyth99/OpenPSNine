    目前版本除了CD（CD可能是因为用的IOS端的最新版本已经过滤掉debug指令，待验证）和任务模式外 皆可debug模式选关并解锁所有奖杯，甚至包括解锁用的星星硬币。如果是老玩家且只为了奖杯可以用此方法。否则极为破坏游戏体验。
    因为前两部作品和ios一样所以，Apple tv指令如下（SEGA偷懒这里直接给差评）
    X代表跳跃
    前两作分别输入上下左右X, 二代↑x3↓x3←→←→X。直接选最后一关看完制作人名单都算通关。
    方法周年庆模式进入在press start 直接输入。3k不知道要问taxman他们。

    白金可用debug，建议二代，激活选关菜单后进入在sound test激活debug在选择bgm代码 01 09 09 02 01 01 02 04后，然后再输入04 01 02 06获得全部翡翠。然后最终关突入用debug调出50rings变身通关就行。
    击杀杂鱼，ring收集奖杯同理，任意作debug打完即可，但必须要把这关过掉才可以。如果更懒可以直接调过关牌。

    更新3k选关，press start 界面↑行x3 ↓x3 ↑x4 成功后有rings声音，选择sound test 进入。debug需要输入3k的发售日期，具体和2代同理

    关于本作的诚意。
    怎么说呢。我不觉得这次remaster有什么诚意。你能明显感觉到12 CD就是直接IOS搬过去。音乐质量和3对比差距直接就出来了。3k明显用的是模拟器音色也就算了，而且还是录的。另外都2022年了，Michael Jackson以及相关人员的版权音乐你说你SEGA搞不定也行，你要有诚意ICE CAP, 游乐场，Launch base zone你音乐重写一下行不。结果最担心而且不想听到的 Prototype那一版真的被直接拿来用了。所以，除非是真的很有情怀或者对这个系列特别有爱。综合以上因素不太建议入手首发，可以等打折。
