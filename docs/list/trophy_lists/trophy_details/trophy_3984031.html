Environmentalist

Kill 3 targets or more using an explosive object in any level

奖杯心得 (TIPS)

油桶油罐煤气罐一类的爆炸物不是一打就炸的，因此最好开着Overkill的情况下去做这个
scottzheng2015-04-19 22:02

开第二扇门时 让队友射掉门锁 自己破门 然后立刻射向防爆兵边上的煤气罐
多试几次 肯定能解
