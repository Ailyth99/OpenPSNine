    Free Dance 全COMMU FEVER组合奖杯早见表


    对着本列表从头到尾打一遍FreeDance即可解锁所有COMMU FEVER组合的奖杯,适合TA

    图例:
    难易度 : 初见就应选择的难易度
    MAIN : 该曲主舞者
    PARTNER : 初见选择的搭档
    |_NONE : 该曲不支持更换or选择搭档
    |_不限 : 随意选择
    |_指定 : 初见必须选择的搭档
    |_AUTO : 初见对应难度下默认设置
    AUTO / 指定 : AUTO或指定搭档所对应的角色
