@godaisky推荐的小夜打法很受用，而且经过我在小夜最后一管血怒了后，死了超过30次亲自测定。基本确定出地面火焰奥义躲是基本不用想的。要么就屈死她让她不要再地上怒，或者地上怒了直接大太刀连击蓄力打倒地一直倒到死，不然她一旦缓过来立马开始无缝奥义后，那么就是只能堆石头无限奥义顶过去顺便输出打死她。 理提到的可以躲过的空中dash和躲避到到她背后的方法不推荐，这办法一旦在她背后下地时遇到她一轮火焰完无缝再接一轮的时机时（基本很容易撞上）就是活靶子。顺便说决死判定和bug防御锅在这里无效，决死不用说了对一击断刀都无效的。断刀不死的bug锅料理也没有用，因为火焰太密集，打到你第一次断刀时就会顺便硬直或者浮空然后立马后面的火顺便鞭尸第二次第三次...，bug加防御锅可以让你断刀不死是其实活用了决死bug，由于断的刀无法在防御，所以之后在首创硬直时还是会被打死...

○小夜
果然是最难BOSS，小夜攻击套路很简单，碰到地面必杀不用砥石可以说就直接GG了，躲啊甚么的都是骗人的
个人推荐练到18级，装备上朝岚村正(烈风)、孙六兼元、雾雨村正(丁半斩),这些都是攻击比较高的太刀，准备2~3个伊予砥石（三河有售）
地面时，尽量避免正脸扇的防御，一下半灵力伤不起，多使用闪避饶后打，一开场的灵符不用管，多点用居合斩来解决
小夜怒是有套路的，要么是被烟花炸弹炸到，要么是自己丢3次烟花炸弹，所以在第一次飞天后尽量在第一次丢烟花炸弹的时候把她打下来，千万不要直接把她炸下来， 怒了以后就看运气了。。如果地面必杀了用烈风靠近，等时间过，运气好不出地面必杀注意稳扎稳打，空中必杀毫无威胁除非不小心被大烟花炸弹给爆了_(:3」∠)_

1.山城(黒猿)
2.山城 → 伊賀save(大百足)
3.伊賀back → 伊勢 → 尾張save → 美濃(虎姫)
4.尾張back → 三河 → 遠江(小夜)
5.尾張back → 伊勢 → 伊賀 → 大和(土蜘蛛)
6.尾張back → 美濃 → 飛騨 → 信濃(血狂毘沙門)
7.信濃 → 甲斐 → 駿河save(龍神)
8.駿河back → 伊豆 → 相模 → 武蔵(狗頭龍)
