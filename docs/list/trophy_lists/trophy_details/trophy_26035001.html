The Chosen One

Unlock all other trophies.

奖杯心得 (TIPS)

  • altair1165111
    1.拿起拖鞋[Silver The Slippers]
    2.检查床头柜，离开房间下楼进入客厅。使用电话并拨打666-3498[The Call]
    3.与电视互动[Black Box]
    4.与大时钟进行交互，与电视顶部架子互动获得钥匙。与书架左下角的书籍互动，上楼梯。去父母房间（最开始房间），楼下的电话会响起，回到楼下客厅，手机旁边有一个黄色物品，互动[Strange Thing]
    5.回到你父母的卧室(最开始房间)和梳妆台互动得到电池，和墙上的歪图互动得到打火机，然后回到楼下的客厅。进入楼梯旁边的门口。与挂衣互动获得毛衣，在鱼缸中互动获得钥匙。回到楼上。进入左边的门，这是你的卧室。检查门旁边的梳妆台以获得钳子。离开房间，你会听到敲门声[Knock Knock]
    6.去父母房间的浴室。与马桶互动以获得钥匙，再次下楼梯，与屏幕最左侧的墙互动，互动直到用钳子打开水。回到你父母的卧室，然后回到浴室并与水槽互动会给你个瑞士军刀。回到楼下，进入走廊左侧角落的门进入厨房。与炉子互动以获得钥匙。回到楼上[Always busy]
    7.进入走廊右侧的门进入办公室[Father’s Office will pop]
    8.办公室里，与书架中间的盒子互动；使用代码 1956 打开它。里面是另一个电池。去客厅和桌子上的电视遥控器互动。现在回到父母的房间，屏幕会闪烁并摇晃。进入浴室[Other Dimension]
    9.进入走廊并与墙上的扬声器互动以接收试管。前往你的房间并与右侧的铁架子互动以获得烧杯。回到你父母的卧室，然后是浴室，与屏幕左边的管道互动，获得一些油脂。回到你的房间，与墙上的轮子互动，然后回到楼下，这时候你会以为是游戏机坏了，但别担心一会儿会恢复正常。回到客厅，进入楼梯左侧的门口，进入地下室。
    与右侧墙上的电线互动以获得树脂，然后与木板互动以获得木板。回到你父母的浴室，与浴缸互动获得烧杯的血液，然后与浴缸左侧的凹槽互动获得器官。现在再次回到楼下，然后回到客厅。与左侧的凹槽互动，出现个眼睛[The Eye]
    10.回到你的卧室与你的电视互动。打开后会出现一堆符号。回到楼下，与墙上的时钟互动，输入时间 04:35，然后与楼上的扬声器互动，铃声就会响起。回到你父母的卧室。在你父母的梳妆台上有一个时钟，与它互动，回到楼下，去时钟输入 07:45。与旁边显示耳朵的凹槽互动，并获得钥匙[Traveler Key]
    11.返回你的卧室并再次与铁架子互动。会显示一段话注意看里面有个化学物质代表的符号，最好把这段话拍下来。回到楼梯，左边的门会让你回到正常的世界。去最右边的门，来到房子外面。与热水器互动，输入刚才那段话所说的元素在元素周期表中序号，这时候可以等一会温度上升到一定程度再走，然后返回走廊。与中间的柜子互动获得个碎片。去你父母的卧室，和左边的衣柜互动，得到另一个碎片，然后回到浴室和水槽互动。会把两个碎片粘起来。回到里世界。走廊最右边的门进入厨房。过场动画后再次与水槽互动，听见咔哒一声后退出厨房进入地下室，穿过大门。进入下一个场景并与房间中间的基座互动，继续前进后来到外面[Where's Everything]
    12.与左侧的平台互动来到最后的场景，屏幕两边各有一台机器，在左边机器的答案是 A = 50，B = 37.5，C = 75，D = 62.5，右边机器的答案是 A = 50, B = 50, C = 50, D = 100。都输入正确之后点击中间的电梯，动画后最后选哪个都行[Without Escape]
