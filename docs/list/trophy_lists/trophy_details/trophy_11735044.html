  • kashima_naoki
    You must discover every gray square on the entirety of both maps. They must each be at 100%.

    Use the map indicator located above your health/stamina numbers and to the right of your thirst percentage. It looks like an antenna with radio waves around it. When you enter an unclaimed square, the antenna will disappear, meaning you've now discovered it. If the antenna is visible, then you are in part of the map you've already discovered. Also, LISTEN for a chime when the antenna turns on/off.

    Return to base camp to save any uncovered map. You can hit the back button and RETURN TO BASE CAMP to save (so you don't have to run back to a transporter, but you will drop any loot you picked up).

    You will have to run in all sorts of nooks and crannies to claim certain squares. To discover one particular square, I had to hang & shimmy along a ridge above the river in map 2 to discover it.

    There is a patch of deadly dust on map 2 behind Ruins 3. Try to run into it and then run back out to claim any gray squares you're missing. You don't have to run too far but running into the river will Force the game to revert back to last save. This will cause you to lose any progress. If you die, be sure to choose to return to home base, as this will save your claims. Do not revert back to last save.

    This is a bit of a grind. You'll have to choose when to fight or move on to conserve O2 for more exploration. I found it helpful to try to run the perimeter. Save. And then shave off rows of gray squares inside that perimeter. Use the GUIDE MARKER feature to mark the beginning, then the end of a row you want to discover and run the line straight as possible.

    Be safe and sure that you can make it back to base camp to save your work.

    The achievement unlocks after you discover your last square and return to base camp. The percentage for each map is shown by hitting Start and RB over to map, in the top left corner under EXPLORED.

    I never used any equipment to discover parts of the map except for one gray square only. RAMPS can be created at a Gadgets Workshop -> under Other Units. So it may be helpful to have some ramps on you to use as you try to climb rocks. You can jump from the end of the ramp. But be aware that the ramps are hard to place as it needs to be a near flat surface.
