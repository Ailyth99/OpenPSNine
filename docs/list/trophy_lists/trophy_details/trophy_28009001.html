  • w_h_hunter
    先 Normal 难度把除了 hard 难度通关的奖杯都拿了，除了熟练度奖杯，其他奖杯都无任何难度，那些条件杯如连击数等根本不用刻意去触发，基本属于流程杯了。
    每张图最好仔细搜刮一下或者直接照着全收集攻略走，确保战技、符文、花和支线任务别遗漏。
    第一章是肯定需要打两遍的，因为有个符文需要等放电 NPC 入队后才能拿。剩下的主线章节尽量确保一遍 S 评价，在 Normal 难度下还是不难的，多召唤 NPC，多用战技，死亡后采用回到标题的方式读档而不是直接检查点复活。建议武器轮着用，多用 C3、C4 技，如果熟练的话也可以多用一用反击、逆转等。练到 C 就换下一把。都 C 了之后就挑一把喜欢的一直用，如果到 SSS 了就换下一把。这样在最后通关时平时战斗如果打的够花里胡哨的话，应该能有 1 把 SSS、4 把 C 和 1 把 A+，能稍微节约一些刷熟练度的时间。
    最后一章可以采用章节内 SL 的方式触发所有的结局，而不是重复打 4 遍最后一章（虽然依旧无法避免要打三次 BOSS 连战，但至少不用清半天小怪了）。
    其他奖杯清理干净后最后刷熟练度，没什么好方法，参见那个奖杯 tip 或者搜一搜视频吧。其实很简单，容错率也很高，手残也能轻轻松松刷。后面熟练了的话，一把武器 C -> SSS 大概需要 1-1.5 小时吧，很枯燥。
    最后是 hard 难度通关，其实 hard 难度不难，我感觉只是略微增加了怪的血量和攻击力，大幅降低了平时回血灵魂的爆率，以及限制了技能只能点 430 点（实际上到通关都不一定能点出这么多技能）。只打主线，每章都不管不顾直接速通即可，武器挑一把自己用着顺手的，只升级那把，技能优先往下开，因为速通的话资源不一定够点出所有技能。
    最后一关其实有个小技巧，如果贴着湖边走、跳进房子里走，能避免掉所有的战斗，直接跑到 NPC 那儿，一个敌人不杀的通关，能节省不少时间，毕竟后期的敌人都挺烦人的
    一切顺利的话，应该能在 35-40 小时白金
