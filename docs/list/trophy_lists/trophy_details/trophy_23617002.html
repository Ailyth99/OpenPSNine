  • alexshyun
    ♩♩♩♩ dd o
    节奏为：4个四分之一拍（♩，注意八分音符是♪），2个二分之一拍（d），1个一拍（o）。
    打开辅助，外圈从舒张到与内圈重合为1个跳动，从重合的时候开始打节拍，光圈1个跳动为四分之一拍，2个跳动为二分之一拍，4个跳动为一拍
    具体方法：按（起手）- 按 按 按 按 X 按 X 按 X X X 按 （每个节点均为一个跳动，X节点为不按键）
    注意留意每次按键除了起手都会有一个对应的音符，没有就是节奏错了
    重复以上5次

  • mryuui
    兄弟们，这个杯真的不难，选择爆米花关卡，你不用理会音乐，只需要按住option打卡助手。然后看着左上角，先起手按，再跟随圆圈收缩连续按四次，再隔一次圆圈收缩按一次，再隔三次圆圈收缩按一次，这一系列操作叫做一个combo，combo成功了之后会有简单的动画提示。每次combo完成按L2重启节奏。我理解其他人说的可以失败指的是只要不按出其他的combo动画，就没问题，但是每次combo内必须是正确且连续的，不然你按一辈子也按不出来combo动画。
