  • lepingps
    原理：利用困难难度的焰返技，终结BOSS

    以下详细方法：

    将游戏难度换成困难难度，装备五本マズルカの結魂書（中文版估计叫玛祖露卡的结魂书，魔女叹愿里55000一本），装满人，我装的全是未转移过的普通99级，阵法中我用的是攻击阵，好像可以增加状态免疫力

    心脏BOSS将按以下固定模式攻击，我们对应操作，BOSS行动缓慢，都是在我方行动后它再行动的

    1.カンパニュラの鼓動 … 全体に即死攻撃，第一合全体即死攻击，我们全防御好了，会死几个，没啥大事，也总有人活下来
    2.アストルムの鼓動　 … 全体に状態異常耐性激減，第二合BOSS对我们用状态，不会对我们造成伤害，所以，活着的人复活死去的人，多余的人继续防
    3.メルムの鼓動　　　 … 全体に弱攻撃＋全状態異常，第三合BOSS还是不攻击，只用状态，我们对第二合里中状态，导致用不了技的小组使用道具解除状态
    4.フェーヌムの鼓動　 … 全体に物理耐性とガード率の激減，第四合还是对我们用状态，我们尽量让更多的人可以恢复到使用技的状态
    5.ウンブラの鼓動　　 … 全体に焔撃，第五合对我们全体火焰攻击，我们全部用焰返技，据说困难难度最高可以返50万伤害，这一合就可以结束BOSS了，不过我打的这次，活的人能力不强，只返几万，最多的10万多点，没干掉BOSS，没干掉还有机会的
    6.ロサテンプスの鼓動 … 全体に魔撃耐性とスタン耐性の激減，第六合不会伤害我们，我们可以复活人，恢复状态
    7.テネスの鼓動　　　 … 全体に攻撃力・防御力半減，第七合，我们继续准备，下一合就真正结束BOSS了
    8.オルニットの鼓動　 … 全体に打・焔撃＋深淵＋スタン，第八合，BOSS又用火焰了，我们活人全部焰返技，我这一次打出两个20多万，干掉了BOSS
    9.以后攻击模式未知，如果第8合还没干掉，不如重新来过吧。
