  • narbfius
    解出来R=215，G=229，B=999，但是流程图里G变成299，大概率bug
    R：结合表格得到图案对应的数字，根据柜子上图案的方向转换成字母
    G：走过的墙壁会消失，把脚印对应方向的墙全部消失就能得到数字
    B：第一个时钟看成一半的箭号，第二个字母结合前面表格的位置，第三个对应魔方公式u=up，r=right
    三个字母相乘的数字对应表格得到LOVE
    三个字母对应数字从右往左再想加，对应表格得到JIN

  • satsukiye_
    到下层，桌子左手边的楼梯上去，最里面的书架：眼球1
    下层桌子右手边的电视里：眼球2
    巨大的球：调查
    电脑1：按下Enter键
    门右边的墙角：眼球3
    地上的纸：调查
    先通过地上的表来计算图形对应的数字
    方块：输入4
    三角：输入3
    圆：输入7
    六边形：输入9
    从中间的神秘人背后发现五角星，对应0
    然后根据柜子上和墙上的角度差旋转数字得到字母
    蓝色柜子：输入MOM
    绿色柜子：输入HELP
    黄色柜子；输入ME
    门：输入215

    Lock1 解锁

    神秘人：调查
    电脑2：按下Enter键
    可以穿墙，在地图右上角，一开始看到的图画右边墙角：眼球4
    走过的墙会消失，再次走过会重新还原墙。三个可调查的物体○□◇，站在边上的脚印处，对应的方向尽头各有一幅画，把中间碍事的墙消除掉就可以看到对应的数字。
    方块：得到2
    菱形：得到9
    圆圈：得到2
    门：输入229

    Lock2 解锁

    球体：调查
    电脑3：按下Enter键
    门左边的墙角抬头：眼球5
    教典：调查
    根据表针还原箭头
    按钮A：左
    按钮B：下
    按钮C：右
    教典：调查
    看之前的字母表，以N为原点，选择M O T H四个字母相对于N的方向
    按钮A：左
    按钮B：右
    按钮C：下
    按钮D：上
    教典：调查
    根据Left，Right可以推断出U指Up，R指Right
    按钮A：上
    按钮B：右
    按钮C：上
    按钮D：左
    按钮E：右
    门：输入999

    Lock3 解锁

    215 * 229 * 999 = 49185765，根据字母表49 = L， 18 = O， 57 = V， 65 = E
    球：输入LOVE

    Lock4 解锁
    把三个数字反着相加512 + 922 + 999 = 2433，根据字母表2 = J，43 = I，3 = N
    球：输入JIN
