On Target

Beat the game without using a Continue.

奖杯心得 (TIPS)

比听起来简单 这里的continue指的是defcon升到1之后再死或者被发现会进入倒计时 60秒没有关闭最后一个房间的机器就会发射核弹 gameover 这个时候会问你要不要continue 所以只要这关打得不太好直接退关重来就ok
