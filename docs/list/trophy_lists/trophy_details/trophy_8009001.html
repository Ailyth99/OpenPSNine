奖杯心得 (TIPS)

  • kaikaiiiiiiiwu
    所有奖杯都可以在一局内解开：

    1. Play Monopoly -> Local Mode -> 随便选个存档
    2. Living 和 Classic Mode 都可以，Classic 观察棋盘比较方便，刷杯推荐 Classic。
    3. 下一步，规则选右下角的 Property Improvements: House Rule，不要选 Classic Rules
    4. 1 Controllers for all，一个手柄轮流控制所有角色。
    5. 一路按添加 Player 2 到 Player 6，按开始，这样六个人都是你操作的，但成就只计本 ID 不计 5 个Player。
    6. Roll 点决定顺序，不是第一就按重 Roll。可以快速重试先 Roll 个 11 12点，再看其它 5 个 Player 能不能 Roll 小。

    进游戏：

    A. 棋盘上地块格有不同的颜色，相同颜色的两块或三块为一组，称为 sets。
    B. 所有人都从右下角出发，我们的目标是让主 ID 角色获得成就要求的各项内容。
    C. 一阶段操作所有角色依次出发，每到一个无人地块时，按 Auction 开启拍卖，其它角色都放弃，主 ID +1 就能拍到。同时就可以开始执行步骤G了。
    D. 运气不是极差第一轮都会跳 Precocious 奖杯，实在脸黑的话单开二周目解。
    E. 有一个颜色的地块拍齐以后，与任意 Player 交易，一块钱卖给对方两个不同颜色的地块，再买回来，解锁奖杯。
    F. Player 钱不够要破产了也可以用步骤 E 送钱。
    G. 开始在棋盘下边和左边猛建酒店，路过就建，连按 R1 直到图标变为红房子，上边和右边务必空着别动。直到跳杯。这个是最费时的。要求是两组颜色上面建满地产，我们选便宜的四组哪个先满算哪个，选便宜的是为了减少支援破产 Player 的次数。
    H. 在此过程中应该有 Player 开始破产，用步骤 E 给它们续费，直到跳 M700 租金杯。
    I. 上面两个杯跳了以后，开始回收红房子并随意在地块上放置绿房子。由于我们是 House Rule 模式，可以直接放四个绿房。放到 24 个跳杯。
    J. 看看 M100 奖杯有没有解，没解就按R3主动破产四个 Player，留一个和主 ID 二人转直到跳杯，如果已经解了就直接让它们全破产赢下棋盘，跳三个胜利杯。
