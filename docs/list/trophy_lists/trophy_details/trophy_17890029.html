  • aliveinsnow
    方法：
    1. 用枪把鸡笼上的油灯打掉，燃烧的油把下面所有鸡都给烧了
    2. 随便找个油灯（倒处都找得到），丢到鸡笼里烧掉所有鸡
    3. 丢任何燃烧道具，使鸡笼点燃（燃烧瓶、地狱火，游戏商店里带火焰标识的消耗品）
    4. 诱饵、照明弹、信号枪，用这几样工具打掉鸡笼上的油灯，油把下面的鸡给烧掉

    地点：
    三角洲地图：
    1. 冶炼厂去往炮台方向的，围栏旁边，固定鸡笼一个。地图上“Works和冶 ”这两字中间位置，也是固定鸡笼一个
    2. 地图右下角农场，白色“小别墅”与红色伸缩门的之间位置，那个伸缩门地上有一堆玻璃渣子，固定鸡笼一个
    3. 地图右侧马场，在“Run”字上方的房子那，固定鸡笼一个
    4. 在地图上的教区位置，从地图上看，右边4个小别墅的第二个房子右边方向（离房子很近），固定鸡笼一个。还没完，“教”字上方，固定鸡笼一个
    沼泽地图：
    1. 地图上，在Davant牧场，从右边河流到牧场最近的白色小别墅旁边，固定鸡笼一个
    2. 地图上，屠宰场的位置上也有一栋白色小别墅，旁边固定鸡笼一个，这位置不远处还有一个烦人的狗笼
    3. 地图上，Stillwater镇（原名静水弯地），在字母ter的位置，固定鸡笼一个
    4. 在地图的左下角，棚屋位置，在字母s的位置，固定鸡笼一个
    5. 在地图中上部分，畜牧场的“畜”字上方位置，红色伸缩门旁，固定鸡笼一个。还没完，畜牧场旁边的Alice农场（爱丽丝农场），在地图上字母“A”的位置，固定鸡笼一个。（这两鸡笼距离刚好斯帕克斯有效射程250米...）
    6. 地图右上角，Scupper湖（原名泄水湖），这个位置经常走过但容易被忽略，在地图上，字母“S”的左边位置，有一小栋房子，房子旁边，固定鸡笼一个
    7. 位于达文特牧场 最大的房子与有红色伸缩门的白色围栏处，也就是达文特牧场的正门有一个固定鸡笼
    （有几率是黑色笼子）
