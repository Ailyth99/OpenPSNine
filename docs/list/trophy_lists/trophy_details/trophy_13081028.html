Professional Rider

Complete a race with Normal Physics in any game mode

    搬一条（看到TIPS空白心里痒！）
    What this means is to set the Physics option in Race Options to Normal. This will cause the bike to be more difficult to control, especially off jumps. The requirement here is to finish the race, winning isn't required. Using Rewind and falling is also okay. If struggling with this, go to Single Event and select a simple custom track where there aren't many turns. Just lag in the back of the pack and finish the short race to earn the trophy.
    这意味着将“比赛选项”中的“物理”选项设置为“正常”。这将导致自行车更难控制，尤其是在跳跃时。这里的要求是完成比赛，不需要获胜。使用倒带和下落也是可以的。如果遇到困难，请转到“单一事件”，选择一条转弯不多的简单自定义赛道。只要落后于队伍的后面，完成短距离比赛即可赢得奖杯。
