    如果要做符文之语装备的话，放入带孔装备的符文一定要按照先后顺序放入，如果用魔盒打孔，白装不能有“超强的”之类的前缀，只有A5第一个任务打孔可以用“超强的”等带前缀的装备，每个角色只能任务打孔3次（每个难度1次），如果3次机会都用完了，也可通过在线开小号找朋友带的方式快速完成打孔任务。
    魔盒打孔公式（B站 @爱迪生玩游戏）：
    塔尔(7#) + 书尔(10#) + 完美黄宝石 + 普通衣服 = 同类型带孔衣服，孔数随机
    拉尔(8#) + 安姆(11#) + 完美紫宝石 + 普通武器 = 同类型带孔武器，孔数随机
    拉尔(8#) + 书尔(10#) + 完美蓝宝石 + 普通头盔 = 同类型带孔头盔，孔数随机
    塔尔(7#) + 安姆(11#) + 完美红宝石 + 普通盾牌 = 同类型带孔盾牌，孔数随机
