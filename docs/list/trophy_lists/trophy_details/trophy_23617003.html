  • alexshyun
    跟慢速相反
    o dd ♩♩♩♩
    节奏为：1个一拍（o），2个二分之一拍（d），4个四分之一拍（♩，注意八分音符是♪）。
    打开辅助，外圈从舒张到与内圈重合为1个跳动，从重合的时候开始打节拍，光圈1个跳动为四分之一拍，2个跳动为二分之一拍，4个跳动为一拍
    具体方法：按（起手）- X X X X 按 X 按 X 按 按 按 按 按（每个节点均为一个跳动，X节点为不按键）
    注意留意每次按键除了起手都会有一个对应的音符，没有就是节奏错了
    重复以上5次
