巨乳は美しくない……

内藤昌秀の合戦会話を全てみる

奖杯心得 (TIPS)

这个算是这游戏最麻烦的奖杯了，因为你不知道到底有多少个是需要发生合战对话的，也不知道发生的条件是什么，所以LZ参考了UCG的PSV专辑

事件发生条件备注
内藤昌秀 と 木下昌直 壹味方龙造寺路线限定
内藤昌秀 と 木下昌直 贰敌对武田路线限定
姊小路良赖 と 内藤昌秀 壹敌对
姊小路良赖 と 内藤昌秀 贰味方
前田庆次 と 内藤昌秀敌对
内藤昌秀 と 远藤直经敌对
内藤昌秀 と 足利义辉敌对
内藤昌秀 と 鬼庭良直敌对
内藤昌秀 と 宇佐美定满敌对
内藤昌秀 と 织田信长敌对
内藤昌秀 と 三好长庆敌对武田ハート
内藤昌秀 と 龙造寺隆信敌对
内藤昌秀 と 大友宗麟敌对大友ハート
羽柴秀吉 と 内藤昌秀敌对
高桥绍运 と 内藤昌秀敌对大友ハート
古田织部 と 内藤昌秀敌对

上面已经都标出了合战对话的发生对象跟条件。只有5个对话是限定路线的，因此在攻略路线时尽量收集，实在不行最后在专门收集也是可以的。需要注意的是，足利义辉是在攻略了一个路线之后再新开始时会有足立路线，而足立路线攻略之后玩别的路线，足立会在游戏中期加入你，因此跟足立的对话最好是能够在通关足立路线时收集到，还有就是大友家跟龙造寺家，这两家离武田家都很远，因此在攻略各两个路线时，要是没有收集到的话也不用急，后期专门开新档，带上几个2000兵的武将一开始就直奔目标，今本上30回合左右就能够遇到了。
