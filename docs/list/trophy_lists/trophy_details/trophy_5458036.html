lvyxia2015-06-11 09:24
来自A9， 做个笔记备忘：

在游戏中总共有7次和Endurance Crew船员对话的机会，需要完成所有可能的对话才可以解开这个奖杯，一旦错过只能重开游戏，二周目不能补缺，必须完整完成，所以千万小心。

有时候这些人互相之间还在谈话，需要等待他们谈话结束才能和他们中的每个人进行对话。除了和Whitman的对话以外，和其它人对话完事后右下角会有TR标志显示正在储存。

#1：沿岸森林：游戏初期，在Lara和Whitman试图开一扇大门后，Whitman会让你去拿50个残料（salvage）去升级斧子，先不要去寻找残料直接和他对话。假如你在之前已经拿到了超过50个残料，别从他身边走开直接开始对话。
#2：沿岸森林：在你找到50个残料回到Whitman那里，先和他对话再升级斧子。假如你进入了营地菜单，快速暂停然后重新读检查点。这里有个比较麻烦的地方，假如你在出发寻找前一个残料都没拿过，那么50个正好。假如你之前拿到过一定量的残料，之后收集到50个也不一定会触发对话，你只能继续寻找残料直到劳拉说：” Now Let's see if i can fix the axe.“为止，之后回到Whitman那里才可以触发对话。
#3：山中村落：找到Roth时他正在击退狼群，在过场动画后你的任务会更新为：找到Roth的包。回来之后Roth会给你一个攀登斧让你去攀登面前的岩石，先别去攀登和他对话。
#4：山中村落：拿到绳子弓箭后你会再次回到山中村落并遇见Roth，任务会更新为寻找驾驶员，先别走开和Roth对话。
#5：船难海滩：在乘坐缆车来到此区域后，你的任务会更新为找道具，别离开，和Sam，Reyes还有Jonah对话。
#6：船难海滩：找到道具回到营地后，Whitman会出现然后给你一把新的弓箭，在离开之前和Whitman，Sam，Jonah和Reyes对话，Reyes在小船上。
#7：船难海滩：从Endurance轮船回来后，在去研究基地前，和Whitman，Reyes还有Jonah对话。

messiahex01-06 21:32
1. 从 #3 号与罗斯腿伤恢复后拿到登山镐开始即可，走完过场回头与罗斯对话。

2. 注意与每个 npc 对话结束后，右下角都会跳出『TR』保存标志。（#1#2 不跳保存标志，大概这也是为什么奖杯无关）

3. 千万！千万！不要在与 npc 对话后立刻传送！！！对话完务必继续往前推进度推到下一个场景。尤其是 #4 条刚拿到打火石旁边又有火堆，容易想着去把之前的回收了。

4. #6 条拿到的船帆滑轮回到基地后，最好不要跳过与约拿的对话过场。过场结束劳拉拿到复合弓会自动坐火堆边开启升级菜单。稍等一会背景音中约拿会开始说第二段话，此时再起身约拿就处于可对话状态了，否则约拿可能会卡住不能对话。如果真的卡住了，尝试直接使用火堆传送来回刷新一下。

5. 承上，为避免 3. 的情况，在与约拿对话前先不要与其它人对话。

6. 利用存档位切换或云存档或 U 盘多备份。

--------来自一个打了 3.5 遍的苦逼的经验猜测，不知道哪条触发 bug，以上全做了处处小心才跳的杯。
