  • goldencorn-1996
    遊戲個體類型：恐怖+轮回攻防型
    游戏好玩程度：5/10（个人）
    游戏整体难度：3/10（个人）
    游戏通关时间：50小时
    游戏推荐程度：2/10（个人心得和时间长度的结果）重复性极高

    存档共用 能4用在5&不能秒跳杯但可以卡点跳杯

    遊戲整體概念：
    ①像楼上老大们所说，第一件事情就是先去设定把全部辅助开和满

    ②游戏至少5周目，顺序周目分别要
    -hardcore 硬核模式(收集43录音器)
    -hardcore NEW GAME + 硬核模式存档新周目(NG奖杯和硬核NG奖杯)
    -contagion 传染模式
    -normal 最低难度（最终boss门口存档）
    -normal NEW GAME + & DLC
    Normal和NG之后还没解决全装备奖杯，直接去DLC，注意武器被破坏剧情之前解决棍的全部技能，还有其他各种错过杯    (收集8个录音器)
    -normal maximum security (解决完dlc全部，最后杯来白金)

    自发现和推荐
    1.游戏我个人会先搞电击棍的第三排第二个，因为很多怪都会挡你攻击（吃时间+烦）直接打断了，不止剪短闪避次数和时间而且怪打你两到三次直接变一次，超方便
    2.游戏发现BUG，在有一边电梯出来有呕吐怪，打死他不要上楼，走回去开门再进来，电梯会重置，开电梯又有呕吐怪，可以这样刷钱！（在传染模式超方便！）
    3.如果以这样bug来运算，可以直接四周目解决
    -a.hardcore模式
    -b.hardcore模式NG
    -c.contagion模式
    -d.normal with maximum security + DLC （记得收集43+8还有全装备）

    ③再强调注意错过杯 43+8 ，错过很麻烦，没save真的要重新打

    ④打riot mode解决五十回合（一次就过，shotgun加魔力手套牛B）
