Speed Thrills

ELITE PASS any A+ Class Challenge

奖杯心得 (TIPS)

  • ohmuro_himawari
    玩家通过提升等级，可以逐步开启A+ Chanllenge，它们隐藏在赛事地图中，当看到地图上有个隐隐约约的“>”状记号时，触碰一下即可开启。它们均为竞速赛事，每个挑战代表一个赛道，速度级别为A+，须跑十圈。

    A+速度级别是游戏中除区域生存赛中最快的速度级别，对于像本人这样基础较差的玩家而言，A+实在是太快了，一眨眼功夫车就开到墙上了，非常难以操控。更要命的是在A+挑战中，AI的强度有了明显的提升，即使在较为拿手的赛道中，跑5、6圈还是最后的情况屡见不鲜，即使领先了，一次小小的碰擦都可能让前排AI赶上1到2秒以上。难道此模式的AI就是无懈可击吗？答案当然是否定的。虽然AI的稳定程度极高，几乎很少碰擦，但是它们有两个主要的缺点：

    1.它们较少用落地加速
    2.它们并不是每次都走最短分支

    这种情况下，为获得此奖杯，玩家首先肯定还是要选自己开得最好的赛道，其次就是要选落地加速机会多、最短分支又较为好走的赛道。只要玩家能保证次次用上落地加速，并且都走最短路线，即使有一些碰擦，对抗AI仍有不小的胜算。
    这种情况在使用道具上又有了新的要求。

    第一，如果获得了武器道具，除非玩家是在和第一第二名缠斗，否则应立刻吸收掉。在这个模式里，打击处于中下游的对手没有任何意义，即使把他们击退，甚至击破，没有任何好处，唯一产生变化的就是多了几个会在你后面捅娄子的麻烦。然而在A+级别中使用落地加速的机会更多，通常能量都是嫌少的，那么吸收道具而得来的能量槽，可以让玩家多用一次落地加速，每多用一次，玩家就相对于所有对手多推进了一点儿，其重要程度可见一斑。若在能量不够的情况下获得加速道具，都吸收掉它。
    第二，自动驾驶将会是该模式非常贴心的朋友，但是千万不要指望它能来进入某个特定分支。即使是在分支直前使用，它也经常会绕到较远的路线，那就没有意义了。就算是以一个必定撞墙的架势强行手动进入分支，再立刻使用自动驾驶，它都会在一阵匪夷所思的调整后以最快速度完美通过所能想象到最难走的弯道。

    个人第一个金掉的A+挑战是Empire Climb，但是在这里要推荐的又是“Sol”赛道了。想必如果挑战过Sol区域挑战赛的话，A+级别的速度肯定已经难不倒玩家了。而这条赛道不仅符合分支和落地加速的条件，更发现即使是A+级别下的AI，居然也是会掉崖的。经过试验，FEISAR Speed赛车、走左右右的路线、尽可能使用落地加速、不掉崖，差不多从第四圈以后，就可以保持3秒左右的领先了。可能有朋友要问为什么最后一个分支要走非近道？毕竟在A+级别下左行的危险系数和右行不可同日而语，又是需要跑到10圈，更何况试验下来只要没有失误最后分支走非近道也不会让对手拉近太多，那么毕竟我们的目标只是拿奖杯，又不是比奥运会，当然还是要速中求稳了。
