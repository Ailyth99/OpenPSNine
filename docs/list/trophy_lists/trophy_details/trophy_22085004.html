    冰之金字塔第一层密码：
    蓝↑ → → ←
    红→ ↑ ← →
    白← → ← ← ← →
    黑→ ← → → → ←
    燕子旁边是女神和猫头鹰
    黑豹在最左边
    女神的旁边是燕子和黑豹
    雕像摆放顺序为：黑豹 女神 燕子 猫头鹰 乌龟

    冰之金字塔二
    女神的右手边是会飞的东西
    左手边是不会飞的东西
    燕子和海龟因为晚到所以在外侧
    雕像摆放顺序为：燕子 猫头鹰 女神 黑豹 海龟
    这里的左手边指的是女神像的左手边，按照站姿是反向的

    冰之金字塔三
    3座石像将会给予提示
    要留意，3座石像中有1座石像在说谎
    1女神在最左边，海龟是从右边来数第3座
    2燕子在最中央，猫头鹰是从右边来第2座
    3黑豹是从左边来第2座，猫头鹰在最右边
    第二第三句都包含猫头鹰，故1是真话。
    又因为海龟在最中央，故第二句是假话，第三句是真话。顺序为：
    女神 黑豹 海龟 燕子 猫头鹰
