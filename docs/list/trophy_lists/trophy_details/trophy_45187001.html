    转的psnp的速白攻略，半小时左右就能白，如果已经玩过的可以速白，没玩过还是老实打一下吧
    From Main Menu
    Options
    Text Speed ---> Fast
    Skip ---> All
    Scroll to the very bottom to Unlock All Scenarios ---> YES
    Choose Option Chapters 5 - 7
    Choose Option 7. Myth
    From Main Menu
    Select 'Read' to start the game

    Jump to:
    7. Myth
    Some Time Later - Raid
    (Press R1 twice to skip)
    True Ending Reached (Trophy 1/31)
    Revelation Mode Unlocked

    From Main Menu
    Options
    Revelation Mode ---> ON

    Everything is now ready to get all trophies in a few minutes.
    Your objective is to jump to every section that has a red X through it, signifying an ending. This won't always give you a trophy, but is required to unlock the Easter Egg trophy.
    All the endings so far should have coincided with Keys in the upper right hand corner of the section selected which will net you the 'key' trophies.

    Clean Up Section
    Jump to:
    1. Start
    Recall
    Choose Option 1: Forget it...

    Jump to:
    2. Dense Mist
    Fourth Lock: Follow the Scream? – Unlocked!
    - Pick Option 2: Keep still.
    Continue to end of section for trophy

    Jump to:
    4. Infiltration
    Decide Arrangements
    - Pick Option 1: Agree with Hashimoto.

    Jump to:
    4. Infiltration
    Decide Arrangements
    - Pick Option 2: Enter prefab myself

    Jump to:
    5. Wit
    Investigation I
    Choose every option to unlock first batch of 'truth' trophies

    Jump to:
    5. Wit
    Investigation II
    Choose the options you haven't yet to unlock the rest of the 'truth' trophies

    Jump to:
    6. Darkness
    Despair?
    - Pick Option 2: Call for ‘that person’

    Almost done

    Now to get the two alternate endings. Both of them are in Chapter 6. Darkness
    Jump to:
    6. Darkness
    Despair?
    - Pick Option 3
    Alternate End 1 Unlocked

    Jump to:
    6. Darkness
    Answer God
    - Pick Option 3
    Alternate End 2 Unlocked

    Lastly
    From Main Menu
    Album
    - Pick 27 (Easter Egg)

    Platinum Unlocked
