    目前已经把等级提升到115级了，不可以直接通过代码方式获得。
    现在的方法是首先到了100级，然后通过 setplayerpos -97500 -51700 46000 代码到Tek Cave 这个地方，地下是个火山口，下去后看到一个需要放东西才能打开的大门。
    具体的道具通过如下代码获取：

    Broodmother_Alpha.PrimalItemTrophy_Broodmother_Alpha'" 1 0 0 1 0 0

    Gorilla_ALpha.PrimalItemTrophy_Gorilla_Alpha'" 1 0 0

    cheat giveitem "Blueprint'/Game/PrimalEarth/CoreBlueprints/Items/Trophies/PrimalItemTrophy_
    Dragon_Alpha.PrimalItemTrophy_Dragon_Alpha'" 1 0 0

    cheat giveitem "Blueprint'/Game/PrimalEarth/CoreBlueprints/Resources/PrimalItemResource_
    ApexDrop_AlphaCarno.PrimalItemResource_ApexDrop_AlphaCarno'" 1 0 0

    cheat giveitem "Blueprint'/Game/PrimalEarth/CoreBlueprints/Resources/PrimalItemResource_
    ApexDrop_AlphaRex.PrimalItemResource_ApexDrop_AlphaRex'" 1 0 0

    cheat giveitem "Blueprint'/Game/PrimalEarth/CoreBlueprints/Resources/PrimalItemResource_
    ApexDrop_AlphaMegalodon.PrimalItemResource_ApexDrop_AlphaMegalodon'" 1 0 0

    cheat giveitem "Blueprint'/Game/PrimalEarth/CoreBlueprints/Resources/PrimalItemResource_

    cheat giveitem "Blueprint'/Game/PrimalEarth/CoreBlueprints/Resources/PrimalItemResource_
    ApexDrop_AlphaRaptor.PrimalItemResource_ApexDrop_AlphaRaptor'" 1 0 0

    道具都获取以后都放到控制台里面，并且点一下前面的第3个道具。这时大门就已经打开了，进入到洞穴里面会看到很多怪，按下 DestroyWildDinos 可以直接瞬间去除所有里面的怪，然后一直顺着大陆走到一个蓝色光的洞穴，最尽头是一个发光的平台，按下中间的三角提示等待一会儿退出游戏。新的DLC的alpha奖杯同时会跳，一直等待到显示结束画面开始滚动制作人列表，SKIP掉后再新开游戏，发现可以导入原来的100级人物，导入后就可以再输入升级代码升级到115级，然后跳杯！

    其实解这个奖杯之后才发现原来这游戏最后的目标就是逃出这个地方，原来以前一直玩的就是个Alpha测试版本啊！！。。

    1. 删除你的保存数据。
    2. 选择主机/本地游戏，选择岛屿地图，然后玩单人游戏。
    3. 暂停菜单，用L1+R1+Square+Triangle一起调出管理框。
    4. 现在再次打开管理员窗口，输入'Gmbuff'，然后按右边的管理员命令按钮确认。
    5. 现在进入你的库存，将等级提高到125级，以获得最大生存者奖杯。
