解锁10张唱片骑师电台的乐曲专辑。

奖杯心得 (TIPS)

  • chenzhechn
    这个奖杯要求玩家使用特定的三名角色通关，方可解锁各个音乐专辑，但解锁条件并没有写全。游戏里写着「？？？」的地方需要玩家自己补齐。

    1、《拳皇 13》：「伊丽莎白」「琼（King）」「夏尔美（Shermie）」
    队伍名称：French Natives / 法国出身 / フランス出身 / 프랑스 출신
    很好理解，都出生于法国。
    感谢 u/Xelae93 分享！

    2、《拳皇 12》：「草薙京」「凯（K’）」「阿修（Ash）」
    队伍名称：Legendary Heroes / 历代主角 / 歴代主人公 / 역대 주인공
    KOF 主线故事「大蛇篇」「音巢篇」「阿修篇」的主人公。
    这个是最好猜的，首先猜到的就是这个。

    3、《拳皇 11》：「伊丝拉（Isla）」「凯（K’）」「柯隆（Krohnen）」
    队伍名称：Bad Behavior / 特立独行 / 素行不良 / 품행 불량
    看起来都是不良少年……简体中文版应该是把队名和谐了。
    感谢 u/kaedesilly 分享！

    4、《拳皇 2003》：「坂崎百合（Yuri）」「雅典娜」「伊丝拉（Isla）」
    队伍名称：New Girls-High-School Team? / 新生女子高中队？ / 新生女子高校チーム？ / 신생 여고 팀?
    看起来好像都是大姑娘。不过队名里有个问号也挺搞笑的……
    感谢@HIBIKI_MOE分享！

    5、《拳皇 2002》：「德洛丽斯（Dolores）」「暴龙王（Dinosaur）」「克拉克（Clark）」
    队伍名称：Grapplers / 投诸九霄云外 / 投げっぱなし / 내동댕이
    好像都是投技型选手……虽然我不是很确定德洛丽斯……
    感谢 t/swagnarok 分享！

    6、《拳皇 2001》：「坂崎良（Ryo）」「柯隆（Krohnen）」「安琪尔（Ángel）」
    队伍名称：Bike Enthusiasts / 摩托资深爱好者 / バイク大好き / 바이크 애호가
    他们都喜欢摩托车。
    感谢 u/Banes5150 分享！

    7、《拳皇 2000》：「拉蒙（Ramón）」「安琪尔（Ángel）」「暴龙王（Dinosaur）」
    队伍名称：Mexican Reunion / 墨西哥·再度集结 / メキシコ・再結集 / 멕시코·재집결
    KOF 14 墨西哥队原班人马。
    这个也很好猜，这个是第二个猜到的。

    8、《拳皇 ’99》：「明天君」「库拉（Kula）」「克里斯（Chris）」
    队伍名称：Juvenile Troop / 少年组 / 年少組 / 미성년 조
    都是小朋友。
    感谢 u/sayo-bao 分享！

    9、《拳皇 ’98》：「二阶堂红丸（Benimaru）」「罗伯特·加西亚（Robert）」「七枷社（Yashiro）」
    队伍名称：Hunks / 仪表堂堂 / 男前 / 미남자
    这几位确实都是有着独特魅力的帅哥。
    感谢 u/Classy860 分享！

    10、《拳皇 ’97》：「瞬影（Shun’ei）」「莉安娜（Leona）」「八神庵」
    队伍名称：Rampage Warning / 失去控制 / 暴走注意 / 폭주 주의
    他们都会暴走。
    感谢 u/Alvin92 分享！

    11、《拳皇 ’96》：「不知火舞」「梁（Lương）」「神乐千鹤」
    队伍名称：Classic Beauties / 沉鱼落雁 / トラディショナルビューティー / 전통적인 미녀
    东方的长发熟女组合。日语原名太随便了，简体中文翻译只好自行发挥了……
    感谢 u/xALBARx 分享！

    12、《拳皇 ’95》：「特瑞（Terry）」「温妮莎（Vanessa）」「德洛丽斯（Dolores）」
    队伍名称：Guardians / 监护人 / 保護者 / 보호자
    他们家里都有个娃。
    这个有点难度，一般玩家可能知道温妮莎已婚，但不一定知道她还有个娃。这是第三个猜到的。

    13、《饿狼传说》：「特瑞（Terry）」「安迪（Andy）」「瞬影（Shun’ei）」
    队伍名称：Disciples of Tung Fu Rue / 唐福禄的弟子们 / タン・フー・ルーの弟子達 / 텅푸루의 제자들
    特瑞、安迪和瞬影都是唐福禄的弟子。
    感谢 u/TJLynch 分享！

    14、《侍魂》：「拉尔夫（Ralf）」「马克西马（Maxima）」「安东诺夫（Antonov）」
    队伍名称：Buff Bods / 体魄强健 / フィジカル / 피지컬
    就是大块儿硬汉的意思呗……
    感谢@HIBIKI_MOE分享！

    15、《合金弹头》：「拉尔夫（Ralf）」「克拉克（Clark）」「哈迪兰（Heidern）」
    队伍名称：First-Generation Ikari / 初代怒队 / 初代怒チーム / 초대 이카리 팀
    就是 KOF ’94 的怒队。
    感谢@HIBIKI_MOE分享！

    16、《Neo Geo 其他作品》：「特瑞（Terry）」「坂崎良（Ryo）」「草薙京」
    （只解锁专辑，没有特殊组队）
    SNK 在 Neo Geo 时期的三大格斗游戏：饿狼传说、龙虎之拳、拳皇。
    感谢 u/Strawberryboi2 分享！

    17、《SNK 其他作品》：「拉尔夫（Ralf）」「克拉克（Clark）」「雅典娜」
    （只解锁专辑，没有特殊组队）
    SNK 在 Neo Geo 之前的早期街机游戏《怒》和《雅典娜》的主人公。
    感谢 u/Strawberryboi2 分享！
