    这奖杯有BUG，友站上有人讨论过，原文解决方法:
    Step 1: Pass first three worlds using the red character (Including bonus levels in Yellow and Red area)
    Step 2: Pass Deep Marsh - Area 1 with red character
    Step 3: Exit Game, start game in slot two, Pass first three worlds using the yellow character (Including bonus levels in Yellow and Red area)
    Step 5: Exit Game, start game in slot three, Pass first three worlds using the blue character (Including bonus levels in Yellow and Red area)
    Step 6: Exit Game, Select red character, pass the first boss for red character
    Step 7 Exit Game, Select yellow character, pass the very first level for the yellow character.
    Step 8: Pass the bonus level for yellow character's first world
    Step 9: pass the first boss for yellow character
    Step 10: Pass Deep Marsh - Area 1 with the yellow character
    Step: 11 Exit game and select Blue character
    Trophy Unlocked on both versions at the world map.
