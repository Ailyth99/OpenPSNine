  • cdmikechen
    首先你需要个篝火：
    cheat giveitemnum 39 1 0 0
    旁边随便用拳头打一点木头放进去

    然后需要个解药：
    cheat giveitem "Blueprint'/Game/PrimalEarth/CoreBlueprints/Items/Consumables/BaseBPs/PrimalItemConsumable_CureLow.
    PrimalItemConsumable_CureLow'" 1 0 0
    注意这里没空格，是因为P9显示问题我手工回车的

    保险起见退出游戏备份下存档

    最后再来个水蛭：
    cheat summon Leech_Character_Diseased_C
    注意这里可能水蛭会跳起来把篝火打坏，所以离远点刷水蛭。

    这样水蛭会吸附到你身上，跳到篝火身上烧死水蛭，喝了解药，跳杯。
