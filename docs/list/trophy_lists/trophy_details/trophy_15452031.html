Martial prowess

Slay the Owlking boss using only kicks

奖杯心得 (TIPS)

  • hackjak
    枭王动作和移动速度都慢，招式大开大合。踢的方法为+为防误操作可以把武器盾都卸掉，建议二周目或新开周目第一次打普通版枭王时解，英雄版BOSS第二阶段胸口炮留地雷比较难走位踢。
    攻击时机:主动小碎步追来的交叉斩接平刺破绽很大直接迎着交叉斩走到背后踢一脚看它向哪侧蓄力，走背后踢两脚。突进刺如果接单刺一下可以踢一脚。第一阶段发胸口追踪弹，如果是面对玩家发可以走位绕背踢三脚，背对玩家则可翻滚到它正面等飞弹飞出踢一两脚
