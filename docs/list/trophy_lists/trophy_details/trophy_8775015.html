  • luna_noire
    普通结局（即坏结局）只有8章，回避普通结局的方法是：
    1. 在第5章自由选择帮助黑白绿三国女神时，在紫国城镇触发 IF 和 Compa 的两个事件；
    2. 帮助黑白绿三国女神的任意一位后，回到紫国，此时紫国城镇会添加一个事件需要触发，如果你直接将三位女神都帮助的话，这个事件将无法触发；
    3. 在第6章紫国城镇触发 Peashy 的两个事件。
    如果5个事件全部触发，你应当有下列关键道具：Nep's Pudding，Compa's Syringe，IF's Notebook，Stuffed Doll，Peashy's Drawing。
    这样就会在第8章末尾触发不同剧情，走向好结局/真结局。

    进入第9章，首先你要拿到绿国的地下城 Nekutoki Forest 的添加计划书。这个计划书在白国的地下城 Luji Plateau 里。进入 Luji Plateau 向东北方向走，跳下台阶，你能看到2个可捡起的道具，其中一个就是 Nekutoki Forest 的添加计划书。

    Nekutoki Forest 的添加计划书的使用要求是：
    Jet-Black Scales：地下城 Luji Plateau 的普通/变化模式下捡到，或杀死其中的 Death Stalker 掉落。
    Meow Claws：地下城 Haneda Mountain Peak 捡到，或杀死其中的 Meow 掉落。

    接下来在 Nekutoki Forest 里捡到地下城 Keraga Dimension 的添加计划书。
    Keraga Dimension 的添加计划书的使用要求是：
    Plaid Dolphin Tail：杀死白国地下城 Ario Plateau 中的 Plaid Dolphin 掉落。
    Magical Contract：地下城 Ario Plateau 捡到，或杀死其中的 Hachibei in the dung 掉落。

    在 Keraga Dimension 中可以找到地下城 Pii Shii Game Factory 的添加计划书。
    Pii Shii Game Factory 的添加计划书的使用要求是：
    Elastic Thread：地下城 Keraga Dimension 中捡到，或杀死其中的 Mega Spider。
    Ganglord Mask：地下城 Anonydeath's Lab 中捡到，或杀死其中的 Ganglord Panther 掉落。

    只要你在前八章将每个关联主线剧情的地下城里的道具都捡过了，正常游戏进程到这里你应该有了 So Shal Forest，Graphic Pass，Vida Dimension 这三个地下城的添加计划书。

    So Shal Forest 的添加计划书的使用要求是：
    Lost Holy Cloth：地下城 Kobaba Ruins 的变化模式（Dungeon Changed）下捡到。

    Graphic Pass的添加计划书的使用要求是：
    Fragrant Thread：地下城 Pii Shii 中捡到，或杀死其中的 Queen Spider 掉落。
    Bird's Kingly Crown：地下城 Magma Cave 中捡到，或杀死其中的 King Cardbird 掉落。

    Vida Dimension 的添加计划书的使用要求是：
    Cat Paw：地下城 Pii Shii 中捡到，或杀死其中的 Tom。
    Molten Stone：地下城 Metroid Shelter 或 Magma Cave 中捡到，或杀死 Metroid 中的 Magma Stone 掉落，或杀死 Magma Cave 中的 Magma Crab 掉落。

    此外还需要在第9章剧情进展到打败 Fake People Heart 之后在紫国大地图右下方的海滩处捡到 Powerlevel Island 的添加计划书，使用要求是：
    Magical Beak：地下城 Graphic Pass 中捡到，或杀死其中的 Nue 掉落。
    Green Block：地下城 Extradimensional Space 中捡到，或杀死其中的 Terist 掉落。这个地下城为剧情地下城，是神次元世界的最后一幅剧情地图，所以你要进去，把东西刷出来，然后出来开启 Powerlevel Island 然后看完隐藏事件，再回到 Extradimensional Space 与 Boss 对战。

    六个地下城 Keraga Dimension、Pii Shii Game Factory、So Shal Forest、Graphic Pass、Vida Dimension、Powerlevel Island 分别有一个回忆事件可以触发，全部触发后即解锁真结局条件；若有一个没能触发，则会走向好结局。
    注：真结局也是本作的正统结局。
