Sinister Legacy

Use the code from Finster's head to get into Darwin's underground complex

奖杯心得 (TIPS)

  • avger17
    在Ranger's Citadel的博物馆，你会看到展厅里陈列的Finster的头颅，调查后得到密码PROTEUS（7768387）。前往Darwin Village，先清除街道上的僵尸，然后在西南方的卷帘门上输入密码进入地下设施。

    你必须在Ranger's Citadel接到去Darwin Village找药的任务后才能在大地图上探测到Darwin Village。

    如果Rose医生在队伍，她会直接打开卷帘门而无需密码。你可以先将她移出队伍，开门后再重新邀请她加入。
