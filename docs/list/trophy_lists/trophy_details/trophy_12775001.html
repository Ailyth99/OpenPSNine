  • zsk1225
    Special Volume的计时赛比RIDE 2的计时赛要难，因为Special Volume里的计时赛的车是固定的，不能改装。我在玩RIDE 2的时候除了Auto Break和Joint Break关闭了，其他辅助选项都是按照系统默认的。但在本作中，我在打第一个Special Volume时碰到的第一个计时赛就碰壁了，用和RIDE 2一样的辅助怎么跑都离金牌差四五秒。后来看国外的tips说建议用pro驾驶风格加上手动挡，刚上手很不适应，不过跑了几圈开始顺手了，很快就拿到了金牌。第一个Special Volume打完后我就直接开DLC了，DLC里的计时赛基本都是一圈过，偶尔跑第二圈。

    驾驶设置大概如下：
    Off Track AID - Off
    Physics Simulation Level - PRO
    Anti Wheelie - No (try another, if not helpful)
    Tuck-In - Automatic
    Ideal Trajectory - On
    Gearbox - Manual
    Rewind - On
    过弯的时候先减档，然后一边减档一边减速。比如前方是一个发卡弯,你目前正在5档，那就先减到4档，然后开始一边减档一边减速。因为关了ABS，所以减速猛了车子会开始晃，先减档就不会那么容易晃。过弯没过好就按R1开始Rewind，这游戏难度低主要就是因为可以Rewind。

    如果买的是黄金版，可以直接对照DLC奖杯把DLC赛事打了，这样Special Volume也解锁的七七八八了，再回头把Special Volume挨个打完，再打一下联机杯，最后拿一下翘车头和车尾的杯就100%了
