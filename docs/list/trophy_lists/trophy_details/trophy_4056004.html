1 Credit Hard Mode

Clear game in Hard Mode using 1 credit

奖杯心得 (TIPS)

这个游戏最难的奖杯，重点就是第五关的无伤，建议之前先通一次最难难度，可以接关，因为这样就可以解锁全部武器，按LR可以切换，这样游戏难度就会降低很多，不过想要不死过第五关依旧需要反复背版，先拿选关模式直接选第五关练习吧，另外记得千万不要去隐藏关给自己找麻烦
