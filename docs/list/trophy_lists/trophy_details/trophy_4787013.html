Machete

Kill 5 guards in 15 seconds using only the Sugarcane Machete (without using the Chain Kill).

奖杯心得 (TIPS)

  • bobeexiao
    我是在第3章的第4个任务，救下被捕的黑人朋友这个任务里完成这个奖杯的，提前换上甘蔗刀，还有烟雾弹，跑到高台上先从天而降干掉白衣军官，然后立即扔一个烟雾弹，再迅速做掉周围的敌军，如果过场动画时没有跳杯，立即重启读盘点，再试一次，注意尽量引敌人聚团时扔烟雾弹，敌人聚团后可以减少移动时间。
