  • mukure_usaki
    白金难度:3~7/10
    白金时长:一周目20h+六周目*5h=50h+++
    多周目：7.1周目需要
    错过杯：有，七周目随时可以补
    难度杯：无

    游戏内容不多，每周目只需要打4个boss战就可以通关。
    类似于复活同盟设定，没有经验值，技能法术靠觉醒，属性战斗后随机提升。
    与复活同盟不同，由于没有全技能和全法术奖杯，因此只需要对boss最基础的技能即可通关。
    游戏的小怪战斗比较难，参考下方速通攻略，除了提到的刷级位置，其他小怪打怪都不需要打。明雷遇敌，随手保存方便SL。
    以下内容基于日站攻略：http://hrs-game.main.jp/legend-of-legacy/

    可错过奖杯：
    輿圖大師
    將所有地圖填滿至100%
    沸腾之海在收集全三个精灵去废村触发新大陆后会消失，记得提前100%。

    是個不錯的鈴鐺吧？
    在無底窪地不跟可可搭話就通關了劇情
    可可在速通不会经过的路线，无须担心错过。

    貓族的樂園
    首次進入貓族森林。
    新大陆出现后，去火遗迹1图左侧，会解锁沙漠地图，沙漠随机出现海市蜃楼，有一张图为中间一颗大树，进去找上方可可对话即可跳杯。
    具体参考：

    装备阵容推荐：
    辅T(主角，攻守)+输出(艾露伊斯，打击)+主T(欧文，攻守)，一周目还没拿到攻守就用防御。
    主角用初始盾，到大裂隙后找可可对话后换13防盾。首饰为风召唤水召唤。
    艾露伊斯用48风属性弓(交易船5000金随机获得)，神将大剑（神将随机掉落，没有的话换任意大剑），首饰为防毒和奶（最终boss换火召唤）。
    欧文用陨石盾(交易船获得)，首饰为防毒和奶。
    上面提到的武器都是珍品，多周目开局就可以在商店买到（没有的话就旅店刷新），想要白金的话推荐一周目多sl几次神将拿到神将的大剑（66火）。

    多周目速通路线，一周目，全地图都可参考：
    全程尽量避免与非刷级点的小怪战斗，没有任何意义。
    前期准备：
    森林-前庭-歌声森林 （获得水呼唤）-> 主城 -> 城主门口小兵拿森林废墟地图，进旅店睡觉 -> 招募欧文，艾露伊斯，购买珍品装备，交易船1000sl获得高级药箱或急救箱和防毒首饰 -> 换装，旅馆换人编队 -> 准备工作完成
    一阶段：
    潜伏森林进图，右（拿水盾）上右获得山谷地图 -> 上右上右（上坡解锁海角废村地图）下下 -> 山谷往上风遗迹，左侧启动水遗迹后从左侧进入塔，上到2F获得风呼唤，下到1F启动风遗迹，回2F最内侧的传送门打风精灵 -> 2F中间传送门到1F，出撘，走正门进入1F，获得利刃风暴 -> 回主城买沸腾之海地图后卖掉 -> 到达沸腾之海右侧，开风遗迹后开始刷级
    主刷海底的阴影蜥蜴人，技能为盾反，利刃风暴，盾反。主属性刷到20以上，HP大概会在200左右
    沸腾之海出口解锁火遗迹，碾压(?)速通三个遗迹。
    以全地图为目标的话此时清掉沸腾之海的地图，新大陆出现后沸腾之海会消失。
    二阶段：
    去海角废村过剧情，回主城后再去海角废村石头的左侧解锁大裂隙 -> 大裂隙进去左边找可可拿盾，打蜜蜂解锁大剑暴怒 -> 离开大裂隙到达船只森林 -> 回主城购买地底地图，进地底后回主城，商店更新到最强防具 -> 来到船只森林，进右图后在营地旁进行第二波刷级。
    第二波刷级看情况，能两刀暴怒杀掉黑船兽就行，不涨属性不想刷就可以不刷了。200血也足够打过最终boss，所以不用强求。
    后面就一路通关就可以了，地底一图进右下角入口，5图可不点灯，触发水遗迹后，摸黑拿宝石触发boss战。回船森林打3黑船+龟龟，进入瀑布后，点击瀑布进入神顶，再次打败3精灵回神顶1图进入幻都，在幻都破坏所有神人来跳过最终boss1阶段（一周目不可）。 打败神将，打败最终boss通关。

    boss战打法：
    所有战斗都大同小异，开局辅T上水召唤水盾风召唤，输出用大剑暴怒（对多）或5费40攻的（对单），主T盾反。
    输出血量低了（例如最终boss复读用粉碎，暗影3基佬触发即死）就改为副T盾反，输出大剑防御，主T奶输出。

    龟龟+3黑船：约3回合盾反，暴怒，盾反，3个黑船兽杀完后慢慢打龟龟
    暗影3基佬：辅T直接水盾+复读水召唤（boss战前先开水遗迹，开局风召唤，后面被boss抢了就补一下），输出用大剑的暴怒，主T盾反。
    神将：辅T水盾+复读水召唤（开局风召唤，水盾一定不要断），输出用弓的3费25攻技能（大剑会被boss反击），主T盾反
    最终boss1阶段：辅T水盾+复读水召唤（开局风召唤，被boss抢了就补一下），输出用弓的3费25攻技能打星杯（大剑会被boss反击），主T盾反
    最终boss2阶段：辅T水盾+复读水召唤（开局风召唤，被boss抢了就补一下），输出用大剑的5费40攻技能（开局火召唤，后面被boss抢了就补一下），主T盾反。
