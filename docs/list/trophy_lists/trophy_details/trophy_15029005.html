Complete the game with all possible heroes still alive.

奖杯心得 (TIPS)

  • heyoutui
    首先，导入存档非常不利于该奖杯的获得，因为有些角色在1代或2代已经死亡。虽然我没有尝试过，导入存档导致的角色死亡是否会影响奖杯获得，但保险起见，建议在一个非继承存档的周目中解锁该奖杯。
    以下说明基于非继承存档：
    《旗帜的传说3》当中（排查好半天才发现敏感词在这里）共有41名可操作角色。其中Kivi是预购奖励角色不算在内；而在第20章结尾处，你会面临是否开城门的选择，选择开门会获得一名黑裔同伴，但主人公会死亡；主人公的死亡不影响该奖杯的获得。换句话说，最终保证特定的39名角色在队伍中即可获得该奖杯。
