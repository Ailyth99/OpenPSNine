    需要完成迷宫图鉴、怪物图鉴和事件图鉴。
    有几点需要注意！
    普通结局的boss和真结局的boss是两种，不想打二周目的话，最好先触发一次普通结局，然后选择不继承存档，再触发一次真结局。
    普通结局的触发方式：没有收齐另外三个女神，就去挑战最终boss。
    真结局的触发方式：收齐另外三个女神，再去挑战最终boss。
    （我想先哭一会儿，恢复点儿勇气再打第二周目。一周目直接真结局的我，内心在咆哮！）

    -------------5天之后更新--------------
    当我开始补齐全怪物图鉴，才无奈的发现：有些敌人只会出现在主线剧情的迷宫里，错过了就再也遇不到了。所以，要么第一周目要严格按照攻略打，要么一周目随意打，然后二周目去补齐漏掉的。
    我推荐后者，一周目等级高了以后二周目打起来很快的，反而是每次进迷宫都要查显得更麻烦。
    下面是我打的时候对照的攻略，方便后来的玩家少走弯路。
    部分怪物图鉴（不全，但是每个怪物都有出处）：怪物图鉴（伪）
    全怪物图鉴（很全，但没有写明怪物的出处）：怪物图鉴（真）
    迷宫图鉴（很全，写明了每个迷宫会出现的怪物）：迷宫图鉴
    事件图鉴（主线一定会发生的事件除外）：事件图鉴

    更新 1：发现一个问题，就是【クリオネ / Clione / 클리오네】这个怪物，一共有 7 个，但是它攻略里写明让你去打的，只有六个。这六个分别是图鉴里的第 1、2、3、4、6、7 这六个，经验值分别是 1300、13000、52000、150000、600000、2000000。缺少的是第 5 个【クリオネ / Clione / 클리오네】，经验值应该是 332000。我正在满世界找这个怪……

    更新 2：根据游戏里的数据，5 号【クリオネ / Clione / 클리오네】的所在地点是：
    - 試練の洞窟：マウスロープの洞窟 / Challenge Cave : Kuterogi's Cave / 시련의 동굴 : 마우스로프의 동굴
    - マードの森：英雄の神殿が隠された森 / Maado Forest : Hero Shrine's Forest / 머드의 숲 : 영웅의 신전이 숨겨진 숲
    - ディース雪森：広大な森の一角 / Dis Snow Forest : Big Forest's Edge / 디스 눈숩 : 광대한 숲의 구석
    - ネオ・ジオフロント：第二区画 / Neo-Geo Front : Sector 2 / 네오 지오프론트 : 제2구획
    - ディース雪森：第三階層 / Dis Snow Forest : Floor 3 / 디스 눈숲 : 제3계층
    - 仙院廟：竜の巣 / Convent : Dragon's Nest / 선원묘 : 용의 둥지

    更新 3：游戏后期把【ルウィー / Lowee / 르위】的 share 提升以收【ホワイトハート / White Heart / 화이트 하트】入队的时候，攻略上说的是反复玩【VS偽ホワイトハート / VS Fake White Heart / VS 가짜 화이트 하트】这个任务，但是这个任务是从【リーンボックス / Leanbox / 린박스】抢 share 的，按他的攻略打到这一步肯定是【ラステイション / Lastation / 라스테이션】的 share 比较高，而【リーンボックス / Leanbox / 린박스】的 share 已经趋近最低点，因此抢不到 share。我的打法是换一个任务【逆転サバイバー / Reverse Survival / 역전 서바이버】，这个任务是从【ラステイション / Lastation / 라스테이션】抢 share，因此可行。

    - ATW-9：需要下载 DLC【暴走ロボを討て】
