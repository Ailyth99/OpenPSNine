  • matthewlai84
    说真，这游戏不难。我买的时候以为有网址计算器，玩的时候什么都没了，个人又懒得去搞JAVA，还是自己计算出来容易。合起来玩了就1天多，之前玩了半天，不懂方法。简答说下心得给同样懒搞java的朋友，无论1 2模式，都是一样的，先把小项算好，大项的留最后一个，然后通过换非关联数（就是非公用方格的数字），慢慢移动，一定要保证只有1个项目是不等下的条件来换数，多次尝试下就解开了。最后模式很简单，先算好一个方向的，然后再搞剩下的一个方向，最后应该就只有2排或2列是不等的，然后把这2个项目大数一排，小数一排，计算之间各列项数的差额，还有项目的目前跟目标的差额，通过之前计算的各列项数差来组合得得出与目标差的一致的项目进行交换。当然不会是一次就成功的，如果不匹配的时候，就要用任意1排已于完成的项目跟剩下的没完成的1排，进行同列项数交换，一定要保证已完成的项目在交换后也是完成的状态下重复之前的步骤，基本5 6分钟就可以解决掉了。

  • dirushui
    https://www.playstationtrophies.org/forum/energy-balance/316832-energy-balance-how-download-use-solver.html

    https://www.playstationtrophies.org/forum/energy-balance/316833-energy-balance-trophy-guide-roadmap.html

    下载第一个链接里1、2、3对应的三个文件（第三个点clone or download的按钮下载），安装前两个，解压第三个，用第一个软件打开解压出来的Energy_Balance.java（其它几个文件不要删掉也不要移动位置）。
    首先找到
    （1）public static int[] original_values = {3, 1, 3, 7, 9};
    （2）public static int[][] addends = {{0, 1, 2}, {0, 3, 4}};
    （3）public static int[] sums = {11, 15};
    这三行。
    然后去第二个链接，2、3楼有每关对应的代码（hardcore的四关分别对应4，6，9，11关），要解哪关就复制哪关的代码，替换掉上面（2）里的代码。
    之后进游戏，把游戏里的数字替换到上面的（1）里面（顺序不要紧，但是最好是第一行从左到右，然后第二行从左到右，以此类推），记得数字间用逗号隔开。
    然后回到第二个链接，每关还对应一张图片，上面有作者自己标的1、2、3……。按作者图片上标的顺序，把你当前游戏里的求和目标值替换到上面的（3）里面（比如4/3，那就填3，不要填4），也要用逗号隔开。
    都输入完毕以后，按一下工具栏绿色加号右边的小红人，如果输入没有错的话，应该能看到有数字一直在跳，说明已经开始计算了，接着就可以等结果了。最后会按每行从左到右的顺序列出结果，填到游戏里就行了。
    前几关的计算是不需要消耗多少时间的，甚至可能你刚按下按钮，结果就出来了。问题是后面几关的计算时间会越来越长（我第一次算12关用了6个小时还没有出结果），所以需要用一些方法减少计算量。
    我个人的经验是，先刷新出求和目标值都比较小的关卡，然后可以自己解一部分，能解多少解多少，满足尽量多的目标就行，之后按顺序（上面有说）输入到里面，这样可以有效减小计算量（我花了几分钟自己稍微解了一下，之后再交给电脑，几秒钟结果就出来了）。
