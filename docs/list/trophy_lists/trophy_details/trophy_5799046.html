Aerial Break

Destroy any armored enemy using Web-Strike

奖杯心得 (TIPS)

  • allan_hu
    不得不说一下这个杯的解法，走了很多弯路得出的心得。网上有攻略，可能自己的理解有误。正确的解法是，必须解锁 web rush的第二个技能和离子网（就是技能表里的最下边那排）的最后那个技能（破甲专用）。然后找个重甲兵，用o键将重甲兵的盔甲打到发黑（不可打碎），然后用web strike（r1+方块键）将敌人破甲，跳杯。

  • oletsgame
    一周目会错过的奖杯，打完「The Hunter - Kraven」之后会解锁其技能、也就是最后一个技能「IONIC WEB」，注意只升级第一个「Weakening I」等级，不要再升级「Weakening II」第二个等级了，否则就要二周目才能解。
    在紧接着的剧情中，从货车中出来，遇到一个装甲兵，先下掉枪械，然后他溶解其盔甲，最后撞碎盔甲就可以跳杯。
