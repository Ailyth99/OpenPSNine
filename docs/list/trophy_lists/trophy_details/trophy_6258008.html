Offensive Line

As an All-Star, Vanquish 5 Players using the Sprint Tackle in a session

奖杯心得 (TIPS)

使用All-star的冲刺杀死5个玩家,需要一局之内完成。注意冲刺秒杀不了满血的植物,但是撞之前扫几下或者捡队友人头都可以。
