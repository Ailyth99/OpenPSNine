  • shizhengzhi
    1.北方标志:灵魂烙印 + 恶魔烙印
    灵魂烙印(打败最终boss，插在地上的剑)
    恶魔烙印(灵庙打老王得到)
    2.发狂触媒:木头触媒 + 3-3boss魂
    3.熔岩弓:复合短弓7 + 2-1boss魂
    4.削取枪:短矛+7 + 1-1boss魂
    5.切肉刀:棍棒 + 4-1boss魂
    6.幽暗剑:短剑+8 + 4-3boss魂
    7.迷糊斧:战斧+6 + 4-1扩散尖兵魂
    8.蓝血剑:已折断直剑 + 5-3boss魂
    9.无尽苦恼针 :匕首+7 + 3-2boss魂
    10.寻物大剑:弯刀+8 +4-2boss魂

    11.神手:打死神殿铁匠老伯或是去2-2boss前面的洞穴
    12.碎龙骨:第2关白化，去2-3有障碍物挡到的地方，就可以拿到。
    13.风暴滋生:去4-3boss，拿插在地上的剑。
    14.伊斯塔雷鲁:第5关白化，去遇到第一位拿了棍棒巨大人前面，附近会有楼梯出现，上去就可以拿到
    15.月光大剑:去5-2，很多水蛭地方，剑会黏在上面，走旁边木板上去砍水蛭，剑就会掉下来。
    16.妖刀诚:第4关白化，死神背后的洞穴尖塔附近，就可以拿到。
    17.神秘剑:去3-2第一座塔上楼梯，无扶手的地方，慢慢跳下去，就可以得到。1-4杀王子暗灵也可。
    18.洁内小剑:去3-1救贤者出来，就可以得到。
    19.婴儿厉爪:去3-1教堂祭坛(boss一开始站的地方)，就可以拿到(从左边过去拿，右边会被石像鬼抓去3-2)（暗杀任务杀男法师也可以获得）
    20.克里斯匕首:去2-1打魔王前的升降梯，附近的木板，就可以拿到
    21.野兽护身符:黑化第2关，去2-2一开始右边那条路(很多矿工那一条)，慢慢跳下去，有个洞穴平台，里面有个黑化怪物，打到他即可拿到。
    22.白弓:1-4打倒3个暗灵，其中一个会掉。
    23.火炬剑:白化第5关，去5-3，打到黑化卡尔文兰，就可以得到
    24.磷光竿:黑化第3关，去3-1希望牢狱西侧2楼，打到黑化(please help me)雷德尔
    25.无柄:黑化第4关，去4-1，打到一开始红色怪物，就可以得到
    26.盲刀:黑化第5关，去5-2，打到黑化塞莲，就可以得到。
    27.贯穿剑:去1-4打倒3个红色怪物，其中一个就会掉。(也可以捡到具体哪里有点忘记)
    28.神秘盾：去3-2第一座塔上楼梯，无扶手的地方，慢慢跳下去，就可以得到。1-4杀王子暗灵也可。
    29.暗銀盾：流程5-3杀卡尔·文兰。
    30.審判者之盾：4-1第2次遇见双刀黑骷髅的地方，可以捡到。
    31. 塔盾：救魔女的流程可以取得，或者1-4杀刚开始的三个暗灵掉落。
    32.锐细剑
    33.恶魔烙印
    34.灵魂烙印
    35.紫染大盾：1-1杀死米兰达之后，往下走，可以捡到。或者完成暗杀双剑比约尔任务，掉落。
