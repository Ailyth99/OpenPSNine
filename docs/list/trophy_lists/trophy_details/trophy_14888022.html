Heroic Hoarder

Return all barnacles to Plumpet Island

奖杯心得 (TIPS)

  • return_cn
    收集全部藤壶，除了岛上初始的一只外，需要手工收集5只

    前4只找蛙岛的蛙宝接任务，在地图上战斗胜利后回去对话交任务，再到沙滩上放生即可
    3只在世界地图上，1只在雪原地图上，位置用红圈标注



    第5只是支线任务获得。在林中湿地和旅行商人对话（可能需要完成前置的捉虫任务），帮他跑腿。注意任务有时效，操作机器人按会移动飞快，是最简单的方式。接下任务之后把东西送给Suiva城里最深处的渔夫后获得第5只，回去沙滩放生即可跳杯。
