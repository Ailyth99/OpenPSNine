  • satsukiye_
    不是很难，死了别点continue，直接撕游戏就行。

    下水道可以在法师边上的售货机买皮手套，幽灵开门任务右上角售货机买个角斗士铠甲，拿角斗士之剑过。boss戴净水器往水里扔个毒瓶子，放风筝等boss毒死。

    第二个场景用角斗士之剑 + 皮手套 + 决斗者铠甲，一路很简单。雪女左边场景的兔子建议拿一下，雪女不用打。

    第三个场景记得在初见兽人boss的地方捡一块神秘的肉吃掉，+5最大生命值。boss就左右翻滚等她累了可以一次打掉半血，二阶段同理，只不过她可以吃肉回血。

    打完碰到加奈可以在右边的场景拿了刀还给她，钥匙可以开右边场景的门拿到隐身术。

    到墓地，这里如果求稳的话在遇到哥布林商人的电梯右边拿个吸星大法，可以很安逸的吸过一整关。也可以用火腿 + 兔子 + 弹弓 + 生命护符 + 隐身术，这里带生命护符都可以有23血，容错率很高。隐身术火腿扔一下50伤害，小怪基本都是瞬秒，boss丢4-5也就过了。打完可以用百合的法杖替换兔子。

    庭院直接隐身跑路，打boss前可以选择收集齐四属性魔法跟法师换一个转剑魔法，也是个比较厉害的套路。用兔子 + 法杖 + 魔法宝珠 + 吸血鬼斗篷 + 魔法护符。boss弓箭妹，只需要注意二阶段跳起以后需要朝一个方向猛滚躲落地的震荡波，其他招式无威胁，隐身火腿或者转剑都是随便过。

    恶魔城场景开局依旧隐身跑酷，注意地上的冰冻陷阱别把自己冻住了。女仆任务拿到的香肠直接吃了+5最大生命之。boss一阶段火腿3下丢死，二阶段丢火腿的话容易打到她的剑，实在掌握不好节奏可以考虑转剑上去换血，如果拿到弹幕减缓的法术会简单一些。建议就换血好了，挨打即时开包吃药，这也是唯一一个需要消耗品的boss。

    最终场景还是隐身跑酷，最终boss无论是转剑还是火腿都很简单。
