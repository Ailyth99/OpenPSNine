    int onoff;

    combo_run(Marie24);}}
    combo repeat {
    set_val(PS4_OPTIONS,100);

    set_val(PS4_OPTIONS,0);

    set_val(PS4_CROSS,100);
    wait(60);

    实在过不去的请收下这段代码，需要编程手柄。
    按三角就是暂停1秒后返回游戏0.1秒之后再立刻暂停的无限循环代码。
    这样就可以做到0.1秒一次操作，什么鬼畜的弹幕你都能躲开的。甚至还能从最终BOSS那个鬼畜的弹幕里面钻出来~
    R1键需要在暂停菜单里使用，按了之后就是立刻切回游戏0.1秒之后再暂停一次。

    不过得注意，游戏掉帧的地方很容易让上面的操作出现错误，一错误基本就要翻车。游戏掉帧频繁的地方是魂收集结束的时候，还有弹幕特别特别密的时候，还有一堆怪物死的时候。这些地方就注意不要用上面的代码了的。
