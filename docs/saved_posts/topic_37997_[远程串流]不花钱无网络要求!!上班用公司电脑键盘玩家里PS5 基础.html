fishsama_108 2024-06-12 11:11编辑     114评论

基础

基础
废话少说先上画质和延迟

地狱开局：广州城中村出租屋10人共享的公交车移动100M宽带
但我在公司远程游玩画质稳定1080P，延迟稳定5~8毫秒，几乎等于没有，不用UU加速器不用任何花钱的东西


本文分 基础 与 鼠标玩游戏两部分，但鼠标部分目前我没完全完善，大哥要是会的话记得教我

确保你有以下3个条件：
热爱游戏、愿意串流、克服困难的心
PS5插网线
公司电脑插网线，配置很低也无所谓

没有也行，但远程串流效果可能要差一些

第一步：


我们先设置PS5，打开设定——系统——省电——待机模式中可使用的功能

这两个选项都打开，以后上班出门前记得把PS5弄成待机模式

不要怕费电，有up主测过大概待机一年也就1~2度电，这点花销忽略不计


打开设定——系统——远程游玩

把启动远程游玩勾选上


打开设定——网络——连接状态——查看连接状态

找到ipv4这一行，记住自己PS5的网络地址 一般为192.168.xx.xxx

第二步：
配置家里WIFI路由器
这一步是最难的，但只要克服了后面的操作就没有难度了

具体要看你家里的wifi路由器，如果你家的是小米或者华为路由器，那我推荐买个华硕路由器小旋风pro
不超过550块钱就能拿下，其他路由器例如小米AX6000我用来串流也有1年多了，真不好用，价格还差不多，如果你有更好、更方便方案也欢迎指点

以华硕小旋风pro为例，我们刷梅林后要给路由器装上一个简单的小插件：zerotier
注册好后就能用了，家里的全部配置就搞定了

第三步：
配置办公室电脑

在公司电脑上也下载一个zerotier，填上家里zerotier注册的networkID就好了，这样子就等于加入家里的局域网了

下载串流软件：Chiaki

不用官方的PS remote play因为它不能用键盘玩游戏
虽然它的确是最方便的，我手机和ipad上玩就用它

另一个第三方的PSplay，估计它是最好的，但是PC版的Psplay目前是新出来不到1个月，版本1.00似乎bug不少，所以我先不讲，等更新完善后用它串流应该不错。

所以我现在推荐用Chiaki来玩

chiaki支持自定义键盘控制全部按键

左边的串流配置照抄我的就可以


第一行直接填你PS5的ip地址，就是上文提到的192.168.xx.xx
下面的AccountID可以直接看B站的这个教程获取：https://www.bilibili.com/read/cv10858980/
最后的PIN码其实就比较尴尬了，你需要有人在家帮你打开PS5远程配对，获取8位数的pin码告诉你
只能叫身边的人帮帮你了，但这已经是最后一步了，相信你进行到这的话，不会轻易倒在这里

然后就是即刻开始爽玩!

如果你想查看延迟具体情况，可以按下键盘win+R键，在弹出的运行窗口里输入cmd

然后在里面输入这个ping 192.168.xx.xxx -t就能查看了（ip地址为上文提到你ps5的ip地址）

用键盘玩圣兽之王没有任何问题

鼠标玩游戏

脚本详情

脚本详情
我不懂脚本不懂代码的写出来，哪里不对了直接骂我就行

代码详情：
Dim LastX, LastY, x, y, ex, ey
BeginThread shoot
Call GetCursorPos(LastX,LastY)
While True
Delay 5
Call GetCursorPos(x,y)
ex = x - LastX
ey = y - LastY
If ex>10
KeyUp 190,1
KeyUp 188,1
KeyDown 190, 1
//鼠标向右移动，把 KeyDown 190,1中的190改成你想要的键位
ElseIf ex < -10
KeyUp 188,1
KeyUp 190,1
KeyDown 188, 1
//鼠标向左移动，把 KeyDown 188,1中的188改成你想要的键位
End If

If ey>10
KeyUp 101,1
KeyUp 104,1
KeyDown 101, 1
//鼠标向下移动，把 KeyDown 101,1中的101改成你想要的键位
ElseIf ey < -10
KeyUp 101,1
KeyUp 104,1
KeyDown 104, 1
//鼠标向上移动，把 KeyDown 104,1中的104改成你想要的键位
End If
If ey = 0 And ex = 0
KeyUp 190,1
KeyUp 188,1
KeyUp 101,1
KeyUp 104, 1
End If
LastX = x : LastY = y
Wend

Sub shoot
Rem start
key1 = WaitClick()
If key1 = 32769
KeyDown 80, 1
//鼠标左键开火，把 KeyDown 80,1中的80改成你想要的键位
ElseIf key1 = 32770
KeyUp 80, 1
//鼠标左键开火，把 KeyUp 80,1中的80改成你想要的键位
ElseIf key1 = 32773
KeyDown 82, 1
//鼠标右键瞄准，把 KeyDown 82,1中的82改成你想要的键位
ElseIf key1 = 32774
KeyUp 82, 1
//鼠标右键瞄准，把 KeyUp 82,1中的82改成你想要的键位
End If
Goto start
End Sub

eternal_sc30 2024-06-20 08:33
关键一步没说，默认配置的话外网设备是访问不到家网路由器下的其他设备的。

记下ps5设备在家庭网络的ipv4地址

1、去zerotier的网页端找到创建好的网络并点击NETWORK ID打开网络设置。
2、找到members一栏确认路由器和外网电脑两台设备是否已加入并授权，正常menbers列表会显示两行设备信息，并且前面的auth?选框要打上勾。并在这里记下路由器的Managed IP。
3、【关键一步】还是zerotier网络设置页面找到Advanced-Add Routes，在Destination填写家庭局域网段（例如ps5的ipv4地址是192.168.1.104，就最后一位抹零加上/24即填写192.168.1.0/24），在Via填写zerotier映射的路由器地址即上一步记下来的路由器Managed IP。
4、chiakia的host填写ps5的ipv4地址。

都没问题了还报Regist search failed就检查chiaki的账号id是否填写正确或者关联设备码是否已过期吧。
其他网站上教学的其他设置都不用管，和远程玩儿主机的目标没啥关系。
