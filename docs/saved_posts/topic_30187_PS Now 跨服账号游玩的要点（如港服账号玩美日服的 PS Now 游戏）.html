PS Now 跨服账号游玩的要点（如港服账号玩美/日服的 PS Now 游戏）

luna_noire 2016-12-28 01:04编辑     54评论
下列游列作废，原理已经搞清。
跨服游玩只需要从主菜单的 PS Store 进入商店，然后找到 PS Now 的游戏列表即可，不能从主菜单的 PS Now 进入。
日服在商店左侧列表就有 PS Now，里面的定额制项目就是我们要找的商店游戏列表。
美服左侧 PS Now 里面的 Subscription Games 不要点，会直接进入 PS Now 的APP，这样是无法在PS4上下载认证的，要点入购买 PS Now 服务的页面，然后往下拉有 All Games ，里面就是 PS Now 的全部定额制游戏的列表了。
