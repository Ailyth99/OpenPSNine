关于《初音计划F2如何》cross save的攻略

liuyuningniua 2022-01-29 21:47编辑     23评论
如何将project diva F2的存档在psv/ps3上进行同步？

（1）进行保存
选 その他
选 1ゲームオプション（game option）
选 4クロスセーブ（cross save）：改成 手动
返回
选 5セーブ（save）
这里会弹出一个“接下来，是否进行cross save（クロスセーブ），选择yes（はい）
现在就上传了（有四个方块的标志在粉色和绿色之间闪烁）。会显示一个上传成功的弹窗。

（2）在其他端进行读取
首先，像刚刚一样，将cross save改成“手动”

之后，进入游戏的标题画面，也就是（タイトル）画面，让 PRESS START BUTTON 这里。
先不急，我们确认一下网络通不通。网络通了的情况，ps3可以看到右上角出现“已登入”，psv可以长按ps键看到 网络的标志。

确认网络通的话，
按下start
选 load game
随便载入一个存档。
载入之后，会弹出一个“クロスセーブを实行しますか？”（要不要进行cross save）？
选择是（はい）
会开始读取，有四个方块的标志在粉色和绿色之间闪烁。
读取好了。
之后，我们将刚刚改的设置改回去，crosssave改成off，免得下次还读cross save存档。
