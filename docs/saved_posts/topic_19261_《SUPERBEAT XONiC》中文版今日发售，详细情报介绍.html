《SUPERBEAT: XONiC》中文版今日发售，详细情报介绍

sai8808 2015-10-22 16:48编辑     10评论
NURIJOY近日公开了PSV音乐游戏新作《SUPERBEAT: XONiC》的众多消息。




视频只能使用 youku 和 bilibili 哦


开场动画的主题曲「STARGAZER」由音乐家江口孝宏作曲，绘师DORAE担任原画。
虽然预算吃紧直到最后一刻才製作完成，动画仍然展现了十足的角色魅力。述说着本作形象角色Riu、Sion追寻着闪耀的新星XONiC，摇身变为音乐人登上舞台的故事。

充满活力的旋律也表现出《SUPERBEAT: XONiC》作为一款新作，变革后重新出发的气息。



绘制国产音乐游戏《DEEMO》里「Niflheimr」、「Recollections」等10曲的选曲画面插图以及《DJMAX》里「Raise me up」、「Starlight Garden」等背景动画的韩国绘师PUNEW，这一次也直接参与了《XONiC》的游戏製作。身为系列作老玩家的他，力求让《XONiC》于设计上使用全新玩法的同时仍保有些许《DJMAX》的感觉。


在17日的体验会中对到场玩家说明游戏系统的PUNEW。




《SUPERBEAT: XONiC》的难度达到了一个全新的平衡，一方面采用POWER OFF等对新手较为友善的设定，另一方面也为有经验的音乐游戏玩家准备了高等级谱面。

如同《DJMAX TECHNIKA TUNE》，《SUPERBEAT: XONiC》在三种难度的谱面之外还有所谓EX谱面的存在，这种最高难度的EX谱面只会在「WORLD TOUR」模式出现，提供玩家更上一层的挑战。不只如此，后期还隐藏了某些要素，将游戏完全征服之后，似乎还有什么在等待着玩家......。


●和ARC SYSTEM WORKS合作的要素




《SUPERBEAT: XONiC》和 ARC SYSTEM WORKS 合作的要素公开！游戏中将会收录《圣骑士之战 Xrd -SIGN-》的主题曲「HEAVY DAY」，ARC SYSTEM WORKS旗下游戏的角色也会作为DJ ICON在《SUPERBEAT: XONiC》内登场。




制作人Planetboom曾表示自己是石渡太辅的支持者，也在FB上暗示过与ARC SYSTEM WORKS的合作。




往后预计会以DLC的方式提供更多的曲子，让《SUPERBEAT: XONiC》的曲风更加辽阔！包含《苍翼默示录 刻之幻影 扩充版》的主题曲： J-POP风格的「BLUE desire」
以及音乐益智游戏《MAGICAL BEAT》裡，由风格特异的VOCALOID P主Kikuo（きくお）作曲，电子歌姬初音未来（初音ミク）演唱的「TUNTUN-MOONLIGHT」......等等。


●未来的更新计画



对音乐游戏来说音乐本身是最重要的。过去《DJMAX TECHNIKA TUNE》发售后也有不少声音希望能有DLC延续游戏生命，无奈因公司组织的变动无法实现。

这一次的《SUPERBEAT XONIC》则不同，NURIJOY公开了DLC的情报，打算在游戏发售后週期性的推出新的DLC，为《XONIC》注入活血。目前已知日本知名乐师Cranky、在本作负责谱面制作的7 Sequence会有DLC新曲。

之后还会出现什么样的作曲家及新曲，详细的贩售情报有待后续公开。

视频只能使用 youku 和 bilibili 哦





《SUPERBEAT: XONiC》繁体中文版已于10/22上市。
初回特典为DLC XONiC PACK #1的下载序号以及2片CD组成的游戏原声带（限量）。
