「E3 2017」噬神者团队新作《CODE VEIN》登录平台确定，预计2018年发售

yz93yz 2017-06-12 10:48编辑     39评论



《CODE VEIN》是一款由制作《噬神者》系列游戏的创作者们所企划开发的完全新作游戏。富泽祐介担任企划统筹、吉村广担任游戏总监、以及新招募的饭塚啓太担任游戏製作人。游戏以以近未来世界为游戏舞台，玩家要扮演一名吸血鬼，为了追求吸血鬼到底是为何而生，以及这个世界是不是有「出口」等真相，而踏上探索世界的旅程。











PS4 / Xbox One / PC STEAM《CODE VEIN》预定在 2018 年内推出。
