因为刺客被削弱并且剑气伤害加强，剑士成为了ACE中最强的职业之一。很多人问怎么配卡，其实配卡没有绝对的，有很多替代方案，效果都一样或者差不多，不必太过于纠结于某个卡。本帖写了一些个人玩剑士的心得，抛砖引玉，欢迎共通探讨

基础

剑士基础攻击力160
技能叠加攻击力上限是250

平板攻击力：

最终攻击力≈ 基础攻击力 + 技能攻击力 + 武器攻击力
理想状态下，最终攻击力 ≈ 160 + 250 + 280（后期的世界树大剑精炼到+10）= 690左右

做个记录

SP最佳化 x 2 = ODS【7星獸人薩滿】
高風險DS x 2 = DS威力上升Lv3 x 2 ＞ 超载【烈阳拳刃抽卡】＞ 兽人爆气【10星獸人薩滿】
夏哈尼爾利刃 和 残月 可以一起使用
