PSV《东京Xanadu》公布各角色专属必杀技「X 强袭」与新战斗系统「X 驱动」

sai8808 2015-04-30 17:06编辑     19评论
日本 Falcom 在今日（4月30日）公布了预定在 9 月 30 日发售的 PSV 游戏《东京 XANADU》的新战斗系统与人物专属必杀技介绍。




本次所要介绍的是「X 驱动（X ドライブ）」与「X 强袭（X ストライク）」两项本作原创的新战斗系统。

关于「X 驱动」，当消耗游戏画面由下脚的「驱动能量表」时，操纵的人物与伙伴将能获得移动速度上升、能力低下无效、状态异常无效等使能力大幅上升的效果。
而关于「X 强袭」，当消耗画面右下角以六角形标志代表的「强袭点数」时，角色将能发动专属的必杀技。



有许多使用这个充满歼灭力与爽快感的战斗系统的方式，例如在 X 驱动的效果期间打出连击累积关键的强袭点数，再发动必杀技一口气扫荡敌军的使用方式便是其一。这些都还有待玩家们慢慢探索。

战斗系统「X 驱动」

使用普通攻击与技能击中敌人时，画面右下角的「驱动能量表」就会逐渐累积。当达 100% 时，将方向键往下压即可发动「X 驱动」。

在量表从 100 % 降至 0 的期间，将可获得人物移动力上升、能力低下无效化、状态异常无效化等使操作角色与伙伴的能力大幅提升的状态。

另外，随着队员组成的不同，X 驱动所追加的效果也会有所改变，人物之中也有能使 X 驱动附加超强力效果的角色。




发动前 → 发动 → 发动后

战斗系统「X 强袭」

随着普通攻击与技能攻击击中敌人所累积的连击数，画面右下角以六角形符号表示的「强袭点数」将会慢慢累积。而消耗这个点数能使各角色发动专属的必杀技「X 强袭」。



洸的 X 强袭「无尽之锚」，将锚剑如鞭子般挥出切割敌人后，朝敌军飞射巨大化锚剑，一口气击飞



明日香的 X 强袭「华晶天幕」，突进并打出快速连击后跃上空中对敌人投置灵魂装置，降下巨大水晶攻击



空的 X 强袭「风尘虎吼掌」，以超高速度形成分身进行连打，再将气凝入双手间，轰出特大号能量弹



祐骑的 X 强袭「畸变陨星」，将灵子壳增殖并以曲线轨迹攻击敌人，再融为如陨石般的巨大灵子壳坠落
