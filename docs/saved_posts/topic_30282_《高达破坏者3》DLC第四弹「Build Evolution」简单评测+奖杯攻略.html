《高达破坏者3》DLC第四弹「Build Evolution」简单评测+奖杯攻略

retridge_rii 2017-01-09 15:39编辑     16评论
第二弹DLC“Build Beginning”奖杯贴
第三弹DLC“Build Rising”奖杯贴

高破3的第三弹收费DLC(总计第四个DLC)，奖杯配置和上一弹相同。

前略，这个DLC我决定最后给他一个6分。主要的要点在于:

-剧情和游戏体验还是那个样子，这都第四个DLC了，再怎么有耐心的玩家也该对DLC套路差不多腻了。
-之前未公开的第四台新机体居然是个BB战士，神秘神秘真神秘。
-随着这次DLC的更新补丁好像还大砍了导弹系OP回收觉醒槽的性能，之前我的一个帖子里说到过的无限觉醒流不好使了。
-萝卜太你废话真鸡儿多。
-干你娘的权天使浮游炮。

+干你娘的权天使浮游炮。

BUILD EVOLUTION：任務完成
以任一難度通過第四波DLC的所有任務

DLC的关卡配置还是和前几弹一样，四关打完跳任务完成杯。

BUILD EVOLUTION：武器/機體零件收集完成
持有第四波DLC內任一層級的全部追加武器及零件
新增的MS分别是:
铁血孤儿的高达基马利斯,本体6件套+近战1把，这个套件有两种背包，但是商店卖的MG里面是只有一种的。只缺一种的话，用另一种派生合成是最省事的办法；(MS出处:DLC4-01最后一张图)
高达ORIGIN的机动工人，本体5件套+盾；(MS出处:DLC4-02第一张图)
高达00第二季的权天使，本体5件套+近战1把+盾，近战武器包含射击模式。(MS出处:DLC4-02最后一张图)
BB战士骑士高达史伯利奥龙。(出处:DLC4-04打爆给)

收全跳杯。

BUILD EVOLUTION：製作者零件收集完成
收集到第四波DLC裡的所有追加製作者零件
这一弹总共更新8种BP，掉落规则改过，精英怪/BOSS单位在被击飞零件时也会进行BP掉落的概率计算，所以大概比前几弹DLC要容易刷一些。精英怪/BOSS单位及相应的出现关卡、掉落BP如下:
1.龙高达
出现关卡:DLC4-01第一张/第二张图
掉落BP:
龙头。以上一种。
2.高达基马利斯
出现关卡:DLC4-01第三张图
掉落BP:
斩击盘(左)，战斗小刀(右)。以上两种。
3.杰钢队长
出现关卡:DLC4-02第一张图
掉落BP:
膝装甲。以上一种。
4.机动工人
出现关卡:DLC4-02第一张图
掉落BP:
驾驶舱(左)，灯(右)。以上两种。
5.权天使高达
出现关卡:DLC4-02第三张图
掉落BP:
GN獠牙荚舱(左)，单侧天线(右)。以上两种。

另外这一弹DLC新加了一个限制解除器，算在强化塑胶类里面，任意难度打一遍4-04随机给4~5个，作用是可以把原本最大强化的零件(等级☆)进一步强化，消耗一个加一个☆，最多强化到☆☆☆。
最后贴一个现在的刚破烂。权天使出了以后我对这个游戏已经没有任何追求了，除非他剩下的两弹DLC敢给我把00剧场版的机体吐出来。
