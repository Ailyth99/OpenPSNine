机器人社自动刷素材刷钱脚本

一、脚本运行前准备
1、当前比赛轮次为刚领取了30000元经费的轮次

第一条的限制已经在最新的脚本里解除了，通过的操作顺序兼容领取奖励的轮次和普通的轮次


2、当前游戏画面为对着机器人社社长冲手河童梦并且出现交谈按键


3、将比赛中的切换AI按键调整为Ps Remote Play的esc键(ps4对应的是x键), 操作路径: 比赛中按options键 -> 键设置



4、打开Ps Remote Play并连接主机，并在脚本开始倒计时完成前将Ps Remote Play做为主窗口


二、运行脚本
依赖于JRE, 运行报错的话请尝试安装
win: 直接点击审判之逝机器人刷钱脚本.exe文件即可
通用: 命令行输入 java -jar ./审判之逝机器人刷钱脚本.jar

三、脚本运行参数

1、等待比赛结束的倒计时
不放心的话可以直接设置成180
2、等待比赛加载的倒计时
按照实际的加载实际设置，我是ps4 + ssd 设置的是15
3、选择比赛对手，支持多选，反选和全选。如果只是刷钱的话选择第一个对手进行了

刷双六技能使用脚本
请确保
1、八神正对着双六小姐姐并出现交谈按键
2、将使用技能键调整为上键
3、在倒计时结束之前将Ps Remote Play调整为主窗口，并将鼠标移动到options键处
