PS4《底特律 化身为人》制作人访谈 — 揭晓一切开端的彷生人「卡菈」

sai8808 2017-11-08 08:33编辑     49评论
Quantic Dream 开发，索尼互动娱乐预定 2018 年推出的 PS4 独占冒险游戏《底特律 化身为人（Detroit: Become Human）》，制作人纪尧姆‧戴‧冯多姆（Guillaume de Fondaumière）与总监大卫‧凯吉（David Cage）日前在自家大本营（Quantic Dream 位于法国巴黎）举办的巴黎游戏周（PGW）中亲自示范了最新游戏画面，值周接受了媒体访问，分享游戏的核心理念。


编剧兼总监大卫‧凯吉（左）与制作人纪尧姆‧戴‧冯多姆（右）


《底特律 化身为人》是 2012 年 Quantic Dream 为了展示 PS3 新游戏引擎制作的实机技术展示《卡菈》为题材所延伸出来的作品。时代设定在距今约 20 年后的近未来，模彷人类外型与能力所制作的彷生人已经成为普及的量产商品，肩负起许多原本人类所从事的劳动工作，如工人、侍者、保姆等。虽然彷生人的外表与行为都跟人类没什么两样，不过却只被视为没有人格的 “物品”。

为了与人类区别，彷生人的臂上都标有蓝光三角标志，额上也会有圆形的光环。

不过原本应该依照设定程序运作的彷生人却陆续开始出现异常（Deviant），有些无故失踪，有些突然狂暴攻击人类，有些则是产生不该有的情感，严重动摇了人类对彷生人产品的信任。



游戏中玩家将从 3 名彷生人角色的观点来体验故事剧情，去年 E3 揭晓了负责处理异常彷生人案件的干员「康纳（Connor）」，今年 E3 则是揭晓了具备自我意志的彷生人革命家「马库斯（Markus）」，而这次 PGW 终于揭晓了整个企划开端的关键角色女僕「卡菈（Kara）」。


卡菈


冯多姆表示，卡菈是 Quantic Dream 为了展示在 PS3 上开发的新版游戏引擎，而于 2012 年制作的一个技术展示中登场的女性彷生人角色，原本并不是一个具体的游戏企划。不过因为该展示影片公诸于世之后，全球玩家都对于影片后续的情节发展很感兴趣，所以 Quantic Dream 在结束 PS3《超凡 双生》的专案之后，就着手开始将影片的情节发展成一个完整的故事。

启发了《底特律 化身为人》这个专案的卡菈，是家务用途的 AX400 型女僕彷生人，虽然出厂前就被发现异常，出现了彷生人不该有的情感与自我意识，但是检验员一时心软，仍让她照常出货。游戏中她来到单亲家庭父女陶德（Todd）与爱丽丝（Alice）的家中，负责家务与儿童照护。



现场比照之前展示的模式，由冯多姆亲自游玩，分别展示两种不同路线的发展。冯多姆透露，卡菈并不是在展示的片段中初次登场，而是会在早先的情节中现身，这部分留待玩家自行体验。

卡菈的主人陶德是个有暴力倾向与药物滥用问题的中年男性，妻子已经离他而去，只剩下年幼的女儿爱丽丝与他同住，因此他购买了女僕用途的彷生人卡菈来协助照料家务以及女儿。失业的他变得愤世嫉俗，认为自己的工作是被彷生人抢走，成天自怨自艾，还经常迁怒女儿与卡菈。


陶德


在第 1 轮的示范中，卡菈依照游戏选单指示料理好晚餐，但用餐之际却因为陶德没由来的发火而毁了这顿晚餐。受到惊吓的爱丽丝不顾陶德的命令，跑回楼上的房间躲藏。原本卡菈想上楼安抚爱丽丝，但是却被盛怒的陶德下令不准移动。徬徨的卡菈就在这个时刻面段关键性的决择。


爱丽丝


正常的彷生人无法违背主人的命令，但身为异常彷生人（平常有刻意隐藏，未被发现）的卡菈可以进入「心灵宫殿模式（Mind Palace）」模式，主动打破命令的桎梏。此时冯多姆决定让卡菈打破心灵宫殿中被具象化为一道牆的命令，上楼安抚爱丽丝。但盛怒的陶德随即跟上楼，打算毒打爱丽丝洩愤。卡菈为了保护爱丽丝而与陶德扭打成一团。但卡菈只是家政用途的彷生人，完全不敌壮硕的陶德，最后损毁而停止运作不支倒地。而爱丽丝则是在陶德的暴力之下失去了性命...


卡菈挺身保护爱丽丝免于陶德的虐待


紧接着冯多姆进入第 2 轮示范，直接从心灵宫殿的抉择处开始。这次他选择不打破命令之牆，而是转头与陶德理论。不过才刚开口陶德就将她甩到一旁，直接上楼去毒打爱丽丝出气。为了保护爱丽丝，卡菈做出了一般彷生人不可能出现的举动－拿着陶德的手枪冲进房间威吓陶德住手。此时陶德也发现了卡菈异常之处，企图制服卡菈。就在一阵同样的扭打后，卡菈又再次陷入损毁危机，不过这次爱丽丝勇敢出手，拿起了掉落的手枪击毙失格的暴力父亲，解除了两人的性命危机。



家暴阴影下的不幸家庭


虽然很多场面是一致的，但是事前的抉择将会彻底改变接下来的命运...

惊觉事态严重的卡菈，急忙带着爱丽丝夺门而出，搭上公车消失在街头...


卡菈与爱丽丝踏上未知的旅程


示范告一段落后，冯多姆向在场媒体展示了游戏的流程图系统，玩家可以透过这个系统来确认自己之前走过的剧情路线，之后想要挑战别的路线就能很轻易地确认走法。而且游戏中可以让玩家自由跳到流程图中走过的点，就像这次示范一样，不用重复之前玩过的部分，可以直接前进到关键抉择处。游戏目前已经 “快完成了”，排定在 2018 年春季推出，制作团队正在努力赶工调整。

随后编剧兼总监大卫‧凯吉与媒体进行圆桌会议，进一步解答游戏的疑问

首先被问到这次《底特律 化身为人》游戏的型态是不是比较接近同样有 3 名角色与多重路线的《暴雨杀机》，对此凯吉表示，虽然《底特律 化身为人》与《暴雨杀机》都是剧情冒险游戏，但制作团队并不是要复制先前的作品，而是打造一个全新的作品，所以讨论游戏到底比较像哪一款前作，其实没有太大意义。这次《底特律 化身为人》提供更多样的故事发展路线，这是之前所没有的。

而每一次的作品制作都会带给团队更多的经验，让团队知道还有什么可以改善。例如之前的《超凡 双生》曾尝试把大部分 UI 资讯隐藏起来，结果有不少玩家反应不知道下一步该做什么，所以这次《底特律 化身为人》会改变作法，让 UI 资讯更清楚的呈现。毕竟这款游戏并不是要考验玩家的操作技术，而是考验人性的抉择，所以操作方面会力求简单直觉。


PGW 展出现场跟之前的 TGS 一样，请到真人演员在橱窗中扮演彷生人产品


对于游戏想阐述的理念与选择彷生人为题材的理由，凯吉表示，《底特律 化身为人》的故事编写比起一般线性故事剧情的游戏难上很多，而且编剧时并不是放入一些随机的选择给玩家，每个选择都有它的意义在。与其说彷生人是游戏的主题，还不如说更多是关于人类、人性与人类的未来。

而挑上彷生人主题，则是因为一本名为「奇点临近（The Singularity Is Near）」的未来学书籍启发，这本书以「科技奇点（Technological Singularity）」的概念为核心，认为科技的进步多半是在很短的时间内有很大的进展。而人工智慧的发展突飞勐进，总有一天会超越人类，并以人类所无法想像的速度爆发性成长，让科技一日千里。因此当初才会动念制作卡菈的技术展示影片。

不过游戏并不是想要描述 AI 是什么，而是反过来探讨人性是什么。

游戏中玩家可以从 3 个能力与背景都不同的彷生人观点，来体验整个故事剧情。不过这 3 个角色的故事并不是一个演完才换下一个演，而是会打散成不同章节，交织在一起。其中康纳是服务人类的立场，马库斯则是站在彷生人争取自由的角度，居间的卡菈则另有其定位。每个角色各自的故事，都会影响其他角色故事的发展。至于 3 人最后是否会碰头，这部分目前还未能公布。


感觉 TGS 的演出技巧比较高明，给人一种似假非假的感觉，PGW 的演员就...还好


相较于康纳的推理，马库斯的骇入等能力，这次卡菈只展示了打破命令之牆的能力，而且手无缚鸡之力，能力似乎显得有点薄弱。关于这点，凯吉表示，卡菈其实还有其他的能力，只是这次尚未公诸于世。而且制作团队并不打算让彷生人变成超人，毕竟游戏探讨的是人性，如果彷生人超脱人类太远，那么就很难将之带入人性的范畴。

当被问到如今的 “卡菈” 是否就是当初影片中的 “卡菈” 时，凯吉微笑表示：「是的」。


当年的卡菈


如今的卡菈


卡菈的造型与演出是由女演员法萝莉叶‧柯瑞（Valorie Curry）担纲


凯吉强调，Quantic Dream 与 SIE 的合作已经超过 11 年，在 SIE 的协助下，Quantic Dream 持续精进游戏制作技术，这次的《底特律 化身为人》将实现 Quantic Dream 有史以来最佳的演出效果，不论是硬体撷取设备、软体制作平台或是游戏执行环境，相较当年都有大幅的进化，而且会充分支援 PS4 Pro 的 4K HDR 画质。目前制作团队正努力打造最后的场面，并持续进行测试。因为游戏有数千种路线，想要确保从头到尾都有一致的品质是相当艰辛的挑战。

最后凯吉顽皮地表示，游戏还有很多惊喜的部分还没公布，请大家多多期待上市前的发表！
