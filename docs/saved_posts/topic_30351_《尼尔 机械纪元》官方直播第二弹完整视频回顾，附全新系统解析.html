15:00 隐藏地图
Demo 中随着快节奏战斗不停变化的视角是游戏的一大特色，但此视角的存在意义不仅仅是让游戏的战斗更加新颖，比如说在某些场景中，你只需要换个思维和视角尝试一下，你会发现别有洞天。



24:20 支线任务
游戏的定义是 A·RPG，所以在你的 2B 冒险中可以接到一些支线任务，就像视频中根岸先生演示的一样。2B 和 9S 在反抗军的前线基地，觐见了新登场的反抗军首领 Anemone，随着剧情的推进，基地中的 NPC 会随之解锁，视频中就向基地中的武器店老板相关的支线任务。



35:50 芯片技能系统
这是游戏的技能系统，也是游戏最核心的玩法，游戏中角色的所有技能都是以“芯片”的形式表现的，而“芯片”的载体则是“内存插槽”。内存插槽的容量有限，随着游戏进度的推进会随之增大，更强力的芯片会占用更高的内存；芯片还可以进行升级，升级后的芯片用 XXX + X 的形式来表现，原本占用内存很小的技能，在升级后占用的内存就会增加。

芯片的种类繁多，从视频中出现的芯片大体上有 3 种，不代表正式版也如此，我总结如下：

·属性型芯片：增加人物的属性数值；
·技能型芯片：人物的攻击方式得到改变；
·功能型芯片：种类繁多，可以让游戏的 UI 发生变化，在地图上辅助显示一些功能，例如显示任务的目的地；伤害数值的样式变化，以及去掉敌人的生命条等等。

·冲击波 - 占用内存【8】格，让 2B 姐姐在攻击时产生向前破空的气刃，随着等级的增加，距离和伤害会更高。


·防反 - 在敌人即将攻击到 2B 的瞬间，然后防御（可能是左摇杆向攻击来自的方向推 By @flameshot233），即可弹开敌人的攻击并对其造成伤害，随着等级的提高伤害也会增加。


2B魔女时间 - Lv3 占用内存【14】格，喜闻乐见的慢动作系统，施展后敌人的动作会变慢，视频中的插槽等级为 3，大约 3 秒钟持续时间，Lv +1 +1秒嘛，好可怕…


子弹时间 - 占用内存【7】格，和魔女时间不同，在敌方的弹幕攻击即将碰到 2B 时，会自动触发时间减缓机能，再躲不过就说不过去了。

弹幕引爆 - 占用内存【7】格，配合“子弹时间”食用更佳，可以在 2B 被敌方弹幕击中后，清除屏幕上的所有敌方弹幕，连老妈都可以玩…


去除UI - 游戏 UI 的也是由各个芯片组成的，田浦贵久还蛋疼把所有 UI 芯片都卸载了，这会让游戏的难度陡然剧增。


45:30 芯片预设与登录
游戏中的技能芯片花样繁多，配出一套适合自己的组合耗时不菲，你可以将你配好的芯片组合登录至预设槽中。另外，为了让非核心向玩家也能畅玩，游戏还通过算法预置了一些推荐的芯片组合，你可以自由选择在重视平衡、攻击、防御 Type 之间。
