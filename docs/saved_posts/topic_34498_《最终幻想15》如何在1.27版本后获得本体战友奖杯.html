《最终幻想15》如何在1.27版本后获得本体战友奖杯

liok_3187 2019-02-07 13:47编辑     121评论
战友独立后，如果升到了1.27或以上，本体的战友奖杯就无法解锁了。
可以通过安装1.26版本来继续玩单机版战友。
方法参考https://forum.psnprofiles.com/topic/68912-how-to-get-comrades-trophies-after-update-127/

这个方法不算作弊，降到1.26版后也是要靠自己离线打出所有奖杯，原文如下：
https://forum.psnprofiles.com/topic/52663-what-constitutes-a-flag/
Playing on a different game version - Deleting update data to use a in-game glitch or play a game on a previous version of the game is also allowed and not a flaggable offense. This includes having to update a game through use of a proxy.
玩不同版本的游戏 - 通过删除更新补丁或者用不同的版本是被允许的，这包括了使用代理安装旧版本的游戏。

简单翻译一下步骤，详细步骤请看原文：
数字版和实体版都可以用这个方法。
首先你需要有战友DLC，皇家版没问题，如果不是皇家版，以前必须买过这个DLC，这个DLC现在已经下架了。
1. 删除PS4上的FFXV，存档不要动，没有问题，不放心可以备份一下。
2. 把PS4和PC用网线接到同一台路由器上。
3. PC安装Charles，按原文图片配置。
4. 查看PC的ip地址，在PS4上设置代理服务器，转到PC的Charles上。
5. 在PS4上，数字版重新下载游戏，实体版更新补丁，这时候Charles能抓到这个网络请求。
6. 在Charles上做json的Mapping映射。
7. 在PS4上删除更新到一半的内容，重新下载或者更新。如果成功，不管数字版还是实体版，下载的东西应该会变小，减少了4G左右。
8. 安装战友DLC，断网打开FFXV，如果自动下载更新补丁，马上暂停或者删除。

解释一下原理:
PSN所有的更新补丁在服务器上都会留着，虽然你用PS4直接更新就会用最新的。
我们用Charles把一个最新json文件指向了1.26版本的json文件，这个json描述了游戏的更新档补丁应该用什么版本，去哪个地址下载。所有json文件和所有更新补丁都是在PSN的服务器上，只是说我们用Charles转到了1.26版本，不要最新的。
