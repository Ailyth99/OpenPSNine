由日本电脑娱乐协会（CESA）主办、日经 BP 社协办的日本最大规模电玩展「2017 东京电玩展（Tokyo Game Show 2017）」已于今（21）日正式开展，活动主题为「来吧、感受超越现实的体验。（さあ、现実を超えた体験へ。）」。



本次东京电玩展同样包下幕张国际展场全部的场地举办，分为家庭游戏乐园、游戏新星区、一般展示、游戏学校区、罗曼史游戏区、日本游戏大赏、一般展示、商业解决方桉区、智慧型手机游戏区、贩售区、VR 区 / AR 区、e-Sports 区、饮食与休息区⋯⋯等各式各样的内容。



这次 TGS 主要大规模参展厂商包括 DMM Games、KOEI TECMO Games、Sony Interactive Entertainment、BANDAI NAMCO Entertainment、CAPCOM、SQUARE ENIX、SEGA Games、Wargaming、KONAMI、Bushiroad、Twitch 等。本文抢先送上现场摊位样貌的照片报导，供玩家抢先一窥展览样貌。
