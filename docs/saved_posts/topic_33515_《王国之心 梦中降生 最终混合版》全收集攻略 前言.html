竞技场任务

竞技场任务
即报告里第3项的第1子项，共计30个，每获得1个即提高1竞技场等级，许多任务需要达到一定等级后才能解锁，最后的Peering into Darkness需要达到30级才能解锁。

任务1~6是收集奖牌，需要累计获得9999个，推荐刷Monster of the Sea。1P使用核爆（Mega Flare）秒全场，2P开场使用Lightning Ray，然后趁鲸鱼攻击时候用雷突（Thunder Surge）反击，并寻觅机会再次使用Lightning Ray速杀，一次耗时2min左右（我最快纪录46s），获得675个奖牌。
刷No Heart一次能获得2000个奖牌（我不信你25min还打不死他），如果你在拿到9999解锁他后还需要奖牌可以来刷他。

任务7~23是完成竞技场任务，打法可以参考小峰攻略31P和32P，Boss基本无脑雷突即可。
其中，A Time to Chill（呪いの輪廻）只在阿库娅篇出现，阿库娅将其通关后其他人处才会解锁。Harsh Punishment（駆ける紡ぎ手）则只在泰拉篇出现，泰拉将其通关后其他人处才会解锁。

任务24~27是一定时间内完成赛车小游戏，注意在游玩的时候一定要是5圈的，而且必须是在竞技场游玩，不能是在迪士尼小镇，不然不算数。敌人数量无所谓。具体请看后文小游戏部分。

任务27~30是在大富翁里获胜，需要7场，注意在必须是在竞技场游玩，不能是在存档点。敌人数量和胜利目标无所谓。具体请看后文小游戏部分。

音乐冰激凌
共有5首曲子，每首曲子2个难度，通关简单（Beginner）难度解锁困难（Master）难度，cool评价或以上算作通关。
初期解锁第12首。通关第12首的简单难度解锁第34首，通关前四首的困难难度解锁第5首，只为全收集的话第5首的困难难度不一定要通关。
在第三首（Dessert Paradise）里打出Fantastic评价可以解锁Frozen Fortune风格及其终结技。全部通关可以获得一把还不错的键刃。
要点方面……KH系列的音游都很玄乎，有些时候按键时机要比节奏提早一些……总之加油。

Maestro

水果球
全收集路上的最难点
共有3次，最后一次面对的是皮特，打败他可以完成小游戏收集并获得射击技能Chaos Snake。
为什么说难呢？
首先，小游戏里你是不能调视角的，这也意味着对面打高球你根本看不见有多高，只能通过影子判断位置，这也让你很难反击高球。
其次，小游戏里你是不能锁定的，你难以辨认你的角色瞄准的是哪里，进而很可能无法控制他的行动。有可能角色瞄准的对象是视野外的某个高球，然后你一按按键他就跳了起来，然后无数球从你脚下飞过……这也让守门变的十分艰难，基本就别想着守门了进球得分要紧。（但是皮特可以扑救）
其次，皮特本身是有碰撞判定的，即使他晕掉了也是可以躺在地上守门的，而我们不能。就算我们站在门前，球会把我们撞晕后进球。
其次，和我们这几个被球碰一下就得晕半天的丢人大师不同，皮特是很难被打晕的。
最后，皮特的扣杀极其强力，瞄准精确，还是无间隔连发，甚至可以你打过去一个球，他直接在你硬直时候打回来，并把你打晕，顺带进一箩筐的球。当门在空中的时候，他的射门几乎是无法阻止的。
对付他的要点：
多多利用香蕉干扰他的行动。
多多利用△+X的扣杀得分，记得用左摇杆瞄准。
其次葡萄最好漏了也就漏了，尽量不要打碎，不然你可以看到他一个连续扣杀一大堆的葡萄飞进了你的门里，而以我方的硬直是绝对不可能挡的下来的。基本碎一个葡萄大势已去，碎两个葡萄就大局已定可以重开了……问题是有时候葡萄进门后还会碎出来一堆小葡萄哦？？？？
祈祷多发香蕉，少发葡萄。

大富翁
除了要拿奖励方块的情况没必要玩普通版，只玩竞技场版就好。
基本上就就是个运气活，而且演出效果就算调到快也很浪费时间，全收集过程中最烦人的一步了。边看番什么的边刷吧。
在前面6个地图全部获胜过1次（普通模式竞技场模式皆可），即可解锁第7张地图。
基本上多带魔法卡，使用Two Dice（消耗1个魔法卡）或者Three Dice（消耗2个魔法卡）一马当先跑出去，快速以最短路程过检查点就不难获胜。必要时候以Stun（消耗1个攻击卡）或者Confuse（消耗2个攻击卡）干扰NPC。然后就是祈祷NPC不要捡到白皮特自己不要捡到黑皮特了。
升级的时候优先把钱花在获得了连锁奖励的格子上，这是可以直接提升你的总资产的。
永无岛的特殊方块的干扰效果简直丧病，而且NPC总能十分精准地踩上去瞬间翻盘，加之彼得潘还会使用Two Dice或者Three Dice，加之这地图很大，你一马当先冲出去造成的差距不大，导致这成为最难的一个地图……加油拼运气啰orz
需要注意的是，左下角原本在右侧的奖励方块，如果你不把他弄到左侧去，NPC永远不会主动弄，而且当你弄到左侧去后，NPC会立刻把他弄回来并获得奖励。所以就别把他弄到左侧去便宜NPC了，这个奖励方块的奖励还十分丰厚。

这里我解释一下：
以Second Chance为例，里面说“Second Chance can be attached to a type N or type P Command recipe through Command Melding, either randomly or by applying a Pulsing Crystal.”即是你要在进行一个属于Type N或者Type P的合成使用Pulsing Crystal，产物就会获得这个技能（不使用也有概率获得）。点击type N可以链接到一个分类的页面，里面是所有可以通过type N的方式被合成的指令。再以Ars Arcanum为例，里面说“The first recipe can be melded by Ventus, and is a type N recipe. Aerial Slam (LV4) + Blitz (LV4)”就是说Aerial Slam (LV4) + Blitz (LV4)是一个Type N的合成。在把他们合成的时候使用Pulsing Crystal就会获得Second Chance了。
