游戏开发者大佬们2017年最期待的游戏！猜猜最高票数花落谁家

ashinlee 2017-01-06 18:43编辑     49评论
SIE 近日拉拢了一批游戏制作人，来让他们为自己在 2017 及未来几年最为期待的游戏投票。作为玩家，我们也可以去了解下这些开发者们在他们开发项目的间隙，都在玩着怎样的游戏。以下就是他们的选择。

老外的口语太诡异了…你们凑合着看吧

《黑暗教堂》（The Church of Darkness）


评语：《黑暗教堂》是一款了不起的游戏，玩家将在游戏中跟随主角一起探索邪教，以及偏执狂和乌托邦的阴暗世界，如果你对此题材感兴趣，我向你推荐。
John Watson // Stoic 工作室《旗帜的传说(The Banner Saga)》技术总监


《底特律 我欲成人》(Detroit: Become Human)


评语：这款法国开发商 Quantic Dream 打造的《底特律：变成人类》，游戏里的设定十分有趣。
Audrey Leprince // 开发《Furi》的法国独立游戏工作室 The Game Bakers 的联合创始人


《梦境》(Dreams)


评语：我知道这些工作室成员，他们非常有才华，对待工作的态度也非常的疯狂。这是必要条件，所以我相信他们也能克服许多困难的问题，创造他们想要表现的游戏。这就是为什么我这么强烈地希望他们开发出这款游戏，并再次开发一个惊人的游戏出来。
David Cage // QuanticDream 的创始人


《Farpoint》


评语：《Farpoint》是我最期待的 PS 游戏，我觉得这是一部堪称 VR 游戏里程碑的 FPS 游戏，在我迄今为止玩过的所有 VR 游戏中，这款游戏中的玩家移动和射击力学展现最为流畅且直观。在这一点上，同样可能是 VR 游戏发展史上的里程碑，并将推动 VR 这个新平台向前更进一步。
Andrew Wilans // 冰岛游戏公司 CCP《EVE：瓦尔基里》的首席设计师


《最终幻想7》高清重制版(Final Fantasy VII Remake)


评语：最期待的 PS 游戏?我选择《最终幻想7》重制版，1997年的《最终幻想7》曾经是我的最爱，可惜的是这次的重制版可能还要很长一段时间。
Chris Davis // 《逃脱者 The Escapists》开发商 Mouldy Toof Studios 的创始人


《荣耀战魂》(For Honor)


评语：我期待着育碧的《荣耀战魂》，玩游戏的 20 多年来，我大多数情况都在游戏中射击或者菊爆各种敌人，所以这种近距离的物理肉搏类游戏更容易感染我。此外，这个游戏是武士题材的，我也十分喜欢。但我觉得让我来作这款游戏的监制，游戏角色的外观和他的用剑的方式会得到更好的表现。
原田胜弘 // BNEI《铁拳》的项目总监


《战神》(God of War)


评语：我期待着新《战神》，即便我是一个动作游戏苦手，但我还是有欲望去挑战这款游戏。就以游戏的视角来说，原先系列的特点是鸟瞰视角和动态摄影，但现在的已经变成了越肩视角，这是一个大胆的挑战。我对《战神》新作如何去改变系列剧本和动作表现方面很感兴趣。
河野一聪 // BNEI《皇牌空战》的项目总监

评语：我期待着《战神》新作，这个超出预期的游戏将会 100% 令动作游戏爱好者满意，但不一样的是，新作中拓展了更多的角色，奎爷被迫地在遏制自己无尽的愤怒，并且还成为了一个有天赋的年轻小伙子的老爸，拥有者令人敬畏的权利。我已经等不及要玩了!
Tim Wileman //《乐高星球大战 LEGO STAR WARS》开发商 TT Games 的制作人


《GT赛车 Sport》(Gran Turismo Sport)


评语：纵然很难抉择……但在赛车游戏上，《GT赛车 Sport》肯定是最佳赛车游戏。
Paul Narducci // Ivory Tower 游戏制作公司《飙酷车神》的首席设计师

《地平线 零点黎明》(Horizon Zero Dawn)


评语：我对《地平线：零点黎明》最感兴趣，是因为这款游戏有着独特的艺术风格，巨型机器人与恐龙的组合让人感到兴奋!
David Hensley //《杀戮空间2》Tripwire Interactive

评语：《地平线：零点黎明》绝对是我最期待的 PS 游戏。我期待着去探索游戏中精美世界的每个角落，此外，女性主角加弓箭的组合，也是非常对我口的设定。
Scott Walters // 水晶动力《古墓丽影崛起》的艺术设计师

评语：《地平线：零点黎明》的美丽的背景，到处烧杀抢掠的机械恐龙这些设定都非常出色，我悄悄地希望Guerrilla Games公司会考虑包括我们自己的恐龙Rextro Sixtyfourus在内的支持游戏的NPC。
Andy Robinson // 开发商 Playtonic《Yooka-Laylee》的编剧

评语：我没有深入研究这款游戏。有一天，我在澳大利亚的一个贸易展上，看见有人正在操作这款游戏，我瞬间就被吸引了：“那是什么?”我问我身边的人：“哦，这是《地平线：零点黎明》” ，我想：“我必须尝试这个!”于是我玩了这款游戏的 Demo。游戏的角色设计超强，游戏中机械作为演化轨迹的整个概念是惊人的，我喜欢游戏中的动作与 RPG 元素的融合，迫不及待地想玩这款游戏了。
Jason Vandenberghe // 育碧蒙特利尔旗下中世纪题材游戏《荣耀战魂》的创意总监

评论：当我们正在巡演宣传《蝙蝠侠：阿卡姆VR》的时候，我很幸运地在 PS4 Pro 上试玩了这款游戏 4K 的 Demo，游戏令人惊叹的视觉效果真正把游戏的神秘世界带进我们的生活里。
Gaz Deaves // Rocksteady 工作室旗下《蝙蝠侠：阿卡姆VR》的社交媒体营销经理

评论：我可能有点偏见，但来自我们在荷兰阿姆斯特丹的姐妹工作室《地平线：零点黎明》将会在明年来一次爆发，我很幸运有足够的时间来好好地试玩这款游戏的 Demo，如同史诗一般的体验!
Alex Kanaris-Sotiriou // Guerrilla Cambridge 工作室《RIGS：机战联盟》的艺术总监

评论：必须是《地平线：零点黎明》，我迫不及待地想看 Guerrilla 团队的作品，我一直在关注这款游戏的发展历程，我期待着它的发售。Guerrilla 正在以新的类型和新的游戏挑战自己，我认为这是太棒了，女主 Aloy 看起来像是我想要了解更多的一个角色，所以我真的期待这款游戏的发售，游戏还有 RPG 元素，这让我角色很过瘾!
Michelle Ducker // Media Molecule工作室《梦境》的制作人

评论：这就是适合我的游戏，能够四处走动和探索的开放世界，加上硬派的动作风格。我喜欢游戏里世界的创意，它的外观，角色，RPG 元素，真的，我已经迫不及待想玩了。
Scott Campbell // WhiteMoon Dreams 公司《血战星穹》的创意总监

评论：女主看起来很酷，这款游戏的开放性在于玩家们如何解决问题，我是《孤岛惊魂》那些大型开放世界动作游戏的狂热粉丝，这款游戏看起来比它们都要漂亮。
Richard Rouse // Paranoid Productions 的总监


《王国之心2.8》(Kingdom Hearts 2.8)


评论：得知《王国之心2.8》将于 2017 年发售时，我很惊讶但又狂喜，我已经玩过所有的王国之心游戏，当然这款新作也会不例外，《王国之心2.8》将帮助我做好玩《王国之心3》的准备，我真的非常喜欢 PSP 上的《王国之心：梦中降生》，可能是我最喜欢的系列，再次在全高清的环境下扮演 Aqua，这真的令人兴奋的一款游戏。
Anthony Solaire // 法国育碧《舞力全开》国际产品经理


《最后生还者 第二幕》(The Last of Us Part II)


评论：虽然离《最后生还者 第二幕》的发售日还遥遥无期，但我真心希望能尽快玩到它，因为我喜欢《最后生还者》。
川田将央 // CAPCOM《生化危机7》制作人


《质量效应 仙女座》(Mass Effect: Andromeda)


评论：我喜欢《质量效应》系列，你总是可期待 BioWare 制作一款惊艳的游戏，一些真正特别的能让你回头买买买的游戏，事实上，我玩过的《质量效应》的次数比我玩《使命召唤 现代战争2》的次数还多，玩得太多了甚至影响到了夫妻关系，这款游戏早期的游戏画面是华丽的，我真的很期待看到他们用 4K 技术会做出些什么。
James “Jlo” Lopez // GearBox《无主之地》的制作人

评论：我很期待《质量效应 仙女座》，我是这个系列的粉丝，并且被游戏故事情节所吸引，花了很多时间在以前游戏上。
Sarah Hebbler // 开发商 Psynoix《火箭联盟》的执行制作人

评论：我看上去像是 EA 公司的托，但我真心喜欢《质量效应 仙女座》，所以我为《质量效应 仙女座》感到高兴，喜欢永无止境。
Paul Keslin // DICE 的星战新作《星球大战 前线》制作人


《美国职业棒球大联盟17》(MLB The Show 17)


评论：我是棒球的狂热，所以《美国职业棒球大联盟17》是我明年(也是几乎每年)最期待的的游戏，这是一款很赞的棒球游戏，今年的封面人物是 Ken Griffey Jr，我是他的粉丝，并把自己的狗也命名为 Griffey，所以很明显我对这款游戏的到来感到超级兴奋。
Eric Monacelli // Infinity Ward 公司传媒总监


《尼尔 机械纪元》(Nier: Automata)


评论：在 E3 大展上我有机会体验了一点点这款游戏，白金工作室开发的流畅的战斗系统给我留下了深刻的印象。
Spencer Yip //《堕落军团》总监

《逃生2》(Outlast II)


评论：没有什么比惊声尖叫和《逃生》更好的方法来在一个安静的夜晚里放松和享受的啦，这就是为什么我热切地期待着 Red Barrels 的朋友们开发的《逃生2》，我们已经准备好了波旁威士忌，牛肉干和尿布来等着玩这款游戏了。
Guillaume Voghel // Rogue Factor 公司《莫德海姆 诅咒之城》的制作人


《女神异闻录5》(Persona 5)


评论：《女神异闻录》系列带来了一些最时尚和最具凝聚力的因素，我喜欢在这款游戏中的音乐，音效，HUD，菜单和战斗演示获得的沉浸感，这一切都让我沉浸在角色们的心态里。
Jason Schroeder // 育碧旧金山公司《南方公园》高级制片人

评论：《女神异闻录5》!我要重返校园和我的朋友们一起玩。
Dave Crooks // Dodge Roll公司《挺进地牢》的游戏设计师

《掠食》(Prey)


评论：《掠食》，当我在 Karnaca 城迷失自我之后，我对 Arkane 工作室的下一部游戏作品很感兴趣。
Matthias Worch // Hangar 13 工作室《黑手党3》的设计总监


《荒野大镖客2》(Red Dead Redemption 2)


评论：《荒野大镖客2》，因为 Rockstar，我就已经对这款游戏迫不及待了。
Jake Solomon // Firaxis Games公司《幽浮2》的创意总监

评论：我很期待《荒野大镖客2》，第一是因为这是一款开放世界的游戏，还有西部题材的游戏非常稀少，我很兴奋看到 Rockstar 团队在技术上的发展，在玩过他们的《GTA 4》和《GTA 5》之后，我更为期待在现实世界中再次看到他们另一款成功的开放世界游戏。
Michael Gamble // BioWare 工作室《质量效应 仙女座》的开发者

评论：《荒野大镖客2》，我迫不及待地想看他们如何改进在树林里隐藏的熊，并用我贪婪的欲望来单手猎杀许多野生动物品种，来获取它们的肉。
Brian Hayes // EA 加拿大团队《EA Sports UFC 2》的创意总监

评论：我真心期待《荒野大镖客2》，孩提时代，我是少数几个不喜欢西部题材电影的人，但因为一些奇怪的原因 Rockstar 团队完全把我拖进了《荒野大镖客》的故事和世界里，游戏中的故事和人物都很惊艳，我迫不及待在 PS4 上玩到它了。
Martin Rabl // Giants Software 公司《模拟农场17》的公关经理

评论：明年会有很多款令人惊艳的 PS4 游戏，但我的首选是《荒野大镖客2》，我喜欢 Rockstar 团队具有野性的雄心和创造性的勇气，西部蛮荒地区的开放性情景设定是西部世界和力学的完美结合。
Ian Milham // 水晶动力《古墓丽影》的游戏总监

评论：必须是《荒野大镖客2》!我真的喜欢它的第一部，在更一般的水平上，我真的进入了牛仔的幻想之中，只是在乡下骑马，进入这种令人难以置信的狂野西部氛围为我带来一种独特的快感。
Stéphane “Fergus” Beley // Ivory Tower《飙酷车神》的游戏总监

评论：我明年最期待的游戏是《荒野大镖客2》，《荒野大镖客》是我最喜欢的开放世界游戏之一，骑着爱马环游西部世界，沉浸在这个世界里将会是一个特别有趣的经验，我等不及想再玩这款游戏了，这是 PS4 上更大型更优质的游戏的典范。
Eric Jensen //Bend Studios 工作室《人生尽头》的高级游戏设计师

评论：我最期待的游戏是《荒野大镖客2》，我是系列的粉丝，我喜欢 Rockstar 在整个《荒野大镖客》系列中完全保留老式西部片的感觉，从音乐，照明一直到人物，让玩家们因为游戏中好的，坏的和丑陋的而尖叫，考虑到在 2010 发售的《荒野大镖客》仍然看起来不错，我期待看到《荒野大镖客》新作可以在 PS4 上大放异彩。
Dominic Matthews // Ninja Theory 公司《地狱之刃》的产品开发

评论：我最期待的 PS 游戏必须是《荒野大镖客2》，Rockstar 总是能推出一些令人信服的叙事和优秀的角色刻画，并和游戏的视觉一起组成了史诗般的游戏体验，《荒野大镖客1》太棒了，是我过去十年中觉得眼前一亮的游戏，所以当我听说要这款游戏要出续作时，我把它放到最期待游戏的首位，虽然《GT赛车》和《战神》也是我的必玩游戏。
Phil Nightingale // Secret Sorcery 工作室《Tethered》的高级设计师


《生化危机7》(Resident Evil VII)


评论：我预测《生化危机7》将是第一款成熟的 VR 游戏，它所使用最新游戏引擎 RE Engine 很有意思，我的意思是，在 VR 设备上体验一款《生化危机》将会是一次地狱一般的旅程。
Jean-François Dugas // Deus Ex 公司执行游戏总监


《莎木3》(Shenmue III)


评论：《莎木3》，任何游戏的工作人员都必须向这款有着悠久历史的游戏致敬!《莎木》的回归对我们来说是一个伟大的时刻，迫不及待想看他们会把这款新作做成什么样子。
Nouredine Abboud // 育碧巴黎《幽灵行动 荒野》的高级制作人

评论：有许多很棒的游戏可供选择，但对我来说，我选《莎木3》，我永远难忘在世嘉 DC 上玩这款游戏的回忆，游戏的配乐也很棒，我现在仍然会定期听这些音乐。
Mathijs de Jonge // Guerrilla Games《地平线 零点黎明》的游戏总监


《铲子骑士 幽灵的折磨》(Shovel Knight: Spectre of Torment)


评论：我们整个团队的人都为这款游戏感到一致的兴奋，这是我们的创始人一直喜欢的游戏，现在我们有系列的新做玩啦!
Gavin McCarthy // Household Games 股份有限公司艺术总监


《蜘蛛侠》(Spider-Man)


评论：我双手投给 Insomniac 的《蜘蛛侠》，《瑞奇与叮当》团队在现代都市纽约用丝滑般流畅的动画开发了一款惊艳的游戏。
Ken Chua // 水晶动力《古墓丽影崛起》Assoc. Sft. 的设计师


《折磨 扭蒙拉之潮》(Torment: Tides of Numenera)


评论：充斥着老式的校园怪杰，我选择《折磨 扭蒙拉之潮》，这是我最喜欢的一款黑岛工作室巅峰之作《异域镇魂曲》的精神续作。
Dominic Guay // 育碧蒙特利尔工作室《看门狗2》的高级制作人


《艾迪芬奇的记忆》(What Remains of Edith Finch)


评论：这是一款我从未见过的游戏类型，游戏的操作方式是独一无二的，它让我笑死了…好吧，我不能继续说下去了。
Martin Kvale // Krillbite 工作室《Mosaic》的音频指导
