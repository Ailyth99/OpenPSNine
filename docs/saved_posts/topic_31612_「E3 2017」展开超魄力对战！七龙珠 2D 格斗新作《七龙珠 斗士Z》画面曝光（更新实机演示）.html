「E3 2017」展开超魄力对战！七龙珠 2D 格斗新作《七龙珠 斗士Z》画面曝光（更新实机演示）

sai8808 2017-06-12 22:05编辑     68评论


微软于 2017 年在美国洛杉矶举办的 E3 电玩展 Xbox 展前发表会中曝光，全新的七龙珠 2D 格斗新作游戏《七龙珠 斗士 Z（Dragonball Fighter Z）》将会于 2018 年初上市，并且释出了游戏画面。

E3宣传片


实机演示


根据影片的展示，《七龙珠 斗士 Z》将会以经典的 2D 横向格斗游戏玩法，玩家将可以 3 vs. 3 组队战斗，并以高画质的 3D 角色模组，展现出大魄力运镜的战斗场景。

《七龙珠 斗士 Z（Dragonball Fighter Z）》将于 2018 年初登陆PS4、X1、PC。
