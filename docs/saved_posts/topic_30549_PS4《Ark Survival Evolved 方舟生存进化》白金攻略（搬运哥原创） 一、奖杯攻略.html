游戏简介：
《方舟：生存进化》是一款由独立工作室Wildcard所制作的生存类沙盒动作角色扮演游戏。故事主要讲的是，一群醒来便出现在神秘海滩的人类为了生存而探索远古开放世界，玩家需要通过合作/竞技来获取生活必需品。人类需要打猎、收获、研究技术，甚至要根据动态天气变化（酷热白昼、冰冷夜晚）搭建住处，而危险的远古动物会随机出现威胁人类生存。

游戏中除了人类之外，恐龙绝对是主角、当然也有长毛象、灭绝鲨鱼复活，人类正遭受着来自海陆空的全方面威胁。玩家可以使用技巧来驯服和骑乘恐龙和其他原始生物。游戏还有一个RPG元素，玩家可以打造自己的角色，获得物品，技能和宠物。并且玩家也可以和其他上百名玩家一起组成一个部落，然后在部落斗争中取得胜利，并最终发现方舟的真实目的。

2016年12月，蜗牛游戏宣布代理《方舟：生存进化》，并将其更名为《方舟：生存进化Online》，采用道具收费，国服由2017年1月1日正式上线。

第一阶段：生存与驯服类
在ARK方舟里的第一天，你主要专注于生存和驯服就行了。在单人游戏世界中使用管理员命令代码，不仅能猎杀驯服最大的恐龙，而且还能快速获得奖杯。

Your first day...
你在游戏里生存了一天！
对于不用代码的萌新来说，第一天主要解决吃喝问题就可以了，很快就熬过去。有国外玩家反应没跳杯的问题，如果是这种情况，目前最好的解决办法是删除所有的保存和内容数据，并重新启动游戏。

Rex Rider
你骑上了霸王龙！
这三个杯可以一起解决，先输入代码summon rex_character_bp_c刷出霸王龙，等霸王龙出现后对着它再输出代码forcetame强制驯服，不用鞍也能坐上去，三个杯到手！

Giga Rider
你骑上了南方巨兽龙！
输入指令summon gigant_character_bp_c刷出南方巨兽龙，再输出forcetame强制驯服，骑上去即可！

Maximum Survivor
人物升到满级！
输入命令：addexperience 10000000 0 1（你可能需要重复几次步骤才能升到满级）。然后，在背包加点技能中随便点加技能，把技能点完了就满级了，这时应该就跳杯！

Maximum Dinosaur
你驯服的恐龙中首只升到满级！
恐龙等级满级好像在原等级基础上升到不超过63级，具体忘记了。。先骑上恐龙，再输入命令：addexperience 10000000 0 1（你可能需要重复几次步骤才能升到满级）。然后，按在恐龙背包加点中随便点加技能，把技能点完了就满级了，这时等一会就跳杯！

阶段2：勘探
现在你有基础知识了，你准备探索辽阔的岛屿和解锁一些神秘奖杯。

Cure-All
你治愈了水蛭病毒！
这个杯如果不用代码的话挺麻烦，需要4种材料制作解毒药Lesser Antidote，想自己体验游戏乐趣的话请忽略跳过以下步骤。这个杯放在阶段2第一个来解，因为不能先用消除负面状态的代码与god模式，用了的话感染水蛭是无法吸到你身上的，你也就不会得病，如果已经开了无敌，要重新启动游戏就清除所有代码了。

步骤开始，首先，输入代码giveengrams解锁全部制作蓝图，接着输代码giveitemnum 107 1 0 0获得研磨器皿，然后是准备解毒药材料了，代码如下：①稀有花朵giveitemnum 250 10 0 0 ②稀有蘑菇giveitemnum 251 10 0 0 ③麻醉药 giveitemnum 123 1 0 0 ④水蛭血液3个(需要先刷出无感染的水蛭summon leech_character_c，接着用长矛或者其他武器打死后挖尸体，就会得到水蛭血液了，数量不够可以再刷出来打)。

好了，现在材料都齐了，因为研磨器皿无法放在地上，需要一个平台载体，做个简单的茅草地基即可，把研磨器皿放上，站在研磨器旁，在制作列表找到解毒药Lesser Antidote，把4种材料拖进去中间的格子才行，放背包里是不行的，在研磨器制作文件夹里看到一个名称为drug的文件夹，点进去就可以制作解药了！然后还需要准备一个篝火(一旦感染了水蛭病毒，就会出现咳嗽，接下来你的血会掉的很快，需要用篝火把水蛭烧死)；接着，输代码刷出感染病毒的水蛭summon leech_character_diseased_c，先不打它，它会吸在你身上，你就得病了！站在篝火上面一会，水蛭从屏幕中消失了，证明已经被烧死，注意左下角还有个debuff生病的图标，喝掉背包里制作好的解毒药Lesser Antidote，病毒消失！跳杯！


Lowest Depth
你到达了海洋最深处！
这个按正常流程来做的话，需要先准备潜水服，潜水镜才行，还要驯服海洋里的生物，例如鱼龙，鲨鱼这些，再骑着游到海底最深的地方，太过麻烦了。这里可以先输入全属性无消耗(包括饥饿，寒冷，氧气，体力，弹夹等等)的代码infinitestats，以防止后面跑图奖杯出现各种意外，接着输入瞬移代码setplayerpos -102000 317800 -50000，立刻出现在海底，跳杯！

Highest Peak
你到达了最高峰的顶端！
这个奖杯的地点在火山的西南角，比较高的一座山顶上，不用代码的需要先驯服飞行生物才能飞上去。这里可以利用瞬移快速到达目得地，接着解完上面的杯后，在原地就能输入代码setplayerpos -97000 -51000 46000瞬移到山顶，这时不要着急，在原地来回走两步等一等，就可以跳杯了，会有些延迟！

Map Maker
你探索了小地图80%以上！
按下键可以查看小地图的进度，这个杯不难！强制驯服飞行生物，骑着它把小地图都逛完很快的，加点可以都加到它的体力与移速上！

阶段3：神器、图鉴与BOSS
最耗时的阶段，也是问题最多的阶段。开始第三阶段之前，建议将存档数据备份到U盘或者上传到云储存。如果收集完还没跳杯，则唯一的解决方法是删除所有的存档数据，重新开始。
