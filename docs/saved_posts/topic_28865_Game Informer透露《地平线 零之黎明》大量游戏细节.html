Game Informer透露《地平线 零之黎明》大量游戏细节

vgtimes 2016-09-07 12:43编辑     19评论
著名电玩杂志《Game Informer》的2016年10月号已经上线，这个月登上GI封面的是PS4独占大作《地平线 零之黎明》。本期的封面故事透露了大量《地平线》的新情报，从主角Aloy的故事到游戏中的各种设定，信息量十足。



视频只能使用 youku 和 bilibili 哦


10月号的GI为我们带来了以下这些信息：

相比他们的代表作《杀戮地带》，Guerrilla希望制作一款“宏大而美丽”的游戏。他们想构筑出一个你愿意沉浸其中的世界，而不是想要逃离。
《地平线》的开发工作2010年就开始了。
《地平线》拥有自己的引擎，和《杀戮地带 暗影坠落》的引擎不同。
GI在游戏设定中看到了沙漠、极地在内的各种环境，还有潜伏在水下的机械生物（这防水性能，不愧是索尼的游戏……）。
Guerrilla承诺他们会在游戏中解释这个世界为什么会变成这副模样，并且明确表示这是一个科幻故事，不含奇幻成分。
游戏中有多个不同的部落，他们对于机械生物的看法也不一样，有人认为机械生物变得残暴是神明的惩罚。

你不能招募其他部落，但可以在他们那儿购买装备，这些部落都会有各自的故事和支线任务。
Aloy最关心的她的身世，她想知道自己的妈妈是谁。
Aloy成长于Nora部落，这是一个由女性统治的部落，然而大家都对其视而不见，似乎是因为某种原因刻意回避。
把Aloy拉扯大的是一个名叫Rost的大叔，但是部落迫使他这么做的。
Rost非常严厉，有时又会过分溺爱，但他都是为了Aloy好。

游戏中将可以操作童年的Aloy。
地图上会有名为“Corruption Zones”的区域，里面有极其危险的敌人，游戏中最激烈的战斗基本上都发生在这里。
支线任务很多，但还没到《巫师》的规模。
游戏的深度、智能的AI、流畅的操作都让GI非常惊喜。

你可以制作各种服装改变Aloy的外观。
有一张截图中Aloy身着金属胸甲，而不是我们看到的动物皮毛。
Aloy的技能树有3个分支，分别对应潜行、战斗和资源收集。
在打败敌人后，可以获得装嵌在武器和防具上的配件，这些东西可以提升装备性能。


《地平线 零之黎明》将在2017年2月28日正式发售，同时台湾FACEBOOK也确认了将有中文版（时间尚未确定）
