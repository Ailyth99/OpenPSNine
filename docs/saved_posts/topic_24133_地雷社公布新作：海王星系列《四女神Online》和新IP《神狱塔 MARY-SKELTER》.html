地雷社公布新作：海王星系列《四女神Online》和新IP《神狱塔 MARY-SKELTER》

luna_noire 2016-03-13 14:11编辑     60评论
在13日的「游戏的电击 感谢祭2016」上地雷社公布了两部新的游戏。
《四女神Online》只公布了一段简短的旧作剪辑PV，具体平台和游戏类型详情将于4月14日的『電撃PlayStation Vol.612』进一步公开，根据闪过的截图推测应该还是RPG。（我是很希望真的和标题一样能联机啦……早前海王星制作组曾接受海外媒体采访，并表示有兴趣开发联机、沙盒、像素风格的海王星新作）









《神狱塔 MARY-SKELTER》作为電撃文库 X 電撃PlayStation X COMPILE HEART推出的动作3D迷宫RPG，剧情原案由乙野四方字担当。本作舞台为日本，位于充满谜团的地下666米处的监狱“Jail”。
