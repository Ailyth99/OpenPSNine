PS4《魔界战记5》公布主要角色能使用的「合体技」以及能制作原创单位元素

sai8808 2014-11-22 16:55编辑     9评论
日本一 Software 于近日公布，该公司将于2015年3月26日预定发售的PS4专用游戏《魔界战记 5（魔界戦记ディスガイア5）》的最新情报。

在这次的报导中，将为玩家们介绍关于游戏中所实装的新系统「合体技」的内容。这是仅有主角齐利亚（キリア）等主要角色才能使用的专用技，是在故事中具有 “相互爱慕” 或 “宿敌” 等这种关係的角色们才能够发动的技巧。此外也公布了新的主要游戏角色，为了要达成暗杀魔帝渥伊特达克的目的，而企图利用齐利亚的「克里斯特（クリスト）」（声优：间岛淳司），在齐利亚面前出现，并无条件帮助他的「杰洛肯（ゼロッケン）」（声优：柿原彻也）。当然，两位新角色也能够使用合体技。

以及也公布了关于能够製作出原创单位的「招募屋（スカウト屋）」的情报。包含新登场的闇黑骑士 / 女僕 / 妖精族在内，玩家能够选择的职业，及各单位可设定的「魔技能（魔ビリティー）」「特殊技」等概要内容，也请玩家们仔细确认。

■ 登场角色

克里斯特
配音：间岛淳司



「我是克里斯特。因为某些缘故而成为了身负暗杀魔帝渥伊特达克之使命的人。」

隶属『雄伟巨大魔界』的其中一位魔王。

盯上了齐利亚所拥有十分强大的潜在魔力，并企图利用他来暗杀渥伊特达克。

作为军师及参谋的能力十分的出色，为了胜利将会不择手段。



杰洛肯
配音：柿原彻也



「好啦、该是门外汉退场的时候啊！与洛斯特战斗可是我的职责啊！」

自称为传说中的大魔拳哥鲁迪翁（ゴルディオン）的弟子，与洛斯特为敌孤军奋战的年幼魔王。

出现在与洛斯特战斗的齐利亚等人面前，并不求回报的协助着他们。

是个根据对手的习性来改变其态度，具有着柔软般思考方式的人，也有着轻浮的一面。




■ 新系统

魔王们的共同战线！
藉由与伙伴们的 “共斗” 所产生出的 “合体技”

在《魔界战记 5》里，除了与相邻的我方单位一同协力进行攻击的 “连携” 外，还实装了被称为 “合体技” 的新战斗招式。

这是仅有齐利亚等主要角色才能够使用的专用技巧，藉由两位主要角色的共斗所施放出的招式。



合体技只要使拥有特定关联性的伙伴间成为相邻状态之后就可以发动，并藉由「相互爱慕」或「宿敌」等各种角色们之间的关联性＆组合来予以发动。

■ 齐利亚＆赛拉菲努（セラフィーヌ）的合体技 “零下心灭”


主角与女主角、相互爱慕（？）的两个人的合体技。将齐利亚的冻结能力与赛拉菲努的魅惑之力合体！


藉由故事的进行，让角色们之间的关係持续发展，使齐利亚等人学会新的合体技。

■ 马格纳斯（マグナス）＆杰洛肯的合体技 “炎风十字杀”



由热血的男子汉们结合力与力的合体技。身上各自缠绕着火焰与烈风来进行攻击，以十字攻击来撕裂敌人。

■ 乌莎莉亚（ウサリア）＆克里斯特的合体技 “咖哩爆发”


十分喜欢咖哩的乌莎莉亚与谋略家克里斯特间十分奇妙的合体技。沉默寡言的魔王们也有展现出华丽一面的时候･･････？


■ 育成系统

让顽强的恶魔们成为伙伴，强化反乱军的战力！“招募屋”

《魔界战记》系列裡玩家们所熟悉的，能够製作出原创单位的创造系统。

由玩家自己製作出一起战斗的战斗单位，藉此来强化我军的战力。

在本作品内玩家们可以从四十种以上的种族＆职业中选择其中一种，并可根据玩家本身的喜好来设定以姓名及能力值为首的各类数值。



■ 招募屋

单位的招募（角色製作），能够在位于据点内被称为「招募屋」的设施内来进行。

在招募屋内可进行设定种族或职业、各种能力值、等级、性格等反乱军所需人才的“条件”，将能从各地裡找出适合条件的人才。

■ 职业＆种族介绍

能供玩家选择的单位职业・种族存在着多样化的种类。
在此除了公布基本的职业外，再加上系列作中首次登场的新职业・种族！

闇黑骑士 NEW CLASS

操纵黑暗的黑衣骑士。
施展有如波纹扩散开来般的攻击，对施予对象的周遭造成影响。




主要擅长武器：剑、枪、斧
固有魔技能：「波纹力」
（攻击相邻的单位时，朝向攻击对象周遭的敌方单位造成连锁伤害。）




女僕 NEW CLASS

忠实侍奉主人的女僕。
能够更为引发出道具的力量。




主要擅长武器：枪械
固有魔技能：「具有效率的工作」
（在一回合内仅限一次，除了原有的行动外，还能够追加进行使用道具的行动。）




妖精族 NEW CLASS

手持大型提灯的光之妖精。
可吸收周围的魔力来回复 SP。



魔变化（魔チェンジ）：杖
固有魔技能：「魔力生成」
（回合结束时，SP 将回复 10 %）



战士（男）

拥有强韧的肉体及不屈的精神的战士。
在不利的状况下能发挥出其力量。



主要擅长武器：剑、枪、斧
固有魔技能：「狂野嘶吼（ワイルドクライ）」
（能够提升自身（相邻的敌方单位数量 X 15 %）的能力值）




战士（女）

兼具灵巧身躯及强大力量的女战士。
擅长一点突破的电击战术。



主要擅长武器：剑、枪、弓
固有魔技能：「冲锋攻击（アサルトアタック）」
（提升自身（移动格数 X 10 %）的伤害值）



魔法使（男）

过于认真的魔法使男孩。
能够使用具有高威力的技巧与魔法。



主要擅长武器：杖
固有魔技能：「魔力收束」
（使用特殊技时，攻击力＋ 30 %，但相对的消耗所需的 SP 也会增加 50 %）



魔法使（女）

天真活泼的魔法使女孩。
能够有效的运用技巧及魔法。



主要擅长武器：杖
固有魔技能：「魔力节制」
（使用特殊技时，消耗所需的 SP－ 50 %）



僧侣（男）

在惩戒自身的同时使用治癒力量的僧侣。
在回复时如果有异常状态将会同时予以治癒。



主要擅长武器：杖、枪
固有魔技能：「医疗治癒（メディカルヒール）」
（使用回复魔法时，也会一併回复状态异常）



僧侣（女）

能治癒受伤的同伴的女僧侣。
更够进行更为强力的回复术。



主要擅长武器：杖、弓
固有魔技能：「精通治癒者（练达の癒し手）」
（使用回复魔法的效果＋100%）



弓箭手（アーチャー）

个性善良的魔界猎人。
以护援射击支援伙伴。



主要擅长武器：弓
固有魔技能：「支援攻击」
（对我方单位所攻击的敌人，将有一定机率会进行追加攻击）



◇ 魔技能

各单位有着因应其种族或职业，被称之为“魔技能”的特殊效果。

除了有着影响伤害值或能力值这种较为单纯的魔技能外，也存在着增加行动次数、扩大攻击范围的特殊魔技能。

不论哪种技能效果都十分强力的缘故，玩家必需在意识着魔技能的情况下进行角色的培育及制定战法，这点可是相当重要的。


装备魔技能时，各个魔技能所设定的“成本（コスト）”总和，必需不得超过单位所拥有“插槽数（スロット数）”的上限。


“职业熟练度”提升后，单位将会学习到新的魔技能。等级越高将会使单位的插槽数获得扩增，并能够同时装备複数的魔技能。

◇ 职业的熟练度

在各职业・种族裡，存在着表示对该职业熟练情况的“熟练度”。

熟练度是藉由战斗来提升，根据上升的情况将可使魔技能获得学习或成长补正等奖励效果。

熟练度即使是在进行变更职业・种族的“转生”后，也会持续继承的缘故，因此能够製作出拥有複数职业熟练度之奖励效果的强力单位。



◇ 特殊技

各单位皆能够根据其各自不同的职业或种族、使用武器来施展出相对应的“特殊技”。

不论是哪种特殊技都有着固定特有的演出与效果，从华丽到独特的表现等，存在着十分丰富的种类及数量。

武器技･･･是根据装备的武器，而能够学会并加以运用的特殊技能。




魔法･･･是 “魔法使” 或 “僧侣” 等，特定的职业才能学会的特殊技能。



职业技･･･是因应各种职业的固有专用技能。
