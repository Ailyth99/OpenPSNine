序言

序言

趁着10月会免 幻痛。大佬们跟我开玩笑说 把我的视频放上来凑个热闹，我就认真（伪）一回。

闲话就不多说，语言组织能力有限，劲最大努力组织语言通顺。
奖杯没什么难点 ，基本上都是流程 和全收集 每个奖杯都有 TIPS 我就不再多说。

游戏开始：

任务1到8：

武器按照

麻醉枪 → 潜入服 → 富尔顿 → Idroid → 假手套件和假手定位声呐 → 狙击枪（为开发麻醉狙击枪做准备）→ 望远镜 → 能穿墙看到敌人的药 的顺序来开发。

任务9到31：

麻醉枪 → 子弹时间药 → 假人娃娃 → 富尔顿 → 电磁地雷 → C4 → 烟雾弹 → Idroid → 狙击枪（可以跟静静对狙了）→ 望远镜 → 火箭炮 → 电击假手 → 重狙 → 富尔顿（可以回收车辆了）→ Idroid → 望远镜 → 防水纸箱 → 寄生虫装甲（注1）→ 生化服 → 烟雾发射器 → 高等级火箭炮 的顺序来开发。

任务32到50：

高级火箭炮 → 儿童富尔顿 到剩下的随意开发。

错了，请指出 ，以便不误导大家刷奖杯。

按照这个流程开发再加上前期全清各个据点资源，人员。手里的资源是不会缺太多的，母基地也按武器需求去升级就好，

武器开发不一定要开发高了，适当的升级。不要去急着开发同伴的衣服和直升机升级套件，没用的，纯粹是白花钱，出击费用还会高的吓人。 最主要的还是富尔顿 气球，钻石狗人贩必备神器。
前期手里只有气球 把俘虏请搬运到离其他正常敌兵30米外开阔地带再放气球，30米外士兵是听不见放气球声的 这点注意

后期有了虫洞，为了安全不触发警戒，也请离远了再放。

毕竟这回 触发警戒后 不会消失 。敌兵们会一直搜索该区域，为了S评价和英雄度，还是要猥琐之上。


重点任务心得

重点任务心得

关于每个任务task挑战，有的确实要重复做才行，不用强求一次就全部做完。

就说几个重点的

第4关 C2W 默认降落地点 仔细找 萌萌的DD狗 在等你把它带回家。

出击任务带哪个同伴呢，我推荐带狗。需要注意的是 在进行对敌兵小队进行抓捕时，一定要让狗 等在一边，要不有可能你还没到跟前，狗先撞到敌人窝里去，都回头 你就悲剧了。。。。。所以要管好狗，不要让它乱跑。
静静虽然厉害 但不能有效标记出敌人位置，容易漏。马 最后再刷亲密度吧 最慢的，流程都熟悉 ，一边开发核弹一边 刷 马 的 亲密度。


任务9的 人质 回收 位置 ：

第1个 最关键 默认降落 地点 右手边 的山坡峡谷 中 ，会有一个敌兵小队去搜索，被他们找到话，人质小命不保。
第2个 默认降落地点 一直向前走 就会看到
第3个 第10关关押人质的宫殿遗迹 旁边的牢房中
第4个 直走第一座桥 左边河流中 搜索
第5个 山猫会通知你，有个吉普车上押着人质，会标出来
第6个 在第5关关押工程师的位置

找不到的话 ，记得带上DD狗

其实这个全人质回收和全车辆，摧毁直升机是可以 一次达成的，那就要靠马了，这要看运气好不好，一开始降落在第10关最后关押 马拉克 的据点 后的山坡上 ，快速去 据点 回收 第一台 装甲车 ，之后迅速返回 小山坡 底下的大岩石处 放置电磁地雷 等待 装甲车和 坦克 自己压上去， 回收 押解人质吉普车和 带火箭炮 的卡车 要快速 奔跑到 离回收 车辆 100米外 的地方放 电磁地雷，进行回收 ，这样避免相互被撞见 被发现 ，全部快速回收后，最后提示出现的坦克和直升机不要管，快速 潜行通过 前方据点 上马向第一个人质所在峡谷奔去，烟雾弹和睡眠手雷，电击手套之类的道具就上吧，回收人质1号达成后，速度不够快的话就只能悲剧了，刚跑到峡谷下就会提示人质已被杀 ， 人质2号已经在 默认降落地点 后方的 沙漠中的一颗树下了，其他人质位置不变，途中可以回收多出来的 坦克和 直升机 ，这样玩可能拿不到S评价了。

第10关 ：

前3个人质 也要快速回收 都在宫殿里和旁边的牢房。剩下的在关押马拉克的据点里，切记不要触发警戒和警报，人质会被杀掉的 ，注意

其他任务 慢慢来就好。

BOSS战

注1 ：在刚到第15关的情况下 不被骷髅兵 发现的情况 可以用 假人娃娃 和 睡眠地雷 和手雷 扔他们 ，不够就叫 空投 ，提前回收 骷髅兵的 寄生虫装甲 为 29关正式 收拾他们 做准备。请保持骷髅兵35米远。

如果在这关碰到司机回收不达成，请使用气球回收。我就碰到了这个BUG ，我都打到收尾了，才发现司机回收没达成，当时就石化了 记得提前备份存档，第一次回收记得查看 TASK 列表 是否回收达成 。

第28关 同 EX 42关
初期遇到4个女骷髅兵 ，还是从瀑布底下的石缝爬上去，溜掉的好，后期有重狙和烟雾反射器，寄生虫装甲后 会轻松很多。 网上也有 CQC 大法 大家可以去参考。


第29关

到这里应该手里都有 重狙 和子弹时间药，寄生虫装甲 了 ，上去狙他们的头吧，至于BOSS们的 近战 刀砍 ，看他们有动作就不要再去做多余的动作了，专心等待QTE 出现 。即使有寄生虫装甲，BOSS的爆突石头还是要躲得。


至于静静 的非武器 击倒 ，
11关是可以用 空投的。

EX的第40关 ，可以用各种非致命武器打到一格气，找出她的位置，担心就穿上隐身服和寄生虫装甲 ，上去给她一拳 就搞定了。这是比较笨的方法。欢迎各位大佬们分享 简单又方便的 攻略之法。




其他任务流程人质回收，后面的视频有演示。

草药全收集 嗯，你只要遇见了 多按 圆圈 就好。
动物全收集 后面放个视频好了。
武器蓝图全收集 前期全清据点 多拷问小兵 就好




至于支线任务 就留给个大家 自行漫游和 探索了 。

为了获得无痕迹，你不能：

使用R2（触发）按钮发射任何武器或使用任何辅助的武器（如：手雷等）
攻击使用伙伴。
伤害任何人（包括碰撞 和 CQC抓取或者击倒 敌人 包括假人，但不包括BOSS）
进入子弹时间模式/警戒和警报状态
呼叫空投
使用特殊道具，如无限弹药头带，隐身布，隐身迷彩，Dwalker的回收枪等。

你只能去举枪 俘虏或者 叫小兵躺下 马上去挂气球把他清出战场，要不一会会被其他小兵发现，从而进入警戒状态。
无痕也就是刷刷排行榜还有些作用，所以我不推荐大家一开始盲目的去追求 无痕过关。
