《That's You!》白金攻略

fyjan 2019-10-27 22:32编辑     6评论
美服PS+曾免费送过
想白金首先你需要准备6个手机或平板（部份Android模拟器听说也行，但我没用过）。

IOS版美服可以下（日服、国服都没有），Android版在这里下下载地址。

本攻略参考了psnp的英文攻略。主要没人玩这奖杯神作，所以就随便写了写。（主要是找不到人联机……）
第一次写如果有错误还请指正。

奖杯白1 金6 银9 铜12 总28
白金难度1/10
白金时间5-10h（有1个奖杯需要刷的时间比较长）
容易错过的奖杯无
联机奖杯1个（玩的人好少）



开始游戏前，在游戏主界面按L1选择Short。


2人游戏
手机app主界面，其中1台选择MAKE DRAWINGS，另1台选择WRITE QUESTIONS，

通过右上角刷新，刷新为两个不同的主题，分别添加新图片、新文字。

开始游戏时把自己建立的问题添加进去。


10种主题都玩过一遍，自己提供问题会锁定主题，让统计起来更容易，不至于最后都不知道自己漏掉哪个，当然最后真能有遗漏的话，那rp要多差……
Well-Travelled
体验过游戏中的全部十个区域


12种CARDs 都选择一遍，进行游戏，刷的过程肯定会解。
Unique Profile
使用过所有的个人资料卡片背景



拍自己头像时选择小丑标志，选择一种效果。
Impressionist
对自己的社交头像使用一个滤镜效果



开始第一遍游戏，保证每个问题2人都选择一样，且有JOKERS瓶盖就用掉（回答问题时，屏幕最下面可以选择是否使用JOKERS瓶盖）。


出现画画的游戏，把每种颜色都用一遍。
Color-full
在一局游戏中作画时使用每种颜色至少一次



通关后会解锁以下奖杯。
Bestseller
在“填词游戏”中达成一致同意

Maximum Return
在一致同意的情况下打出小丑牌

Two's Company
完成一次双人游戏

Doodling
用你涂鸦的画作，玩一局游戏

Authority
用你拟定的问题，玩一局游戏

Planning Ahead
游戏结束时不剩下小丑牌

Peas In A Pod
在双人游戏中取得高分



3人游戏
准备3个手机或平板开始游戏，第一个问题3人都选择一样，且都使用JOKERS瓶盖。
Good Start
在游戏的第一个问题就达成一致同意

Joker Full House
在3-6人游戏中，每个人都在达成一致的情况下打出小丑牌


出现Photo Game时（4张图片选一张的游戏），3人要选择一样；随后出现用这张图画画的Drawing Game，3人要选择一样。
Artist
在“画图游戏”和“照片游戏”中，同一个玩家的画图获得一致同意


再次出现画画的游戏Drawing Game，每人选择自己画的那副画，且其中1人使用JOKERS瓶盖。
Baby Carrot
为你的画作打出小丑牌，但无人投票给你

Anarchy
在一场3-6人游戏中获得一次“混乱”评价


通关解锁
Three's a Crowd
完成一次三人游戏


再开一轮3人游戏，第一个问题3人都选择一样，之后的问题3人都选择不一样，且始终不能使用JOKERS瓶盖。
Close Call
以彼此间400以内的分差，完成一局3-6人游戏


这个奖杯是每种游戏在当前局人数下都选择过同样的答案。刷奖杯过程中自然会解。
Broad Knowledge
在每种游戏模式中达成一致同意

Decision Game - 2 Players ONLY
Gallery Game - 2-6 Players LOCAL ONLY
Drawing Game - 2-6 Players
Mimic Game - 2-6 Players
Question Game - 3-6 Players
Picture Game - 3-6 Players
Word Game - 3-6 Players
Final Game - 3-6 Players
Photo Game - 3-6 Players LOCAL ONLY


4人游戏
准备4个手机或平板开始游戏，选项随便，通关一次。
All 4 One
完成一次四人游戏



5人游戏
准备5个手机或平板开始游戏，选项随便，通关一次。
High Five
完成一次五人游戏



6人游戏
准备6个手机或平板进入app，其中3台选择界面下方的MAKE DRAWINGS，通过右上角刷新，刷新为不同的3个主题，新建图片；剩下3台选择界面下方的WRITE QUESTIONS，通过右上角刷新，刷新为与上面相同的3个主题，新建文字。添加自己建立的问题，开始游戏，选项随便，通关一次。（6人游戏很漫长，有6轮…………）
Full House
完成一次六人游戏

Prolific Creators
用六个由你们创作的画作或问题来完成一场六人游戏



唯一联机奖杯
必须加为PSN好友才能联机，每人需要准备1个手机就行。
Friends Without Borders
玩一局网络对战



剩下的刷刷刷
刷2人游戏吧，30场也是很漫长的（其他奖杯15场以内就能搞定）。
Novice
玩过五场游戏

Master
玩过十五场游戏

Veteran
玩过三十场游戏



然后你就白金了……
Know-All
收集全部奖杯
