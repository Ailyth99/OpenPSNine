「2017东京电玩展」各厂商特设网站总整理

sai8808 2017-09-21 07:38编辑     2评论
不知道去哪查展区资讯和舞台活动情报吗？看这篇就对啦！

献给对游戏非常热爱的你



在2017年9月21日至24日，日本最大规模的游戏祭典「2017东京电玩展」终于要开幕啦！
为了更加有效率在各厂商的展区和舞台之间来回参观，必须得先好好预习呢。
未能前往会场的玩家，也能透过这篇总整理报导，查询各生放送的行程表哦！


■Wargaming Japan
http://tgs2017jp.wargaming.net/
以《战车世界》表演赛为中心举办各项活动。
此外也有电视动画《少女与战车》合作舞台活动。


■CAPCOM
http://www.capcom.co.jp/tgs/
由于在一般公开日能够试玩《魔物猎人：世界》，必须在1～8厅侧边的入场处领取排队整理券。
注意不要排错地方囉。


■GungHo（《LET IT DIE》）
https://event.gungho.jp/tgs/2017/
本次只以《LET IT DIE》单独出展。
除了试玩、特别节目外，还将举办钢管舞表演活动。


■KOEI TECMO Games
https://www.gamecity.ne.jp/tgs2017/
展出旗下各品牌游戏。
能够试玩《真・三国无双 8》、《火纹无双》、《莉缇＆苏尔的鍊金工房》等作品。


■KONAMI
https://www.konami.com/games/event/tgs/
以《潜龙谍影 生存》、《LOVE PLUS EVERY》、《纯爱偶像》等游戏出展。
于国外引发话题的独立游戏《VA-11 Hall-A》也会在这裡登场。


■SQUARE ENIX
http://www.jp.square-enix.com/tgs17/
推出《DISSIDIA FINAL FANTASY NT NT》试玩服务。也能抢先游玩《恶灵附身2》。


■SEGA Games
http://tgs.sega.jp/
展区内能试玩《如龙 极2》、《索尼克 力量》等最新作。
《D×2 真・女神转生启示录》，则是能以 VR 欣赏恶魔的威猛造型。
其他还有 Atlus、5pb、日本一、FuRyu、Warner 等各厂作品也会于该展区出展。


■Sony Interactive Entertainment
http://www.jp.playstation.com/events/tgs/
以未上市 PS4 游戏为中心出展。
成为 PlayStation Plus 会员就能抢先预约各项服务，千万别错过囉。


■D3 PUBLISHER
https://www.d3p.co.jp/tgs2017/
于昨(19)日公开的最新作品《地球防卫军 钢铁之雨》，马上就能试玩


■DMM GAMES
http://www.dmm.com/netgame/tgs/
以《少女摇滚！》、《Terra Battle 2》出展。
最后一天还会举办话题新作《绝地求生》的舞台活动。


■BANDAI NAMCO Entertainment
http://tgs2017.bandainamcoent.co.jp/
能够试玩《噬血代码 CODE VEIN》、《巨影都市》、《七龙珠 战士集结》等游戏。
另外，要观赏舞台活动「IM＠S GAME SHOW 2017」必须先领取排队整理券，请事先确认。


■Bushiroad
http://bushiroad.com/sp/tgs2017/
以《动物朋友亭楼》、《新网球王子 RisingBeat》等手机游戏为中心出展。


最后附上「2017东京电玩展」官方网站：http://expo.nikkeibp.co.jp/tgs/2017/
