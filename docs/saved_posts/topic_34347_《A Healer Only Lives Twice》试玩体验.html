《A Healer Only Lives Twice》试玩体验

yzh321 2018-12-09 00:54编辑     2评论

先认识一下游戏基本规则和画面内容。
左下角是当前火把数和燃烧进度条，火把携带上限是5，火把烧完就game over哪怕你的勇者能一招秒最终boss。火把燃烧速度大概是一支火把能让你攻击8次。所有道具都可选中后按烧掉来续火把且续的量所有道具都是一样的。如果UPGRADE项升级过再利用的话有几率获得所烧道具的一部分合成素材。
左边中部是当前获得素材，各素材携带上限依然是5，看到满了就去合成最低级的烧掉续火把吧。
右上经验条，攻防数值，防御力右边还有一个槽是brave槽，满了后下一次攻击是打击一横排4个敌人的攻击。
右边是身体5个部位的血量，各部位被打掉会有debuff。头部会不能使用brave攻击，左手攻击力下降30%，胴体全身血量下降30%,右手防御下降30%，腿大幅下降的切换目标速度。被打掉的部位进下一层时会1血复活。

每层都固定出现一支火把和一个赤い石(除main模式的tier 25，效果是秒全部怪，但是在怪后面的道具也一起打飞了)

3个模式通关杯：
Beyond the Nightmare
Complete MAIN Mode or EX MAIN Mode
Their Journey Continues
Complete EX MAIN Mode

EX模式所有主菜单升级项无效，但是技能不需升级项支持也能升到5星满级。
需打完tier 25的红龙，血量在727-769之间应该是750血量。MAIN模式与TRIAL模式不同，无法合成赤い石，tier 25也不同于其他层不提供火把红石，所以tier 25必须靠实力打过去，如果前面关卡都靠红石一路秒不注重提升等级的话是过不去的。

技能升级顺序建议：
高潔（半防2连动）习得
信義（生成mana药）升满
忠節（复活）升到2星
高潔（半防2连动）升满
信仰（10秒后全身回复）在tier 25前留1,2个升级点外全升这项
胆勇（加攻击）在tier 25前起码要习得技能，升1,2星更好

总的来说就是用信義生成mana药，低级药一概烧了续火把，中级的看情况基本也是用来烧，高级的留着用。其余除了怀表和红石外也全部烧了。注意火把持有数上限是5，手上有5把的时候等烧完再捡火把别浪费。
用高潔的2连动作为主要输出手段，刚进下一层的时候如果马上用是会在第一次攻击前就效果消失的，最好在挨完第一排4个敌人的攻击后马上用技能秒了第一个敌人后马上LR切到第二个敌人身上连杀。时刻注意自己的腿别死了不让会降低切换目标速度第2击很难及时切到别的敌人身上。
基本保持等级与层数相当就打得很舒服了，保持火把3个左右尽量杀怪升级别用红石秒全部，在tier 22打狮鹫注意拿这层出现的+12攻道具。存几个高级mana药到tier 25打红龙，先用几次加攻后2动就能1回合秒了。

Behind You
Complete TRIAL Mode
必备UPGRADE项：
第一页第四项(多品种生产)[最优先]
第二页第二项(再利用)[次优先]
第二页第三项(燃烧效率改善)
全部技能最大等级提升项
第五页第四项(升级时体力回复)
第五页第五项(升级时mana回复)

这个模式需要打完tier 40，敌人配置随机。
注意提升经验倍率，利用UPGRADE项多品种生产的效果作成不同的道具提升经验倍率。我的做法是树叶和蘑菇各存够3个时在进层时交替合成最低级的hp和mana回复药提升5或6次经验倍率，然后时间基本刚好够切出来把第一击用技能变成2连动秒2只怪后再把合成出的低级药都烧了续火把。配合自行用经验倍率提升技能基本能让倍率保持在300%到400%之间，大部分敌人都杀掉的情况下等级会保持比当前层数高7到12级，在前30层基本能无视敌人攻击不用回血。操作得当在tier 28就能升到40级满级。


注意红石基本用来烧掉，基于UPGRADE项的再利用，有可能会获得铁块用来合成加攻防的道具，到30层的时候大概够合成3次+12攻防的道具。都加到攻击上，20层往后不加攻击的话有时会有即使有最高级2连动的+100%伤害加成还秒不掉敌人的情况。

要注意的敌人：鸟人看到有发动技能提示(★号提示)马上打死，会偷道具。树也是看到发动技能提示马上打死，会求雨让火把消耗飞快。

30层往后敌人攻击很高一下500伤害往上，2下打死一个部位。这个阶段建议options第二项中断用U盘等备份存档，靠脸刷红石来秒整层。
