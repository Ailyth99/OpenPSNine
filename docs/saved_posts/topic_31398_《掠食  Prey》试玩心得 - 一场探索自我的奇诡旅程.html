《掠食 | Prey》试玩心得 - 一场探索自我的奇诡旅程

power87 2017-05-16 16:27编辑     14评论


《掠食》是一款由 Bethesda 发行的， Arkane Studio 制作的第一人称射击游戏，该工作室曾经制作过大受好评《耻辱》系列游戏，而本作确实可以感受到《耻辱》与 Bethesda 的风格，如高自由度的玩法、钜细靡遗的环境设计。科幻兼具黑暗的世界观也非常有《网络奇兵》的既视感，再加上异能枪战的战斗设计又与《生化奇兵》似曾相似，让本作有一种夹杂多款大作风格的奇特感受。Arkane Studio 这次果然没有让玩家失望，《掠食》不管在世界观设计、游戏体验、剧本深度上，都下了不少功夫，而让玩家担心的优化问题也没有发生，游戏体验非常顺畅，让本作获致各界的正面好评。

其实这款作品是重开机作，初代《掠食》早在 2006 年由人头工作室 （ Human Head Studios ） 制作，采用与《毁灭战士3》同样的 Id Tech 引擎制作，游戏体验与《毁灭战士》非常类似，是一款淋漓畅战的老派射击游戏。二代作品本应延续初代爽快战斗风格，然而一连串因素导致 Bethesda 硬生生腰斩二代的计划，将制作权从人头工作室转交给 Arkane Studio ，并将作品整个重开机，游戏风格从快步调的激烈战斗演变成为保守的策略交战，引起老粉丝相当大的不满，甚至扬言抵制引发不小风波。不过撇开这些历史包袱来说，这款重开机的《掠食》表现确实相当出色。




掠食 2006


胎死腹中的 Prey 2


Arkane Studio 呈现不同以往的《掠食》，实际上也大获成功


这次的《掠食》背景同样设定在外太空，玩家扮演摩根‧余（Morgan Yu），为转星企业的重要职员，在太空站塔洛斯一号 （Talos I） 上任职，身为太空科技领航者的转星企业，发展出许多惊人的科技成就，例如能改变大脑结构的「神经调整仪」，能保存人类的脑神经结构，并複制到他人身上，可以让人一夜之间变成某领域的天才。而这些成就都来自研究外星生物「风暴异魔（Typhon）」而来，这些如梦魇一般的漆黑生物潜藏许多未知祕密。毫不意外的，意外终于发生了，风暴异魔的脱逃让整个塔洛斯一号瞬间变成人间炼狱，这些毫无人性的生物疯狂的屠戮人类，肆虐整个太空站。高科技、具有质感的先进设施，在异形肆虐下披上一层阴影，设备损坏、物件四散、遍地尸体。

本作采用非线性的设计，玩家将在塔洛斯一号内四处探索，塔洛斯一号是个佔地辽阔的太空站，内部分为数个大型区域，每个区域都各有特色且区域间四通八达。随着主线任务的进行开启更多的区域，各区域也设计了不少有水准的支线任务。游戏体验饱含探索、交战两大要素，玩家将在各区域收集物资、解决地形障碍、并在必要时作战。摩根必须善用各种资源与这些异形生物周旋，试图力挽狂澜。玩家除了决定如何分配技能点以外，也将引领摩根做出各种关键抉择，决定他人的命运，而这些选择将会进一步形塑主人公的性格，影响最终结局发展。


塔洛斯一号，内部相当辽阔，有许多不同部门







场景非常富有变化，设计的相当用心


太空站内甚至有植物园


本作有轻度惊悚成分


尽情探索，谨慎交战

Arkane Studio 的《掠食》，大幅改变初代的玩法，相较于弹药多到捡不完的初代作，本作的资源较为稀缺，号称《网络奇兵》精神续作的玩法，玩家必须多多搜刮场景上的垃圾，分解成材料来制作弹药、医疗包、护甲。现成的资源大多藏在密码柜、隐藏地带、或是地形障碍后。本作给予玩家强烈的诱因去四处探索搜括，除了子弹补品外，攸关玩家技能点数的「神经调整仪」大多也是靠制作获得，想要有充分续战力、或提升人物性能，拾荒是不可或缺的要素，让笔者想起在《辐射4》中捡拉机的生活，只不过场景从废土转移到高科技的外太空站。

本作的敌人也颇为难缠，初级怪物「拟态」不仅喜爱伪装偷袭玩家，灵活的动作让许多第一人称苦手的玩家头痛，高等怪物更会使用各种异能来对付玩家。因此，必须慎选交战方式和场合，因为交战意味着消耗辛苦蒐集的资源，思考最有效率的作战方式乃生存之道，胡乱交战一通，浪费子弹或医疗包就得不偿失，有时甚至需采取迴避策略来节省弹药。而当人物性能成长，资源运用愈有效率时，独当一面的感受也相当不赖。策略交战的设计大幅放慢游戏节奏，让玩家能够细细品尝游戏世界，配合幽暗诡谲的氛围着实非常到味。

多样化的游玩方式，玩出自己的风格

就像 Arkane Studio 旗下的《耻辱》一样，《掠食》也是采用高自由度玩法的设计，透过蒐集或制作神经调整仪，玩家可以取得各式各样的技能，技能共分两大类，人类技能可以强化枪械伤害、加大背包空间、强化补品效果、搜刮材料量上升等等，大抵是基本能力的提升。而风暴异魔技能则是本作卖点之一，可以习得各种有如超能力般的魔幻效果，如拟态、动态爆炸、静电爆破、上升力场、火焰陷阱等等。透过风暴技能系统的引入，让本作添增了异能枪战的色彩，运用控场技制伏敌人，再用手上枪械予以击杀，很像是《生化奇兵》的套路。

各种技能与特定玩法搭配，发挥出更有效率的作战方式，玩家可以强化扳手攻击威力、提升耐力、搭配扳手硬直力强化晶片组，开发出近战流玩法。或者点出搬举重物技能，搭配搬举加速晶片组，来个节省资源的掷物流作战法。骇客机枪塔使其为你作战，并运用修复技能修理、强化机枪塔。上述玩法也可以搭配潜行技能组服用，偷袭伤害翻倍大幅提升作战效率。喜爱正面驳火的玩家，可以善用风暴异魔技能，开场心灵冲击瘫痪高阶敌人的特殊能力，或使用上升力场制服地面单位、静电爆破晕眩敌人，控场后任你宰割。或使采用拟态技能伪装，搭配潜行技能神不知鬼不觉的绕过敌人。

总结

尽管大幅改变了初代的风格，让部分老玩家不满意，但对多数玩家而言，《掠食》毫无悬念地是一款非常杰出的作品。《耻辱》的高自由度配点法、《辐射》的拾荒要素、《网络奇兵》的环境风格、《生化奇兵》的异能枪战，很难看到一部作品能夹杂各种大作的风采，但又具有其独特性。Arkane Studio 这次的新尝试，算是非常的成功。流畅的游戏体验、高自由度的玩法、细腻的场景营造、深度的剧情脚本，除了一小部分的技术瑕疵外，几乎没有什么明显的大缺点。喜爱上述提及大作的玩家，不妨来尝试这款重开机的《掠食》。
