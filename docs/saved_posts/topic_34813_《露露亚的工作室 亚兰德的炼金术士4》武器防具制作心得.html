《露露亚的工作室 亚兰德的炼金术士4》武器防具制作心得

hero32167 2019-06-08 19:00编辑     7评论
话说研究了好几天时间，并做了大量的无用功后，个人终于总结出本作在武器防具制作方面的一些心得了……

其中本文用到的制作公式虽然参考自日站（https://gamewalk.jp/game/lulua/top/），但个人在此基础上对某些地方进行妥协，并成功使制作难度大幅降低。

因此本文适合已经通关并学会制作999品质，并准备挑战机械领域的玩家进行阅读。然后若有出现错误的话，也欢迎大家能够指出来。那么下面开始制作心得：

首先本作在制作武器时，需要用到7个物品：3块砖、1个固定素材或调合物、2个可选择调合物、以及1个加成物。

其中3块砖是决定武器数值的关键。当使用3块带有“制作装备 攻防+10”和“制作装备 全能力+5”效果的砖来制作武器的话，那其最后就会比使用没带这两个效果的砖的面板数值要高45攻和45防。

然后1个固定素材或调合物。当使用到的是固定素材时，其就成为拉低品质的罪恢祸首。

再来是2个可选择调合物。这2个调合物是决定特性、觉醒效果的关键。其中一个调合物通常会选择武器，这样就可以在不考虑PP的情况下把特性带进来。而另一个调合物则一般用于对4大元素的调整。

这样当上面所有物品都选择好后，最后只需要使用加成物对4大元素作最后调整，那目标武器就完成了。

整个武器制作过程的难点有几个：1、砖的选择；2、觉醒效果的最终获得；3、4大元素合理化；4、最终品质做到999。

首先是砖的选择，在没有机械领域特性的前提下，个人使用到的砖如下：

*完美钢块
制作装备 攻防+10
制作装备 全能力+5
觉醒 品质上升+50
觉醒 成分·雷土+3
究极的逸品
超级品质



其中该砖的数值加成已经是最优状态，然后元素加成也适用于所有角色的武器制作。而要制作这块砖，需要用到的材料如下：

*完美钢块
神灵钢铁岩*2
永世之火
贤者之石（觉醒成分火+5，觉醒成分土+5，觉醒矿石）
誓剑沃尔堤凯特（觉醒龙素材）
贤者之派
冰雷元素

其中调合的基础是拥有“究极的逸品”和“超级品质”特性的“特级中和剂·白”。而觉醒类别所用到的公式如下：

湛蓝色的翅膀 -> 亚兰德石晶（觉醒万灵药）
亚兰德石晶 -> 贤者之石（觉醒矿石）

龙核 -> 誓剑沃尔堤凯特（觉醒龙素材）

当准备好砖并登陆量贩后，接下来就要考虑觉醒效果的获得。以露露亚的“光杖艾伯特因”为例，其武器的目标觉醒效果是“道具强化Lv5”，那材料就需要选择“灵刀·翠磷太刀”。但由于“灵刀·翠磷太刀”是属于剑，并不能直接用于调合，因此其还需要通过多次觉醒转换才能使用。其中过程公式如下：

湛蓝色的翅膀 -> 特制女神像（觉醒万灵药）
特制女神像 -> 万能中和剂·虹（觉醒宝石）
万能中和剂·虹 -> 特制女神像（觉醒中和剂）
特制女神像 -> 史塔隆合板（觉醒宝石）
史塔隆合板 -> 灵刀·翠磷太刀（觉醒木材）

最后是最终成品要做到999品质。在日站攻略的追求极限数值公式中，最终成品是不添加任何品质加成特性的。但由于这样做制作难度会超级高，因此个人在牺牲一个特性用于品质加成后，制作难度也变得更为简单。

这里还是以露露亚的“光杖艾伯特因”为例，其最终成品结果如下：

照亮未来之光·大
道具强化+15%
觉醒（金属）
道具强化Lv5
共鸣之全·万象
共鸣之刃·真打
究极的逸品



其中所用到的材料明细如下：（3块“完美钢块”都用的上面配置）

*光杖艾伯特因
完美钢块
创生树枝
完美钢块
完美钢块
灵刀·翠磷太刀（觉醒木材，共鸣之全·万象，共鸣之刃·真打）
创生树枝
火冰元素

虽然这样最终成品的数值只有日站极限成品数值的70%不到，但其已经可以用于挑战EASY难度机械领域的门卫了。

接下来是防具制作。以“女武神链甲”为例，其材料组成和武器很相似，也是3块布、1个固定材料、2个可选择调合物、以及1个加成物。

其中个人用到的布如下。其也适用于“女武神链甲”和“灵银甲誓约十字”的制作：

*天之亚麻布
制作装备 防速+10
制作装备 全能力+5
觉醒 成分·雷+3
觉醒 成分·火冰+3
究极的逸品
超级品质

*天之亚麻布
湛蓝色的翅膀

灵刀·翠磷太刀（觉醒神秘之力）
四精灵之瓶
龙眼

觉醒类别所用到的公式如下：

彩虹噗尼珠 -> 愚者之石（觉醒宝石）
愚者之石 -> 灵刀·翠磷太刀（觉醒神秘之力）

然后“女武神链甲”制作过程中所使用到的材料如下：

*女武神链甲（其中梦锚伊迪盖涅通过完美钢块来觉醒金属）

世界灵魂

天之亚麻布

最终成品结果：

*女武神链甲

使封印无效化
觉醒（布）
技能上升Lv5

究极的逸品



注意影响武器和防具最终数值的，还在于材料中有多少“制作装备”的效果。以武器为例，3块砖肯定是会有数值影响，而固定素材则一般没有影响。因此若要追求极致的话，那就需要在固定调合物和另外2个可选择调合物中做文章。

但由于2个可选择调合物是会直接影响特性、觉醒效果和4大元素，而本作的跨类别觉醒公式也非常复杂。因此这方面研究起来会非常的麻烦，个人也是直接放弃的节奏……

所以最后总结一点：如果裸裸娜脱脱莉梅露露的调合难度是10的话，那本作露露丫的难度就是100，不花一定时间研究的话，是根本玩不过来的……

PS：经个人测试，以上述武器防具的配置，再加上象征饰品和19次终极卡夫和复活药，就可以直接打通NORMAL难度的机械领域了。当然再上去的难度的话，就需要用机械领域里的新砖和布来重新做武器。这个就有待大家自己去尝试了。
