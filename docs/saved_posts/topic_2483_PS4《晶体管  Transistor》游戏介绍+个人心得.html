PS4《晶体管 | Transistor》游戏介绍+个人心得

sai8808 2014-05-26 14:12编辑     12评论
Author.emerson119

故事属于科幻类型，在城市Cloudbank很出名的歌手Red，受到Camerata跟The Process的袭击，后来得到了神秘的Transistor展开了跟Camerata与The Process战斗的旅程。

接下来简单介绍一下游戏各特色

技能
技能看起来很简单，而且一次只能装备４种主动技能，
但其实各个技能的组合可以算是千变万化。
各种技能都有３种用途
１：主动技能
２：强化其他主动技能
３：强化Ｒｅｄ的被动技能

就以游戏最早期第一招Crash()为例


用来当主动技能的话，可以击晕敌人，造成伤害
用来强化其他技能的话，其他技能会得到击晕效果
用来当被动技能的话，会有减伤跟抗缓速效果

以下是Crash()被其他技能强化的效果，一个主动技能还可以最多被两个技能强化。


每个玩家玩法都会变得很不一样，大家可以按照个人游玩风格来搭配各种技能。
注意每个技能都有ＭＥＭ限制，技能介面最左边有写ＭＥＭ限制。
升到一定级数能选择提升ＭＥＭ上限( 最大32 )、被动技能装备上限( 最大4 )、主动技能强化上限 ( 最大装备量各2 )


战斗
战斗有分即时制跟回合制，回合制要不要用都是看玩家的个人玩法，
回合制功能在游戏中叫做「Turn () 」，需要自行发动，
即时制时可以自行使用各种主动技能，没有限制，
发动Turn()时时间会停止，并进入「计划」模式。


计划时所有技能都会消耗Turn() Cost ( 最上面的蓝条 )，就连移动都会消耗



计划好就准备发动吧！发动时Red会变成无敌状态，而且技能都会用很高速的速度去完成。

注意Turn()发动后要等Cost回复完毕才能再次发动，而且回复完前连主动技能都不能使用。
唯一例外的就是有用Jaunt()强化的技能 ( 可在回复Cost时使用 )


技能Overload
在战斗中不幸HP归零的话，会令一个技能Overload ( 超载 )
超载后HP会回复全满，所有技能都超载后才会Game Over

超载后的技能要经过1个Access Point后才能再次使用

Access Point是换技能的地点，同时也会自动存档。

Cell
打败敌人后都会掉Cell，如果没有在限定时间内吸收Cell，敌人就会重生！
每个Cell都会写重生时间还剩多少。



Limiter系统
觉得游戏难度太低? 可以选择装备各种Limiter来提升难度


Limiter有各种功能，例如敌人攻击力双倍、掉Cell时会为Cell放盾要击破才能吸收、在Access Point解除技能时会超载技能等等。
装备Limiter也有一个优势，就是有经验值加成。


10个都装备会有32％加成。


故事、世界观
这游戏的故事、世界观与细节都需要玩家自行发掘
例如查看OVC终端机可以看到各种新闻，也可以看到Red的想法




也别忘了四周探索一下可以调查的地方



其实Transistor的「技能」本身是吸收了「人类」的思绪...?



查看技能可以看到各种人物的背景






哼歌?


只要紧按一个按键，Red就会开始哼歌
个人测试过只要BGM没有歌词，Red都会哼出来。
只要不是在战斗中，都可以随时使用此功能。
另外Red发动Turn ( ) 时也会随着BGM去哼歌，
这功能虽然完全不影响游戏本身，但个人认为这种细节实在太讚了。


沙滩?


沙滩要通过「后门」才能进入，游戏早期就会自动拿到发现后门的能力。
沙滩算是让玩家休息的地方，Hammock那边也可以听听Transistor的想法。



逛一下你会发现有几个门，这些都是小测试 / 小挑战，
例如Planning Test会要求使用特定技能在一回合内解决所有敌人、
Speed Test要求限定时间内使用特定技能解决所有敌人等等。
完成Test都会解开新歌曲，可以到Music Player那边播放。


音乐


不得不说，这游戏最迷人的地方就是音乐，再配上游戏的画风跟场景，真的令人觉得是一款很优美的游戏。

官方Supergiant也很良心的将整个Soundtrack上传到Youtube上
有兴趣的可以去听听
官方播放清单

我个人最喜欢的歌曲就是The Spine

最可惜的就是没有包含Red哼歌的版本，只能在游戏中自己哼来听了。


缺点
要说缺点的话，就是游戏有点短，流程也比较直线向，能四周探索的地方其实不多。

整体来说的话，我是觉得比之前的Child of Light还要讚，
可能因为我个人也比较喜欢这种科幻风格。
故事上也有很多需要动动脑筋思考的地方，因为很多事都没直接解释清楚，
有点像Dark Souls那样需要自己发掘。
