S+达成条件：
1、至少杀150个敌人
2、死亡次数最多只能有1次
3、道具收集至少20个
4、通关时间35分钟以内

推荐使用角色：Vector
主动技能：Mimicry　变身为敌人的样子，持续90秒（LV3）

心得：本关美军非常多，如果遇到弹药和药草补给不够的情况，可以利用Vector的变身技变成敌人的样子收集药草再打，而且很多地方可以变成敌人的样子直接跳到下一个场景，节省时间。我们的主要杀敌数是靠丧尸来累积的，美军皮糙肉厚会反击。尽量少打或者不打。通关时间方面，由于不太好把握，最好开始游戏的时候做个时间记录。
S+评价只需要达到3个S，1个A就可以了，当然4个S最好～




剧情过后，回头拿到一个Collect Data

下一个房间只有一个解毒喷雾，一路跑到头进门，在尸体旁的手提箱中拿到Collect Data

上楼梯进门，在蓝门调查会被丧尸抓进去，干掉门口两只后，点射红色的汽油桶，干掉下面扎堆的几只
旁边的架子上拿到Collect Data


下一个房间中有很多丧尸，不过还算分散，逐个清理吧，没什么难度


开门之前先从左手边的橱柜上拿到一个Collect Data


门右侧的橱柜上也有一个Collect Data


开门触发剧情，遇到了UBCS的尼古拉，玩过生3的玩家应该不会陌生吧


剧情之后进门，可以补充一下弹药，地上可以拿到两个Collect Data


上楼梯进门，清理丧尸。场景中间的台子旁边可以补充弹药


清理完毕之后先别急着进门，门左侧的地上拿到一个Collect Data


标记门右侧的过道中有一个房间可以进去，里边的书柜上可以拿到一个Collect Data，另一侧有一个治疗喷雾


从刚才的房间出来往左看，着火的旁边地上也有一个Collect Data（很不明显）


从标记门出去马上找个掩体，接下来面对的是一场美军车轮战，首先正面涌出4个
打完之后右手边出来4个
最后还是正面出来4个
这里笔者做过很多尝试，P难度下推荐缩到刚才进来的门里边打，远距离逐个点射（旁边有弹药补给）


全部清理完毕后收集一下绿草吧，这个场景中共有3个绿草


这个场景中有2个Collect Data，一个在右侧美军出来的地方杂物堆旁，很容易发现


从左手边楼梯上去，倒下的柜子旁边会发现另一个Collect Data


接下来进入下一个场景，4个美军，这里推荐进门后就一直往右跑，让自己的队友吸引火力，你从侧面干掉他们


干掉美军后跟着标记开门，屋内也有两个美军，先不要急着进去，从门外点射死一个，然后进去干掉另一个


这个房间的Collect Data一共有5个，首先从尽头的门旁边的柜子上拿到一个


转身电脑桌上拿到一个Collect Data


进门的电脑桌和屋子中间的柜子上拿到两个Collect Data


仪器上拿到最后一个Collect Data，屋子的角落有一台电脑，可以提交一下数据


接下来根据标记调查仪器，触发剧情


剧情过后从门里和屋子里涌出n只丧尸，全部干掉


出门右转可以拿到一个绿色药草，旁边的茶几上拿到一个Collect Data


跟着标记走，又是一波丧尸，清理完毕尽头门旁边可以补充一下弹药，开门后是两个背对你的美军，这里推荐打腰部出血，然后聚集一下丧尸，再扔颗雷清理干净，往前跑到下的柜子后面可以拿到一个Collect Data


尽头有大片血迹旁边的门是可以进去的

然后从标记的门出去，干掉两只丧尸往左看，又是背对你的美军，这里还是推荐打出血，然后让丧尸上去吸引火力


干掉几个美军之后对面还有一波，远距离点射吧，尽头角落有2个药草


而刚出来的门右手边尽头有一个治疗喷雾，清理完这个场景所有的杂兵后，一路跑到头还有一个治疗喷雾


跟着标记进门，接下来的两个场景什么都没有，一直走就是


来到一个满是尸体的房间，这里的任务是打掉所有的7台服务器，打完之后地上的尸体会变成丧尸攻击你，注意


接下来跟着标记一路跑出来，进入之前未打开的栅栏门，要面对的又是成群的美军，这里的方法还是先打美军出血，然后让丧尸和美军去战斗，从中寻找机会


清理完一波美军后可以从右手边尽头的橱柜上拿到一个Collect Data


左手边的橱柜上也有一个Collect Data


接下来尽头挡门的美军不推荐打，因为这个场景的补给太有限了，很容易挂掉
如果你用的是Vector直接三角变身敌人的样子，一路跑过去开门触发剧情
如果不是Vector，推荐你直接仍一颗闪光弹然后跑过去开门
进门之后的任务是销毁5个证据，进门和出门的桌子上分别有一个，其他3个位置也很好找


房间的Collect Data收集一下，左手边的电脑上有一个


绕一下，还是左侧，橱柜上有一个Collect Data


要出去的门右侧的橱柜上有一个Collect Data


完事后进入下一个场景，房间右手边的电脑旁边拿到一个Collect Data


靠近阶梯旁边的柜子上还有一个Collect Data


下楼之后是茫茫多的补给，搜刮一番吧


出门后触发剧情，舔舐者的盛宴...


可以移动之后先往前走，小阶梯上拿到Collect Data


接下来要做的事就是无视所有舔舐者，从阶梯下来，一路跑到中间的尸体旁取得Black Card（切记，要先打掉红外线！！）
