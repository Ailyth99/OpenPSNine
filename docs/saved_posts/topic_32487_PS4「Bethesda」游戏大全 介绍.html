介绍

介绍
第五个，大名鼎鼎的美式RPG大厂Bethesda-B社。就是俗称MOD大厂。BUG多，优化差，MOD才是本体是很多玩家对他的印象。喜欢美式RPG玩家对B社肯定耳熟能详。上世代的老滚5可谓是一时风头无限，也推出了中文版。虽然翻译质量不高。
旗下的首席大作上古卷轴V可谓是一代神作，叫好又叫座。引领了开放世界RPG的大潮。其获得奖项无数，奠定了B社在RPG历史上的地位，也让开放世界和美式RPG开始在全球流行起来。其中的“直到我的膝盖冲了一箭”这句游戏台词已经变变成互联网中很流行的一个梗，乃至之后很多大作和ACG领域都有这个梗或彩蛋这款游戏也带动无数民间MOD，不断的为其续命。其中除了功能、系统和材质MOD之后，最主要就是咳咳，美少女MOD了老司机带带我。
辐射系列更是最经典美式RPG IP之一，B社接手了辐射之后，接连推出了几款上世代的经典之作。本时代的辐射4也到千万级别的惊人销量，当然于此同时B社祖传的引擎和BUG也是饱受玩家诟病。尤其是本世代的PS玩家辐射4的优化和补丁推出时间我就不说了。bug和优化差也是B社作品主要问题。还有辐射4赶工和砍内容的问题
B社的游戏数量不多，但是游戏质量和评价很高。属于款款经典，但是本社的游戏优化和画面的都不算顶级，画质和技术力算第二梯队。一些二线作品也有固定的受众，整体感觉就是老派硬汉作风，对经典游戏的重塑和创新，有一种老派复古的老炮的感觉。历史、人性、黑科技等等元素出现在旗下各类作品之中，风格鲜明，极富辨识性。
中文化数据13款游戏，9款中文 中文化不错，文本量居多的RPG也有中文好评就是翻译质量有提高。
统计13个游戏

按照版权发行来统计，代理发行的不统计 B社是3A厂商，所以只统计PS4平台的。由于全靠手动收集，还望大家帮忙补充
B社本社的RPG在主机平台优化不好，PC体验更好,Mod也方便安装

厂商历史

厂商历史
Bethesda Softworks,LLC,是一家美国游戏发行商，并且为Zenimax Media 的子公司。公司于始创于美国马里兰州的Bethesda（Bethesda，Maryland），并且于1990年迁址至马里兰州Rockville（Rockville,Maryland）。旗下有诸多不同题材和类型的游戏发系列，包括角色扮演（RPG），竞速，模拟类和体育类型等。Bethesda Softworks （以下简称Bethesda）是一家有着逾20年交互式娱乐（包括游戏在内）开发，发行历史的公司。在2001年Bethesda 成为了一家专注于游戏发行的公司，并将原有的开发部门组建成了Bethesda Game Studio 用以专注于游戏开发。
公司由Christopher Weaver 于1986年始创于Maryland Bethesda，然后在1990年搬迁至Rockville。公司在此阶段开发了很多PC和命令行游戏。在1999年，公司创始人Weaver和Robert Altman成立了ZeniMax，Inc。
Bethesda是游戏物理引擎的开拓者和先驱者，其于1986年在Commodore Amiga 和 Commodore 64/128 平台开发出了具有物理引擎的模拟游戏Gridiron
总公司旗下工作室 B社全部负责发行
Bethesda Game工作室 id Software工作室 Arkane游戏工作室 Tango游戏工作组 Machine游戏工作组 ZeniMax Online游戏工作室

数据分析

数据分析
B社游戏数量整体不多，绝大多数都在主机上发行，是一个非常典型的单机厂商，也是美式RPG的老大哥。本世代本社仅推出一个赶工痕迹严重的辐射4依然大卖1000多W，滚6在世代还没推出新作，预计会创造新的纪录。
Bethesda本社的辐射4由于赶工导致成品质量有些问题，虽然媒体评价高分本作大卖。但是玩家评价不是很好。
Id Software工作室制作的画面和优化都相当不错。 DOOM的评价和销量都是不错。玩法硬核复古而有挑战性。
Arkane工作室的羞辱也算近几年的黑马作品，评价和玩家反馈都是不错的。最近推出了独立DLC也是好评。
恐怖之父三上真司的Tango的恶灵附身评分和销量不错。二代销量应该不错。
老滚6这张王牌一定要足够优秀啊

希望能早日公布上古卷轴6的消息，在本世代推出正统续作，体验B社创造的这款伟大的RPG巨作。
