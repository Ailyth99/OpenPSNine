白金流程

void TheEnd()
通关游戏
1.准备工作需要少女的满腹度最高、容器清洁（没有便便标志）、没有污染度（下方横条目测不超过1/4），最后这个如果之前没有刻意准备攒食物或者控制污染度还挺烦，因为需要吃没有污染的食物才能降下来；
2.完成准备工作之前如果有未完成的奖杯记得存档，通关之后读档回来补；
3.最后结局2种分歧，都可以跳杯。

培育心得

培育心得
对于一个每次进迷宫都从Lv1开始的Rouge-like，所谓培育，个人认为主要体现在技能二选一和机器人定制上。

每次升级的时候，会随机从技能池里挑选2个技能，然后可以选择其中之一学习。技能又分为被动和主动。
先说主动技能，因为主动技能涉及到计数类奖杯，因此还是多尝试用一些自己喜欢的技能（虽然我也是平A战士）
void SkillUser(500)
发动主动技能500次
主动技能里还是有一些比较好用的，正常在流程中多用应该也不需要刻意刷，个人推荐如下：

技能名威力消耗EN冷却时间推荐原因
ストライク20010５ターン初期自带，想不用都难……
ショット302なし无冷却，最多10格的射程，用来追击老鼠和硬壳甲虫非常合适，也可以用于大型怪近身前的耗血
ブルズアイ2000１フロアストライク的升级版，每个楼层可以用一次，可以用于对付大型怪，必定出暴击的特性也可以用于对付硬壳甲虫
ボルト3002050ターン简单粗暴效果好，房间内最多攻击3个敌人，威力基本保证毙命，还附带麻痹状态，适合在怪物房使用
スイング505４ターン攻击周围一圈同时带击退，解围神技

另外如レイジ、ホロデコイ、ハッピースロット等技能也都有玩家推荐，但是个人觉得泛用性不如上述技能好，就不逐一讲解了。

PS：技能是可以享受Critical Chance+和Critical Damage+的，因此无论是技能战士还是平A战士都推荐适当堆一些。

然后说被动技能。被动技能在早期一般等级在15级以内的时候，以见效快的技能为佳，如ATK强化、回避率强化，同时要考虑迷宫内续航问题，自动回复量强化和EN节约都是很好的，如果有吸收就是最理想的了，能兼顾战斗和续航。到了后期，就要考虑一些长期才能见效但是效果是按乘法计算的技能了。

机器人定制也分为两部分，一部分是角色，一部分是部件。
部件比较简单，除了可以提供初期自带主动技能之外，还有一些特殊效果的部件，如Ｗゲットチャンス可以25%概率同时取得升级时的2个技能，スキル抽選枠アップ可以让升级时从二选一变成三选一，等等。スキル抽選枠アップ和Ｗゲットチャンス设计蓝图要刷，但是正常推图的过程中出1个应该还是不难的，有和没有是质变。スキル抽選枠アップ等于提高抽取技能的质量，Ｗゲットチャンス等于变向白送技能点，或者理解为白送等级也是一样的效果。

角色主要是改变的是抽取技能时各技能出现的概率。具体的抽取率影响RRPG写的很详细了。
初期建议用ファイター（Fighter），出ATK强化会比较多，ダブルクロー（概率攻击2次）、死神のかま（平A改为攻击正面横向3格）、回転斬り（平A改为攻击周身一圈）、必中也都是亮眼的攻击被动。炎の拳（攻击带有爆炸效果）如果有也可以选，但是因为基本对主要目标以外的范围伤害打不死人，只能打个半残，所以不是非必须的。
后期可以选择不止一个角色，建议一定要带ショルダー（Shoulder），首先作为一个辅助角色，对所有技能都没有-概率的效果，简直百搭。另外有几个技能后期真心神技：
キャラメイク：根据持有道具种类获得对应加成，武器提升伤害，防具提升减伤，MOD提升暴击率（一个MOD提升50%），电池提升回复量，修复组件提升暴击伤害
アイテムを力に：每个道具提升1%伤害，相同的技能可以叠加，比如30个道具、抽取到2次アイテムを力に，可以+60%伤害
アイテムを盾に：每个道具提升1%减伤，但是Fighter会对此技能-2概率，算上Shoulder的+3，出现概率也不高
デバステーター：取得特定种类道具时不会获得道具，改为增长属性数值，ガジェット或者POT改为HP+1，武器或者炸弹改为ATK+1，防具或者烟雾改为DEF+1，在基本的武器和防具都已经入手的情况下，这技能简直是平A战士的福音，还能节约大量的道具栏给续航的修理组件、电池以及素材
上述技能配合Shoulder自带提示概率的リュック拡大（道具栏数量增加），效果真可谓谁用谁知道。

另外，可以选3个角色的时候，因为Fighter和Shoulder的设计蓝图是自带的，所以可以比较简单的入手，强度也足够推图。如果还想用足，建议去专门刷一下Assassin，能大概率出吸收、必中、ダブルクロー这些技能。

关于同一个被动技能的叠加，アイテムを力に这类可以正常叠加的没有问题。但是回避率和CRI率強化好像有些不同，第一次点CRI强化率I的时候是Critical Chance+7%，第二次是+9%。但是看日站介绍这个技能是+2%，不排除最初有5%的概率是内置属性、不显示，或者这个技能确实就是第一级收益最高。回避率同理，第一级是Dodge Chance+17%，第二次是19%。所以选了一次CRI强化率I之后，再重复选，收益就很低，但是可以继续选CRI强化率II，可以正常叠加到27%。

运营心得

运营心得
void CreateFurniture(30)
收集30种装饰
迷宫内所有获得的道具在退出迷宫时（无论是何种方式退出，被击破还是到达终点）都会折算成资源，不同的道具会有不同倾向，比如说修理组件可以获得更多的有机资源，炸弹可以获得更多的无机资源。所以如果缺的是无机资源，去 溶鉱炉 打机械敌人就比去 植物プラント 打生物敌人要好。
另外污染资源是道具被污染了才会得到，推图的时候要刻意保留一些污染道具带回城，后期缺的很厉害。另外去 海洋プラント 这张后期图的时候体感污染道具出现率很低，要刷的话不如去无限废墟或者其他图。

推图的时候有素材包裹记得优先带回家，类似于炼金系列的“发想”，素材是产生制造蓝图的前提条件。

初次制作家具会奖励永久技能，推荐优先完成那些ATK+2和HP+5的家具，前期非常有用，可以说前中期有没有家具送的技能打底推图完全是两个难度。后期完成了其他奖杯之后可以做一些少女不容易饿、家里不容易变脏的家具。

小Tips

小Tips
1.在迷宫内拿到高速修复MOD可以带着身上用于回血，血满了再卸装；
2.グロースPOT污染为橙色的时候效果是等级下降1，但是千万不要怕，最好用的时机是刚升级的时候用，会退回到之前等级但是经验条全满，随便打1个怪就可以重新升级，等于白送1个技能点（等级下降的时候不会收回技能点）；
3.如果能拿到WクローMOD但是没拿过ダブルクロー，可以装备上，然后加点多往Critical Damage+和Critical Chance+上靠，通过增加攻击机会提升暴击率；
4.打毛毛虫出污染，可以适当利用消耗敌人血量；
5.一些两回合一动的缓慢敌人，可以打一回合后撤一回合，放风筝式的打法；
6.反伤铁球很容易把自己打到暴毙，要注意回血；
7.ビットくん这个技能没必要主动追求，但是也可以选，选到了就可以放风筝，遛到残血回头补刀+；
8.进怪物房最好是回撤一步卡在门口逐个击破，如果被吸进怪物房可以用マイティPOT这样的道具来站住，或者用脱出装置逃跑。
