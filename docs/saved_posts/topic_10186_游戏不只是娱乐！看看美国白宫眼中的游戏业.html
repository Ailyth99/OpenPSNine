游戏不只是娱乐！看看美国白宫眼中的游戏业

mechille 2015-03-26 14:44编辑     16评论


Mark DeLoura，他曾经任职于美国任天堂、Sony Computer Entertainment America (SCEA)、THQ、UBISOFT 等游戏大厂，在 Game Developer Magazine 担任过总编辑，而且是《Game Programming Gems》系列的作者之一（*），拥有超过20年的游戏业经历。更酷的是他在2013年进入美国白宫，在科技政策办公室担任数位媒体高级顾问一职，成为相当罕见的，资深游戏业人士成为政府官员的桉例。

在 GDC 2015，DeLoura 以《A View from the White House: Games Beyond Entertainment》为题的演说，向大家分享美国政府对游戏应用的看法。

]

*注：《Game Programming Gems》以及《Game Programming Gems 2》的作者

以下是笔者对此次演说的一些重点心得整理：

游戏具有各种可能性

「大家一定很好奇为什么政府会对游戏有兴趣。」对于这个疑惑，DeLoura 首先提到了三款游戏的名称：

《The Oregon Trail》
这款游戏的首发在1971年，是一款让玩家体验西部拓荒时代的冒险游戏，具有很棒的历史教育价值。

《Dance Dance Revolution》
知名的音乐节奏游戏，将游戏与运动融合在一起的玩法在当时相当创新并受到欢迎，因而掀起了极大的热潮。

《Wii Sports》
伴随着 Wii 一起发售，在全球造成疯狂的休閒体感游戏，并且成功的吸引了很多的休閒玩家。



接着他特别讲到《Foldit》这款游戏，《Foldit》是一款实验性质的蛋白质折迭游戏，由华盛顿大学的电脑科学和工程学系以及生物化学系共同开发，《Foldit》以已知的蛋白质结构为题目，希望能利用人脑特有的三维图形匹配能力，让玩家操作类蛋白质来解谜。并且透过分析玩家在解谜时的思考途径，进一步改进现有蛋白质折迭软体的演算法。



另外一方面，美国政府自二战以来开始重视利用媒体来推广政府政策，从二战时期的宣传影片，到如今甚至会利用游戏来进行政策的推广。例如《Budget Hero》，这是一款模拟联邦政府制定预算的游戏。另一款《Moonbase Alpha》可以在 Steam 上下载到，玩家要扮演一名在月面基地上的太空人，必须为了保护在月球上的研究成果想办法完成各种任务。




这些游戏除了有趣之外，背后都蕴含了更深的价值。这代表游戏除了娱乐性之外还能够拥有更多的意涵。

「现在越来越多的团队投入手机游戏开发，要获取玩家的青睐变得更加困难，如果你做出了一款相当棒的游戏，大家会马上一窝蜂複製你的游戏。」DeLoura 提到：「这现象让我感到好奇的是，还有没有其他要素是能让游戏开发者感到兴趣而关注的呢？」

接着 DeLoura 提出更多的桉例：

《EyeWire》--藉由群众力量解决问题的公民科学

《EyeWire》跟《Foldit》相当类似，《EyeWire》希望能解决的问题是如何从电子显微镜的剖面照片去重建错综複杂的3D神经网路模型。游玩这款游戏并不需要任何专业知识，玩家要做的就只有上色这一件事情，在上千张的显微镜剖面图中，将你认为是同一个细胞的区域用滑鼠点击着色，电脑就会帮忙将你建立的3D神经细胞树图显示出来。视网膜的神经网路系统庞大无比，利用公民科学的力量，能够帮忙解决这种无法依靠电脑、需要庞大人力来处理的问题。



《NeuroRacer》--透过游戏训练改善脑部老化

《NeuroRacer》人类同时执行多项任务的能力以及认知控制的能力会随着年龄增长而逐渐下降，而有研究表明多工处理带来的紧张感能提高老年人的工作记忆以及维持注意力。《NeuroRacer》正是基于这种多工处理训练的概念而设计出的游戏，玩家在操纵赛车的同时，也要一边分辨突然会出现的蓝色、红色以及黄色路标。根据研究显示受测老年人的多任务能力以及认知能力都有所提高，并且能持续数个月。

这个研究的相关论文曾经登上《Nature》的封面，后来实验室主任 Adam Gazzaley 创立 Akili公司开发出另一款游戏《Project:EVO》，并与製药大厂 Pfizer（辉瑞）协议为《Project:EVO》进行临床实验，测试该游戏是否能用来早期诊断阿兹海默症。



《DragonBox》--玩游戏也能产生教育效果

《DragonBox》是一个很有趣的游戏，它的设计非常巧妙，能够让小朋友在没发现自己正在学习数学的状况下，学习到複杂的代数观念。游戏以左右两块面板代替数学式等号的两侧，左右两侧各自分佈有数个图桉，有的图桉会有黑色的版本，将同一侧的黑白两个图桉相迭加便会变成一个黑洞图桉（代表0），点击黑洞图桉就可以把它消除。游戏的目的便是要让某一侧只留下名为 DragonBox的图桉。随着游戏进度的攀升，原本的抽象图桉便会慢慢的被抽换成适当的数学符号，根据实验当游玩《DragonBox》90分钟之后，在4000名孩童当中约有90%的小孩在代数运算方面的能力获得明显提升。




透过游戏进行教学

2011年，美国总统 Obama 曾经号召要开发出跟最棒的电玩游戏一样有趣的教育软体，他也曾经呼吁更多人来一起学习写程式。美国白宫在2014年的美国电脑科学教育周邀请了20名中学生到白宫使用程式自学网站 Hour of Code 学习程式设计，Obama 虽然并没有完成 Hour of Code 的程式教程，最后却还是透过 Google 开发的 Blockly Games 以 Javascript 写了一小段程式码，在萤幕上画出一个方型色块，成为史上第一位写程式的美国总统。




SRI International（史丹佛国际研究中心）做过一个关于数位游戏与学习成效间的相关研究，结果显示在课程中加入数位游戏可以提升认知学习成效12%，并且能改善解决问题的能力及记忆力。



2012年的国际学生能力评量计画（PISA），共有65个国家及经济体参与，15岁学生样本超过51万人，而美国学生的平均分数落在第36名，只有超过整体平均值一些些。对于此现况 DeLoura 表示：「如果我们可以透过游戏来提升学习成效，我们就应该去尝试看看。」



加拿大的一名高中物理教师 Shawn Young 与家人合作开发出了一款教学游戏系统《Classcraft》。由老师担任游戏的GM，5～6名学生组成一个团队，各自扮演像是战士、法师或僧侣的角色，老师依据游戏规则以及学生在课堂上的表现来加减学生角色的经验值或生命值，例如学生表现积极、答对课堂上的问题可以获得经验值，迟到、欠缴作业则会被扣除生命值等等。许多使用《Classcraft》进行教学的教师纷纷表示该游戏让学生变得乐于投入课堂活动，并且改善学习态度。



在《Minecraft》爆红之后，美国有名教师 Joel Levin 尝试希望能将该游戏应用在课堂教学上，获得许多教师支持甚至连开发商 Mojang 也与他接触，最后他与其他人合作创立了 TeacherGaming，让这款教育版的《MinecraftEdu》得以诞生。教师可以利用《MinecraftEdu》创建各式各样的教学内容，并且让学生进行探索，例如在游戏中重建历史建筑、建立生物模型，或是让学生建立一座城镇并尝试运行社会机制。



David Hunter 则是设计出了一款名为《Zombie-Based Learning》的教材，这是一个有关殭尸入侵的剧本，内容超过70堂课程。学生必须在各式各样的活动、作业，以及讨论之中思考如何闪避这些殭尸以及维持资源的供给，并试着进一步重建这个被殭尸入侵的社会。




由白宫所举办的 Game Jam 活动



2014年9月6日～8日，白宫举办了一个名为《White House Education Game Jam》的 Game Jam 活动，希望开发出能运用在老师课堂教学上的教育游戏 prototype。该活动一共有105名游戏开发者加上35名老师、学生，以及教育研究者共襄盛举，其中包括来自 SCEA、EA，以及 UBISOFT 等知名游戏开发商的成员。这些来自各地的人士分成了23个小组并经过48小时的脑力激盪，最后完成了许多可以应用在教学上的 prototype。




《Endemos》是其中一个游戏，是由 Team Wolfshark 完成的作品，玩家可以自由编辑出各种生物，设定像是大小、寿命、生命力、移动速度，以及肉食草食等数值，然后将生物配置在虚拟世界裡面。希望能够让学生藉由观察生物在虚拟世界中的行为，来学习生态系统相关的知识。




另外一个介绍的游戏是《Global Doomination》，在游戏中玩家可以产生一种名为 Gloob 的生物，随着游戏时间经过他们会遇到一次又一次的天灾，大部份的 Gloob 会死于天灾，但其中一小部分会因为身体产生过变异而能从灾难中活下来。玩家的目的就是不断创造出能从这些海啸、火山喷发，以及细菌感染袭击下存活的生物，并且观察他们如何藉由进化来度过这些灾难。

持续探索游戏的未来

在演说的最后 DeLoura 表示，自己并不是要号召大家投入教育游戏的创作，而是鼓励大家去思考新类型的游戏。



「作为娱乐游戏相关的开发者，我们已经走过了一段很长的道路。」DeLoura 说：「游戏变得越来越棒，并且拥有相当多样化的平台，我们正在探索游戏到底是什么，而这些蕴含的可能性总是能让人感到惊奇。」
