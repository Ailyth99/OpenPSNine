PS4《血源诅咒》1.07版更新，增加联盟系统、协力NPC

sai8808 2015-11-21 09:09编辑     44评论
新誓约 联盟



1. 这是个有着相同目标的猎人互助会（误）誓约，跟会长 Valtr （ヴァルトール）缔结誓约，获得联盟誓约符文「不洁」(协力时血量 +2%)



2. 会长的位置在禁忌森林第一个灯点旁的捷径建筑内（风车小屋走出来往蛇森方向那个电梯）。

3. 在协力游玩时配戴誓约符文「不洁」可于完成协助后(打赢 boss)获得「害虫」，使用后可以提高芳名录内的排名。
P.S. 联盟的目标就是碾碎这些看不见的害虫，把人类从罪恶（？）中解放出来。

4. 联盟成员可以呼唤联盟 NPC 猎人参战（地板上看起来像是红色骷髅头的特效）。

召唤来的猎人若是联盟成员会显示「盟友」。

第一次使用（踩碎）「害虫」后，再去跟会长讲话，会给予「联盟手杖（内藏芳名录）」跟获得动作「联盟誓约」，一个拿出手杖比出元首万岁的动作（误

拿过动作后被盟友召唤时，刚显现就会自动做出联盟誓约的动作。


协力NPC追加




1. 满足特定条件后，可在猎人梦境中获得老猎人的召唤钟，之后就可以在特定地点的老猎人专属印记上进行召唤，老猎人出没的地点包含某些「金杯迷宫（？！）」。

2. 依据区域不同，最多可以召唤两位 NPC 猎人，但若你已经召唤了一名玩家（真人）的猎人，你就只能最多召唤一位 NPC 猎人。


金杯共享制度改变




非始源金杯（一般金杯）现在增加「共享的固定地城」额外仪式选项，使用后（无须额外仪式材料）创造出的金杯将是 Server 共通的，可以更容易让人加入（协力、入侵）。

如果你想单机也可以跟过去一样创造自己的独特金杯铭文。


协力「幻影」改变




现在当你摇动召唤钟的时候，你可以看到其他寻求合作的猎人的幻影，若是在召唤范围内，你会看到这些寻求合作的猎人幻影脚下产生涟漪波动（就跟平时自己摇钟时脚下那沱一样）。


其他还做了一些修改、变化跟完善配套。
