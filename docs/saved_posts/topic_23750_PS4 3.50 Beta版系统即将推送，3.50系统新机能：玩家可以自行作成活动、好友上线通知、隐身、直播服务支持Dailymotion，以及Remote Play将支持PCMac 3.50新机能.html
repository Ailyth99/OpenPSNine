3.50新机能


PS4 Beta版系统固件3.50（内部版本号为「MUSASHI（武藏）」）将于近日推送。

注意：即将推送的Beta版不包含Remote Play对PC/Mac的支持。
之前的申请

3.50新机能
3.50新机能：
· 玩家可以自行作成活动





· 在「派对」中可以通过「Section Play」一同游玩




· 好友上线通知



· 显示不在线（即隐身）

· 直播支持Dailymotion



以及...


Remote Play将支持PC/Mac（即将推送的Beta版不包含Remote Play对PC/Mac的支持。）

以下内容感谢 @nathaniel_wu 提供。
· 在线状态可以设定为隐身
Other features in v3.50 include - finally - the ability to be notified when friends come online and the ability to set yourself to appear offline.

· 设定自动邀请特定玩家参加派对聊天
You'll also be able to set a scheduled event - such as a gameplay session with friends, and set for specific people to be automatically added to a party.

3.50 Beta版系统升级前须知


PS4 Beta版系统固件3.50（内部版本号为「MUSASHI（武藏）」）将于近日推送。

注意：即将推送的Beta版不包含Remote Play对PC/Mac的支持。
之前的申请

3.50 Beta版系统升级前须知

在Beta版系统升级前须同意《システムソフトウェアベータテスト参加契約》（即《保密协议》）。

To 不想看冗长的协议的人：


建议在升级Beta版系统之前先备份PS4。
建议在升级Beta版系统之前仔细阅读系统更新日志。


测试期间如遇BUG，可以前往这里（日服链接）报告。

Beta测试的时间
2016年3月2日 ～ 未定*
　* 3.50正式版系统推送后测试结束。


关于兑换码
· 兑换码只能使用一次。
· 兑换码有效期：2016年3月2日 - 4月30日。


兑换码使用方法
（Step1）启动PS4。
（Step2）在PlayStation®Store内、选择「输入兑换码（コード番号の入力）」。
（Step3）输入当选邮件中带有的兑换码。
（Step4）选择「システムソフトウェア ベータテスト 2016 3月」并开始下载。


Beta版系统的安装方法
下载完「システムソフトウェア ベータテスト 2016 3月」后，
进入[設定] > [系统升级（システムソフトウェアアップデート）]，
选择[3.50 Beta]并按画面指示安装即可。


从Beta版降级至目前最新正式版系统的方法
进入[設定] > [系统升级（システムソフトウェアアップデート）]，
选择[バージョン3.15]并按画面指示安装即可。


售后维修问题
请先降级至目前最新正式版系统后再前往售后。


From. PlayStation Japan
