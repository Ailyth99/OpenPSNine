为了迎接即将在5月14日于台湾与香港推出的人气动作冒险游戏《如龙》系列最新作 PS4 / PS3《如龙 0 誓约的场所（龙が如く 0 誓いの场所）》繁体中文版，巴哈编辑部获SEGA 日本亚洲营业部邀请，独家飞往日本 SEGA 总部进行《如龙0 誓约的场所》游戏相关内容的深度采访。

其中包含了《如龙》系列总监督「名越稔洋」的近距离访谈、《如龙 0 誓约的场所》主角桐生 一马配音声优「黑田崇矢」以及真岛吾朗声优「宇垣秀成」对谈、还有实际于游戏内以 NPC 角色登场的性感女优「纱仓真菜（まな）」以及「大槻响（ひびき）」的访谈。
