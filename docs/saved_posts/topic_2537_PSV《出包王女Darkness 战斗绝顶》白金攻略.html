白金攻略

没写攻略的奖杯就是顺其自然就能解到的，假如有不懂可以留言问

1.难度攻略
ノーマルクリア
Normal难度下通关
ハードクリア
Hard难度下通关
ベリーハードクリア
Very Hard难度下通关

第一轮建议先从简单开始破 比较好练等和刷素材
可以先把房子事件全GET 还有别忘了挑战 不然再困难刷会吐血(俺就是困难刷那位

第二轮直接跳过普通 去打困难 因为出包奖盃颇特别的 打完困难连普通都会一起拿
困难真的颇血尿的
30阶打完一隻龙又跳三隻 害我打两次才过




2.挑战奖杯
はじめてのチャレンジ
完成任意挑战任务
芽生えはじめた挑戦心
完成5个挑战任务
燃えてきた挑戦心
完成10个挑战任务
成熟された挑戦心
完成50个挑战任务
最高潮の挑戦心
完成100个挑战任务

每1阶有5个挑战 第5个挑战要跑完此阶才会拿到 还要折回去
还有第5个奖盃通常是到达挑战 从头走到尾 所以记得先清怪

比较特别的是22阶跟30阶的挑战
无法折返 22阶是不用清怪就能直接挑战
30阶是先跑第5挑战 在重置地图再跑其他挑战

30阶攻略 跑错会回29阶 第二个选项是重置地图
其他图攻略自己摸索或是留言问



3.装备奖杯
発明の喜び
收集了10个发明品
発明の快感
收集了20个发明品
発明の衝動
收集了30个发明品
発明の極み
收集了所有发明品

比较麻烦的就 S开发品跟拳鬪士和ミカ
S开发品 去二楼ログイン找MOMO刷的
拳鬪士则是找芽亚OR黑肉刷的
ミカ可以找美柑刷
也可以到前几层刷 只要重置地图就能无限刷 只是我忘了是哪几层有了ˊ_>ˋ
其他大致上多玩几轮就有了

4.事件奖杯 最麻烦的奖杯
見習いラッキースケベ
触发15种Room Event
職人ラッキースケベ
触发30种Room Event
一人前ラッキースケベ
触发45种Room Event
ラッキースケベキング
触发所有Room Event

这东西要刷两轮
一轮是30阶结局 一轮是20阶结局
30阶条件 20阶打黑肉前现有女角好感全用到5心
20阶挑件 20阶打黑肉前现有女角好感未满5心

房间事件的刷法
好感心心
0~2会出现随机事件1
3~6会出现随机事件2
6~10(MAX)会出现随机事件3
降好感方法 去二楼打ログイン TTT时乱摸 摸到强制退出 两次可以减一心
还有建议再打玩20阶时刷 再刷完前不要打图
要打图时要记得把现有女角好感全刷满 因为21阶尾会把好感度满的女角收走
要再22~24阶把她们找出来 有事件跟CG
