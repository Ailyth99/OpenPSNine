关于海沟

关于海沟

起点

最早于主线15章通关后，完成圣塔利亚支线任务【起锚】及【起锚·续】后开始加入重力暴风系统。




进入海沟矿区
一般来说，经过一定自然时间，5个矿区会产生重力暴风，此时即可进入岱尓夫海沟矿区。

海沟一共50层，每个矿区有进入海沟的最高初始层数，主线通关前后不一样（通关前最高30层），详见下表
矿区名主线通关前最高初始层数主线通关后最高初始层数
0号采矿场11
13号采矿场611
魔境地带1121
烛光地带1631
托弗拉古礁石2141
矿区的海沟入口层数为【当前可到达的层数，该矿区最高初始层数】的较小那个值，举2个例子（设主线已通关）
- 当前已通关海沟第5层（假设从第6层返回）则下次进入经过自然时间产生的重力暴风的矿区，0号采矿场入口仍然是1层；而13号采矿场及以上都是6层；
- 当前已通关海沟第21层（假设从第22层返回）则下次进入经过自然时间产生的重力暴风的的矿区，0号采矿场入口仍然是1层；13号采矿场是11层；魔境地带是21层；烛光地带及以上是22层；

也就是说高级采矿区的初始层数高，低级采矿区还得慢慢爬才能到达相应的层数。
当然41以上也只能从41层开始爬起。


进入矿区之良性bug
大家也都试过改系统时间对矿区没什么影响的样子，而前面提到的是经过自然时间产生的重力暴风，这样就造成了海沟出来容易进去难（前期懵懂的时候可能还是死出来的），也不知道官方怎么想的，难道真要一个通关全部游玩要素的玩家天天等重力暴风...(这点确实该批

于是这个不知道算不算bug的可能唯一解成了最后的救赎。主要还是依赖存档，下面举个例子：
1. 现假设某个存档【烛光地带】产生了重力暴风，当前已通关22层（23层出），则前往烛光地带下至矿区（不进入海沟），可以看到当前入口是23层，存一个档作为备用；
2. 假设当前主存档【烛光地带】没有重力暴风，当前已通过24层（25层出），则前往烛光地带但不下矿区，存档覆盖主存档或不存档；
3. 读取步骤1的备用存档；
4. 读取自动存档或者步骤2覆盖后的主存档；
5. 此时可以发现当前存档【烛光地带】产生了重力暴风，不过由于这个重力暴风是备用存档产生的，所以入口仍是23层。

从例子中可以发现几个问题：
1. 仍需依赖自然时间产生的重力暴风，所以一般情况下，高级矿区产生重力暴风就应该去存个档备用；
2. 需产生重力暴风的存档必须与需读取的备用存档应处于相同的矿区；
3. 入口层数是跟备用存档的，所以步骤1提到的备用存档是需要更新的（只能由自然时间产生的重力暴风更新）；例如【托弗拉古礁石】最高初始层数是41，若该矿区备用存档初始是23层，之后想打高层都得从23层爬起；
各大矿区更新成最高初始层数即可。

PS：无论是通过采矿笼或者死回船上，未进行其他操作（比如回城再到矿区），然后利用当前自动存档重复了上面的步骤3、4准备再次进入海沟，那么当你准备进入海沟前，你会发现已经有一定数量的矿石了；实际上这还是上次结算的矿石，建议回到船上结算一下再下去。本以为是多出来的可以成倍刷矿石
至此，海沟也能随进随出了。具体也可参考下面的视频：
https://www.bilibili.com/video/BV1AF411f7mY

此时，只要不通关主线，可重复刷26~30层，每层必掉落5个光彩品质的矿石；可将月球模式攻击力提升至目前最高：

5. 对于第4点的补充；也可以选择死亡回到船上整理一番再挑战；
下次再到这个死亡点时

靠近即可获取上次死亡损失的矿石（如果有）

关于护符
- 矿石品质与护符技能对应，不存在交叉，比如“气愤”技能只有耀眼才会出
- 变形护符是一个固定顺序，只能由 一般→月，月→木，木→一般，每3个相同的类型转换成另一个
- 回收护符有点迷，比较靠谱的是用5个最高级的技能去合礼物碎片（比如气愤、盛怒、牺牲），虽然概率也一般；SL大法？没有这个必要，因为重新LOAD以后又要从老诺华传送到普雷茱，这个过程太漫长了...
- 冷知识一则，持有护符最多999个

- 通关50层后的采矿标准配置

- 另外看到有说建议拉文采矿（出护符概率高），变身凤凰的时候还能炸BOSS半血以上；从开采出护符的角度来说确实是拉文概率比较高，但是因为不能佩戴护符，没有其他重力模式的支撑，显得很单一而且很容易死（45层一定不好打），所以总体来说个人还是喜欢操作凯特（特别是有了最最终阶护符以后）

示例：https://www.bilibili.com/video/BV1YP41147rB
备注：官方叫你来biubiubiu啦！看起来时间挺充裕的，主要需掌握莲花奈必的位置，可能需要多打几次就熟悉了；考虑到与莲花奈必的距离，多丢几次小的投掷物不确定是否打死比较浪费时间，建议全程选择木星模式的重力抓取，大约吸个4、5个就可以一次砸死一只了；最后一块区域可以用“聚气粉碎领域”多吸一些可以一次砸死3只（巨人+2核+镭射眼）；
PS：这层的石碑就在初始地点直线不远处，但是如果慢腾腾地走就准备吃弹幕吧

第45层
要求：击败5只稀有奈必
难度：中
示例：https://www.bilibili.com/video/BV1E94y1e79w
备注：
- 稀有奈必全程霸体不会被打硬直；攻击非常高；隔一段固定时间也会出现下一只；简单的方法还是依靠第二只莲花奈必等其他奈必出现再慢慢砸；
- 小奈必：扫尾+远程子弹，偶尔扫完第一下会接着反方向再扫一次，所以不能贪；如果你在空中则可能直线向空中发射光线...
- 莲花奈必：甩出的石头带毒，被砸中了就就进入毒状态，好像其他没什么了；
- 蚊子奈必：可以是说最麻烦的，除了月球模式的投掷能控住以外，其他任何攻击打一下就消失,过一下再出现，重力踢击的话打到何年何月...所以至少要用莲花消灭这只蚊子；
- 盾哥：攻击太高了，不能硬直不敢硬拼，一种花式的打法就是靠接下来的镭射眼直接射死它；
- 镭射眼：光线射出后也有补正追踪，月球模式也有可能被打到，一碰就秒杀；接近的话会放一圈速度一般的追踪子弹

PS：这一层接近书架会有额外的4句话，不知道算不算剧情补充(看不出说的是谁...)

2. 他们一直出现，我不可能永远撑下去...
3. 我想回家，再见我家人一面
4. 只剩我了，我得完成任务
详见视频 https://www.bilibili.com/video/BV1dj411672X

PPS：如果这一层开始关卡前没有消灭周围的奈必，通过后也不立即进入下一层，一段时间后会再次出现，然后...然后就准备受死吧

BOSS突出一个磨叽，要么打了就跑（46），要么敌人数量多还得一波结束下一波才出现（48），要么不让你打（49吹飞你、50雷电保护）;血厚攻击还高...

第46层
要求：尤尼卡&贝尔玫
难度：中
示例：https://www.bilibili.com/video/BV1eu411H7eU
备注：
- 2个家伙自己攻击完立马全天空到处飞，过一会再进行下次攻击；
- 尤尼卡主要是追踪火弹和自身的蓄力直线冲撞；
- 贝尔玫的蓄力光线攻击后紧接着是追踪飞弹，所以要在飞弹发出击后才能虫洞踢击；
- 重力抓取选择木星模式，其他大部分都是虫洞踢击，毕竟满图飞还想用重力踢击不成，没什么太多技巧

第47层
BOSS：卡丽&熙熙
难度：中
示例：https://www.bilibili.com/video/BV1k94y1e78C
备注：
- 熙熙不吃投掷伤害但是可以用月球模式的抓取物砸上空中，先与熙熙空战；尽量保证足够高否则卡丽的空中抱摔还是很疼的（只要不碰到地面、建筑物就不会有伤害）；
- 熙熙开盾的时候可以用投掷物砸掉然后接虫洞踢击（你不会选择重力踢击的）；如果有一圈子弹围绕那就得等熙熙把子弹扔完才能接近了；
- 卡丽多了几个技能，但依旧是踢屁股，只是血厚要有耐心

第48层
特殊：碧斯马利亚300勇士击败300个敌人
难度：中
示例：https://www.bilibili.com/video/BV1L8411d7jj
备注：全程月球模式闪避+投掷；敌人分为几波出现，抓起来的人不算消灭要丢掉才会更新下一波；说实话每次打得想睡觉，但是可能突然中一发导弹就挂了...中间的喷泉是个干扰，离得太近没有可抓取物时容易抓到水滴，抓到了就赶紧扔出去

第49层
BOSS：突变的卡丽
难度：中
示例：https://www.bilibili.com/video/BV1px4y1X74P
备注：
- 有2管血；部分脸的位置似乎是随机的；
- 木星模式的重力抓取吸个4、5个投掷物就能砸掉一张脸，但是投掷物距离卡丽的位置太远了，来来回回也很慢；
- 卡丽所在楼的顶端有一颗体力宝石，虽然看不出，但是这里大约有够一次重力抓取的物品；
- 就算是重力踢击也得选择木星模式，但是有时候蓄力会被吹飞很烦...
- 黑豹除了回血没什么太大意义，毕竟打一次眼睛就闭起来了

第50层
BOSS：伊蕾崔斯蒂
难度：高
示例：https://www.bilibili.com/video/BV1mh4y1c7sT
首次通过奖励：最最终阶护符（恢复技能点数）
备注：
- 接近飞弹但先不要重力抓取飞弹，看到真身后再抓，然后飞奔；没有雷电环绕的时候先虫洞踢击一轮，然后缠绕雷电后大概会闪一次再丢飞弹；
- 就算不丢飞弹，隔一会雷电保护也会消失，所以也可以选择木星模式重力抓取去砸
- 接近或者捡起飞弹可能会被雷击到爆炸，所以还是得闪避
- 黑豹可以在没有雷电保护的时候打一会儿
- 全程不是在找飞弹就是在找真身...
