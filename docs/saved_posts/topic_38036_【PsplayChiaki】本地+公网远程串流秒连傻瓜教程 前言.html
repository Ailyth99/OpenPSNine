光猫DMZ绑定游戏主机

【光猫】wifi/网线直连【PS4/PS5】 设置教程（半一劳永逸）


串流路线图：【PS4/PS5】→【DMZ地址绑定光猫】→本地串流or远程串流


※以下教程使用电信的【网络管家】为例：

步骤①：确认PS4/PS5的IPv4地址
（备注：作者我本人的光猫默认 192.168.1.1）

如下图所示：PS5连接【光猫】后的IPv4地址：【192.168.1.4】



步骤②：进入【网络管家】-【DMZ主机】绑定【PS4/PS5】在【光猫】上的IPv4地址
（如果能在光猫里把IPv4地址永久固定到设备上更好 电信光猫不给设置 缺点是PS4/PS5的IPv4地址可能会变动 比如192.168.1.5/192.168.1.6等 其他设备连接光猫wifi会占掉机器的ip）




光猫-PS4/PS5的准备工作就完成了！只要PS4/PS5的IPv4地址不变动 算一劳永逸了。
如果变动的话 再到PS4/PS5中查一下IPv4地址后 改光猫的DMZ地址就行。
现在可以去串流APP进行设置了！


路由器DMZ绑定游戏主机

【路由器】wifi/网线直连【PS4/PS5】 设置教程（一劳永逸）


串流路线图：【PS4/PS5】→【IP与MAC地址绑定路由器】→【路由器DMZ地址绑定光猫】→本地串流or远程串流


※以下教程使用电信的【网络管家】＆【路由器】为例：

步骤①：确认PS4/PS5的IPv4地址
（备注：作者我本人路由器默认 192.168.0.1）

如下图所示：PS5连接【路由器】后的IPv4地址：【192.168.0.108】



步骤②：浏览器或路由器APP进入【路由器】管理界面：【192.168.0.1】



2-1：进入路由器的【IP与MAC绑定】（绑定后PS4/PS5在路由器内网的IPv4地址永久固定不变）
把PS4/PS5在路由器上对应的IPv4地址绑定到【路由器】上即可。



2-2：进入【路由器】的【DMZ主机】
把PS4/PS5在路由器对应的IPv4地址绑定到【DMZ主机】上即可。



步骤③：进入【网络管家】-【DMZ主机】绑定【路由器】在【光猫】上的IPv4地址

3-1：确认【路由器】在【光猫】上的IPv4地址（如下图：192.168.1.2）




3-2：【光猫DMZ主机】绑定【路由器】的IPv4地址




路由器-PS4/PS5的准备工作就完成了！一劳永逸！现在可以去串流APP进行设置了！

※非同一网络下自动远程串流（建议手机4G/5G网络或其他WIFI网络）



步骤5-2：回到软件打开时的一级页面 测试【远程连接】→【手动】输入家里宽带公网IPv4地址（5~7秒连接成功）
※非同一网络下手动输入IPv4地址串流（建议手机4G/5G网络或其他WIFI网络）

推荐外出时想查询家中公网IP时 可以在手机/平板+家中电脑上安装远程控制软件 比如：ToDesk

Chiaki串流

Chiaki串流

步骤①：获取用于Chiaki绑定用的PSN帐户ID

点我获取→输入自己的PSNID→获得【Encoded ID (for Chiaki)】框中的ID信息




※Q：PSNID输入正确 但无法查询到绑定Chiaki绑定用的ID如何解决？



※A：登陆自己的PSN帐号（官网网页/PSAPP/PS主机上）修改隐私设定
【隐私设定】→【朋友丨社交关系】→所有的关系选项都改为【任何人】→等大约5分钟
→再前往获取ID网站（点我获取）输入PSNID



※获得绑定Chiaki的ID后别忘记再恢复之前的社交关系状态！


步骤②：将PS4/PS5＆手机or平板连接在同一个网络下（同光猫下or同路由器下）

比如：手机或平板连接路由器WIFI：【ABC】；PS4/PS5网线连接路由器 或 PS4/PS5连接路由器WIFI：【ABC】

步骤③：绑定PS4/PS5与Chiaki

以PS5为例：【设置】→【系统】→【遥控游玩】→【启用遥控游玩】→【连结装置】→【显示8位数字】



打开Chiaki→【点击同网络下自动显示的主机】→【输入图2信息】→【点击最下方注册按钮】→【绑定成功】
※提示：手动输入Chiaki绑定ID 哪怕输错一个字母或大小写 绑定都会失败！





绑定成功后返回一级界面会看到ID右侧多了个（registered）-已注册的标识。

显示蓝色灯条ICON的PS5主机代表可以直接点击【本地串流】



显示橙色灯条ICON的PS5主机代表需要点击【Wakeup】唤醒后再【本地串流】



步骤②：【远程串流】设置
※确保串流设备切换至 手机4G/5G或其他WIFI

【Chiaki一级界面点击右下角“+”号】→【Add Console Manually】→【Host：PS4/PS5公网IPv4地址】
（IP查询网站：点我查询）→【Registered Console处选择步骤①中绑定过的主机】→【SAVE】


回到一级界面 点击设置好的主机 就可以远程串流了。



步骤③：Chiaki串流参数设置（一级界面右上角“3个点”进入设置）
由于Chiaki设置项目很少 不做表格了 参照下图设置即可
