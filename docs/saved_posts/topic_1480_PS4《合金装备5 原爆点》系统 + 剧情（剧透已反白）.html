PS4《合金装备5 原爆点》系统 + 剧情（剧透已反白）

目前主要是玩PS4版的，对于还玩不到的玩家底下简略介绍一下这代玩起来的感觉，希望能起到一点借大家望梅止渴的作用。剧情也用比较小说的形式让想知道的人能了解个大概，当然有提及剧情的部分肯定会反白的大家请安心（笑）

另外好像发心得都要说一下自己的MGS经历？
总之大概就是系列做全都有玩（包括一些AC!D或是TTS....etc）

曾经big boss只有MGS4跟MGS1。总之是个不值一提的骨灰级玩家一枚啦


SCREEN









CONTROL

简单说一下操作。这代延续四代跟PW不断精炼简化的精神，我们的big boss操在手上又可以用更简单的方式进行更複杂的动作。不过操作的部分这代其实打掉重练的程度没有过去那麽高，可以说是四代的改良版，操作又更加接近市面的射击游戏，所以应该不难熟悉才对。

其中最令人期待的想必就是Demo裡有看到的这代新增的追加动作「全力冲刺」以及「紧急回避」了！

整体的按键配置大体如下
「左类比」移动 / 下压冲刺
「右累比」镜头控制
「口」紧急回避
「△」动作键，开锁 / 翻越围栏 / 攀爬 / 悬吊
「Ｏ」轻点换弹夹 / 长压与场地上的武器作交换 / 搬运人
「Ｘ」轻按低姿势 / 长压匍匐
「左半touch pano」（PS3版应为Select）暂停选单
「右半touch pano」（PS3版应为Start）iDroid Menu
「十字键」切换武器
左 - 夜视镜
右 - 手榴弹
上 - 主力枪隻
下 - 小枪
「L1」无线电（任务提示）
「L2」长按举枪
「R1」望远镜（第一人称视角）
「R2」轻点肉搏 / 长按CQC

举枪中操作
「R1」轻点切换模式（准心模式 / 第三人称模式）
「R2」射击

CQC中操作（R2需长按）
「L1」胁迫
右类比左 - 呼叫附近的同伴
右类比上 - 逼问情报
「R2」连打勒晕
「△」刺杀


SYSTEM

这里简单说一下明显感受到与过去不一样的地方。

1.潜入系统

当你在有些距离的情况下暴露于敌兵的视野时，Snake会感受到敌兵的视线。
此时画面中央会出现白色的圆弧表示视线来源的方向，是类似踪横谍海的感知系统有玩过的应该懂我的意思。

不过不用紧张，进入视野到触发警戒还有一段时间，敌兵只会先慢慢拿出望远镜来查看。此时只要拉开距离或是用紧急回避之类的闪进障碍物后方，过一段时间即可解除敌兵的疑惑。

而如果画面进入「慢动作状态」则代表敌人已经认定你是入侵者。

这个时候再进行暱踪已是毫无用处，唯一能够挽救劣势的机会就是按下L2进入枪搆状态，Big Boss就能发动「子弹时间」并且自动瞄准敌人的方向。而只要不是对射击游戏相当苦手的人，相信此时有相当充裕的时间让你瞄准敌人的头部。（根本超神）

如果你与他距离够近，在敌兵开枪前进行CQC也是一种方式。

然后在潜入时，只要用枪搆或是望远镜把准心移动到小兵身上一次，便可以将小兵的位置记录进iDroid裡，此时不只会在地图上秀出他们的位置，即使敌兵身处于障碍物后方也能够透视看见他们的动向。感觉这会是MGS5潜入系统上很重要的重点！

警戒的情况在画面左手边会不停跑出动态消息显示现在的情况。这次没有警报倒数，不知道还有多久会解除警报让人很焦虑。不过相对这代的敌兵比较没过往那麽缠人，即使被逼进了死巷裡，时间到了小兵也会自动离开。

整体算是很单纯好懂，不过MGSV似乎不像是历代有那麽多各种有趣的AI反应

2.而iDorid的概念其实就有点类似多功能的军用战术PAD（还附带黑科技般的立体投影）
可以查阅地图、查看收集到的线索、呼叫空中支援、甚至是变换背景音乐......等等

3.空中支援这个东西呼叫后大约要等待30秒至一分钟左右就会降落在指定地点，任何能扛在肩上的人都能扔进直升机裡打包回家（包括敌兵），可以先把要拐走的人全部集中在降落地点再呼叫直升机一次扔乾淨。

4.搬移没有行动能力的人质之类的这次变得很多变，由于是扛在肩上Big Boss能跑能跳甚至可以边扛边战斗（限定单手小枪无CQC）。不过有些身体状况不好的人质会不停发出呻吟声，如果太靠近敌兵会吸引到他们的注意力。也可以把它们扔上军用卡车之类载他们离开。

5.而这次还有一个很不一样的地方是道具系统，Big Boss这次就不把百宝袋放身上了，能用的东西就只有十字键四个方向。不过有趣的是过去敌兵的的武器会被DNA ID Lock的设定没了，看到好枪想带走也无彷，不过一种类型的枪一次只能带一把.....目前感觉还是Big Boss原先那把有消音器的机枪+zzz手枪好用。

6.然后子弹道具的部分更是跟地图融合在一起，有时在地上有时在柜子上，多了真实性不过变得相当难找。

7.血条也是与一般射击游戏差不多的系统，受到伤害会随着时间自动回血。当受到重伤时Big Boss身上还有止血喷雾可以使用。

8.其他值得一提的，还有双眼镜的部分。这次我们的望远镜竟然还内建指向性麦克风，不只能观察到远处还能听到该方向的声音。（是很酷的设计啦可是怎么感觉1975年的科技越看越是比一二代的还先进）


WALKTHROUGH & STORY

目前只玩过主线任务的普通难度，因此以下的流程基本上会以此为基准

一开始的任务简报目的就是解救Paz

在演示过一段发售前就播放过很多次的片头动画之后（顺带一提过场动画的放大是按R2）。Big Boss会先从东方岩壁的方向开始潜入。

此时我们要前往的是位在北方海岸的监牢地区。在地图上确认好方位之后，出发的岩壁位置其实正好处在地图的制高点，建议在出发前先按R1开启双眼镜把沿途能见范围的敌人先一一标记Mark起来。如此一来在往后的攻略便能更加得心应手。

而首先要面对的，就是位在岩壁下方的监视塔。

该塔的视线等于聚光灯扫过的位置，往右方可以爬往更高处绕道监视塔的侧面，从此处即可轻鬆的狙击驻守塔裡的小兵。

解开铁门的门锁出去的方向左右前各有一隻小兵，如果事先已经在制高点上做过标记这裡应该不难才对。而要注意的是在前往北方的路途还有另一座监视塔，可以狙击也可以沿着楼梯爬上塔把它欧昏。不过要注意的是塔底也是有敌人的，贸然行动很容易被发现。

接下来面临的是帐篷区。这裡的走道范围较小，很容易与敌兵在转角相碰不过外围大马路上大多是站着不动的敌兵也不容易应付。一样如果有事先把敌兵标记起来用处就很大了，小兵的动向非常清楚。

通过这个区域后，走到底以可看到两座山壁中间的路线即是通往监牢区域的方向，门口有几位士兵，其中一位会站在哨口，通通狙击解决。

过了关口正前方即是监牢区域了，也是这次任务首个目的地。而左方的岩岸是个安全又隐秘适合呼叫运输机的地点。

监牢区域没记错只有一位士兵会在内巡逻。潜行CQC无力化他之后，最靠海岸角落的牢笼即是CHICO所在的牢笼。

看来监牢惨无人道的惨遇，让Snake与Chico的相逢并不是那麽愉快。Chico似乎受到了极大的心理创伤。仔细一看，甚至发现为防止他逃跑，Chico的脚踝甚至被打了铁钉，几乎等于没有行动能力。

而令人震惊的是据Chico所言Paz已经被杀了，一旁Paz的牢房现下也已经空无一人

结束剧情之后，便得把Chico送上直升机。开启iDroid，往左一页的选单裡即可看到呼叫直升机的选项。此时个玩家可以选择自己喜欢的降落点，而如果从安全面来看监牢区域旁边的海岸是个很棒的选择。

而运送Chico的过程，其他牢房的囚犯看到Big Boss也会纷纷嚷嚷着带他一起走，如果之前没先把监牢区域的敌兵解决这个骚动会引起他们的注意。当然有馀力的话把其他牢房的人质一起带走也是OK的。

等待直升机的过程，Snake会详细问Chico关于Paz被杀的原委，Chico会把一开始的光头男丢给他的录音带给Snake听，内容似乎是Paz被杀的过程的录音

呼叫后稍等一段时间直升机便会到来，待直升机的门打开即把Chico送上直升机。此时Chico会把光头男的录音带交给Big Boss（好想吐槽一台有浮空投影的机器可以播tape这时代的落差感也甚妙）之后Kaz便会与Big Boss透过录音带的内容一起研判Paz最后的所在位置。

而几个关键的声音，巨大的铁门、汽车声、直升机声等等等，玩家可以透过这些讯息寻找地图上符合条件的地点，或者是用CQC逼问几个敌兵，敌兵也会透露出Paz的所在位置。

然而不管方法为合，地点都在西方的资工所。

潜入资工所的路线有蛮多，但是这裡是错综複杂的窄走道，也是很容易在转角跟敌兵相撞的地图。除此之外这裡一楼还有为数不少的监视器。觉得麻烦可以将之击毁。

而我们的目标是往地下室前进，Paz就位在地下室角落的铁门后，开锁后即可入内拯救她。

一进去便能看到，双手被吊在牆上Paz模样是非比寻常的凄惨，不仅妹妹头被剃掉了、衣服上满是血迹、漂亮的脸蛋也被打的东肿一块西肿一块。不过令人庆幸的是在Big Boss把她放下来之后，发现她还有气息并不是真的死了。但即使如此她的身体仍是非常虚弱毫无意识，只会不停重複着肚子好痛之类的

接下来的任务便是带着Paz离开这裡，这裡玩家依然可以决定自己喜爱的回收地点，不过这裡与当初解救Chico的地方不同，两个较偏僻安全的地点都有些距离。不过有趣的是回到一楼，先去开启车库的大门（与录音带中的大门声音是同一个），即可驾驶装甲车或是吉普车迅速的带Paz离开。

不过要注意的是看起来相当痛苦的Paz沿途的呻吟声很容易招来敌兵，当然即使扛着Paz也是可以战斗，但是在解救完Paz基地会呈现黄灯状态，Paz又特别容易被打死，建议先清理出一条安全路线一口气突破这裡。

到这裡原爆点的主线剧情也差不多告一个段落了，剩下的只剩剧情动画。真的很短两个小时轻而易举。之后会有个成绩总结对这次的任务进行评价（发现解救人质加的分数特别多）

送上直升机后本以为终于大功告成了，还来不及庆幸Paz还活着，Chico就发现Paz的腹部有异状，一掀开衣服发现Paz的肚子上有刚被切开过又缝合的痕迹！Big Boss见状立马呼叫医护兵前来调查（看Paz身上那麽多血早就该叫了好吗！），医护兵伸手一摸发现不对劲，肚子裡似乎有异物！Big Boss直觉认为这是陷阱，没时间做全身麻醉了，医护兵立刻剪开缝线，撑开伤口手一伸就伸进Paz的内脏裡翻找异物的来源！（ㄍㄍㄍㄍ！真的很不好意思，但是我真的忍不住！！ㄍㄍㄍㄍㄍㄍ！！！！！）

想当然尔Paz叫的之痛苦凄惨，血也喷的整个镜头到处都是（这绝对是MGS史上最血腥的镜头！ㄍ！！！），之后挖出来的，是一块砖头大小、上面印有Peace符号的包裹。Big Boss一把将它扔出直升机。

然而事情刚过，接近Mother Base时Big Boss才赫然发现海上基地被他人奇袭了！（应该就是片头光头男所率领XOF反FOX部队）Kaz与他的兄弟们都在奋力抵抗。直升机尚未落地Big Boss便立马加入战局，无奈即使如此仍寡不敌众，一群精英部队就这样被人在自己家裡给灭了，仅连带Kaz在内的少数几名无国界士兵登上了Big Boss乘坐的直升机，Mother Base也毁于战火之中。

刚死裡逃生的Kaz极气败坏，对着Paz又打又骂，认为这个双面间谍才是罪魁祸首、击垮无国界军队的根源！而Kaz一连串的骚动，也无意中把Paz的意识摇醒了。醒来看到了Big Boss与Kaz等人
，Paz感到相当的慌张与焦虑。

「我的体内.....有炸弹！」

「别担心，就在刚才我们已经把你腹部的异物给拆除了...」

「我说的炸弹并不是指那个！」

说完，Paz不知何时已经自己移动到了舱门边（泥马小姐你才刚刚被人插过肚子啊！），开启了舱门跳了出去。
就在半空中，Paz化为一道闪光，强烈地冲击震飞了Big Boss，尔后就这样失去了意识.....

半梦半醒之间，眼睛模模煳煳地睁开了，映入眼帘的是没见过的天花板。洁白的牆与绿色的帘幕，以及同样身着白色服装的护士。

发现了病人已经苏醒，不知道说了些什麽后护士急急忙忙跑了出去。再次进入房间时，身旁已跟着一位看似是主治医生的人。意识依然是模模煳煳的，唯一知道的只有自己的左手却不知何时已成为了冰冰冷冷的铁钩，以及，经过了那场爆炸之后时间已经过了9年

MGSVGZ END
