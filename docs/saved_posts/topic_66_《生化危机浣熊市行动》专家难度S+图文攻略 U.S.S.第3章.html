《生化危机浣熊市行动》专家难度S+图文攻略 U.S.S.第3章

mechille 2015-07-17 12:34编辑     0评论
S+达成条件：
1、至少杀110个敌人
2、死亡次数最多只能有3次
3、道具收集至少20个
4、通关时间35分钟以内

推荐使用角色：Lupo
主动技能：Super Soldier 对下一次攻击免疫，同时提高射击精准度和威力（LV3）

心得：本关丧尸茫茫多，杀敌数不是问题。问题是时间，路上遇到舔食者的地方可以用Super Soldier节省点时间。医院着火之后遇到的猎杀者不杀，直接跑过去开门（注意红外线）。尼古拉的地方，站在窗子底下用Samurai Edge点射+扔雷。最后几波美军注意预判出来的门，然后扔雷搞定，不要拖拉，毕竟上面还有一个尼古拉，他的狙击枪很容易打你出失血。最后一波美军不杀，直接直接开SS跑过去安装EMP就行了




一上来就可以在一堆散落在地上的书本旁拿到一个Collect Data


来到地图上中间的房间，屋内的书架上可以拿到一个Collect Data


桌子上的电脑旁可以拿到一个Collect Data


地上躺有一个医生尸体的房间可以拿到一个Collect Data


尽头的房间可以拿到Blue Card（非剧情必须）


来到另一个尽头通过Blue Card开门屋内有很多道具，左手边有治疗喷雾


地上有手雷


右手边的桌子上拿到Collect Data，至此开篇的收集告一段落，进入尽头的门触发剧情


剧情之后是茫茫多的丧尸战斗


优先攻击脑袋上绑炸弹的丧尸，可以节省弹药和时间


屋内一角的茶几上拿到一个Collect Data


桌子上也有一个Collect Data


下一个房间的柜台内侧电脑旁边拿到一个Collect Data


旁边的书架上拿到一个Collect Data


屋子另一边的电话旁边拿到一个Collect Data


往左手边的过道走，会有几只丧尸，依然优先击杀绑炸弹的


进入下一房间，电脑桌的后面有个治疗喷雾


一路往前走，清理茫茫多的丧尸


男厕所的地上拿到一个Collect Data


值班室的轮椅旁拿到一个Collect Data


大厅柜台旁边的轮椅旁还有一个Collect Data


绕到后面的房间拿到垃圾桶上的Black Card（剧情需要）


然后开门后遇到舔食者，P难度下这里不推荐杀


先沿着左手边走，电脑旁拿到一个Collect Data


然后房间另一边靠墙的柜子上拿到一个Collect Data


然后跑到尽头碎掉的尸体旁边拿到这个房间的最后一个Collect Data


打掉红外线进入下一房间，会从天花板上掉下来一个舔食者，干掉它，红外线后面有个治疗喷雾


打掉路上的红外线，上楼，丧尸战


依然优先击杀带炸弹的


房间柱子旁边的架子上拿到一个Collect Data


角落里凌乱的书旁拿到一个Collect Data


柜台内侧拿到本房间的最后一个Collect Data，旁边有个电脑可以提交数据


打坏红外线上楼，开门触发剧情，这里要对付很多只狂暴丧尸，背后有两个草药和一个解毒喷雾，解决掉他们之后上直升机平台，剧情


剧情过后一行人再次掉到医院里，干掉破门而入的丧尸


出来后沿左手边走，干掉两只从火焰中冲出来的丧尸


茶几上拿到一个Collect Data，小心火


出门，居高临下干掉下面的丧尸，注意有几只狂暴的


再次进门，这里往前走会冲出一只猎杀者，无视它，一路跑进去


这里注意躲开红外线跑


尽头还有一个猎杀者，依旧无视，拿到旁边架子上的Collect Data然后打掉门上的红外线，出门


如果费血过多，尽头的医疗车后面有两个急救喷雾


跟着标记走，出来后遇到美军正与丧尸激烈冲突中，这里推荐把美军打出失血效果，然后再逐个点杀丧尸，如果扎堆可以考虑扔雷


一路往前走右转从逃生梯跳下来，地上无头尸旁边拿到一个Collect Data


进入公园，拿到门口的一个EMP，然后沿着右手边走


注意这里全部都是狂暴丧尸，注意走位


从铁栅栏门进去可以拿到本关的小浣熊


出来找到另一个EMP（位置是随机的，不过只有几个点会刷），然后就可以进门了


这里是尼古拉的BOSS战，推荐站到窗下用Samurai Edge点射+扔雷，注意躲避尼古拉仍的雷


打一会儿尼古拉会匿掉，然后外面刷出几只丧尸，用喷子清理一下，然后尼古拉会再次钻出来，重复刚才的打法


不一会儿尼古拉跑掉了，进入旁边的门，钢琴旁边的椅子上有一个Collect Data，两个丧尸都可以杀掉


旁边的门内，干掉丧尸，注意红外线


桌子上拿到最后一个EMP，旁边的架子上拿到一个Collect Data


然后瞄准房间的侧门，一会儿会冲出来3只丧尸，一个带炸弹


出来之后没走多远是和美军的战斗，这里推荐Samurai Edge点射


剩下没几个的时候可以直接换喷子开SS上去喷


进入电厂，剧情过后往前走，楼梯旁边就有一个Collect Data


沿着右手边一直走，快到头的地方桶上拿到一个Collect Data


往里走，铲车上拿到最后一个Collect Data


调查机器开始战斗，这里推荐调查机器之后马上往正前方刷美军的地方扔雷，然后点射


干掉一波美军后过去安装EMP


然后下来调查第二个机器，调查完毕之后马上右转扔2颗雷，炸掉门里出来的美军


然后下来安装第二个EMP


接下来调查旁边的第三个机器


之后开SS一路猛跑到最后一个机器的位置，路上的美军无视


安装最后一个EMP，本关结束
