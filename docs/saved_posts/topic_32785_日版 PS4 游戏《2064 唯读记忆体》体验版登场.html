日版 PS4 游戏《2064 唯读记忆体》体验版登场

sai8808 2017-11-15 21:14发布     15评论
本作正式版预计于11月22日在 PS4 上登场。

Active Gaming Media 旗下，专为玩家精选优质游戏的 PLAYISM。
推出了预计在11月22日发售的 PS4 游戏《2064：唯读记忆体》（2064: Read Only Memories）试玩版。



西元2064年，由于科技高度发展使得人类机械化，加上透过控制遗传因子让人体修复功能大幅提升的「半生化人」增加，搭载「唯独记忆体」这种人工智慧的机械也大为普及，「人类」一词的定义逐渐模煳。
玩家要在一座名为「新方济」的溷乱城市内，与当地民众交流，进而展开一段解谜之旅。
在与身边的唯独记忆体一同行动、调查的同时，你也会逐渐被捲入这座城市的巨大阴谋当中……。



各有立场的角色群

玩家身边的唯独记忆体，是世上首批拥有自我意识的机器人，因此拥有各式各样的烦恼。
至于人类，也同时存在着接受改造与相对的反对者，而身处科学文明和人类之间的人也不在少数。




洋溢对过去日系着名冒险游戏的爱之结晶

本作画面全都由点阵图所构成，只需进行选择就能推动剧情。
难免令人回想起数部名作，但这可不只是个怀旧风格的作品而已。

多重结局

这段解谜之旅会随着玩家的选择、行动，产生不同的变化。
这座未来都市的故事走向，全凭玩家决定！



何谓人类的大哉问

本作聚焦于超人类主义的主题，让玩家在故事中深思随着网路科技发达，不断追寻便捷生活的情况。
