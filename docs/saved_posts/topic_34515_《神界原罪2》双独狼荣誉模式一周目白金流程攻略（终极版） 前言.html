大家好，我是imba（前dota玩家），张总之前叫我七海（尼玛被人称呼为自己喜欢的角色的名字好羞耻啊），时隔了8个月现在重返P9了。但显然还是要回报篇攻略才算数嘛，果断我就拿PC上此前已经通了两周目的神界原罪2开刀。但在想怎么构思这篇攻略的时候，上来纠结了好久，具体在机因里说了说，但最终，我还是狠下心打算把这篇攻略写了，因为这游戏实在太赞了，不分享下心得蓝瘦，同时之前有dalao写的攻略是没有流程的，对一周目荣誉难度（尤其是前期）受苦的新手实在有点不友好。

首先这篇攻略的打法路线已经给你固定了，就是全程物理双独狼来进行通关，其中一个是拿弩的远程，一个是拿力量系武器的双手战士。当然我也不是神界原罪2的build搭配大佬，攻略中间一些加点、技能搭配的问题请大佬们多多指出。

其次本攻略针对的是想要快速白金本作，同时不打算或无法靠蹭别人的多人游戏看结局来拿到最高难度奖杯的新人玩家。假如蹭到了奖杯也可以参考下本攻略快速通关最低难度补完余下奖杯（剧情模式下，本作全程无双，前期独狼就已经可以无压力越1级打人）。荣誉难度就是存档唯一、死队友自动存档、死完直接删档的战术难度，区别仅此而已，玩这个模式，建议把系统自带的自动存档关闭（仅关闭在进特定剧情时的自动保存，死队友后的自动保存就别想了）。

记得有站友在机因跟我说这游戏PS4版保存太慢不舒服，但我细想一下发觉这不是好事嘛。亲测在死了队友，系统在进行保存的时候按并直接退出游戏，再开游戏选继续玩会直接回到上一个存档，卵事没有。

最后本流程是我用PC的终极版本重打一次荣誉模式的流程，PC主机的版本内容基本一样（BUG不知道），必要时加入图片。欢乐堡会比较详细，死亡海岸其次，无名岛、阿克斯城就较为简略，因为本作难度也是由此递减。在这期间相信站友对于这游戏的玩法也会逐步加深了解，到了后面，在等级不太低的情况下难度估计只有1/10，因为可用套路很多，而且也差不多都成型了。

PC版的荣耀模式的存档文件直接复制一份备份好，系统删存档就直接复制回来就行了，比主机简单多了。。
PC版垃圾，我要是失误人死了，根本来不及退出取消自动保存。

进入游戏，选择策略模式中的荣誉难度，进入角色选择界面。
首先在“起源”里，选择费恩（一个有起源故事的骷髅头）。在下方中的“预设职业”中选择骑士
再前往“天赋”中将伺机而动换成踽踽独狼。

先讲一下独狼的特性：行动点数上限从6变为8且每回合回复数量从4变为6，血量+30%，双甲数值+60%，属性和战斗能力（除了变形学派）的点数投入翻倍（也就是升1点当2点来用，但上限分别是40和10，与普通角色相同，可以通过装备和外来加成来超过这两个阈值）。独狼天赋在队伍成员大于2的时候无效（此指左上的角色数，不是指联结起来共同行动的角色数。）。

“标签”里由于你选择的角色的起源故事固定，因而无法改变。“外观”可以凭个人喜好进行改动，在第一章末尾上船后亦可随意进行，音乐也在上船后可以随便改动，个人觉得默认的班苏里笛已经很好听了。最后在“预设”中将下方的属性改为+6力量，能力改为战场技艺+4，交涉+1，技能选择默认。

选完天赋后，别再修改预设职业，因为这会重置之前设定的属性、技能与天赋。

进门后触发与温迪戈的剧情，选项随意。

醒来后拿上一瓶桌子上的酒，打开桌子前面的箱子（以后遇到类似的箱子可以进行一下搜刮），装备上双手剑并且也拿上其他的武器，把旁边3个尸体舔一下，最后切记把自己身上自带的兜帽装备上，在这个游戏里被别人察觉到亡灵时，很多时候会招致无差别攻击（方法包括并不限于不带胸甲、头盔等），然后从身后的楼梯上楼。

这一层先从右侧进入有3张床和一只狗的房间，两下打爆通向船尾的门，潜行进入房间拿床前的箱子里的装备和鉴定镜后原路离开，这一层的死去的净源导师都可以舔，在X:29 Y:-180的尸体上拿到开装有死亡毒雾桶的船舱的钥匙。

在这一层中间通往船头的方向可以找到一个带有骸骨标记的门，也即上边钥匙对应的门，现在门口进行下保存。接下来操作为，与门互动，在对话中选择用钥匙打开门然后在门开的过程中摇杆往舱内移动快速进入舱内，拿上最近的一桶死亡毒雾，没拿到直接载入，拿到的话把桶扔地上看是否自行烂掉，没有烂说明成功，烂的话说明手慢了也载入。

往船头方向的左侧有个楼梯，上去后触发本作第一场对战：两个虚灵幼虫。
打法很简单，原地不动，推迟你的回合等这两个幼虫葫芦娃，你的输出根据加点和装备足够1回合里打爆一个（致残打击伤害高于普攻），差一点也无所谓，这两输出很低打不死你，因而无压力。面前有摊毒性物质可以让你战后回血（反复在那个范围里走）。

这层可以搜的除了旁边的幼虫、净源术士以外，还有船尾的船舵旁也有两个可以舔，旁边地上有改良魔杖，一路上的火可以用船舵面前的水桶直接扔上去扑灭。

这些搞完就可以直接走向图中的红旗处，选择与救生艇活动，在与矮人和小孩的对话进行到有选项时选择“你会回到甲板下面，寻找幸存者”，然后走向新的红旗处，下梯子。

具体过程如下，先在门口的西侧阶梯下还未触发剧情时将你之前带着的兜帽摘下显示你亡灵的身份，然后径直走向大锤，她会直接跟你开战，这一战，大锤是行动链里的第一个，你是第二个。她很可能会走到脸上直接打死你，这里载入的话手速要快。假如会被大锤打死就离她远些的地方靠近大门触发剧情，然后仗着独狼的6AP，我们就大摇大摆走到亚历山大跟前（一个穿着华服的男子，就在那摊净源导师尸体的后面），放上之前装死亡毒雾的桶（如下图），打破它，这时候亚历山大会显示“死亡”，大锤会直接变龙带走亚历山大，然后你就拿到了奖杯，拿到之后迅速退游戏，因为大锤的手下还能动而你的这一回合已经跳过。


上述过程中有两个地方可能直接死亡：经过大锤和手下太近时被他们的“伺机而动”给打死，这一点需要特别注意，或者在大锤带亚历山大走之后，在等跳杯期间被之后行动的手下打死，但由于我们已经备份存档，所以都没关系。

这里有个很大的关于达利丝身世的伏笔：你拿桶砸她脸打破会发觉她免疫毒雾。

拿杯完毕，载入回触发大锤剧情前的存档，我们通过大门旁边的梯子绕过大锤剧情进入欢乐堡（此时未到2级，全游戏唯一一次卡等级），会上到一个城墙之上，此时先别往右边走，先往左边的另一个梯子下去，首先去见我们唯一的队友——希贝尔。

路上可能会被一个叫高文的人主动搭话，我们先选择“离开”，不进行下去。同时途中可能会激活欢乐堡中央的一个传送点“欢乐堡——广场”，在其雕像位置附近（X:203，Y：157）即可激活。

你会先在X:213 Y:121发现希贝尔，但假如没有第一时间和她搭话，过了一会儿她会最终前往X:184 Y:101。

与希贝尔的对话可能会引发与她的交战（见下图），所以保险起见存个档，但不用上传，她前边的对话随便聊，但最后会来到一个分支：“与其杀了你不如加入你”&“已经无路可走了”，显然后面那个就是开干，所以要看清选项，别选错。


选完和平选项之后再随便聊几句就进入希贝尔的职业选择了，这里选择弓箭手，关键词“身手敏捷”——“弓箭手”。

你会发觉希贝尔的初始状态里没有踽踽独狼的天赋，下一个天赋需要在3级才能加，因而我们需要尽量无战的方式来到3级。

回城里后，前方有四个人站在一起（给个坐标X:209,Y:179），其中包括主角之一的伊凡，跟其中的巴罗说话，一通对话后选择“站在精灵的一遍，并要暴徒退开”，经验到手，这个精灵会之后让你跟着她近一个地洞（X:215,Y:90）,进洞之前在洞的前方有一个矮人医生莱斯特问你要酒，把从船上拿来的酒给她即可。

进洞里，跟里面一个叫摩迪的小男孩对话，选择“求他饶我一命”后他会邀请你玩捉迷藏，准备好后，他会跑（当我们是盲人），走到（X：475，Y:564）找到他之后，他会又一次跑，这次他聪明了，在准备之后，开着隐身跑。。这一次他在（X：507，Y:552）,这一次他佛了，带我们去认识个朋友，跟着他到旁边的狭缝中会发觉人和精灵进不去的（矮人的个头可以进去），等一会儿剧情之后在这个狭缝的前方会发觉一个坑，这时候之前的铲子就有用了，挖完会出现一个石板门，进入。

进去之后往前走会见到小孩在与一个被长矛刺穿的人？进行对话，直接拔出“维瑟莫尔大人”身上插的长矛（3级，2级用只有75%的命中，不实用），之后的对话中选择接受他的交易，原路返回。

狭缝外可以找到之前的帮助的精灵，与她一通对话完后可以得到一份奖励，三选一。

一般来说我们需要的是强力的远程的弓、弩、双手力量武器、词条效果强力的戒指&吊坠、有用的技能书和一些符文，因而矛也不考虑拿，假如没有之前所说的急需的物品就直接拿奖励中最贵重的物品用来卖钱。

同时洞里还有一个重要角色——萨希拉，与她对话询问如何出欢乐堡，得知拯救一个叫阿迈罗的精灵，其他的一些剧情信息也可以进行询问。

在欢乐堡中装备的提升作用并不是那么得显著，理由是因为等级低的情况下，目前的装备词条效果都很弱。但之后装备数量会越来越多，词条会五花八门，所以对其的选择需要多加斟酌。

本路线里，对于物理双独狼战术来说，达到最大的DPS数值，能够先手秒掉敌人并尽可能控制剩下的永远是作战的核心思路。因而就装备的选择来讲，面板的护甲数值并不太重要（当然要是面板数值差个好几倍，同时提供的属性差别不大的情况下也可以选等级高的）。而费恩和希贝尔在装备的词条选择上有一定的差别。

计算数学期望：

平A时的伤害数值=武器伤害 x（1+武器对应属性点*5%+对应武器类型点数*5%）x（1+战场技艺*5%）x [1+暴击率x（暴击倍数-1）]

当拿远程武器且有高地优势的时候，平A的数值=武器伤害 x（1+武器对应属性点*5%+对应武器类型点数*5%）x（1+战场技艺*5%）x（1+神射技艺*5%+20%）x [1+暴击率x（暴击倍数-1）]

暂不考虑暴击的情况下，又由数学常识可知，当几个正数相乘时，若要将其中某个数增大一定数值，则当选择将其中最小的数增大一定数值时，所得结果最大。

也即若 0 < a < b < c ,又0 < d,则(a+d) * b * c > a * (b+d) * c > a * b * (c+d)

对于费恩而言：
战争技艺>双手>力量>智慧>刺杀学派>死灵学派

对于希贝尔而言：
战争技艺>神射技艺>远程>敏捷>智慧>刺杀学派>死灵学派

这里神射技艺的作用在高射低的时候效果仅略低于战争技艺，效果依然很好，而且本作很多地方可以提供高地加成，比较实用。但假如是全程平地战，远程又没点满的话可以把其洗为远程。

然后考虑倍伤：站友们应该发现了，本作战斗中一个比较特殊的地方：无论对战双方的先攻如何（这里暂时不考虑第三方因为情况较少），行动顺序一定是敌我或者我敌交替的。对于独狼来说，这个顺序仅在于先攻最高的人是否大于先攻最高的敌人。又因为独狼的属性点数爆炸早早就开始全力拉满智慧，因而先攻的考虑几乎没有什么用（印象较深的只有浮木镇这张图里的石园墓地里的看门烈犬安德拉斯42的先攻对于当时等级来说很难超过）。因而智慧的作用在前中期实在是不太高，但到了后期前面属性爆炸，然后爆伤爆率堆起来的时候就很恐怖了。

刺杀学派可以和双手一样增加武器的暴击倍伤（像双手战甚至可以堆到300%暴击倍率），因而对于输出也有用，但考虑到前中期主角的暴击率没起来，所以排名靠后。

死灵学派的效果是普攻可以吸血，切记此效果不能回甲，但满血的鲁莽之人BUFF效果不错，可以靠攻击吸血来保持满血所以也算是聊胜于无，几个大BOSS有反伤的效果，需要靠这个来抵消。

剩余的抗性效果、免疫效果等见仁见智吧，对于尽可能无伤的独狼战法来说提升都较小。

特别的，需要1点烈火学派来学习火系的两个增益BUFF：心静神宁&加速，这里尽量靠装备来凑（尤其是戒指），别用属性点来使其+2。

接下来主要都是干，为了保证新人们稳干医生，这里我分享下费恩的武器选择，因为现在爆伤的大大提高导致现在他才是主要的DPS。首先我的档里费恩在装备矮人买来的沃尔达拉马斯的时候，均伤在1000多（如下图），暴击率在心静神宁的加成下是100%，打死医生的时候在额外回合里还没开肾上腺素剩1AP。

然后这里我给新手一个再一次大大提升DPS的的方法，也就是搞把更牛叉还更帅的武器，它的名字叫法隆，可以直接在教堂大门进去右转也就是（X:340,Y:384）处的雕塑手中拿到，但需要20级才能拿上这把武器，而我们不够19级，所以首先练到19。方法简单粗暴，也就是把教堂正门口的桥两端站岗的圣教骑士强开宰了，假如还不够就把净源导师兵营里的圣教骑士也一同宰了，难度都不高。然后装上后你会发觉命中率还是只有85%（初始95%+10%鲁莽之人-20%的武器等级惩罚）这15%直接由心静神宁的20%命中率加成就可以直接弥补，同时你的DPS最起码提升了30%。

接下来有点难的成就只有杀死医生了，难度高于最终决战。他的甲加血总计超过30000，而且一旦倒地起来的时候还会重新回复护甲，但同样靠绿茶和其他套路我们还是可以1T里秒他。与门互动：受邀来的。为了求稳还是上传存档并保存（之后做和医生合作的结局时别在线上覆盖这个存档），进去之后，这架的主要问题在于医生在正常对话完开战的情况下一定是先手，这逼的火系魔法可以秒我的7000血甲费恩。为了不让医生出手，我们的打法是潜伏加金蝉脱壳，开启医生南边的门，切记此时一定要让费恩的先攻高于希贝尔（可以靠心静神宁加10智慧），然后让希贝尔强攻医生开战（如下图开战位置，需要S/L防止对话）。费恩的回合里磕绿茶（同时可以把其分给希贝尔，不会消耗AP），然后位移到医生身边干他，我上来1+1T直接把他给打死了（由于绿茶原来的-2AP效果削成了-1AP，因而我的建议是全程除了致残打击以外平A即可，输出也很爆炸）。假如打不死也很简单，破甲倒地总归可以，虽然他起身会回复6000甲，但希贝尔下回合磕了绿茶后应该也能直接带他走，打死就会跳杯，再打他的小弟有点麻烦，直接读打之前的档即可。此外医生不仅坚毅高，报复的等级也不低，但我们应该都有一定点数的死灵学派（费恩应该是10），不至于被反击死。
