bug及解决方案

bug及解决方案

恶性bug

1. 坏档：
没得说，只能多注意备份存档

2. 自动存档失效：
触发机制不明。有时候你玩了很多天以后退出游戏，下次再启动会发现自动存档并没有保存你的记录。
建议：每天自动存档结束后手动存档一次

3. 存档失效：
触发机制不明。但我能肯定这个问题是跟随存档文件存在的。我在PS4上遇到这个问题，把存档拷贝到ps5问题依然存在。反馈给制作组后直到目前也没有回复。
这个情况应该比较少见，可能只有幸运如我才会遇到。具体症状是自动存档和手动新建存档在游戏时看都成功了（产生了新的栏位并且有新的存档带有对应的时间），可是一旦你退出游戏再进入，之前的存档就好像从来没有存在过。。。
建议：在你没有遇到这个情况的时候多建一些新档。一旦出现了这个情况，你只能选择覆盖之前的存档才有可能保存成功。每次存档务必多存几次确保存档成功

游戏性bug
1. 有时候客户要求的配件是还没有解锁的。。。这种任务就果断拒绝吧。。
2. 清灰的任务无法正常完成。官方的解决方案是把电源拆下来，清灰，安装电源。（本来最简单的任务复杂程度直接翻倍）这个bug已经在2021年8月的更新中修复了，但保险起见还是提一下。
3. 上面的bug修复以后我还是会遇到一体式水冷清灰清不干净的情况，这种时候就只能把水冷拆了，涂硅脂，涂硅脂，涂硅脂，再把水冷装上去。。。
4. 时间进行到下一天的时候偶尔会卡住，画面上风扇的图标一直转啊转。。这时候就退出重进吧，只是一天白干了而已。。

金钱相关：

这里需要用到一个良性刷钱bug（虚空刷钱）。具体操作是：
1. 在装机界面选中一个部件。显卡，CPU之类的都可以。推荐用CPU，因为在部件列表的最上面，可以少摁几次键。。还记得我前面说过这游戏对手柄操作不友好么:)
2. 选中部件以后它会出现在装机模式的左下角，这时候不要安装，直接摁圈退出装机模式。
3. 在非装机模式（就是你可以自由走动的模式）中摁十字键打开仓库，卖掉刚才选中要安装的部件
4. 对着正在安装的电脑点叉回到装机模式，你会发现你卖掉的部件仍然在左下角。这时候直接安装部件。注意，千万不要打开仓库，打开配件列表你的这个部件就真的被卖掉了。
5. 安装好再拆下来，部件就又回到在了你的仓库里。如此一来你就白嫖了一个部件三分之一的价格。之后再回到第一步，如此循环就能刷钱永动机了
这里推荐前期不用着急刷钱，到满级以后用最贵的CPU进行操作会省很多时间。

常见蓝屏、错误、超频报错速查 来自Steam社区，作者Famille

关于常见错误：
第二工作台无法解锁：必须在每天刚开始游戏时，不做其他操作直接走到第二工作台位置鼠标直接左键按住解锁，要花费$1500。（我玩的时候没有遇到过这个问题，感谢@David Osborn的测评）
插电源一点反应都没：主板或者电源问题，一般主板坏的情况较多。
插电源电脑启动屏幕不亮：显卡问题
no cpu found：CPU问题
no ram：内存问题
no boot device：硬盘问题
no os found：插u盘并重启自动装系统
蓝屏提示CLOCK_WATCHDOG_TIMEOUT：CPU没涂硅脂或者散热器没插线
蓝屏提示the system encountered an uncorrectable hardware error：一般在多显卡SLI/CROSSFIRE的时候跑分出现，超频时也会出现，CPU和显卡总功耗超过电源承载，更换更高瓦数电源。（极少的情况下是内存不够）
SLI/CROSSFIRE（多显卡）显示uDisplay.drv的错误：显卡温度过高，需要增加机箱风扇。（感谢@Fly1nDutchman的补充）

关于超频错误：
CLOCK_WATCHDOG_TIMEOUT：CPU过热，需要调低电压或者加散热。
the problem seems to be caused by the following file uDisplay.drv：显卡过热，需要调低电压或者加散热
the system encountered an uncorrectable hardware error：总机功耗过高，降低超频水平或更换更大电源。
attempt to reset the display driver and recover from timeout failed：当前电压下显卡超频过度，请适当降低超频水平或者提高电压。
a clock interrupt was not received on a secondary processor within the allocated time interval：当前电压下CPU超频过度，请适当降低超频水平或者提高电压。
PRN_LIST_CORRUPT：当前电压水平下内存超频过度，请适当降低超频水平或者提高电压。
请注意，超频CPU和内存的时候电压不要超过推荐最大值，不然可能烧坏。
