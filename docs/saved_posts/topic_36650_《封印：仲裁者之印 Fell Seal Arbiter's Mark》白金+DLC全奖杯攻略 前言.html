前言
这游戏可以说是大量继承了OB2/FFT遗产的产物，充分体现了西方人对这类老玩法的推崇，日本业界反而已经抛弃了这些曾经的好东西(比如三角战略的养成系统，简直就和手游没什么区别)
在Steam上出了好久，买了一直没开。这次正好上PS4就赶紧把奖杯都打了

相比于FFT的优点：
1. 有自动战斗
2. 有自动存档
3. 有难度选择，并且还可以微调难度
4. 职业等级依赖于技能学习，不再会出现FFT那种职业等级满了技能还没学完，或者反过来的情况
5. 随机战斗改为菜单项选择巡逻进入战斗，方便后期的收集工作
相比于FFT的缺点：
1. 怪物不会生蛋了
2. 去掉了所有弹道系统，不再会因为射击弹道误伤队友(但依然可以强制攻击队友)，降低了策略性
3. 地图不能旋转(官方号称为了精美的2D手绘背景，但我真没觉得有多精美)
共同缺点：
职业平衡性严重不足

要点:
1. 主角团里，近战职业还是武士比各专属要好用得多
2. 技能不会组合开自动看一下AI的玩法就懂了
3. 双枪是本作典型的不平衡玩法，后期除了少数高物理防敌人，其他都是远远就能狙死，所以推荐佣兵都练男性君主双枪，补魔法输出的有耶茨和贝扎罗就够了
4. 实在想练女性佣兵的话，因为不能转君主所以还是建议走法系
5. (适用于DLC)大型战斗需要9人，想练怪物的也可以练练怪物体会一下DLC新加的要素，像收割者，梦魔，光羽鸟战力都还行

条件杯

Sinking Feeling
让一个敌人溺水而死
淹死一个敌人
用位移技能(比如蛮力突击或者回旋镖)把不会游泳(会游泳的角色信息框上会有个脚蹼)的敌人推进水里，很多关卡可以完成

Stand Down!
让一个敌人坠落而死
摔死一个敌人
同样使用位移技能，但是需要一点技巧，最早可以在耶茨的小屋尝试完成。实在搞不定可以去古代之路最后一关做，垂直落差最大的关卡

Double Agent
以一个狂暴，魅惑或者精神控制状态的敌人击败另外一个敌人
用狂暴，魅惑或者心灵控制的敌人杀死一个敌人
