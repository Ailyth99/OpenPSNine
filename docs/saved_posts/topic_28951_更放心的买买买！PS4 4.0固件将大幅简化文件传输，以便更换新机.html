更放心的买买买！PS4 4.0固件将大幅简化文件传输，以便更换新机

ashinlee 2016-09-12 12:17编辑     85评论


索尼于9月8日的新品发布会上一口气公开了 PlayStation 家族的两名新成员，它们分别是 PS4 Slim 和 PS4 pro。再加上我们熟悉的四公主，索尼在本世代的主机数上升至 3 台。

PS4 Slim 的轻薄运行降噪，PS4 Pro 的 4K、HDR 技术，想必不少玩家已经准备好换新机的准备了。不过，有一个比较恼火的问题摆在我们面前。

PS4 游戏动辄几十G的大小，像《巫师3》这种容量已经到了 50～60G 的游戏比比皆是，就算你买的是实体版，重新安装后也需要再更新个十几G的补丁到最新版来体验最完整的游戏内容。还剩下一种方法，就是准备一块移动硬盘，把 PS4 上的所有内容拷贝至这块硬盘上后，再传输到新的 PS4 Pro/Slim 上，还有一大堆游戏存档和 DLC 内容，想想都觉得蛋疼。。。

一切都为了玩家的索尼当然考虑到了这些，索尼 PlayStation Blog 上最新的一篇文章打消了我们的顾虑。

在 PS4 XMB™ System 4.0 更新后，会迎来一个新的功能，来大幅简化上述的问题。玩家只需要将一条以太网线连接两台 PS4，就可以把游戏、存档、视频、系统设定等资料拷贝至新机中，三款 PS4 主机互通。

关于 PS4 4.0系统，索尼暂未公开其上线日期。PS4 Slim 将在 9月15日 发售，PS4 Pro 则是 11月10日。

From.vr98 / Polygon
