PS4PSV《初音未来 歌姬计划X》的开场曲已确认，本次公布其它收录歌曲及其演奏PV截图

siritios 2016-01-12 17:44编辑     36评论
除了此前新闻提到过的「LIVE QUEST MODE」（LIVE委托模式）和「EVENT QUEST」（事件委托）以外，还特别公布了「特别演出」（SPECIAL LIVE）功能。



开场曲

实力创作者ryo（supercell）为本作谱写了新曲《罪の名前》。在歌声中，Miku的故事也将以「歌の力の喪失」为起点而展开。







EVENT QUEST

「事件委托（EVENT QUEST）」是在满足一定条件后解锁的功能，与「AREA QUEST」不同，事件委托会随着游戏的推进相应解锁。


发生事件！



与角色进行对话，大致了解事件内容


特别演出

在继续游戏进度、完成事件委托的过程中会发生比较重要的特别演出。所谓特别演出，是让玩家选择3首喜欢的歌曲形成播放列表后，游玩这个列表的歌曲节奏游戏的功能（请参考IA/VT的My List）。登台角色最多设置3人，在拥满观众的直播大厅里进行有着绝佳临场感的演出。玩家通过这个功能，可以享受选曲、选角、选服装和指挥现场演出的感觉。


满足举办演出的条件，触发特别演出。


选好3首曲子形成列表。游玩3首曲目的时候会随之出现服装的变身演出，要谨慎考虑喔。


最多可设定3个登场角色。服装方面，玩家可随意任由喜好，或者优先考虑加成的效率。


在特别演出中，剧场里全是人。观众席也有不同的元素，营造出各种不一样的氛围喔。


连续演奏三首歌曲的节拍游戏，可以获得比普通游玩模式下更多热度。

舞台（ステージ）

游戏设置了许多与乐曲风格相适应、个性丰富的舞台。本作可以替换舞台背景（但也有根据委托内容而固定舞台的情况）。游戏在此基础上为玩家增加了自由度，让玩家随自己意愿来让演出更为有趣。
