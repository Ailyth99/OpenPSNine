SE发表PS4神秘新作《Project CODE Z》 预定31日揭晓详情

sai8808 2015-01-28 19:12发布     19评论
SQUARE ENIX 今（28）日开设新作预告网站，预告将于 1 月 31 日下午在幕张国际展场举办的 niconico 游戏活动「鬪会议 2015」摊位上，公布代号《Project CODE Z》的 PS4 神秘新作。



虽然目前官方尚未发表具体内容，不过预告网站上已明确标示出 PS4 主机的标志，因此可以确定是 PS4 上的游戏。另外 Z 字标示的背景中则是出现了丛林、侵蚀洞窟等场景。网址与 Twitter 都使用了 “msz” 的字样，推测应该是正式名称的缩写。有兴趣的玩家不妨密切注意后续的发表。
