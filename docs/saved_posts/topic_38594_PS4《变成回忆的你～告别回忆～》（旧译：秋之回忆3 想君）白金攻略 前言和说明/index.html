前言和说明

前言和说明


前言
其实标题旧译我想用的是《想君 秋之回忆系列》（←光谱的杰作）……
翻译基本上就是沿用老PC版，所以选项中令人烦躁的标点以及翻译不统一也一同继承了下来……

说明
虽然应该不用多加说明……
关于存档：在下一行的选项出现时，存档在对应的档位，方便之后读档进入分支/回收遗漏部分
打了括号的存档，是收集全章节标题才需要用到，如果只是白金，则不必理会
关于选项：非加粗的选项（有任意字样），选哪个都不会对攻略产生任何影响，所以也可选择自己喜欢的
有几个选项只有在重玩该段剧情的时候才会出现，没什么意义，不另外说明了
关于剧情锁：彼方和音绪真实路线，需要完成除彼方外6个角色的GE后才会出现，延续音绪GE一路下去

攻略顺序是沙子→那由多→响→环→深步→音绪→彼方，以沙子为主线发散出去，最后有攻略锁限制
如果完全照着来玩，只需要去到「回收」即可白金，后面两页（只）是些资料整理，不用看也可以……
