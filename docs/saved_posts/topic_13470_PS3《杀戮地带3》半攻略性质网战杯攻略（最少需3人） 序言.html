序言
以下内容只有杀戮3的网杯，不包含流程，因为我就喜欢和人刷网杯，网杯刷完之后就根本没玩了，因此就只写一下网杯的心得。大部分奖杯都是顾名思义的事，因此只具体说部分杯怎么刷。单击部分白金奖杯攻略A9有，附在参考网站处。

网络
我当时是找了6个人一起刷的，因为不知道那个杀5个ISA的奖杯可以换号。从结果上来讲，杀戮3联通相对比较好联，电信不太容易，开坑前建议用小号试下。在我刷过的老的游戏中，杀戮3的服务器算的上很好的了。

建房方法
由于网杯不能私人房间刷，只能公众房，因此说下建房方法。
加入有两个小伙伴，那就一起搜需要的地图，搜到的房间没有人（一般都没有人，而且一起搜很容易搜到一起）后，另一个人就可以通过游戏内的好友列表选择加入到好友的游戏，然后就可以愉快的刷了。

网杯索引
杀戮地带本体的网杯有10个，3个dlc都是网杯，因为我都刷了，所以就混在一起写。难度上来讲基本只在于凑齐人，网络很不错。具体刷的时候，难度是0。估计一两个晚上就搞定了。
这个网杯最少需要4人，因此我就按只有4个人来写，分为4人杯、3人杯、2人杯。

四人杯and三人杯

四人杯and三人杯

1.四人杯

四人杯总共有两个，推荐一起刷。选择史塔爾軍火庫来刷这两个奖杯。

奖杯：致命三連環 － 連續殺害三名敵軍---#62
这个奖杯必须在史塔爾軍火庫才能获得，让其他三个基友站成一列，然后快速，必须快速把三个基友一梭子干死，当屏幕上出现350分的时候就是成功了，没出现就再来一次。

奖杯：索敵追蹤 － 使用偵察能力殺害敵人---#76
刷的那个小伙伴选择战术师(Tactician),让三个小伙伴并排站好，都站在你的视野里。
然后使用侦查（Recon）技能（十字键左），会有一道横线从屏幕上扫到下，扫到敌人的时候，敌人连上会有个叉，这个时候把敌人干死，连续干三个就可以解锁奖杯，当时刷的时候好像是必须在一次技能发动时杀三个才能解锁，多次不行（存疑，有点记不清了）

更正：感谢@flynon 的提供的方法，这两个四人杯，三人也能刷。方法见下：

奖杯：致命三連環 － 連續殺害三名敵軍---#62
被杀的两个人医疗兵,杀一个人,医疗兵救 在救的同时杀医疗兵出现双杀。再把起来那个人杀了屏幕出三杀就解了。

奖杯：索敵追蹤 － 使用偵察能力殺害敵人---#76
被杀方选择战术师，选择复活区，被杀后直接在战术复活区复活即可。

2.三人杯

奖杯：近接殺害 － 近接式地雷殺害---#55
刷的小伙伴选择职业时，将手雷换成接近式地雷，默认是手雷。把地雷扔在一处，然后另外两个小伙伴去踩即可。
该奖杯两个人也能刷,只是两个人刷需要另一个人跑的快一点，比较累，因此不刷三人杯的时候一起刷了。

奖杯：最高榮譽 － 取得所有勳帶---#64
勳帶一共有5个，需要在一局完成，推荐选择任务较多，玩的时间较久的战区里的地图来刷。

勋带名完成条件备注
Double Kill XP完成战区任何一个任务即可
Increased Accuracy格斗杀5人
Bullet Damage辅助5个你首先开一枪，然后队友扫死对方
Faster Aim瞄准杀5人，任何枪都可
Faster Reload不瞄准杀10人，任何枪都可

勋带完成后，完成的勋带会在屏幕右下角出现，只有Double Kill XP这个拿到一段时间后悔消失，第一个拿这个确认一下即可。

两人杯

首先，确保一点，刷的时候拿的经验值不要乱用，用来学以下技能。因为有些记不清是需要学的，那些是通过升级解锁，不过刷的时候就知道，除了下面表一的技能，其他你能都不要加。好像是一级的技能都是由升级解锁，这个在游戏中都有提示，看看就明白了。
