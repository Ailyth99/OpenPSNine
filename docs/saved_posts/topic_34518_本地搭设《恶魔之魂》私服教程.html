本地搭设《恶魔之魂》私服教程

jhzthpp 2019-02-20 18:54编辑     121评论
本教程适用于有双机的朋友，可以用电脑做私服给局域网内的PS3用
公网需要考虑路由器，防火墙，很多配置，还需要有公网IP，本教程不涉及
首先要感谢油腻腻（destinybo14）教窝怎么搭私服，这个方法是他从外站看到这个研究出来
同时油腻腻也租了阿里云服务器来临时搭了个恶魂公网服务器，把PS3的DNS地址改成47.244.4.188即可
这个服务器搭建和测试都是油腻控来做的，窝只是搬运工，感谢destinybo14
---------------------------------------------------------------------------------------------------
使用私服之前可以先下载这个
然后安装python-2.7.16rc1和VCForPython27
之后以管理员身份运行CMD指令输入pip install pycrypto
编辑desse-master里的remote.conf（设置DNS地址）记得关掉电脑防火墙
然后把remote.conf放到FakeDns-master下，同时在FakeDns-master目录下执行Python fakedns.py -c remote.conf指令
接着编辑desse-master里的info.ss（设置服务器地址）
然后在desse-master下创建db文件夹
最后在desse-master目录下执行python emulator.py指令
最终打开2个CMD窗口，一个是DNS服务器窗口，一个是恶魂服务器窗口

PS:自搭服务器的话，DNS地址和服务器地址都用自己电脑IP地址替代，PS3里DNS地址改成电脑IP地址
