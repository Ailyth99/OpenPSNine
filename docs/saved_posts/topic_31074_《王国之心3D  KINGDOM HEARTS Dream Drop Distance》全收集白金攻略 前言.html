白金有两种方案：

1、一周目pound难度，通关后全收集全奖杯，白金时间50小时左右

2、一周目standard难度开局，通关后全收集，完成除了Ability Ace奖杯之外的奖杯后(此时奖杯进度98%)，二周目继承开pound难度，用4~5小时完成Ability Ace奖杯，白金。此处并不需要pound难度通关。

之所以有这样的区别，是因为只有在pound难度以上才会有zero exp技能，坑爹

建议系列老玩家直接P难度白金

新入坑玩家，或者担心自己手残的玩家可以选择方案2。

食梦兽性格转换

食梦兽性格转换

Strongest Link
在任何「靈魂」的能力連結面版中解鎖全部16個節點

解这个奖杯需要了解一下本作的食梦兽性格系统，这里挑重点讲

1、在培育室里按弹出玩具零食菜单，选择最后一项蛋糕图标，这就是本作的喂食系统了

2、全部零食一共5类13种，分别是：

金平糖、雪糕、曲奇、巧克力、蛋糕

前4个都有3种等级，对应低中高3个效果

3、所有零食都可以提升最大HP，同时：

金平糖 →→→→ 攻击
雪糕 →→→→→ 魔法
曲奇 →→→→→ 防御
巧克力 →→→→ 魔防
蛋糕 →→→→→ 全4项随机提升

4、金平糖、雪糕、曲奇、巧克力，对应食梦兽4种性格

持续喂食同一种零食即可令食梦兽转变性格，转变性格的时候眼睛颜色会相应改变

5、每种食梦兽4种性格的名字都千奇百怪，但是都对应了物理魔法物防魔防，4种使用倾向

6、食梦兽的技能板一开始都有部分技能被隐藏，转换3次性格后都为16个技能


7、性格改变后建议立刻进入link技能板查看是否有新技能出现

1、索拉和利库每个世界都有6个Special Portals、最终世界3个

3、每个SP门都有一个稀有食梦兽，就是普通食梦兽换了个颜色，但是也单独计算在食梦兽收集完成度中。
食梦兽完成记录在 report → game record → dream eater 内

4、在世界选择界面，选择一个世界后按进入drop菜单，如下



左边的done！就是你当前角色已经在该世界收集了一个Sp门

按Next就是下一个角色的该世界的相应情报
