PS4/3DS《勇者斗恶龙11》体验会试玩，不同主机同一个故事带来不同趣味

sai8808 2017-06-03 07:31编辑     15评论
SQUARE ENIX 预定在 2017 年 7 月 29 日发售的系列最新作《勇者斗恶龙 XI 寻觅逝去的时光（ドラゴンクエスト XI 过ぎ去りし时を求めて）》（PlayStation 4 / 3DS，以下略称 DQXI），在 2017 年 5 月 27 日、28 日两天于大坂举行了「勇者斗恶龙 XI 倒数嘉年华会」。只在日本全国五个都市举办的这项活动，由于主打来场的游客能够试玩本作，我们电玩媒体也尽量在有限的试玩时间内，呈现本作的试玩报导给各位。

特地为活动准备的试玩台有分 PlayStation 4 与任天堂 3DS 两种类，但不管是玩哪一个主机平台，都有两种试玩模式：从游戏开头试玩的「冒险的启程路线（冒険のはじまりコース）」、稍微有点游戏进度且以等级 16 的状态在「萨玛迪地区（サマディー地方）」周围自由试玩的「多样化路线（いろいろコース）」。游戏时间包含替换等大约十五分钟，不过本次大坂会场的限制时间内，打倒「多样化路线」魔王「死亡巨蝎（デスコピオン）」的瞬间，游戏就会结束。


两种难度与两种路线，都由玩家自己选择


笔者一开始选择的是 3DS 版，在 3DS 的上下画面可以体验到「3D 画面」与「2D 画面」上下连动，模式选择「冒险的启程路线」，并能玩到游戏开头的重要事件，十六岁的主角会跟着青梅竹马．艾玛与爱犬路奇（NPC）一起前往一处被称为「神之岩」的场所进行成人仪式。



这个游戏画面就是上、下画面各以 3D 画面与 2D 画面彼此连动。也就是说，如果使用类比钮就是操作上画面的 3D 角色，使用传统十字钮就是操作 2D 角色。而且，在使用十字钮的状态下按「A」键跟 NPC 说话，显示的讯息会出现在下画面，如果碰到敌人的话，也会以 2D 图示展开战斗。



左边是用类比钮操作、右边则是用十字钮操作的样子。跳出来的讯息会因为操作不同而更改位置

有趣的是，在 3D 画面因为视角动向而看不见的宝箱，如果经由 2D 画面的俯瞰视角就可以看见。善用上下画面的特徵能让游戏更好玩。

打倒史莱姆等初期的弱小魔物，两人终于到达神之岩，却突然遭到身影形似地狱兀鹫的鸟形魔物袭击。这裡的画面请务必实际体验，或是在活动现场的舞台等地方观览。事件过后回到村子试玩就结束了。


从上面的 3D 画面看不到的宝箱，透过下面的 2D 画面就能看到这点很有趣


使用十字钮操作时遭遇敌人，会采用 2D 图示战斗


和云雾怪战斗的主角们，普通攻击很难命中，但是用咒文战斗却很有效


只是初期的事件画面，上下画面也有连动


顺便一提，这样以 2D 画面与 3D 画面的连动只限游戏开头，之后如果想要切换两个模式玩的话，必须透过教会或是女神像等帮忙设定才行。选择 2D 模式的话，除了特别的事件画面，上画面也是以 2D 显示，战斗则是随机遇敌开始战斗。



3DS 的「多样化路线」也有稍微试玩一下，2D 显示画面完全就像是以前的《勇者斗恶龙》令人怀旧


在城下町的赛马画面，会自动转换成 3D 画面


接下来游玩 PS4 版时，选择了「多样化路线」玩玩看。在这裡的游玩模式，主角与卡穆、维若妮卡、赛妮雅四人小队加上一位来宾角色席尔比亚（シルビア）加入的状态下，在沙漠王国「萨玛迪城下町」及其周围进行冒险。然后，在冒险途中与法理斯王子（ファーリス王子）会合，一起打倒沙漠杀手以及恐怖的魔物．死亡巨蝎。

PS4 版全部以 3D 画面显示，角色模型也是和鸟山明的绘图，相近的尺寸大小。远处的背景一望无际，身在远方的魔物位置也能清楚看到。按下「□」键叫出地图等操作反应非常快速，整体给人的游戏印象相当良好。



萨玛迪地区画面以及地图，按下「OPTION」键就可以让角色自动跑步


战斗上，可以从系统设定画面选择「自由移动战斗（フリー移动バトル）」或「视角自动战斗（オートカメラバトル）」其中之一。选择前者的战斗，主角可以自由移动、视角也可以任意操作；选择后者的话，视角画面会配合敌我方的攻击，让画面角度充满张力。自由移动战斗的感觉就像《勇者斗恶龙 X》那样，视角自动战斗的感觉，则像是历代 3D 画面的《勇者斗恶龙》系列那样。两种战斗方式差满多的。


「自由移动战斗」主角可以自由移动，也能从任何角度攻击


「视角自动战斗」可以享受到历代的《勇者斗恶龙》那样爽快的战斗


顺便一提，该路线能体验到的不只有战斗，还可以触发各种事件与小游戏。在打倒死亡巨蝎之前，笔者也探索了不少令人在意的地方。

原野地图虽然是一望无际的沙漠，但如果敲响各地的钟，就可以骑乘马匹移动。骑马的时候冲刺（「R2」键）还可以将小型的魔物踢开，既是非常便利的坐骑，骑乘的感觉也很舒服，在萨玛迪城下町时，还可以享受赛马的乐趣。而赛马跟本篇相比，又是另外一种不同的爽快感。

本次的体验会在讨伐「死亡巨蝎」之后就结束的关係，想要玩玩看赛马的玩家，最好能在讨伐之前先往萨玛迪城下町去玩玩看。另外，赛马场在晚上的时候不会开放，这时候只要在宿屋或是露营帐篷度过一晚，到了早上再前往就好。


拿到宝箱了！整体画面连远方的物件都有显示出来的关係，就算是不小心漏掉的东西也能很快发现



遇到了金属史莱姆。不过这金属光泽也太金闪闪了吧。在宽广的地区移动时，相当便利的马。冲刺的时候可以将小型敌人踢开，但不是打倒。如果想要经验值，还是要乖乖停止冲刺，正常触碰敌人才行喔


3DS 版也有的赛马，按下「□」键，就会「飙速（马ドリフト）」炒热气氛


途中，在洞窟裡会遇到奇妙的生物。被称为「时光化身（时の化身）」的生物，身体呈现半透明，身影偶尔会消失，一会儿又忘我的出现在旁边散步。在舞台活动也有介绍到这谜样的存在，跟故事又有什么样的关係也令人在意。会现身于前往死亡巨蝎所在的「魔虫的居所（魔虫のすみか）」途中经过的洞窟，玩家不妨在附近探索看看。


偶然遇到的「时间化身」，但是无法触碰也无法对话


以法理斯王子的助手身分，和死亡巨蝎战斗的主角一行人。虽然是会使用全体攻击的强敌，但是睡眠术（ラリホー）对牠有效


PS4 版与 3DS 版虽然是同样的游戏、同样的故事，但是游戏感触完全不同，非常有趣。如果可以的话，实在很想两边都试玩看看，仔细比较两个版本不同的乐趣。
