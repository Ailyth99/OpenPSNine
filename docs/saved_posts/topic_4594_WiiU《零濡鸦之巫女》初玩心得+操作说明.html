简单说明一下操作吧

本次每章节的开头，有都故事说明，让玩家能先有个头绪，能马上进入状况

序章：是人物基本移动的说明 ( 操纵角色： 雏咲 深羽 )


第一章：是射影机/影见操作说明 ( 操纵角色：不来方夕莉 )


基本操作：
左类比人物移动
右类比视角转向
A键一般物品 捡拾 或 调查
按住ZR键捡拾，某些特殊物品会进行此命令，通常都是玩家心跳加速的时刻 XD（如果鬼手快抓到你手之前放开ZR键有机会躲开被抓！如果不幸被抓了快速抖动左右类比键能加速甩开鬼手少扣点血 Orz）

进行 「影见」的时候也需要按住 ZR 键 ，才能够看着白色灵体移动到何处 ( 跟着走就对啦 ww )
按住ZL键人物奔跑
单按ZL键视角重置(回到人物正后方视角)
轻按Pad的面板能开出地图(显示人物位置，右上角有目前的目标为何) 简单易懂 非常好
更新：如果不习惯点pad面板.....按 + 也可以叫出地图


这是pad上 没有 轻按前的画面


这是pad上 轻按后的画面，左上角显示所在的地区，人物位置是红点标示，右上角是目前的目的


射影机 操作：
X键举起/放下射影机
ZR键在 射影机举起后为拍照键
ZL键在射影机举起后为灵体锁定键，持续按着能锁定灵体！
Pad在拍照时能横的拿、直的拿、斜的拿只要你想照的目标物都能进入pad视角都成！！
左类比在 射影机 举起后 还是人物移动的 控制
右类比在 射影机 举起后 还是视角移动的 控制
十字左右为切换要用的除灵底片
十字上下为切换射影机的强化镜头
