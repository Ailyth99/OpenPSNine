「TGS2015」《圣剑传说 最终幻想外传》重制版将登录PSV/iOS/Android平台

hayato_607 2015-09-17 17:22编辑     21评论
SQUARE ENIX 本日（9 / 17）宣布，PS Vita / iOS / Android《圣剑传说 －Final Fantasy 外传－（圣剣伝説 －ファイナルファンタジー外伝－）》将于 “今年冬季” 释出下载，同时开设官方网站以及释出影片。关于贩售采取买断或是课金制等尚未明朗，有待官方进一步公布。



《圣剑传说 －Final Fantasy 外传－》是 1991 年于 Game Boy 平台推出《圣剑传说》系列第一款游戏的重制版，故事和内容将按照原游戏，画面、操作和 UI 将翻修为符合今日水准的内容。
