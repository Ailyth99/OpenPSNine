PS4《最后生还者 重制版》台北地下街抢先试玩心得

water1022 2014-07-24 15:50编辑     5评论
看了新闻《最后生还者 重制版》台北地下街今起抢先试玩 年度经典强化再登场

炎炎夏季假日，就跑去试玩了一下，分享一下心得了
地点是在台北地下街Y17出口，其实就是PS3首卖时体会送手电筒的地方

刚好抢到手电筒的名额，可惜是没拿到衣服，
PS4的衣服有比较好看（工作人员穿的），有加上艾莉跟大叔
不知道有没有机会拿到

加上这次体验会，感觉这游戏蛮照顾玩家
只是消息曝光度不高...（我也是今天看到新闻才跑过去玩）

18+所以试玩要出示证件！

现场试玩人潮断断续续
我是下午6点多试玩，机台也都没客满过



试玩机台 前后各一台 总共8台！
耳机可以试听，有临场感

GNN新闻是说：试玩的版本已经包含完整内容，可直接进入本篇或额外章节游玩。
我满心期待想去玩DLC的剧情（我是没玩过DLC，想玩玩一小段也好）
但事实记试玩只能选择两章节
序篇跟匹兹堡（匹兹堡就是巴士陷阱那裡）
难度自己挑（太久没玩我只选普通），忘了看是否有生还者模式
另外是有限时的，工作人员会有计时器计时
工昨人员现场都有在控管维持秩序！辛苦了
另外不知道我是太久没玩还是手把配置又不同
还蛮生疏的试玩，死了两次。

提示一下操控（看了对试玩比较有帮助唷！）
GNN新闻
　操作部分因应 PS4 DS4 控制器调整，举枪瞄准 / 开火从 L1 / R1 钮改为 L2 / R2 钮，冲刺 / 聆听模式则是调换到 L1 / R1，START 钮的功能在 PS4 上是以 OPTION 钮替代，SELECT 钮则是以触碰板下压取代，其馀操作没有变化，游戏中亦可让玩家更改操作配置。有部分音效会从 DS4 的喇叭传出，在危急的时候 DS4 前方的光条会闪烁警示的红光，不过游玩途中不容易注意到。
我补充的话
圈是找掩护跟跑出掩护
叉是爬过物品
三角是开柜子、拿/拣物品，敌人靠近可以抓住
方块是挣脱，敌人靠近用近战武器（这个有点不确定）

由于现场布置蓝光有照到我试玩的电视
我无法看得很清楚画质，又加上近距离，我眼睛烂
不过提升是有看的出来，也有比较流畅，让我蛮心动的
到底要不要冲PS4，好挣扎。
以上是我的心得。

夏日无聊又不想晒太阳，可以去试玩！

试玩设备宣传
除了试玩，我觉得似乎也在宣传耳机
所以有空朋友或对耳机有兴趣的可以去试试唷
7/20最后一天


现场也很有心的布置，不是贴贴纸，有用实物布置
很有心！
