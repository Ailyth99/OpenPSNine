MUL《轩辕剑外传 穹之扉》曝光更多家用主机游戏画面，计划推出多国语言版本

sai8808 2015-10-01 10:10编辑     57评论
大宇资讯于近日再次曝光《轩辕剑外传 穹之扉》家用主机（PS4/X1）版游戏画面，同时宣布游戏将登上 Steam 平台，并同步推出多语言版本。

《轩辕剑外传 穹之扉》为《轩辕剑》系列第十二部作品，结合历史背景与神话传说、讲述一段充满神秘与梦幻色彩的动人故事。本作 PC 版本已上市，开发团队 DOMO 小组正针对移植版本与多国语言版本进行各项调整工作。

官方表示：「主机版本方面除了对画面进行优化外，还调整部分系统内容、重制 UI 与操作、并会对剧情与文字的表现做相应的改动，让游戏更加的适用于主机的演绎与操作，让用户在不同的平台有不同的体验。」


主机版调整中部分 UI 展示


不停在调整完善的 UI 展示


让主机操作更加方便快捷


另外，开发团队为了能够使《轩辕剑外传 穹之扉》在主机版本中有优秀的表现，同时也对画面细节与部分镜头展示做了细緻的调整，期望让玩家在体验游戏的同时能够有更加强烈的代入感；剧情方面的运镜也同样也进行适当调整，让玩家能够更加融入到情景之中，体会到人物此时的感受。


主机版调整中画面展示


剧情动画运镜调整中，让玩家更容易代入游戏剧情


《轩辕剑外传 穹之扉》已登上 Steam Greenlight，官方计画多语版本将比主机版早登陆 Steam 平台。官方期望，能藉由这次多版本计画让《轩辕剑》走出国门、并让 DOMO 小组接收来自不同国家的建议，完善并突破自身让下一代《轩辕剑》能够更加优秀。
