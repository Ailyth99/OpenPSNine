《生化危机浣熊市行动》专家难度S+图文攻略 U.S.S.第1章

mechille 2016-04-14 21:03编辑     0评论
本文包含流程攻略要点和S+评价所需的Collect Data收集以及所有的7只浣熊收集，不含剧透
另外，只有专家难度下才能获得S+评价，S（S+）评价所需要点包括：
1.杀敌数（越多越好）
2.死亡次数（越少越好）
3.物品收集（Collect Data，越多越好）
4.通关时间（越少越好）
其他的比如摄像头、队友死亡次数、使用喷雾次数、收集浣熊个数等等一律无关评价
获得的Collect Data需要在笔记本处提交数据


Game Start


开始画面，按START进入游戏


然后选择Free Play单人模式开始游戏，以后的话可以在这里选择已经打过的关卡
R1是设置难度，这里笔者选择Professional专家模式
另外，游戏存档会自动保存


然后就是人物选择画面了
第一个SELECT CHARACTER是挑选角色
这里的角色第一个是自己控制的，后面三个是队友，只有自己控制的角色可以设置技能、主武器、副武器
通过积攒XP来解锁内容（按方块）


第二个PASSIVE ABILITIES是选择技能


第三个PRIMARY WEAPONS是选择主武器


第四个SECONDARY WEAPONS是选择副武器


选择完毕后按START确定，开始倒计时进入游戏

本关S+达成条件：
1、至少杀32个敌人
2、死亡次数最多只能有1次
3、道具收集至少12个
4、通关时间25分钟以内

推荐使用角色：Lupo
主动技能：Super Soldier（以下简称SS）

心得：Lupo大妈的Super Soldier技能非常逆天，增加攻击力不说，还能抗5次攻击。前期美军不必多说，Collect Data也是要事先记住位置。第一个难点是威廉将HUNK甩飞之后无法切换视角的地方，一直后退，等威廉靠近直接开技能即可，根本不需要什么扔雷打硬直了（Too Easy），第二次是威廉从墙里冲出来的三下攻击，同样开SS技能硬抗3下即可。大妈实在太逆天了啊啊啊！



开始的剧情结束后一行人乘坐电梯，这时可以用右摇杆调整视角了，先习惯一下视角操作吧


出了电梯左手边的柜台后面有一个Collect Data


走到尽头机柜后面拿到一个Collect Data


看到那个安布雷拉的标识没？它指示的就是你的目的地，以后配合小地图和这个LOGO就知道你要前往的地点了


紧接着就是一波遭遇战，注意上下两层都有敌人。L1是瞄准，L2是切换主副武器，R1是射击，R2是丢手雷


干掉这波敌人下来的电脑旁边拿到一个Collect Data


绿色药草是直接回复体力的，按x直接使用


枪械或者弹药上按x可以拾取


弹药箱可以补给弹药


然后跟着标识的指引走，顺便瞄一眼队友的枪法……枪枪上墙


清理完这个场景的所有敌人后，右手边的白色的台子上拿到Collect Data


进门触发剧情，HUNK依旧犀利


然后又是一波遭遇战


战斗结束，旁边的柜子上拿到Collect Data


继续前进，拉响警报的房间桌子上拿到Collect Data


房间的内部可以得到一只浣熊


通过过道进入研究室，左手边架子上得到Collect Data


桌子上也有一个Collect Data


角落的黑色机器上拿到一个Collect Data，调查桌子上的电脑将一路获得的数据提交


进入下一个场景，HUNK会拉着便当甲进入威廉博肯的房间，让众人在门口等着


这时你也别闲着，找个掩体躲上，马上又是一波遭遇战


清理完敌兵后，右手的箱子中间得到Collect Data，这个场景中还有两颗手雷和一个治疗喷雾可以收集一下


然后一行人乘电梯上去


上来之后右手边的箱子上拿到Collect Data


往前走了没几步进入剧情，剧情过后，先别急着进门。右转可以拿到一个Collect Data


跑到头还有一个Collect Data


跟着标记进门，一路猛跑，注意躲避路上的喷火，被威廉追赶的时候推荐用L3进行加速跑


进入下一个场景，剧情，HUNK被威廉打飞


此处一开始不能转视角，不要停一直往后走，威廉靠近了就开Super Soldier抗...


往后退一会儿视角自动切换，这时候不要回头一路加速跑到尽头开门
门开的比较慢，需要拖延一会儿时间


往前走遇到3个美军，清理完毕尽头的箱子上拿到Collect Data


旁边的机器与墙壁夹角里也有一个Collect Data


然后陆续出现两被感染的狗，需要提前预判，否则很难躲开


下一个场景用笔记本电脑提交数据，并取得一个治疗喷雾


出门又是一场美军遭遇战


然后威廉破墙而入，需要按提示躲避3次，这里也可以开Super Soldier扛过去，站起来后马上加速跑


尽头又是个有延迟的门，继续拖时间等门开


最后着火的场景一路往前跑，中间会强制被炸两次，跑到头进入剧情


第一章终了
