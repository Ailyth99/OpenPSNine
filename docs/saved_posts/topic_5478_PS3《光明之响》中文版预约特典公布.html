PS3《光明之响》中文版预约特典公布

acg_connarlei 2014-11-14 10:10编辑     17评论
产品名称：

Shining Resonance《光明之共鸣》繁体中文版
　　发售日期：2014年12月11日（四）发售预定
　　游戏语言：繁体中文字幕 / 日文语音
　　对应机种：PlayStationR3
　　预计发售地区：香港、台湾、东南亚
　　发行公司：Sega Corporation
　　销售公司：Sega Corporation
　　游戏类型：与龙共奏角色扮演
　　游戏人数：一人
　　著作权标示：SEGA


　Sega Corporation于今日正式发表，将于2014年12月11日发售的PS3独占游戏《光明之共鸣》（Shining Resonance）繁体中文版的预约特典。

　　《光明之共鸣》（Shining Resonance）的预约特典为“偶像服装：悠玛、雾香、桑妮雅”的Product Code。使用了此Product Code之后，悠玛、雾香、桑妮雅就能以乐队造型的服装在游戏中使用。

　　预约特典数量有限，敬请即早预约。

　　预约特典：

　　“偶像服装：悠玛、雾香、桑妮雅”
　　Product Code

　　※有关特典的附赠与否，请在预约时与各店铺确认。
　　※特典画像是示意图。
