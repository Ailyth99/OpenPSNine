前言

前言

之所以是目指攻略，是因为我自己目前也在冲击100%，内容会随着我的攻略进度增加。游戏虽然难，但是即便是手残，也一样可以完美这个游戏的，这也是这篇攻略的目的，也希望大家可以对自己有信心。
所有内容全部是自己攻略过程中的心得，希望在对大家攻关过程会有一定的帮助。

这是个各方面都跟洛克人十分相像的游戏，只是敌人和BOSS换成了各种萌物与妹子。注意如果更改卷轴移动等待时间的话，可能会造成掉帧，建议用默认设置。

游戏共有两个角色，用姬丝通关后解锁莉帕。两个角色能力上区别很大，除了奖杯要求外，不推荐使用莉帕，因为实在是太不好用了。纯晶石需要射击对应的点，然后才会掉出来，两个角色的纯晶矿位置有区别，暂时只收集了姬丝的，莉帕的迟点我会补上。所有BOSS的攻击在最高难度上都会发生数量上的变化，弹幕会比简单难度多，但是基本躲法没有变化。

我们先来看看奖杯要求
GAME MASTER
獲得所有PLAYING LEVEL項目。

这个实际上就是要完成游戏里内置的全部成就，对于BOSS战成就的描叙有点问题，实际上只需要用基本武器无伤干掉BOSS就可以了。但是必须在对应的关卡，最终关的BOSS RUSH里解不了成就。另外有个累积50次格挡的成就，这个在攻关过程中应该就可以拿到了。格挡的时机，游戏里并没有提及，好像是在子弹攻击到你的一瞬间按相反的方向键的样子。


接着说说推荐的攻略顺序，建议第一遍先简单难度，这一遍首先熟悉游戏的操作和场景配置及敌人的攻击方式。这一周目先不要消耗任何能源蝶强化能力，并且不要捡任何的纯色晶石，装填能力可以用。然后一直打到通关，此时会解锁晶石商店，并且拿到以下几个奖杯
HAPPY END
突破所有的關卡，看完故事結局。
NO SYEGA
在完全沒有獲得純色魔晶礦的狀態下突破所有關卡。
FIRST STATE
在完全沒有強化特殊技能的狀態下突破所有關卡。


然后读取通关存档，此时可以开始强化能力和收集纯晶矿（具体位置见每关的分页）。优先蝴蝶，然后是子弹和体力。等晶矿全部收集完毕之后，会获得最后的装填能力防护罩，这个可以抵消一次攻击。然后就可以开始挑战每关BOSS的无伤了，最后的莉帕注意是必须两阶段都无伤才可以拿到成就。全部能力升级完毕解锁成就后，就可以开新档开始二周目了。

第二遍，就要挑战最速了，开始前最好先在商店里购买一下移动速度增加和跳跃力增加，其余无所谓。另外，记得把装填能力设为完全手动操作。全程记得都不要装填SPEED以外的能力，强化不影响。保险的情况下，建议打完一关就上传一下存档，或者一关里耗费时间过多的情况下，直接撕页，千万不要在游戏里退出，会自动存档并且保存累积时间的。这周目完成可以拿到。
SPEED STAR
在４５分鐘內突破所有關卡。
NO SLOT
在完全沒有裝填「SPEED」以外能力的狀態下突破所有關卡。


此时的蓝晶矿数量应该不少了，然后就去赌梭哈吧，视自己的攻关能力决定要刷多少晶石。然后把商店里能买的都买了吧。

开三周目，这个时候就选择莉帕，难度选择LUNATIC，然后把所有特效都打开吧。坚持到通关后，读档并更改难度到EASY，然后去拿无伤解决姬丝的成就。
此时应该会获得

鉴于游戏里的附加效果需要大量的蓝晶矿，然而光靠完成成就来获得的数量非常有限，所以还是要靠梭哈来刷晶矿。
相比乱赌来说，我的方法相对来讲要效率一些。该方法的主要思路就是赌同花顺和三条加一对的葫芦来刷。
一般来讲牌型为以下几种：两种花色，这种情况同花色的最少有三张。这个时候推荐换两张花色不一样的，不管是不是对子都换掉。
三种花色，这种情况下，同种花色的牌最少有两张，有时候是同种花色的牌各两张，此时就换掉三张不一样的。如果有对子的情况下，可以考虑保留对子换牌回本。
同种花色就不解释了，不用换。
其他情况，两对的情况下换一张赌葫芦，三条的情况下换两张，四条的情况基本很难遇到，遇到了换不换也无所谓了。不伦何时都不推荐赌顺子，太难中，同花顺更难，除非你起手就摸到了。
这赌法最多一次不会换超过三张牌，而且不用太花时间思考，熟练了直接看花色换牌即可。特殊的牌型会有闪烁效果提示，只要手不是太快一般也不会错过。兼顾速度的同时，也能较为效率的提高中奖几率（信不信就随你了，反正我是乱讲的）
