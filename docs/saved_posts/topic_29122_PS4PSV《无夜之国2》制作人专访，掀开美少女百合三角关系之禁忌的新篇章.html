PS4PSV《无夜之国2》制作人专访，掀开美少女百合三角关系之禁忌的新篇章

sai8808 2016-09-20 15:01编辑     72评论
KOEI TECMO Games GUST 副品牌长兼 PS4 / PS Vita 动作角色扮演游戏《无夜之国2 新月的花嫁》制作人菊地啓介特别接受 GNN 等媒体的访问，分享这款「美少女与美少女和美少女互相吸引的 RPG」的特色、理念以及与前作的差别，供玩家参考。



菊地啓介是隶属原 TECMO 的创作者，曾经手过《零》、《影牢》等系列的制作，在 TECMO 与 KOEI 合併并收购 GUST 之后，则是担任 GUST 品牌的副品牌长。而且不只是单纯的管理职，还亲自出任《无夜之国》系列的制作人，而且是实际参与第一线游戏开发制作的制作人。


菊地啓介


谈到这次的新作《无夜之国2 新月的花嫁》的卖点，菊地啓介表示，之前《无夜之国》是以「美少女 x 美少女」为亮点而受到玩家欢迎，所以在这次的《无夜之国2》中，就决定把美少女从 2 人增加到 3 人！（他是认真的），彼此之间所构成的三角关系相信度玩家来说会更具吸引力。



关于可操作角色的部分，菊地啓介表示，《无夜之国2》跟前作一样只能操作主角一人，不过加入了可自由选择伙伴并肩作战的「Lily」系统，算是可以间接使用这些角色参与战斗。而且可以选择的伙伴不只有其他两名女主角，还有很多目前还没公布的角色，但都是女性（笑）。他曾经对制作团队下过一到指令，就是「专心制作女性角色就好，不用管男性角色」。虽然这样的指令感觉有点偏差，但前作问券调查有 93% 的玩家都是男性，所以这也只是顺应时势之举。此外，这次的从魔系统也有大幅强化，前作玩家基本上没办法控制从魔，但这次从魔系统的自由度比较高。



当被问到与前作的关系时，菊地表示，《无夜之国2》与前作是处于同一个世界，时间点则是在前作故事完结之后相隔很久的遥远未来，因此虽然存在与前作相关的要素，但并不是那么直接，玩家可能要游玩一阵子之后才会慢慢察觉。

至于自己是如何以一介男性的身分来掌握纤细的美少女情感表现一事，菊地笑着说，自己从来都不是一个纤细的人，对美少女细腻的情感也是一窍不通，所以这次特别找来两个帮手，分别是《影牢》系列的柴田诚担任编剧，知名动画制作公司 Production I.G 的藤咲淳一担任剧本监修。

对于这次制作平台从 PS3 / PS4 / PS Vita 转移到 PS4 / PS Vita 的具体改变时，菊地语气高昂地表示，的确这次平台转移到 PS4 上让游戏有很大的进展，因为前作基本上是以 PS3 为基准，PS4 版算是画质提升的版本，而 PS Vita 版则是画质降低的版本。这次则是完全以 PS4 为基准来制作，因此整体画质表现有显着提升。另外，先前 PS Vita 版采用的降低画质移植手法，显然效果并不好，玩家的评价也不高，因此这次制作团队有充分反省，不再以此方式制作，而是活用 KOEI TECMO Games 先前制作 PS Vita 专用游戏的经验，来打造《无夜之国2》的 PS Vita 版。



菊地总结表示，这次《无夜之国2》基于前作的反省与改进主要有 3 点。

1.剧情薄弱的问题，所以请来两位帮手协助剧本编写
2.耐玩性不足，所以这次加入可以自由搭配伙伴的「Lily」系统
3.动作性差强人意，主要是因为前作使用的是 GUST《炼金工房》系列的游戏引擎，并不是针对动作游戏打造的，这次《无夜之国2》在 ω-Force 团队的协助下，导入《无双》系列的游戏引擎来强化动作性


此外还有之前提到的 PS4 与 PS Vita 的画面品质全面翻新提升的部分。

至于除了美少女之外，还有什么特别希望玩家注意的特色一事，菊地斩钉截铁地表示，基本上在《无夜之国》这个系列中，美少女就是「天」、是大前提，其他包括画面、剧情、系统等，都只是为了支撑最高等级的美少女游戏所准备的基础建设而已，希望大家能细细品尝美少女的乐趣。



当被问到是否有打算将《无夜之国》系列搬上美少女题材相当受欢迎的 PS VR 虚拟实境时，菊地表示，他也知道很多玩家都希望在 PS VR 上体验被美少女包围、沉浸于温柔乡的感觉，不过目前《无夜之国》的动作玩法并不适合用来表达这样的感觉，应该是要打造一个能与美少女面对面私密互动的玩法比较适合，这部分还要多多斟酌。至于是否能在这次或下次实现这样的构想，目前还没办法确定，但就他个人的立场来说的确很想做。当然想要实现这个构想，就现实上是需要预算的支持，所以希望亚洲玩家能多多捧《无夜之国2》的场，让他有机会能构思与制作《无夜之国》系列的美少女 VR 游戏。

因为前作有推出中文版，对于这次的《无夜之国2》是否也会推出中文版一事，菊地表示，这部分制作团队会积极考虑，但目前还没有具体消息可以公布，之后如果有确定消息的话，会尽快让大家知道。
