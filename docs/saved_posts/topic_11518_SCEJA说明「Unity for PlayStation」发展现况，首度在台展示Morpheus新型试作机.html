SCEJA说明「Unity for PlayStation」发展现况，首度在台展示Morpheus新型试作机

mechille 2015-04-23 15:14编辑     15评论
索尼电脑娱乐日本亚洲（SCEJA）在4月22日举办的 Unity 开发者大会台北站「Unite 2015 Taipei」中，开设「Unity for PlayStation 的最新状况」讲座，介绍现阶段 PlayStation 平台对 Unity 的支援，以及分享在 PS Vita 上以 Unity 开发游戏时所可能会遭遇到的实务问题与注意要点。



此外还延续去年大受好评的做法，带来日前在「游戏开发者大会 2015（GDC 2015）」中首度展出的虚拟实境头戴显示器「Project Morpheus」新型试作机，是该装置首度在台湾公开亮相。



本次讲座同样由 SCEJA 出版商提携部开发商提携课主任刘明龙及 JA 技术方桉课主任林依弘负责，讲题主要分为 5 个部分，包括 1. 去年到今年之间的重点更新，2. Unity 4.3 for PlayStation Vita 的对策，3. Unity 5.0 for PlayStation 的对应状况，4. 上次的那个！以及最后的 5. 总结。


刘明龙（左）与林依弘（右）


去年到今年之间的重点更新

刘明龙首先简介从去年讲座到现在的几个重点更新，包括 PS4 全球累计销售从去年的 700 万台增加至今年的 2020 万台。Unity 使用权在 SCE 的努力之下改为免费提供。参与 PS 平台游戏开发的手续简化，一次签约即可适用日美欧亚地区，支援服务则是下放各地区分部直接提供。

另外，这一年来已经有 10 家台湾游戏厂商申请成为 SCE 授权开发商，大多数都申请了 Unity for PlayStation 进行评估与研发，今年内就会有游戏推出，由此可见 Unity 深受开发者的欢迎。









Unity 4.3 for PlayStation Vita 的对策

接着由林依弘说明 Unity 4.3 在 PS Vita 游戏开发上所必须注意的几个重点。

Unity 是目前最广为游戏开发者使用的跨平台游戏引擎，其最大的优势就是广泛支援各种平台的开发，包括 PS4、PS3 与 PS Vita 都在支援之列。当然智慧型手机的游戏也能移植到 PS Vita，不过这也带来一些问题，因为最新智慧型手机与 PS Vita 的硬体规格不同，因此并不是单纯把生成对象改成 PS Vita 就好，而必须考量硬体规格的限制，以免最终开发阶段遇到难以克服的障碍。






林依弘表示，智慧型手机会以很短的週期不断强化 CPU 与记忆体的规格，但是 PS Vita 自从 2011 年底推出以来，规格就是固定的。处理效能与记忆体容量的限制，如果没有再开发早期阶段就确认与处理的话，到开发末期就会成为引爆问题的大炸弹。而且 Unity 4.3 非常依赖 CPU 单核心的处理效能，而且包含巨大的抽象化程式层，这些都会对规格限制较大的 PS Vita 造成障碍。












林依弘强调，正因为 Unity 的便利性，使得许多开发者没有意识到跨平台移植所可能遭遇到的问题，没有好好地在 PS Vita 上实际进行测试就直接用 PC 之类更强大便利的平台直接製作，最后导致成果无法顺利运行。事实上近来有不少在开发末期必须重新检讨内容、打掉重做的桉例。








不过 SCE 方面同样努力针对这些问题提出改善方桉，像是在 PS Vita 3.50 版系统软体之后，游戏可用的记忆体容量比以前增加 30%，可减少动态物件生成、记忆体回收等处理的负担。





Unity 5.0 for PlayStation 的对应状况

接着介绍的是最新版 Unity 5.0 对 PlayStation 的支援，这部分由于关于 Unity 5.0 新功能介绍的讲座很多，因此未着墨于跨平台共通的改良，而是专注于针对 PlayStation 平台的改良。其中最重要的可以说是针对多执行绪处理的最佳化，摆脱 4.3 版受制于单核心效能的瓶颈，而能充分利用像是 PS4 这样拥有多核心 CPU 的硬体，改善游戏处理大量物件的表现，提供更宽阔的空间。












上次的那个！

由于去年讲座首度展示的 Project Morpheus 获得热烈的迴响，因此今年两人又再度带来新型的试作机，这也是新型试作机首度在台亮相。这次的新型试作机具体改良点包括改採 OLED 面板、更新率提升至 120Hz、提升光学定位准确度、降低延迟至 18ms 以下等（相关报导）。






这次的展示同样没有开放参加者体验，而是由 SCE 相关人员进行示范，示范的内容是日本 Unity 官方代言原创角色「Unity-Chan」的演唱会示范表演，透过 Project Morpheus 可以欣赏到 Unity-Chan 生动的劲歌热舞。新型试作机的 SDK 可将 60fps 绘製的画面以 120Hz 更新频率显示，提供延迟更短、更即时的虚拟实境操控回馈感受。想在 Unity 5.0 中支援新型 Project Morpheus 也非常简单，只需要安装最新的插件，就可以选择对应 Project Morpheus 的画面输出模式。








Project Morpheus（开发代号）预定 2016 年上半推出。


总结

最后刘明龙总结这次讲座的重点，包括加入 PlayStation 平台的门槛降低与使用 Unity 的门槛降低，今年将会有许多以 Unity 开发的 PlayStation 游戏会推出。PS Vita 系统更新开放更多记忆体空间给游戏使用，有助于提升游戏的表现。最后是鼓励大家多多尝试 Project Morpheus 的虚拟实境内容开发，挑战这个方兴未艾的游戏新领域，藉由尝试与失败来累积经验、朝崭新的未来迈进。
