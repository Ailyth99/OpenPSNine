前言

PS4这个魔界战记4R就是PSV的4R移植，我打到白金也没看到任何不一样的地方，所以按照我以前玩PSV版的经验打得非常顺利(就是刷道具和界贼船零件真是刷得我两眼昏花)。
要说PS4版的唯一好处，那就是有官中(听说PS3版也有官中)这一点了。
另外因为从PSV开始的R版就没了复制bug，所以老版的各类攻略里的复制大法都用不了了，现在纯灵只能耐心手刷，没什么好办法，基本上来说因为两个修罗杯都只要打死一次boss而不需要过关，所以要省时间打白金只需要准备一整套强力装备就行了，不用追求全主力豪华装备。

购买数字版的方法：游戏没上港服，但是台服有，直接台服账号+预付费卡走起

注意：本游戏对于非系列爱好者来说，白金可以说是一种折磨，所以千万想好再开坑，慎重慎重。
先放个日站R版攻略，做收集时注意对照
然后是英文版的非R版道具和零件收集攻略，取得方法更详细，但没有R版新增道具和零件

完美难度8 / 10，有两个全收集杯很累，白金整体难度远高于没有收集杯的1R和5
完美时间120+小时，这是最乐观估计，稍微不熟悉的人可能花费的时间远多于此
联网奖杯无
奖杯BUG无
最少通关次数1.5周目(全职业以及部分界贼船零件获取需要进入二周目)
容易错过的奖杯五个额外结局杯以及【你有看到我的私房錢嗎？】 后文会有详细说明

白金流程
1. 通关主线(以及看五个额外结局)进入后日谈
2. 练级和刷道具界，同时在实力够的情况下推进后日谈(包括新党地狱，三个外传)以及里关，并开始进行道具和界贼船零件收集
3. 进入修罗国，继续增强自己，强度足够后完成修罗杯
4. 进入二周目完成角色收集和界贼船零件收集

练级建议
建议进入后日谈后立即开始练级流程
1. 作弊屋强度加到8，去打5-2，系列设定99级EXP等于333级敌人，在盗贼身上装备3个鞋子让移动力到9，5-2一开始让盗贼往右将黄色面板往场直中丢，消灭这黄面板场上就变成exp+150%的黄色区块，然后AOE清场，拳系角色500级可学【魔拳大霹雳】，练功神技，在5-2练几个角色到等级500以上，道具界把武器增强到攻击五千上下，保证最强角色的攻击可以打下面连武山。
2. 作弊屋强度降回0，单一角色累积7000玛那后，议会出现「开始通往练武山的传送门」议题，剧情模式出现【锻炼练武山】，练武山1-2难度不高，3主要是敌人多也不能使用特殊招式，血会被敌人磨光，所以不能硬拼，要去道具界把武器和防具拉高能力及增加移动力，最强角色移动力8及防御值及攻击值有1.5万以上，准备好后3一开始就叠罗汉把我方送到绿面板所在地，下一个回合消灭绿面板，再来就变简单，注意用叠罗汉把最强角色往上能送多选就送多远，到达最顶消灭那只黄龙及黄面板，再清掉僵尸。开启练武山4，这才是练级关，敌人站得整整齐齐。
3. 想一击灭练武山4的敌人：作弊屋强度加到1以上，去10-6，一开始就有2只巨大化怪兽，派出死亡子及拉兹贝莉儿用发怒变成巨大化，打倒任1只巨大化怪兽，议会出现【想使用巨大武器研究所】议题，暴力通过后造一只猫又，放在邪恶标志【巨大武器研究所】格子，到练武山4先出场猫又，再派出死亡子用发怒变成巨大化猫又，再派出阿克塔雷或芬里希，巨大化猫又用魔变换指令，这时候【魔拳大霹雳】攻击范围变成5x5菱形，慢慢提高作弊屋强度保证可以一击灭杀，这样拳系角色都可以练到2000级以上。
4. 随着敌人强度调到8以上，练武山4的敌人等级超过1000后，用5x5的【魔拳大霹雳】杀敌会解开【想使用双刀流道场】议题，通过后把邪恶标志【巨大武器研究所】和【双刀流道场】都加给同一个角色，这样就可进行2次巨大化合体，给剑系角色装上普利尼和猪人族魔变换后用十字加3x3两个技能也能一下全灭练武山4的敌人，于是主角等剑系角色也可以练到2000级以上。
5. 培养出职业【教授】，第5阶有魔能力【教师执照】，可以全图+exp20%且可以叠加，在初期还没用加经验值纯灵的时候特别有用。教授出现条件:
a) 巫医=红魔法师(LV15)+红骷髅(LV15)
b) 下忍=小偷(LV30)+红骷髅(LV30)
c) 阴阳术研习生=巫医(LV30)+下忍(LV30)
d) 教授=阴阳术研习生(LV20)+治疗师(LV20)
6. 培养出职业【蒙面英雄】=小偷(LV25)+战士(LV25)，这职业到第5阶段有魔能力【懦夫之心】，移动力MV+3，同时培养天蛾人到第5阶段有魔能力【紧急离脱】，效果为HP25%以下移动力MV+3，这要装备防具石膏类如【命中石膏Z】、【速度石膏Z】、【命中石膏改】等等，让HP降为25%，学会后转生成蒙面英雄第5阶段，学会魔能力【懦夫之心】后再转生为天蛾人，移动能力就有MV+12，目前再加鞋子2个MV+14，然后找看看有没有一拿到就有MV+1的魔物武器如【狮心】、恶魔法33条【叛乱】等等，这样可到MV+15，这移动力对付中小型道具界图没问题，但稍大一点就不够，如想再加强移动力，可去角色界增加移动能力或道具界增加装备的移动力，要在道具界或是角色界快速离开比较大的地图层，没有守门人的情况下大约要MV+18以上。
7. 有守门人的状况，先培养出职业【红魔法师】到第4阶，做一只【星魔法师】，学会魔能力【术域扩增】，灿星术学到5阶【兆级灿星术】，学会后转生职业夜魔族【恩璞莎】，蝙蝠翅膀的美女角色，主要她会飞，练武山4里等级练高一点，再装备加移动力的鞋子，遇到守门人派她用距离远的灿星术加上魔法射程+1魔能力【术域扩增】，远远消灭守门人后再让上面移动力远的天蛾人一次到达门。
8. 职业【小偷】第5阶可学魔能力【幸运手指】，开宝箱rank+1，初期比较有用。
9. 跑跑道具界开启里关(见奖杯说明)，开启界贼船系统(第一次在道具界遇到界贼即开放)，以准备下面的进一步增强
10. 此时已经可以先去开启修罗国(见修罗国奖杯说明)方便之后的刷装备和收集

然后就可以开始抓纯灵，注意道具界只跑L(传说)级装备，普通和稀有的装备成长能力有限没有意义
1. 【催搬专家】纯灵: 可能会出现在【头号弓手的羽毛】、【恶魔戒指】等道具界，可以搬走还未服从的纯灵，所以可以利用来累积自己想要的那个纯灵，到一定数量去道具界把那个纯灵打死变成服从可以加快培养速度。
2. 抓重要纯灵: 目前比较重要的有ATK专家、HIT专家、重击专家、SPD专家、玛那增加专家、作媒专家、纯灵教师等纯灵，逆海贼会比较容易累积，同时抓逆海贼才有的纯灵如增加EXP专家、使剑专家、使拳专家、防具运用专家、操控魔物专家等等，将所有道具内的纯灵都移转与合体加起来。
3(可选). 作媒生出双重属性纯灵: 放【作媒专家】100可以在纯灵镇触发融合两个纯灵属性的事件，但效率很低且对白金流程来说也不是很重要，按自己需要决定是否要玩融合。

伴随抓纯灵的过程开始进行进一步的增强
1. 修罗国练武山4: 修罗国开启后，ATK纯灵撑够了确保能秒练武山4的敌人的情况下练级就很有效率了，装备上在保证攻击够的前提下插满【增加EXP专家】300，升级飞快。
2. 培养出9999级小偷: 这时候要再增强就是要有更高rank的武器与防具，所以首先培养出9999级小偷，把小偷放在邪恶记号【地狱教育场】，可以吸收10%经验值，修罗国练武山4练级的时候顺便小偷慢慢就到9999级了，高等级的宝箱HP很高，一般要让小偷装备高ATK纯灵才能一次击破宝箱，同时去角色界把小偷的魔能力【幸运手指】给主力角色学习到，这样就可以边打怪又打宝箱，当然要花点时间跑角色界。
3. 敌人SPD很高有可能我方命中率太低，这是就需要里关通关率20%的奖励【选举候选人缎带】(必中效果)，所以跑道具界把它练到至少有8个纯灵与移动力4，跑完90层就可以达成，后面打起来就会轻松许多，遇到高SPD敌人只要你的ATK够强用高伤害技能打中通常就能秒掉。
4. 找R39装备: 后日谈的部分关卡可以偷到R39和R40装备，进去打的时候先注意看一下boss身上的装备(用9999级的小偷从侧面或后面偷，建议偷稀有度0的顶级传说道具，因为如果身上装备的道具稀少度相同，会有额外30%能力加成)。另外这时手边已经有一些R34道具，去修罗道具界，用逆海贼找R39的传说道具，80层以上找绿色宝箱掉率高且宝箱多。防具就找秘宝【阿卡迪亚】，这个可以在后日谈的boss关卡偷，也可以R39的装备逆海贼刷，能不能刷到传说级别就完全看脸了。R39拿得七七八八后就可以开始偷剩下的R40道具。
5. 偷R40道具: 除了之前偷到的R40道具，剩下就必须走到R39道具的修罗道具界，偷100层的道具神，一个快捷的方法是在普通道具界过到99层用道具【脱逃装置】脱离后存档，然后切换到修罗国的道具界，SL到100层有要的R40装备(同样只要传说级)，R40道具有剑武器【巴尔剑】、拳武器【宇宙永恒拳】、魔物打击武器【巴尔手臂】、秘宝【结晶体】等这几个比较优先及重要的主力装备。
6. 完美升级rank40道具: 因为要许多等级天球，需要准备很多的脱逃装置凹等级(参见300级奖杯)，纯灵议会最早出现在10、20、30、40、50、70楼，最多共6个议题，一定要凹出来。首先要把传说等级的【结晶体】练到300级，通过3个ATK议题及3个增加MV移动力议题，再练武器，通过6个ATK议题。
