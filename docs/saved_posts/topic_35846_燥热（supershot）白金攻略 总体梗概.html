总体梗概

总体梗概
大家好，这里是lzy，第一次写游戏攻略，请大家多多包涵。
燥热是一款独特的FPS游戏，甚至可以说是一款战术策略游戏。游戏的核心玩法就是“我不动敌人也不动”，“我一动敌人一起动”的玩法，一开始还比较有趣，如果白金的话会感觉比较重复。看到p9很少人玩这款游戏，故写一下白金攻略做做安利。我这里的攻略是非vr版，请大家不要搞混了。

游戏名称燥热（SUPERHOT）
游戏类型FPS
白金难度7/10游戏的挑战比较紧张，很考验心理素质
白金时间6-15小时
最少周目一周目，但通关游戏后还有大量的挑战，大概相当于12✖25关，之后还有大量的无尽模式的关卡需要刷
注意事项此游戏港服独不占游戏虽没有中文但是对我这个英文盲而言都能看懂
白金流程梗概：通关游戏-补杂杯-完成所有挑战/时限挑战成功

基本流程杯

杂项杯

杂项杯
接下来是一些击杀项杂杯，可以在游戏的特定关卡内自己解开
Conserve Ammo
一发霰弹杀死三个敌人
游戏通关后可以选关，在第21关的办公室门口的拐角会遇到一个拿着霰弹枪的兵，干掉他后拿着他的武器等待着两个没有武器的兵和他们后面拿着小手枪的兵都在攻击范围内时射击即可
It's a me, Mario!
跳到一个敌人身上踩死它
按住L2跳到敌人身上将他们踩碎就能解开
Ratatatata!
用步枪在一次射击中杀死3人
在关卡13中打败开车的敌人拿到突击步枪后，对着巷子里的三个敌人开枪就行
Shishkebap
用武士刀投掷一次杀死两名敌人
摁住圆圈键朝两个靠近的敌人扔武士刀即可，武士刀即使被投掷也不会碎的
So it is possible!
用武士刀切过子弹
用武士刀切开离自己很近的子弹即可，就是这样做很容易死，得多试几次
收集等杂项
Charted
找到所有隐藏的终端
参考大佬发的视频攻略即可https://www.bilibili.com/video/av25230812，推荐通关再收集，因为这些收集藏的太隐蔽了而且过关后收集很快，不需要非得一周目收集，收集后退回主菜单即可，已收集的关卡后面会标记上cracked
Bored
启动系统中的每个应用程序
把游戏菜单内的子文件夹、exe、wir、art、txt、avi等都打开一边即可，全部点开后点击游戏菜单上方的credits.exe就能跳杯
Spread the system
录制一个回放
在打通一关后游戏会连续出现superhot的字样，此时连摁两下方块，游戏便会自动录制你的击杀视频并准备分享，在分享之前按返回键返回到游戏就能跳杯
Deep Web
成为社区的一部分
点进主菜单的apps-groups.exe，然后开这些人进行聊天，等15分钟即跳杯

高难度杯

高难度杯
Hercules
完成所有挑战
拿到此杯需要完成游戏中的所有共12个挑战，需要把前面的挑战打完才能解锁后面的挑战，最难的莫过于一命通关与速通关卡，接下来我会把每个关卡的规则和部分攻略都写下来共大家参考
KATANAONLY在此挑战内玩家只能携带武士刀（不能射击，殴打，投掷除了武士刀以外的东西）并获得更快的移动速度
AD2013此挑战内玩家不能携带近战武器或投掷杂物，但使用的拳头可以一击必杀敌人
GHOST玩家在前期就可以获得一拳必杀和附身的能力，但不能使用任何武器
FULLSTOP玩家使用的枪械都只有一发子弹，且敌人的子弹的速度会更快
THROWING你不能射击且拳头的伤害更低，可你投掷的每个物品都会一击必杀
BAREHANS只能用拳头攻击，不能使用任何武器或投掷任何东西
Stand REDy
以Red完成所有SPEEDRUN挑战
在你不动的时侯时间会过的很慢，所以尽可能别动吧；在一定时间内通过每关的限时挑战才能获得RED评分，视频攻略：https://www.bilibili.com/video/BV1MU4y147rQ
Get REDy
以RED完成所有SPEEDRUNrt（现实时间）挑战
无论你动不动时间都会被计算在内；在一定时间通过才能获得RED评分，视频攻略https://www.bilibili.com/video/BV1Kv4y1f7TP
HOTSWITCH最后的限时挑战，但能提前获得附身能力。由于此关不涉及速通类的奖杯，所以不用RED过关，从头到尾打完一遍就行了
HARDMODE时间过得更快，敌人受到的眩晕时间较短并且子弹速度较快，玩家的拳头伤害很低
IMOSSIBLE跟HRDMOED基本一样，只是玩家的枪都仅有一发子弹
NORESTART需要玩家一命通过全关，需要有较高的心理素质，视频攻略：https://www.bilibili.com/video/BV1qz4y1676r
FUNFUNFUN敌人的头部更大，玩家所有的攻击都能一击必杀，属于一个给玩家奖励的挑战，难度很低
Pumpkin Spiced SUPERHOT
在10.24到11.7间开启南瓜酱挑战
南瓜挑战不涉及全挑战之内，只要将日期改为10月24日与11月7日之内就能在挑战的菜单里发现，只要一进挑战就跳杯
接下来是无尽模式的奖杯
Full Package
解锁所有无尽关卡
无尽模式的关卡都是四个一组，玩家只需在每四个一组中最上面的地图中击杀共150人就能解锁下面的三个挑战，依次循环，一共需要打9个地图，每个地图杀150人即可
Can't get enough
在无尽模式杀死1987个敌人
解锁所有无尽模式后人数应该也不及1987人，建议在BRIDGE这张地图上的楼梯拐角用武士刀一直刷怪即可

一些技巧

一些技巧
1.游戏中按L2可以起跳，在主线和部分挑战中，按住L2不放手可以起到一个类似慢动作的效果，可以利用这个技巧轻松躲开敌人的子弹。另外游戏中的跳也是比走更快的，多利用跳能使速通更简单。
2.游戏中的不死挑战确实很难，但也没有难到一种完全打不过去的地步，当你打通过前几个挑战时你就已经对游戏中的关卡烂熟于心了。调整好心态，我当时打了好几次都打不过去，最后干干别的事情一会儿打很轻松就打过去了，不要害怕。
