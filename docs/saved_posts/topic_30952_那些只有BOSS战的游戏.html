那些只有BOSS战的游戏

deadkid414 2018-09-01 17:01编辑     38评论
游研社之前发过相同主题的文章——“那些奇奇怪怪只有Boss战的游戏”。
一些幕后故事写得还不错，但感觉列得并不够多。所以拖延症患者终于拖到现在搜索整理了一下“那些只有BOSS战的游戏”。其实大多都打不过，欢迎各位大触指正和补充。

本帖大部分内容来源于TV Tropes。这种类型的游戏，TV Tropes以“Boss Game”或“Boss Only Game”如此术语来定义，与只有杂兵无BOSS的游戏相对，Boss Game整体上是持续的BOSS战，BOSS战与BOSS战之间也许存在喘息，但并无过多的杂兵或无必要的陷阱干扰。另外两个术语提一下，Boss Rush通常出现在游戏流程末期，先前击败过的BOSS再度登场，有的情况是通关后才会开启Boss Rush模式；Boss-Only Level顾名思义则是单一关卡中仅有BOSS。

Boss Game可大致分为三种类型：

Type 1
专注于BOSS战的原创游戏。每场战斗间可能存在杂兵（或节奏上的暂停），但这些流程都很简单和短促。
这一类型也是最主要的Boss Game。

▼ 以下对TV Tropes的清单作了一定的筛选，因为那边选录标准略为宽松。
▼ “怪物猎人”系列可松散地看作是只有BOSS战的游戏，因为于猎人而言，最普通的怪物也是巨大的BOSS。
▼ 绝大部分的街机和格斗游戏，按剧情走也就是击败一个个对手。

年份 作品开发平台
1983 Forbidden Forest
禁忌森林		 Cosmi CorporationCommodore 64/Atari 8-bit
由Paul Norman设计的Forbidden Forest前瞻性地运用到视差滚动、动态血液、昼夜变换、氛围配乐，玩家操纵一位赏金猎人，以有限的箭袋在森林里从一个BOSS战辗转到另一个BOSS战。该作前后有过两部续作。
1984"Punch-Out!!" Series
“拳无虚发”系列		 Nintendo Arcade/Nintendo Entertainment System
以拳头代替子弹，每位拳击手都是有着独特模式的BOSS。
1988 Ka•Ge•Ki: Fists of Steel
火激		 Kaneko Arcade/Sega Genesis
街机版只有9个BOSS战，移植MD后加入了部分剧情和杂兵。
1990 Street Fighter 2010
街头霸王2010		 CAPCOM Nintendo Entertainment System
早期8-bit游戏在此类别下的代表。杂兵战可忽略直接跑酷到BOSS处。
1992 Death Duel
死亡决斗		 Razor SoftSega Genesis
目标为击败9个来自银河联盟的对决者。
1995 Alien Soldier
异型战士		 Treasure Sega Genesis
25个BOSS连战，BOSS战与BOSS战之间只有少量的杂兵。
1998 Radiant Silvergun
闪亮银枪		 Treasure Arcade/Sega Saturn
前期有少量杂兵作为间隔，但第五关之后BOSS战与BOSS战再无普通敌人作为过渡。
1999 Spawn: In the Demon's Hand CAPCOM Arcade/Dreamcast
改编自漫画角色Spawn的清版游戏，Boss Rush为主模式，但每一关依然有少数杂兵与BOSS为伴。
2003 rRootage ABA Games (Kenta Cho) PC
30关×5个BOSS×4种模式=600个BOSS战，该弹幕游戏后来被Lahzrog Games移植到iOS平台。
2005 Shadow of the Colossus
汪达与巨像		 SCE Japan Studio (Team Ico)PlayStation 2
16个BOSS，全部都是Puzzle Boss。该作称为独特游戏绝不过誉，在美学上的成就很高，可谓影响了一批游戏人。
2007 Banana Nababa LazraelPC
6个BOSS，免费的单屏平台游戏，玩家操作名为Harry Flowerpower的角色携带着无限的斧与矛，爬灯塔来击败邪恶巫师恢复“新非洲”大陆的和平。
2007 No More Heroes
“英雄不再”系列		 Grasshopper Manufacture Wii
严格说来并不是纯粹的Boss Only Game，存在一定的杂兵，2010年发行的续作No More Heroes 2: Desperate Struggle会更接近一点，除了15个Boss战之外的一切都可以跳过。系列第三作计划2018年登录Nintendo Switch。
2010 Malicious
恶意		 AlvionPlayStation 3/PlayStation Vita
自由选关直接进入BOSS区域，但除了BOSS外还包含附属杂兵。PSV版《恶意：重生》相较PS3原版新增6个BOSS，2017年2月于PS4平台推出的《恶意：堕落》重制版包含前两者的所有内容以及新的独家内容。
2010 Sky Serpents
天空蛇		 Nitrome 页游（Browser Game）- 游玩页面
15关14个BOSS，每条天空蛇都有数个弱点位置。
2015 Titan Souls
泰坦之魂		 Acid Nerve Microsoft Windows/OS X/PlayStation 4/PlayStation Vita
19个BOSS，大部分BOSS都是Puzzle Boss，主角与Boss皆一击毙命。
2015 Jotun
巨人约顿		 Thunder Lotus Games Microsoft Windows/OS X/Linux/PlayStation 4/Wii U/Xbox One
手绘美术风格是一大亮点。BOSS战之间存在少量杂兵和解谜流程，击败BOSS会获取新能力。
2016 Furi
 The Game Bakers Microsoft Windows/PlayStation 4/Xbox One
10个BOSS，弹幕武士的音游之旅，Furier难度下BOSS模式有所变化。
2017 Cuphead
茶杯头		 Studio MDHR Microsoft Windows/Xbox One
受上世纪30年代动画影响的美术风格和复古的游戏模式让本作备受瞩目，预计2017年中期发布。《茶杯头》的创作者Moldenhauer兄弟俩计划以30个BOSS冲击跑枪游戏类别（Run-and-Gun，也就是横版动作射击游戏）的BOSS连战吉尼斯纪录，目前该纪录由上文的Alien Soldier保持。



Type 2
整个游戏是对抗单一BOSS的持续战斗，BOSS会有阶段变化。这类游戏通常短小激烈，大多数为2D弹幕类型。

▼ 这个类型还是略有点偏离核心，更详细的陈列就不展开了。只裱一款。

年份 作品开发平台
2008 You Have To Burn The Rope Mazapan Browser Game - 游玩页面
从进场到打完BOSS只需十秒钟，任何介绍都会影响体验，请直接去玩。Henrik Nåmark创作的片尾曲《Now you're a hero》非常魔性，



Type 3
将一款普通游戏除BOSS战外的其余元素全部移除的特别版，与Boss Rush的体验类似。
