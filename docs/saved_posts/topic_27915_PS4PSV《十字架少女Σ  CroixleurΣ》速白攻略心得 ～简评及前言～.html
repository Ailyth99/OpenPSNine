～写在前面的话～
作为一款同人游戏，这款游戏并没有很多剧情，也没有很多复杂的要素，像鬼泣系列血宫一样一层一层的战斗就是这款游戏的全部。虽说如此，这款游戏的武器搭配、各种取消也是非常考验集中力和耐心的，流畅性和爽快感十足，而且有四只呆呆的萌妹子，各种呆立绘看起来特别可爱~推荐给时间不多喜欢无脑刷刷、喜欢动作类型和2次元呆萌妹的玩家食用

～速白流程～
The Vermilion Whirlwind
获得全部荣誉的证明。今天起你也是Croixleur大师了
本作并没有线性的故事，速白的流程实际就是拿奖杯的流程，难点主要在于各模式积分第一，所以提前收集好全武器和必要饰品，并搭配好武器技能对于速白至关重要。
个人推荐速白流程如下：
①打开option，将两个Assist都设为On，如图
别问为什么……我曾经没打开这两个选项玩了4个小时什么杯都没拿到，我承认我是手残……
②难度改为Easy，打通各人物的Story Mode（控制在10分钟内），并收集全武器（即红+黄毛每个房间都走遍即可），按照我下面所推荐的配置武器（如果有更顺手的亦可，只是个人推荐~）
③难度改为Normal，不放奥义不死人于10分钟内拿到所有Story Mode通关杯以及各个最速杯，并通掉红+黄毛的EXmode，最后不放奥义不死人拿下Ex杯
④完成除D模式以外其他各Mode得分第一杯（注意看右下角，如果有2p coop的选项点上，用另一只手or脚or基友一直按圈即可轻松拿到高分）
⑤刷钱，开满饰品用最喜欢的人物去打d模式最高分吧！至此白金。也可最后刷钱。

Complete Mastery
迷宫模式取得本地排名第一的证明。全部到达全部超越者的荣誉。
并不像楼上所说的那么难，重点：
①打开选项中的两个assist
②1p进行游戏（2个人的话副手死掉会变成鬼魂，本层无法复活也无法造成伤害，顺利打到45关没问题但是后5关因为伤害不足基本过不去）
③我选的红毛，能力相对平均的意思就是相对不容易死。武器配置见上；大范围奥义也很好用；
④不用太在意分数和连击，时间在40分钟（大概）之内就可以，用不了那么久，所以后几层拖npc复活就好了，吸引火力后猛k；
⑤优先打小怪，尤其招雷的；
⑥基本每5关一个boss，或大或小或群，留着奥义给这时候用。
⑦后期保险基本输出循环：攒到怪身边迅速1-4下后离开（因为有格挡判定），npc复活or有安全输出环境就狂配合
⑧几个相对重要的饰品：增加攻击次数的手套（陆地必备）、过一层回血的尖帽子、冲刺取消硬直的道具、增加方块格挡时间的眼镜、收集范围增大的耳朵？（收集金币也是得分点）、偶尔免疫伤害的眼镜，有喜欢的也可以自己配。

B：关於游戏标题「クロワルール (Croixleur)」
先把结论写在前面：
游戏标题中的「クロワルール (Croixleur)」是开发商(社团)souvenir circ.的自造词。不要考虑这标题的中文译名是甚麽，直接念原名就行了。
读法可以照着法语的读法 (qua/lur) 读，也可以照着英语的读法 (croi/lur) 读，也可以照着日语的读法 (kurowa/ru-ru) 读。爱咋读咋读。


一开始请教了Google老师「クロワルール (Croixleur)」的意思是甚麽，结果甚麽都没找到。

然後去问了百毒，找到了两个版本的译名：
「十字架少女」和「黑色规则」


「黑色规则」还好理解是怎麽翻译过来的：
(日) クロ = 黑 ; (日) ワ = (不翻译) ; (日) ルール = 规则


游戏内有”黑色”部分(EXTRA模式)，再联系一下根本没什麽存在感的剧情，
所以这个翻译好像没什麽错。

「十字架少女」的话应该是这麽得来的：
(法) Croix = 十字架 ; (法) leur = (代词，代「他/她/它们」)


然後咱回想了一下这游戏根本没什麽存在感的主线流程，
再在游戏中翻了翻武器，好像和十字架没什麽关系啊…
再加上这是个日本产游戏= =

但只要把标题从法文翻成英文就讲得通了。

即：
(法) Croix = (英) cross = 十字架 / 交错 ; (法) leur = (英) them = (代词，代「他/她/它们」)


合在一起，即：
cross (交错) them (她们)


再经中二式的脑补加工一下：
(命运/时空)交错(的)少女们
