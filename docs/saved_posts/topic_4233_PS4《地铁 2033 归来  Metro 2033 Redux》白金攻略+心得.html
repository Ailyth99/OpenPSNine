Metro Redux（Metro 2033 + Metro Last Light）这个特辑包含两套游戏，奖杯也是分开的，有爱的关系达成双白金了
奖杯入手难度不高，有兴趣的朋友可挑战看看

这里是last light redux的白金攻略



Metro 2033 奖杯攻略
所有奖杯需要的数目(杀敌数，开保险箱的次数等等)都可累计，通关后可任意选择章节
但如果开始了新游戏(NEW GAME)，所有数据会重置(包括收集物、敌人杀死次数、Weaponsmith使用过的武器等等)，开始新游戏时请考虑清楚
要白金最少要玩故事模式两次 (Survivor / Spartan + 两个结局)
可先解决存1000块钱的奖杯Scorge，因为其余奖杯都可以在通关后解决，唯独这奖杯比较难挽救

Survivor 2033
以幸存者模式完成游戏
Spartan 2033
以斯巴达人模式完成游戏
Spartan Mode 通关后解开
