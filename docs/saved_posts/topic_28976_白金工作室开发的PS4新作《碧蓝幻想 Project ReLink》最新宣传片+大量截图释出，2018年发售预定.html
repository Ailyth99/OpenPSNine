白金工作室开发的PS4新作《碧蓝幻想 Project Re:Link》最新宣传片+大量截图释出，2018年发售预定

sai8808 2016-09-13 22:15编辑     31评论


Cygames 在今（13）日下午举办的「2016 PlayStation Press Conference in Japan」中宣布，将于 2018 年推出。由《猎天使魔女》系列的 Platinum Games 团队担任开发，改编自营运中的手机游戏的PS4新作《碧蓝幻想 Project Re:Link（Granblue Fantasy Project Re:link）》。



根据发表会现场公布的开发版本展示影片，《碧蓝幻想 Project Re:Link》将承袭《碧蓝幻想》原作的世界观与角色，类型为动作角色扮演游戏，并预定支援 PlayStation VR 虚拟实境呈现。
