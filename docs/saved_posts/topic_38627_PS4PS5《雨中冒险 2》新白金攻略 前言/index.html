前言

前言
开坑前：本游戏虽然类属肉鸽游戏，但是可以通过局内解密，解锁局外神器，即常驻buff。比如统率神器（随机掉落改成互动自选道具）可以让BD构筑变得非常简单。
本游戏组队游戏可以非常快乐，前提是你队友不跟你抢道具

估计奖杯难度3/10，(某次更新后键鼠直连似乎不支持了)(不使用开局神器：8/10)
白金的大致时间8-20小时
离线奖杯白1 金2 银22 铜13 | 总38
在线奖杯0
可错过的奖杯数量0
难度杯
决定
在季风难度下击败一名精英头目。
PS4/PS5秒白 支持，但是4版后期掉帧严重。秒白我暂时不确定，有人反映被锁白金档了

本游戏作为肉鸽还是非常有趣耐玩的，有着丰富的搭配还有隐藏关卡，但是优化确实一般。
既然作为白金攻略，咱们就奔着最功利白金的打法路线：跑图→解锁几个好用的神器→开始白金流程。

新手教程

新手教程



角色推荐：
第一次游玩推荐使用女猎人，优势包括：
✓ 基础攻击带跟踪效果
✓ 拥有远距离位移和高跳技能，操作灵活
✓ 新手友好，不容易暴毙

缺点：前期伤害稍显不足

难度选择：建议从最简单的细雨模式开始


角色选择提示：
界面左边的装运按钮里可以改变技能形态，部分简单的达成目标可以留意解锁




游戏目标：
每局的核心目标是找到图中的Boss召唤器：
1. 找到并激活召唤器
2. 击败召唤的Boss
3. 占领该区域后通往下一关

六关后或者解锁神器后会跳回第一关并解锁：
既视感？
跳回第一关。

发育阶段：
主要通过收集道具增强实力：
获取方式：
- 开启宝箱
- 激活辅助无人机/炮台
- 击败每关最终Boss也会掉落一个道具，有时候会是boss专属道具

重要提示：游戏难度会随时间动态提升，请尽快增强实力


白金思路：
快速推进到第五关解密解锁以下两个神器：
1. 开局选用统率神器：可自选道具
2. 开局选用牺牲神器：地图不生成箱子，但怪物有概率掉落道具

(具体神器解锁方法请参考后续神器解锁页面)
