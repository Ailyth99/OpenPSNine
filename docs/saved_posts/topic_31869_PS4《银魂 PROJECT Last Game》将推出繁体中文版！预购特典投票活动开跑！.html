PS4《银魂 PROJECT Last Game》将推出繁体中文版！预购特典投票活动开跑！

sai8808 2017-07-17 11:04编辑     62评论
台湾万代南梦宫娱乐今日（7.17）宣布，将推出PS4动作游戏「银魂 PROJECT Last Game」（正式作品名称未定）繁体中文版。







中文宣传片


「银魂」最新游戏作品情报公开！

久违四年的「银魂」最新游戏作品「银魂 PROJECT Last Game」将登陆PS4平台，为「银魂」系列首次推出的正宗动作游戏。

为了纪念久违四年的再次游戏化，本作的预购特典将由玩家公开投票决定！


【特典内容】

「不知道会长怎样」的登场角色下载代码。

（待投票结束后，将再以胜出的角色好好计画）


【投票规则】

为期四周的生存淘汰赛，每周将由两名登场角色一决胜负，当周获得较高票数的角色将进入到下一周继续选拔，

最后将于官网公布人气第一名的角色并作为「银魂 PROJECT Last Game」的预购特典。

※每人每日限投票一次。


【第一周参赛者】





【投票期间】

第一周：7月15日（六）至7月20日（四）

第二周：7月21日（五）至7月27日（四）

第三周：7月28日（五）至8月3日（四）

第四周：8月4日（五）至8月16日（三）

※最后一周投票时间加码延长。

From.万代南梦宫厂商专稿
