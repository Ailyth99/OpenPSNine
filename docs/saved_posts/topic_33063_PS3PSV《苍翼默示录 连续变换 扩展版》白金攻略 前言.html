白金难度4/10
有无联机杯有7个（其中3个组队杯至少需要3人）
白金时间20小时以上视RP而定（同步存档另一版本10小时之内即可白金）

萌新第一次写白金攻略，如果有不足的地方，请指出来，我及时改正
首先我也是格斗萌新，所以玩的时候能用帅气模式的时候都用了帅气模式，本攻略也是面向萌新的
最后，如果2个版本都想白金的话，建议先白PSV版，因为后期刷ABYSS模式的时候PS3版重启游戏比较慢
注意:创建存档的时候要联网才有cross save,cross save在option里选第一项game option里可以找到
因为游戏比较老，不少奖杯下都有大佬写的TIPS，所以这里会引用一些
===============================================================================================
首先，说明一些本攻略要用到的格斗术语，方向键和数字对应的关系如下
789 ↖↑↗
456 ← →
123 ↙↓↘
普通模式下A B C D分别对应，举个例子236A=，5B=
帅气模式下N（普通钮）H（重击钮）S（特别钮）H（投技钮）也分别对应
HG=能量槽，DD=超必杀技，需要消耗50HG,AH=一击必杀技，消耗100HG且对手血量低于35%(此时人物头像会亮)
RC=快速取消（普通模式ABC一起按，帅气模式不可用）
普通模式使用DD技和AH技可以参照招式表，帅气模式长按N（普通钮）或者H（重击钮）是DD技，2个一起长按是AH技

那么正式开始
===============================================================================================
联机杯至少需要三个人，只要能找到人就没有难度，具体看第二页

设置好之后先玩TUTORIAL教学模式，打完前三个就跳杯了，战术指南不用打

隐藏BOSS出现的条件是不接关,多用DD终结对手,或用AH技终结对手一次（诺艾尔，哈扎马和μ-12无法见到最终BOSS）
这里注意按R挑衅可以加满HG（每局仅可用一次）
这里推荐先用帅气模式的帝卡打,接起手再随便打打，挑衅一次最后用DD技或者AH技终结用起来简单粗暴
