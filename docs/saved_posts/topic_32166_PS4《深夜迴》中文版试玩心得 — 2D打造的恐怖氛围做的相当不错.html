PS4《深夜迴》中文版试玩心得 — 2D打造的恐怖氛围做的相当不错

sora0612 2017-08-24 18:17编辑     31评论
简单介绍



《夜迴》为2015年10月 Nippon Ichi Software 在 PSV 平台上开发的惊悚游戏，故事叙述年幼的少女为了寻找失踪的爱犬跟姊姊而不断徘徊在深夜的街道上。

虽然人物是用着 Q 版可爱的二头身画风，但因为搭配黑夜寂静的吓人演出、可怕妖怪的紧追不舍，令当时游玩该作的玩家都回想起了「黑夜」的恐怖，获得了超出日本一想像的优良评价。

为了让玩家更直接地感受这部黑夜惊悚探险剧，日本一推出了正统续作《深夜迴》，让玩家可以在大萤幕上享受令人「暂时停止呼吸」般的深夜冒险。



由于《深夜迴》是以「唯」跟「遥」两个女孩为主角的故事，因此本作的游戏封面自然就由两人担任，看着两人娇小的背影被庞大的黑暗街道所包围，实在很难想像她们手中那只有薄弱光芒的手电筒能为她们带来希望。

封底并没有明白叙述游戏方式跟类型，反而用鬼故事的语调在介绍着游戏，对于惊悚游戏来说，这样的叙述方式反而更能传达出游戏的有趣之处。虽然一般版的盒中只有游戏片，说明书是采电子说明书，但事实上基本的操作在游戏一开始就会教导，当然，建议玩家最好做点「心理准备」面对序章的教学。





游戏心得

整个游戏的基本操作概念就是开关手电筒、走路或跑步、调查或搬动可疑的物件、躲避有威胁性的妖怪直到顺利推进故事为止。

操作虽单纯，但每一个动作都具有关键性。像手电筒的有无会影响玩家探索道具跟发现妖怪的难易度，没有手电筒，就算操作角色头部出现「？」讯息表示周围有东西可以调查，也很难发现到底该调查什么，几乎是瞎子摸象状态；更不用说在听见妖怪逼近的心跳声时，没手电筒的光芒有时还无法判断妖怪的位置，可以说是保命的最重要道具。



这游戏最可怕的地方就在于一片寂静、周围昏暗，几乎只能看到手电筒的光跟听见自己脚步声的前题下，各种充满杀意的妖怪常常会冷不防地突然出现惊吓玩家。

有些是会照一定路径行走，只要没注意到操作角色就没事，但有些却会在走到特定地点时突然就从周围冒出来追杀。

这些妖怪绝大部份都是玩家角色只要接触到就是「一击必死」，也难怪系统会要求玩家聚精会神地看着萤幕，因为一个分心，就有可能来不及反应妖怪的突袭而惨死。






毕竟玩家操作的两名角色都是软弱无力的小女孩，在遇到妖怪时，主要的对应方法也就只有逃跟躲。不少妖怪在移动速度上其实都比小女孩们慢，只要拼命跑多半都可以甩掉，只不过小女孩是有体力的，没有办法一直「跑」，这时候通常有三种解决手段：
1. 一个是运用妖怪的习性，例如用手电筒照一下怕光的婴灵争取时间、丢石头制造声音吸引妖怪的注意。
2. 找遮蔽物躲藏，直到妖怪们离开，不过这对会在特定范围徘徊的妖怪没用
3. 寻找正确解法，因为有些妖怪的袭击就像所谓的「头目战」，必须使用特定的方法、调查关键的道具才能让小女孩捡回一条命。





整个游戏流程就是重复探索黑夜的区域，接着触发剧情，切换操作角色，事实上因为每一段剧情的时间并没有特别长，所以大部份的游戏时间都是在探索，乍看之下会觉得只是在一片黑暗的背景中走来走去，好像有些闷，但其实跟前作相比，因为可以探索的区域不仅限于城镇，所以通常在切换角色时，就要往一个全新的区域冒险，探索部份比前作多了一些新鲜感。而且妖怪的杀意也比前作高出不少，可能走几个转角就会遇到妖怪冒出来吓人，令玩家每一段路可能都要疑神疑鬼、小心翼翼，根本没时间觉得闷。




从《夜迴》开始，这个作品虽然有个乍看之下很明确的主轴，但整个故事的真相却是完全不明朗，而是在游戏中许多地点和桥段中安排着各式各样的线索让玩家自己去推敲真相，就某种意义来说，还含有一点「惊悚推理」的要素。

当然这作也相同，唯跟遥为什么会受到妖怪的袭击？突然出现，如同引路人的存在是什么？最一开始的冲击性展开到底是在哪个时间轴发生的？真正失踪的人是谁？已经不存在的人又是谁？当玩家一路路收集线索去考察真相时，最后得到的真相会是个哀伤的解答？还是令人不寒而慄的事实？这个引人去考究、探查故事的特色也可以说是继承自《夜迴》，《深夜迴》拥有的最大魅力。




和《夜迴》相比，《深夜迴》另一个强化的要素就是增加了不少收集要素。前面也提到这个系列的游戏方式就是 2D 探索，在四周黑暗的各区域閒晃寻找正确的目的地。只是有些时候多少要往下一个目的地的提示不够明确，甚至可能是要到达某些地点才能获得很不明确的提示，容易形成无意义的逛街，再加上妖怪的不断袭击，很容易削减游玩的耐心。

为了避免这种状况太容易发生，本作增加了不少支线剧情跟特殊道具让玩家收集，例如不明人物留下的笔记、道具，这些凑一凑可能就会形成另一桩可怕的故事；同时玩家角色也可以收集护身符，每一种护身符都有独特的效力有助于玩家的探索，只是要注意一次只能带一个护身符出门。就像这样，探索中发现的亮点、垃圾不一定只会是存档用的硬币或小石头，还可能有一些意想不到的物品增添这段恐怖故事的趣味性。





总结

总结来说，究竟《深夜迴》的音乐和光影表现如何呢？这部份我给予较高的评价，这并不是因为这作有着多么动听的音乐、多么华丽的图画，而是出于这作尽可能地活用 2D 风格、黑暗、吓人演出让玩家获得惊吓的刺激感。乍看之下可能会觉得画风这么可爱的作品怎么可能会觉得恐怖呢？但在这个先入为主的观念下，就很容易被吓人的演出给吓出冷汗，可别忘记，这两个令人觉得可爱，手无缚鸡之力的女孩就是玩家要操作来面对生命威胁的角色。



音乐部份为了配合深夜，所以大部份都是一片寂静，玩家通常只会听到操作角色的脚步声。所谓「安静就是恐怖的调味料」，在一片安静的黑暗中，走到一半突然听到妖怪走动的声音，而且作为妖怪接近警告的心跳声也越跳越快时，再怎么感受不到恐惧，也会因为这些声音演出而感到紧张、慌乱，更不用说被妖怪发现时所响起的充满压迫感的曲子，让这游戏的 Q 版可爱风格形成一种落差，压迫着玩家的神经。




一款好的恐怖游戏并不是要做得让人不敢玩下去，而是就算玩家当下不觉得恐怖，还是会在关键时刻被吓出一身冷汗、被吓到手忙脚乱，但又沉浸在依然肯自动鼓起勇气继续游玩的气氛中。就这点来说，《深夜迴》的气氛拿捏可说是非常恰到好处，利用冲击性的开头让玩家想快点玩完探究真相、想快点让两位小女孩再会，但又必须小心翼翼、擦着冷汗地面对突然的惊吓跟杀意，将「夜晚」这个题材发挥地相当好。

再加上这年头的惊悚游戏多半为 3D，以 2D 风格为主的惊悚游戏已算相当少见，对于已经习惯 3D 作品逼真吓人风格的玩家来说，《深夜迴》的 2D 表现会带来一种新鲜的刺激感。
