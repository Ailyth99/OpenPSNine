PS4PS3《如龙 0 誓言的场所》公布调度资金的新系统「黑BANG经营手法」

sai8808 2014-11-29 10:32编辑     3评论
SEGA 于近日公布，将于 2015年3月12日预定发售的系列最新作品《如龙 0 誓言的场所（龙が如く0 誓いの场所）》（PlayStation 4 / PlayStation 3）的最新情报。

在游玩本作品时最不可或缺的就是赚取「金钱」，这次将公布首次于系列中登场的新系统「黑BANG经营手法（シノギ）」。介绍为其中一位主角・桐生一马的「黑BANG经营手法」的「神室町金钱乐园（神室町マネーアイランド）」。

「神室町金钱乐园」是以神室町为舞台的模拟游戏，藉由购买房产或是进行投资来赚取资金，以让区域内的佔有率获得提升为目的。区域佔有率上升后将会故事将会所有进展，并与充满个性的对手们展开火热的战斗。将作为对手的五位亿万富翁「神室町五富豪（神室町ファイブビリオネア）」打倒，就是桐生一马被赋予的使命。为了达成这个目的需进行人才的录用及配置等工作，系统可说是相当複杂。也请玩家们从下列的报导中来确认系统的详情吧。



累计销售数量已突破 650 万片的《如龙》系列。为系列最新作品的 PlayStation 4 / PlayStation 3《如龙 0 誓言的场所》，公布了为了获得于本作品内最为重要的“金钱”，新系统「黑BANG经营手法」的相关情报。身为主角的其中之人・桐生一马的「黑BANG经营手法」为「神室町金钱乐园」。挑战不动产业并打倒盘据于各区域的不动产王们，目标为一获千金。

本作品是以 1988 年的日本作为舞台，在系列史上、最为激动且刺激的街道裡，所涌现出前所未见的繁荣，请玩家们务必注意新系统「黑BANG经营手法」的内容喔。

《如龙 0 誓言的场所》「神室町金钱乐园」内容解说

＜设定＞
以某个麻烦事件为开端，被命令得经营不动产公司的桐生。以被称为「神室町五富豪」的五位宿敌作为对手，在环绕于神室町内的高额房产的金钱游戏中获胜，获取霸权及庞大的财富。

＜游戏概要＞
「神室町金钱乐园」是以神室町为舞台的模拟游戏。由「神室町五富豪」的五人各自支配着的神室町内的五个区域。在那里收购人群聚集的饮食店或风俗店等房产，将收购来的物件进行投资使其成长，持续于各区域内提升自己的佔有率。在提升区域佔有率后等情况下将使故事向前推进，将展开与对手们之间的火热战斗及十分扎实的剧情内容。

● 01＿专用选单
神室町金钱乐园的专用选单画面。可个别进行五个区域的攻略。



● 02＿房产管理
房产管理画面，可进行已获得房产的管理及投资。



＜赚取金钱的方式＞
从已获得的房产裡，将定期汇入获利金额。是个如能增加房产使之成长，将可获得众多收益的系统。如能取得所有的房产并将其培育为最高等级，便可预见到能够经常获得高额的定期收益。

● 03＿利益回收
增加房产使其成长后能够获得巨大的财富。



● 04＿开始回收
作好人才的配置等准备工作后就可以回收金钱了。



＜人材的获得与配置＞
在神室町裡所必需获得的不仅仅只有金钱而已。藉由确保优秀的「人才」，来赚取高额的收益。在金钱乐园裡存在着「经纪人（マネージャー）」「保安（ガードマン）」「顾问（アドバイザー）」等三种职业的人才，可藉由击倒宿敌或是经由通过支线剧情来持续增加。在街道上彻底来回寻找，广招最强的伙伴加入，奠定作为根基的经营阵容。

「经纪人」
在各区域裡进行着资产管理的人才。从来自对手的收购攻势裡守护物件，并能对区域内的景气带来影响。

「保安」
守护各区域的人才。将使发生后会暂时停止收益的「纷争」事件变的难以发生。另外，还有着在治安状态变的良好之后，会使区域内的敌人弱化等优点存在。

「顾问」
为了让获得的房产得以成长・发展所需的人才。支付顾问费后将进行关于房产的建议，并使房产持续成长。

● 05＿支线人才确保
在支线剧情裡所获得的人才相当的优秀。尽早搜寻街道来使他们成为伙伴吧。



● 06＿人才配置
人才可于专用选单裡进行配置。需注意的是如果不勤于更换人才，将无法发挥出实力。



＜房产的收购＞
房产的购买・收购将于街道上的冒险部分裡进行。房产有着由从便宜至昂贵为止等各种不同的价格，其中也有着无法用金钱购买，必须与加深与店长间的感情后才能获得的房产。

● 07＿购买房产
房产有着各式各样的价格。其中也有着高达数亿元的高级房产！



● 08＿支付热血动作
取出大笔金钱的桐生。这似乎也是《人中之龙》系列中，花费最高金额的购物。

＜对手＆战斗＞
在神室町金钱乐园裡以桐生的对手身分阻挡于面前的，是被称为「神室町五富豪」的五位亿万富翁。他们利用神室町作为自己赚钱的道具，并使居民持续痛苦着，在与他们的战斗裡将其制伏，并将神室町从他们的手中夺回。

★ 娱乐王
藉由经营柏青哥而成功的亿万富翁。

★ 电脑王
藉由投资开发半导体及电子机器产业而成功的亿万富翁。

★ 其他的富豪们
「风俗王」「赌博王」「综艺王」等具有个性的实力派们，将等待着桐生的前来。究竟桐生是否能在与他们的金钱游戏中获得胜利呢。

● 09＿娱乐王
最初与之战斗的娱乐王。经营手段虽然不太高明但使用了卑劣的手段使桐生陷入苦战。



● 10＿赌博王
经营地下赌场而获得成功，原本为黑道份子的亿万富翁。拥有众多前景看好的房产。



● 11＿击倒对手
击倒对手后区域将会变为自己的地盘。由于能够学习的技巧也将一口气增加之故，目标是将桐生培育为最强。
