「2016 ChinaJoy」2D格斗厂Arc System Works确认再次参展

watarasejyun 2016-07-14 18:27编辑     18评论
继「2015 ChinaJoy」，今日日本知名2D格斗游戏开发公司Arc System Works确认2016年仍将会参展，今后继续想中国市场提供许可以及取得面向日本的作品许可。



本次「2016 ChinaJoy」Arc System Works的参展作品包含《罪恶装备》以及《苍翼默示录》两大作品的最新作等公司内的作品。
