《生化危机浣熊市行动》专家难度S+图文攻略 U.S.S.第5章

mechille 2015-07-17 12:35编辑     2评论
S+达成条件：
1、至少杀150个敌人
2、死亡次数最多只能有1次
3、道具收集至少17个
4、通关时间30分钟以内

推荐使用角色：Lupo
主动技能：Super Soldier 对下一次攻击免疫，同时提高射击精准度和威力（LV3）

心得：本关非常难，很考验人品。要想单人P难度拿S+需要一点耐力和运气。时间一点都不富裕，杀敌数要求也比较高。前期能看到的丧尸全杀掉，第一个难点是在车库遇到茫茫多的舔食者，这里推荐远距离用Samurai Edge点射。然后开门后绕到另一边，美军不杀。一路开SS跑到下一场景。猎杀者能杀就杀一个，如果HP不多了就放放风筝，保证不死。过一会儿丧尸从栅栏冲出来之后一路跑过去到下一个场景。这个场景的第一波美军可以考虑干掉，然后无视猎杀者一路跑到标记点，会冲出来一波美军，无视他们直接开SS进门。后面的美军一路点射。到出来4个猎杀者的地方，这里是必须要打的了，多注意走位，可以考虑用旁边箱子里的火焰喷射器或者就用RIOT GUN也行，最后双子T-103暴君，用RIOT GUN风筝打头吧。CD一好就开SS


一上来，先转头从背后的尸体旁边获得一个Collect Data和一个燃烧弹
另一侧箱子旁边可以拿到一个治疗喷雾
一路杀下楼梯，往角落里跑，拐过去杀死两只丧尸可以看到一个治疗喷雾和一个Collect Data
解决完空地的丧尸，往前走门里冲出两只着火的，直接喷死
来到下一个场景往前走几步会遇到丧尸犬，点掉
跑到头，警车后面拿到一个Collect Data
上楼梯进门
老虎雕像的旁边拿到一个Collect Data
进门一侧的柜子上拿到一个Collect Data
另一侧的柜子上可以拿到本章的小浣熊，然后调查警长的办公桌，触发剧情
出门遇到舔舐者，无视直接跑路，尽头的门会被丧尸撞开，一路杀出去
路上的丧尸全喷掉，舔舐者全无视掉
下楼梯，继续喷丧尸
中间的办公桌上拿到一个Collect Data
来到警察厅门口，门再次被丧尸撞开，喷掉
这里注意别急着跑出去，在门口附近先把丧尸都喷掉，然后再跑过去触发剧情
剧情完毕，出门继续喷丧尸
往前走遇到LEON，然后调查卷闸门，丧尸从旁边涌出来，这里要先清场再走
进入地下仓库，遭遇大量的舔舐者，这里杀起来一定要快，因为舔舐者是一直在补充的，如果出来的太多会很不利。SS CD一好就开
杀完舔舐者之后，从垃圾堆旁边拿到一个Collect Data
调查卷闸门，这里可以隔着门用喷子喷掉门口的丧尸
门打开之后点杀掉左手边的一个美军
右手边的美军也可以点杀掉1-2个
出来往前走两步，手推车旁边的地上拿到一个Collect Data
然后从后面绕道到另一侧，干掉一个美军
接下来剩下的美军不管了，直接开SS一路跑，注意长椅上的Collect Data顺手拿了
一路往前跑，美军如果正好挡住可以跑的时候按圈撞开，到标记处直接触发剧情
剧情之后出来4个猎杀者，如果HP多的话可以考虑杀死1-2个。如果HP少的话就转圈吧
等到旁边的门里冲出一个着火的丧尸，干掉他就可以从他冲出来的地方走了
拐角处会掉下来一个猎杀者，无视，旁边的地上拿到一个Collect Data
一路跑到头，美军遭遇战，点射
这里点射死4-5个美军，零散的丧尸喷死
右手边可以拿到喷雾和一个Collect Data
然后直接加速跑到正对的一个走廊里，拿到一个治疗喷雾和一个Collect Data
外面会掉下来好几个猎杀者，无视他们，绕路往右走到头，卷闸门里会出来好几个美军，再次无视，开SS进门触发剧情
往前走杀一波丧尸
再杀一波狂暴丧尸
出来之后垃圾堆旁边拿到一个Collect Data
右转走到头红色垃圾堆旁边还可以拿到一个Collect Data
再右转利用墙做掩体，点射死几个美军
从公交车的窗户里可以点射到对面高处的狙击手
往前走没两步，地上的尸体旁拿到一个Collect Data
往前走，遇到一波防守的美军，利用箱子作掩体点射之
拐角的台子上拿到一个Collect Data
继续往前走，还是美军遭遇战，利用汽车做掩体点射之
杀完后跑到小巷里利用垃圾箱做掩体，再杀一波美军
接下来就是一波强制的猎杀者战斗了，这里看着点队友的HP。一定要及时复活他们来做肉盾，然后逐个把猎杀者风筝掉（武器用喷子或者火焰发射器都行），SS好了就开
杀完猎杀者之后会出来一波狂暴丧尸和几个美军，这里推荐打美军出失血让它们自相残杀去
都清理完了之后到左手边的墙边拿到一个Collect Data（看地图
再接下来是右侧的一波美军战斗，找个掩体逐个点杀即可，没什么难度，干掉后过去可以拿到一个治疗喷雾和补充弹药。顺便清理一波丧尸
最后的场景调查门，BOSS战斗T-103暴君，一开始出来一只，之后会再出来一只，这里要不断放风筝，保持一定距离让它刚好打不到你，太远他会冲锋，太近会挨打。放风筝的时候用喷子尽量打头，雷也不要吝惜，干掉两只暴君之后马上进门，剧情本关结束
