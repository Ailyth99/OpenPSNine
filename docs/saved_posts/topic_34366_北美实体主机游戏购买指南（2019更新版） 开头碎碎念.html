自上一版的游戏购买指南完成已经过去了1年有余，北美各大电商的销售策略均有所变化，我本人在不断的购（kai）买（xiang）过程中也积累了更多经验。与此同时，更多坐标北美的ps玩家加入了p9大家庭，也有越来越多的国内玩家开始追求海外版本的游戏/限定特典......

美版普通版游戏首发购买教程

美版普通版游戏首发购买教程
普通版游戏
如果你只是一个“纯粹”的玩家，对于所谓限定版大箱子并不感冒，或者某些ip你并不是特别熟悉只是想试个水，又或者你就是单纯的贫穷。那么下图这样的普通版游戏或许会是你的最佳选择。



普版游戏除了实体发行量有限的独立游戏（Indie Games，会单独分页讨论）以外，几乎不存在断货/缺货的情况。那么在哪家买可以享受到最大的优惠自然成为了购买者最先需要考虑的问题。在2019年的北美，bestbuy GCU停止发行、亚马逊prime会员预购8折也被取消的情况下，想要以低于原价的价格购买到首发的普通版游戏几乎是不可能的。以下列举一些常用购买渠道，并简单讨论各家的优劣。

- 渠道1: 大型电商/实体店（Amazon, Bestbuy, Gamestop）





以上三家便是我，同时也是很多北美主机玩家最常用来购买首发游戏的网站。其中Bestbuy和Amazon均提供游戏发售当日免费送货。而Gamestop在这方面就相对比较坑爹，一般会提供发售日后2-4天的免费送货，以及5刀左右的首发日送货。

Bestbuy和Gamestop还有一大优势便是他们在大多数城市都设有实体店，不喜欢等待的朋友可以趁着发售日一大早店铺刚开门的时候就去实体店pick up，虽然可能会消耗点汽油费，但是省去了等待快递的烦恼。

而与前两者不同的是，Amazon的优势在于可以买到的游戏种类多。一些小众游戏，特别是日系游戏发行的美版很容易在bestbuy，Gamestop这样的店铺出现保有量很少或者根本不上架的情况。但是相对的，亚马逊因为允许第三方发行商的入驻，但凡在北美确实发行了的游戏都能买到。

最后需要注意的是，虽然曾经风靡一时的预购8折已经消失，这三家电商至今还在提供一些限量的、较小的福利给首发购买者。一些游戏会提供pre-order bonus，比如在gamestop预购只狼会送一把拆信刀，在bestbuy购买NS的任天堂大乱斗会送一个实体金币等等。同时，bestbuy有时还会对一些大作提供预购送$10刀礼卡的福利，也可以看做是变相的折扣。

所以购买游戏前在这三家都看一看，比较一下，一般总不会错。

-渠道2：游戏发行商官网（NIS America, Square Enix, Rockstar Games等）



SE的游戏在欧美还挺大众的，所以不愁买不到。NIS会代理一些日本rpg（如伊苏、闪轨，当然还有日本一自家ip）在北美的发售，喜欢日系游戏的可以经常关注一下他们的官网。当然就如前文所说，NIS与amazon是有代销关系的，直接在amazon购买也没问题。

-渠道3: Target, Walmart等大型超市

超市的话，如果你碰巧办了Target的购物返现5%的卡那也不失为一种选择。其他就真没什么优势了...


美版限定版游戏首发购买教程

美版限定版游戏首发购买教程
限定版游戏



没错，就是我之前开过箱子的那些。

如果你和我一样从来不在意保值的问题，就是喜欢首发限定版。那么请注意，北美大多数游戏的限定版在发售初期，甚至是开售前几天都会处于没有现货的状态，而后期根据游戏的风评/制作量会出现1.价格下滑 2.价格飞涨这两种情况。

所以，想要确保自己能第一时间原价购买到想要游戏的限定版，一定要密切关注预购的消息。搜索时常用的关键字有：limited edition, collector's edition, pre-order release date等。热门的游戏、限定版主机别说在发售的第一天了，很有可能在几个月前开始预购的第一天就被抢完了。

购买渠道同前文，三大电商和发行商官网依旧是最好的选择。既然都选择了购买较贵的限定版，价格自然不再是第一考虑因素。相对的，购买限定版游戏的困难点在于很多游戏的限定版都不会放到特别多的店铺发售。

发行商官网一般是最常见的各种限定版发售地。三大电商中，Gamestop因为是一家专门做游戏的店，经常可以拿到专属的限定游戏授权，例如我曾经开过的MHW限定版。

总之想首发限定版就一定要多做功课研究哪里可以买到，另外就是要在预售的第一时间下单。


独立游戏的购买

独立游戏的购买

单独为indie开一业是因为很多独立游戏在欧美以外的地区是不发行的，也就是说国内的朋友想要买到一些独立游戏的实体版只能通过欧美的渠道。


最常用的独立游戏购买网站无疑是Limited Run Games，著名的celeste限定版就是在这个网站发售的。当然在这网站买东西都要按时按点抢购...毕竟存货少


当然他们有时候也代销小众日本游戏的限定版


除此之外，Fangamer也是不错的独立游戏购买网站，undertale的实体限定就是与他们独家合作发售。


港版、日版、欧版游戏首发购买

港版、日版、欧版游戏首发购买

PS系游戏的语言地区自闭性一直是非常让人头疼的问题，可以理解很多身在北美的玩家对于玩中文游戏的需求。其次，日版、港版游戏对于美版还存在一个发售时间差的问题，当然了有些游戏根本就不会在美国发售。至于欧版游戏，有时会有与美版不同的奖杯组，或者是不一样的限定内容物，所以在此也讨论一下不购买数字版的情况下如何买到其他地区版本的游戏。

港版游戏
至少在我所知道的范围内，美国还是没有什么可以以较实惠的价格购买到港版游戏的途径。下面简单介绍一些购买方式，价格肯定是没有在国内买便宜的，但是真的想要玩港版游戏也不想买数字版的朋友还是可以看看。

- 渠道1：PLAY-ASIA

在美国做了很多年的购买亚洲游戏的网站，定价本身就不便宜，运费也有点贵。但是确实可以买到很多港版游戏、日版游戏甚至是韩版游戏。

- 渠道2：淘宝买了转运

这个就不用多解释了吧？

- 渠道3：Ebay
有一些Ebay卖家是专门卖港版游戏的，也可以去搜索一下。

日版游戏

介绍港版游戏购买方法中提到的三个渠道自然都是还能用的，但是购买日版游戏最推荐的还是日亚直邮，不仅速度快价格（特别是运费）也超级良心。



另外日版游戏牵扯到一个店铺限定的问题（如著名的GEO铁盒）



对于想要购买特殊店铺特典的朋友，亚马逊以外的店家除了转运基本无解。至于转运公司的话挺多都是不错的，我没有仔细研究、货比三家过，只能说我自己在用的是Buyee，没什么毛病。

欧版游戏

购买欧版游戏的话，美国亚马逊的账号在英、法、德、意、西五个国家的亚马逊都是通用的，无需另行注册填写信息。

以上亚马逊也都支持美国的直邮。

此外其他已知支持直邮美国的欧洲店铺有：micromania，game（game可以邮collector's edition）




非首发游戏购买、二手游戏买卖

非首发游戏购买、二手游戏买卖

非首发游戏购买
早买早享受，晚买享折扣，在全世界都是一样的道理。很多玩家不买首发可能就是想等一波降价、折扣。

北美最著名的黑色星期五和紧随其后的cybermonday自然是蹲大折扣的好时机，而一年中的其他时间各大电商也会在不同时间段多多少少给出一些还不错的游戏折扣。

至于如何更快更高效地get到折扣信息，我会推荐如dealmoon、slickdeals这样的折扣分享网站。这些网站不仅在购买游戏的时候可以用做参考，生活中其他各方面的购物也都可以查看。其次，以Reddit, CheapAssGamer为代表的论坛也是分享折扣的好去处，这两个论坛也可以作为买卖二手游戏的渠道。

对了，这里还是要推荐一下我非常喜欢的Ebates，这是一个网购返点小插件，平时开着就行。（大家要用的话欢迎找我refer呀~）以及真心推荐留学生早点弄到ssn办张信用卡，也会有很多不错的福利。（每次写到这一part就觉得变成了留学生生活指导）

二手游戏买卖

除了刚才提到的两个论坛以外，学校的二手交易群，facebook的本地二手交易组还有北美微论坛也都是买卖二手的好去处。当然了，说到美国的二手交易就很难不提到那个叫做EBAY的网站。

Ebay相比于其他所有二手交易方式，最大的优点就是用户基数大。在你作为买方时有更多的选择，作为卖方也能有更高的曝光量。同时，Ebay的拍卖系统也可以让用户得到一些通常不太会出现的低价。但是要注意的是，ebay上挂东西卖会被收取高达10%的手续费。另外，碰到不讲道理的买家，由于Ebay的政策对于买方有极大的偏袒性，卖方很可能还会在退货等方面吃亏。当然不可否认的说，在ebay卖东西能卖出得非常快，也相对省心。

这里顺带一提，对于日本店特感兴趣的朋友，日拍+转运单独购买限定物不买游戏也是一种不错的选择。

结语
