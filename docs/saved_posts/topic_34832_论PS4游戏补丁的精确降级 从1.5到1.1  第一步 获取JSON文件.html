第二步 使用PSX

先关闭防火墙或者开启对应端口，

然后启动PSX，只开启本地加速



PSX设置增加 |*.json* 截取，记得点应用



PS4设置代理到PSX，删除应用，然后重新下载应用
等PS4下载进度（下载体积）出现后，删除下载

此时PSX上会出现一大堆记录，找到刚才保存的JSON文件名类似的一条



然后在此项中选取本地文件，对应到刚才保存的那个JSON



现在回到PS4 重新下载

重新下载的文件体积应与第一次下载的体积不一致
并且在PSX上最后的记录也会显示下载的PKG文件是指定版本122

最新补丁的hash计算与下载

使用python运行下面的代码，修改GameID的值为对应的9位ID，运行结果复制到浏览器打开
使用 https://c.runoob.com/compile/6 运行也行


import hmac
import hashlib
import binascii

GameID = "CUSA06120"

byte_key = binascii.unhexlify("AD62E37F905E06BC19593142281C112CEC0E7EC3E97EFDCAEFCDBAAFA6378D84")
hash = hmac.new(byte_key, "np_" + GameID, digestmod=hashlib.sha256)
hash = hash.hexdigest()
print "https://gs-sec.ww.np.dl.playstation.net/plo/np/" + GameID + "/" + hash + "/" + GameID + "-ver.xml"

可以自行保存此文件，之后用此文件的信息可以改出对应补丁的地址hash，效果同 https://ps4database.io 一致
