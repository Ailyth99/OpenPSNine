PS3 《麦登NFL 12》 白金攻略

llagnd 2015-11-13 22:54编辑     4评论
最近在补PS3上一些旧坑的过程中，惊奇发现EA居然没有关闭Madden NFL 12的服务器。一方面可能橄榄球作为美国国民运动在当地有很大的群众基础，另一方面或许是NFL 12的MUT模式依然在为EA带来收入。游戏白金难度不高，需要对橄榄球知识有一定了解并对网络条件有所要求。因为无法在在中文世界里查找到任何关于该作奖盃获取的技巧或资料，所以我把白金过程中的一些心得写出来，希望对有需求的玩家提供一些帮助。



游戏流程：

1.Madden Ultimate Team模式
MUT模式共包含4个奖盃，需要网络连接。然而EA的服务器稳定性并不是很好，希望大家可以顺利的迈过这个坎。
This One is Easy
创建一支Madden Ultimate Team
先升级游戏至最新版本，第一次进入MUT模式，根据系统提示创建自己的Madden Ultimate Team。

This One is Hard
打造一支评价为80的Madden Ultimate Team
方法有以下几种：
1.通过MUT模式下的Head to Head，MUT tournaments和玩家对战赚取硬币，赚取足够多的硬币后在商店购买4-5个Gold Premium Packs基本就可以解锁。
2.通过MUT模式下Play vs. CPU赚取硬币，比赛时间和难度会影响你获得硬币的数量。2分钟一节的比赛通常可以得到300左右硬币，赚取足够多的硬币后在商店购买4-5个Gold Premium Packs基本就可以解锁。
3.美服玩家可以直接在菜单中按下，进入Playstation Store购买硬币。如果显示当前商店里没有可用的物品，过一段时间登录再看看。
4.和别的玩家交易卡牌。

2.Franchise模式
简单的2个奖盃。
Hey, Can I Talk to You?
在Franchise模式下完成赛季裁员指标
进入FRANCHISE MODE后，选择NEW FRANCHISE，随便选取一只队伍，然后在接下来的SETUP OPTION中设置如下
Salary Cap: Off
Update Depth Chart: User
Re-Sign Players: User
Free Agent Bidding: User
Sign Draft Picks: User
Fill Roster: User
Sim Cut Days: User
Sim Scouting: User
进入季前赛按，选择Sim The Game（模拟比赛），接下来选择Simulate This Week（模拟本周），每两周系统会询问你一次是否要手动裁员，选择Go To Cut Players Screen（去选择裁员菜单）。屏幕右上角会告诉你本次裁员最多可以裁掉几人，在队伍中选择队员、裁掉足够的人数即可。需要注意的一点是每个位置上的人数有最少限制，保持满足最少人数要求即可。

准备工作3：这里有两种方法

方法一（又称一劳永逸法或手残法，适用于新手玩家）
由主菜单进入My Madden> NFL Rosters> Player Management，选定一只你所想作为对手的球队，将队伍里所有球员的几项基础属性（Strenth、Speed、Acceleration、Awareness、Agility）调节至0，切记调整完后记得保存游戏资料。由于每支球队的队员会有很多，所以这会花费掉你不少时间。

方法二（又称轻松法或老手法，适用于系列玩家）
由主菜单进入My Madden> Creation Zone> Create New Team，创建一直新球队，队名、所在地什么的都不重要保持默认即可，在Team Type Editor设置中选择Fantasy Team进入梦幻选秀。在OVR属性栏按两次进行逆向排序，保证每轮都选到全联盟数值最低的球员，需要注意的是需要满足队伍不同位置的最低人数需求。经过50轮选秀后保存游戏资料。

准备工作4：有主菜单进入My settings> Settings，将所有Player Skill设置为100，所有CPU Skill设置为0，将Special Team数值设置为100，Penalties数值设置为0.

赛前阶段

方法一，选择被削弱的对手球队。

方法二，按读取资料，将刚才创建的队伍设置为对手。

强烈建议使用两个手柄完成。以下我方简称A，对方简称B。
1.准备工作：先控制B进攻到我方10码处附近，选择一个外接迂回跑位的传球战术；用A布置一个Cover 2或Cover 6防守战术。
