介绍

U.S.S（Umbrella Security Service）
雨伞公司的直属秘密特种部队，专门负责歼灭、排除对雨伞公司利益上有害的对象
部队的成员们如同受过完整训练的猎犬，具有强大的战斗力 及 冷酷服从任何命令的忠诚心
而其他关于部队的讯息都是个谜，就连雨伞公司里也只有少数几人知道他们的存在
部队中也分成了许多小队
大家熟悉的HUNK是属于 α(alpha) 小队的队长
而这次的主角们是属于新的 δ(delta) 小队


δ Wolfpack（狼群小队）

Lupo
本名：Karena LesProux
年龄：38岁
出生：法国
兵种：突击兵

背景：
LUPO是经验丰富的前法国特种部队队员
退役后一心想着与理想的男性组成一个美满的家庭
但是事情却不顺利......
她的丈夫经常对她施暴，但是为了小孩，LUPO一直默默的忍受的家暴
直到丈夫开始对小孩出手
LUPO终于忍不住怨念及怒火
徒手将丈夫活生生的杀死了（不愧是特种部队= =
事后法官由于LUPO可怜的遭遇
所以判决她无罪释放，让她回到原来的生活

但是变成单亲妈妈的LUPO想要回到原来的生活却非常的困难
对于一个经过严厉训练的职业士兵来说
想要找一个普通的工作十分困难
她的本能非常强烈，且流着士兵的血液
最后还是促使她回归杀手类的工作

雨伞公司看中了她的精神力及技术
将她拉拢进公司
并与LUPO作了契约
公司保证了她小孩的三餐
且使他们接受最优秀的学校教育和享有最高的医疗保証
但是！前提是LUPO还活着
意思是只要她在任务中死亡
公司当然也没必要继续这项契约



现在LUPO做为狼群小队的队长
她丰富的经验及技术就像狼的首领一样带领导着队员
因此有了「狼之母」这个称号


Vector

本名：不明

出生：日本

背景：
他的武术如同艺术一般
明显的受过日本武术大师的训练
但除此之外，他的相关资料、经历都是一团迷

过去曾在HUNK的手下训练、执行任务
非常尊敬HUNK并视他为恩师

现在VECTOR被分配到狼群小队
虽然他有向公司提桉想继续呆在HUNK的旗下执行任务（根本超爱他师父= =

或许公司是希望狼群小队中也能有像汉克般能力的人吧


Four-Eyes

本名：Christine Yamata
年龄：28岁
出生：美国（日裔）
兵种：生化兵

背景：
Four Eyes从年幼时就开始专精于研究科学
因此导致她并没有像一般人一样在社会中生活的能力
在她的眼中，人类只是单纯繁殖力高的生物而已
她已经剥离了正常人类的感情
她深爱的只有病毒学而已

她会加入雨伞公司的理由
是因为这裡能够带给她更大的科学视野
运用雨伞公司最先进的病毒技术
且能研究世界各地最毒、最稀有的病原菌

在有许多人类聚集的任务中
她常常将负伤的俘虏独自抓去实验
就算被公司发现，也没有意思反省悔意
也因为这样，常常延误任务的完成时间

但是在这领域，她所实践的病毒知识，也都是原本无法知道的知识
所以公司应该也对她的行为睁一隻眼闭一隻眼吧......

Bertha

本名：Michaela Heinzwaffen
年龄：34岁
出生：德国
兵种：卫生兵

背景：
Bertha能对人体最最大限度的治疗
并且能促发病维持人体的潜在能力
是位优秀的医疗兵
但是，在这领域她却缺乏了重要的道德感......

每当在治疗患者时
她都会详细的记载患者感受到多大的痛苦
她认为让人害怕受伤是她的职责
这也是她几乎不使用麻醉药的原因
但是她工作上的研究，在民间医疗却完全派不上用场......

而她的技术、感伤性和感受性的欠缺正
对雨伞公司来说，正是最优秀的特徵

同时Bertha也是个专业的拷问者
且愿意帮任何人执行这个职务
她活用人体相关的庞大知识
让治疗的同时发生痛苦
行刑后再治疗，再行刑再治疗
不断的将被拷问者从死亡边缘拉回来
直到她得到想要的情报为止

心得

系统：
这次的动作性做了很大的革新
就历代而言，这已经不叫恶灵古堡了
不过他的背景、故事、剧情则不口否认，这的确是恶灵古堡
虽然很多人在争说这游戏不该挂这名子........
嘛.......既然都说这是外传了
这些就不要太在意这个了^^
而且能看到高画质的殭尸跟浣熊市
应该也实现了很多人的期望吧ＸＤ




感染跟出血的设定也增添了许多刺激感
主角终于不像以前一样是免疫体质了XD


近战体技：



刚开始许多人可能会对这招很反感
觉得这款应该会变成恶灵无双......
...................................................
但如果你把它当恶灵无双来玩
当你被一大群殭尸包围时
你就会开始抱怨：「我X 怎么不能用无双乱舞！」
恩.....就因为这点这并不会成为无双
体技通常是在敌人数量少时用的
难度高一点
对多隻殭尸时，血量应该会扣的比你杀人的速度还快= =

我是认为这种近战、体技的作法还不错啦XD
至少让攻击的手段更多变化


射击及伤害判定：
伤害判定有问题，有人说射殭尸不痛？
..........第一次见到殭尸的人也都这样说= =
所以还是听薪水高的人的建议：「打头」



射击判定有问题，瞄头打不到头？
这我是觉得还好
几把高准度的枪瞄的好几乎都能很容易就爆头
沙漠之鹰类的更容易

但是以上两点都有例外.......
狙击枪
这次狙击最难发挥了= =
没有说很好瞄
对敌人的出力更是柔弱
射殭尸竟然也要爆头才会挂.......
再怎么样，打身体好歹也给我弹出去一点吧= =
明明用散弹枪射都会飒爽的爆开的说><

士兵
这吐嘈无力= =
穿防弹衣就算了
竟然内建钛合金脑袋= =（刹那表示：「我们都是钢弹」

为独这两个设计不良= =
猎人的话
我也是觉得还好啦
难得他地位大翻身嘛XD




AI
这代的AI队友除了挡路、当诱饵
真的就没啥用处了............
嘛.......不过换个角度想
如果AI真的很完美，攻击性又强
那就不需要连线了= =


救援系统
救人变成捡枪.......
其实这问题也不大啦
指示键上面也有明确的写出你现在是在救人还是换枪
只有紧急的时候要马上桥到救人的那个位置，会有点急啦
不过通常紧急时我都跑了= =
谁还救人啊.......

而最重要的事：救人请站在尸体上救
ID有时会跟尸体分散在别的位置
这才是真的做的怪怪的= =


剧情
这次可以从别的角度来看浣熊市事件
剧情张力是还好啦
毕竟每关都是短篇的小队任务
故事又是大家熟悉的那个
要出现很剧情化的超展开.......应该很难吧= =
而且这代并不是在塑造英雄
主角们只是很现实的特种部队而已啦.......

当你看到之前四把机枪射都射不退的暴君
被李昂小枪几发就推下楼的那幕
你就会知道凡人跟英雄的差别了= =

而王的话，有！
历代几个经典的王都有出现
只是打一打都送给历代英雄捡尾刀= =



长度则是有些意犹未尽没错啦><
麻....看之后DLC加起来能不能改善


连线
这次的连线模式都蛮好玩的^^
各种兵种、技能的配对
会给战场带来不同的效果

但是，能不能沟通、有没有麦就真的差很多了= =
当敌队的人可以互相沟通，而你不能时......
你就会知道 "X影忍者" 的 "轮迴眼" 有多机车了....
对方一个人看到你的位子，就全队都知道了= =
这种状况还是乖乖回家修练仙人模式吧......
不然至少找一个能交谈的好友再来打才是上策


总结

这款游戏就是叫你去多找几个好友啦ＸＤ

其实单人过关也不会说烂到很不想玩
只是玩久了很闷很腻是真的= =
严格说起来
单人是考验你的个人技术
多人是考验你的团队默契
所以也没那么糟啦><

BUG的部份则是应该再改善一下
有时连线打王，打了半天就是杀不掉.......
我X！！　
好歹你也讲一下老大你现在是BUG状态= =
何苦一直吃子弹呢....
搞的大家也手酸= =

不过中BUG也不是经常出现
所以问题也不大

总之这款我是觉得很好玩啦^^
喜欢连线的人应该都可以试试看
分数、评价什么的，果然还是自己实际玩过比较正确
