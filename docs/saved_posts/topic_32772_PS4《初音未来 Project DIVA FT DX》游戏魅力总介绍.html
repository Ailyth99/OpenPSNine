PS4《初音未来 Project DIVA FT DX》游戏魅力总介绍

sai8808 2017-11-13 17:57编辑     81评论
用进化后的摄影功能和其他初音爱好家交流吧！

SEGA Games 即将在2017年11月22日推出《初音未来 Project DIVA Future Tone DX》。本篇报导将为各位读者带来本作追加新要素、功能、以及受到广泛支持的理由总整理！



第1点 不用顾虑太多尽情享受节奏游戏的乐趣

本作为了让新手老手都能尽情享乐，在各种游戏细节中都有好好考虑到。

．只需跟随着歌曲主旋律按下按钮的系统，非常容易抓住节奏。
．旋律按钮采用「〇」或「△」等等，简单明瞭（可自行更改）。
．准备了4种难度（EASY、NORMAL、HARD、EXTREME）。部分乐曲还能选择「EXTRA EXTREME」难度。
．在「实践」模式中可练习各乐曲的任意部分。例如说，能够不断重複乐曲较难的后半段，来尽情练习。
．实装引导功能，帮助新玩家进行较难操作，包含同时按下多颗按钮等等。
．能够自行设定按钮的细节，配合自己的操作方法来享受游戏。


为了让玩家「沉浸于节奏游戏的乐趣」，游戏中都有各种充实功能帮助玩家。
而有了这样的基础系统，无论玩多少次都非常开心！








第2点 充满魅力的游戏影片，不会妨碍到玩家演奏

于游玩中拨放的影片（PV），也是本作的最大看点之一。
其内容森罗万象，其中有由初音未来等「V 家角色」带来的歌舞 PV，也有配合歌词内容展开戏剧演出的 PV。
这些 PV 不只带来视觉享受，甚至还会与乐曲旋律的节奏谱面联动。



像这样子的演出，让玩家能够更加沉浸于节奏之中。
另外，有些乐曲的歌词内容也会和节奏谱面联动，这些由开发团队带来的小彩蛋，也请玩家们细心发掘。




第3点 收录238首乐曲和396种衣装的超豪华份量

《初音未来 Project DIVA Future Tone DX》，一言以蔽之就是份量超充足。
总共收录238首乐曲、396种衣装，游戏售价只要7,990日圆，实惠到让人觉得「放这么多真的好吗？」的等级。

重点的乐曲阵容，收录自2007年《初音未来》诞生以来至今所有的人气、话题乐曲。
玩过所有乐曲一轮，就可以知道「初音」至今为止的所有历史。
乐曲流派从流行音乐、摇滚、爵士等到约德尔唱法都有，相当广泛。
而初音等人的衣装种类也非常丰富，玩家可随自己的喜好、或是配合乐曲风格替角色们换上不同衣装。



本作亦收录现今最流行、或是造成话题的人气乐曲。
来自「DECO*27」的《幽灵法则》、「米津玄师」的「沙之行星」等等。
配合这些乐曲，也特别製作了各式角色全新模组，也请玩家们敬请期待！






第4点 由「PV 照片」功能让摄影更加进化

各乐曲 PV 随时随地都可以自由鑑赏。在欣赏的过程中，按下「△」即可以将该画面保存起来。

・可拍下各种尺寸的游戏画面！

在本作中，可自由选择摄影画面的大小，有「16:9」、「1:1」、「9:16」、「3:2」供玩家选择。
拍下的照片会保存于 PS4 当中。



・以丰富的图案装饰 PV

角色、乐曲 LOGO、各种颜色的框架、爱心或气球等各型图案都可配置于画面当中。
调整图案的大小、透明度、角度，创造世界仅此一张的照片，考验您的审美！



而这个 PV 照片功能，除了单独享受外，也能投稿至 SNS 分享，并欣赏其他玩家的精美作品。
待游戏上市后，请务必体验看看，享受其中！
