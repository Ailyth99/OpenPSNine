《高达破坏者3》DLC第三弹「Build Rising」简单评测+奖杯攻略

retridge_rii 2016-12-24 12:23编辑     13评论
上一弹(第二弹)DLC“Build Beginning”奖杯贴

高破3的第二弹收费DLC(总计第三个DLC)，奖杯配置和上一弹相同。

按照惯例先整体评价一下。抛开营销手段这类的好恶不提，基于高破3本体的水平，这个DLC本身的质量可以给7分，但是同样考虑到这玩意卖1000岛国币，我决定最后给他一个6分。主要的要点在于:

+DLC关卡的地图都是移植的初代高破的一些地图，总算是有了“新场景”。
+新BGM的水平不错，比较好听。
+比起上一个DLC，剧情里好歹有能让人笑一下的成分在。

-剧情格局依然小，而且还是比较扯淡。这个DLC主要讲述了主角把官方钦定的小女友晾在家里，然后和以岳父为首的一帮大叔大妈去东京游山玩水打电动，同一时间被晾在家里的小女友和一堆本体剧情里没什么存在感的NPC开女生会的故事。
-比起上个DLC，宝贝数量明显缩水。
-这群孙子是不是又把宝贝的掉率调低了。
-总的来说，游戏体验还是那个样子，没什么意思。

大概就是这样。接下来开始奖杯部分。

BUILD RISING：任務完成
以任一難度通過第三波DLC的所有任務

总共四个任务。NT难度上来可以直接选关打，其他难度必须一关一关推，和本体的设定一样。
第一关三张图，第二关一张图，第三关三张图，第四关一张图。任意难度打完四关跳杯。
就像上面说的，这四关用的都是“新地图”，分别是加布罗、赌场桌、火山地带和台场。

BUILD RISING：武器/機體零件收集完成
持有第三波DLC內任一層級的所有追加武器及零件
这一弹DLC共更新四台机体。先说一下铁血我没看，所以机体名字我都音译的，反正有图，你们也不会认不出来是不。

龙再飞。本体五件套+射击武器1把+盾，总共7个部件。

高达古修。本体五件套+近战武器1把+射击武器1把，总共7个部件。

高达阿斯塔洛特。本体五件套+近战武器1把+射击武器2把+盾，总共9个零件。

高达AGE-FX。本体五件套+射击武器2把，总共7个零件。

四个新加入套件的出现关卡分别是:龙再飞DLC3-01第一张图中场精英怪，高达古修DLC3-02关底精英怪，高达AGE FX DLC3-02关底Boss，高达阿斯塔洛特DLC3-03第二张图末尾精英怪。

上述全收集跳杯。HG/MG不论。

BUILD RISING：製作者零件收集完成
收集到第三波DLC裡的所有追加製作者零件
这一弹总共更新8种BP，掉落规则和上一弹一样（精英怪和BOSS单位才会掉落，并且只在击破的时候进行概率计算，已经获得过一次的BP就不会再掉落）。精英怪/BOSS单位及相应的出现关卡、掉落BP如下:
