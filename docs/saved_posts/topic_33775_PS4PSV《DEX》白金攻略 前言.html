前言

前言

大家好，这边是来自弯弯的黑吐。

这游戏还真冷门，只有四个人玩过。

趁着PSV特价时入的游戏，还送了PS4版，虽然战斗很别扭看在双白金的份上就推一下。

不过PSV版的报错机率真的不可小看阿…

这个游戏白金不难，但几乎所有杯都是可遗漏的，纪录个流程方便有兴趣的朋友参考。

请按照分页顺序攻略喔～





流程杯

流程杯

Assassin
同一周目20次背后暗杀敌人。

背後暗杀是在敌人背对的时候按下，全敌人数量远远超过要求数量。看到背对就暗杀一下吧～

Sniper
进行头部射击

随便装备啥枪按住会出现瞄准线给头部来个一发吧！

Silver-tongued
在对话时成功使用说服选项

游戏中的对话超多，只要对话中出现(Convince)提示，选择这个对话选项就跳杯了。
但必須先學會charisma魅力技能之後才會出現(Convince)提示。

Lock, stock and barrel
打开10个锁

也是一个要求远远少於游戏中数量的杯，但要先学会lockpicking开锁技能。

Thug
升至等级5
Mercenary
升至10级
Runner
升至等级15

等级杯与技能挂勾，提升等级会给技能点数。

Professional
提升主技能至6级

主要技能为技能列表前四个，提升至满级每种需要10点技能点，任一技能点满就跳杯。

Software Engineer
将软件升级至最高级

用光盘升级，光盘会散落在游戏中各处，推荐直接找Tony购买比较省事。
共有三种，每种升级五次满级。
Software Disc:Pulsewave
Software Disc:Spammer
Software Disc:Wedge

=====================================================================================

游戏中开始遇到商店时可以用道具金钱复制法来拿取上面有相关的奖杯，
可以参阅tgkonamix大大提供的方法，
https://psnine.com/topic/33372
相当建议用这方式升级，
第一游戏中敌人个个有霸体又超级敏感，
只要面对就基本闪不掉，
Hack小游戏没升级前超级恶梦，
子弹多的跟啥一样还附带自动追击小炮艇，
但不先花些点数拿些开锁对话的技能又让游戏很不顺，
所以还是全加满点数升级吧！
就算全升级满还是会死的。

注意：不要刷过头，等级刷到15，金钱大概20万左右金额就够通关游戏了，刷太过头这游戏会开始有很多莫名其妙的Bug，NPC不能继续任务阿敌人暗杀後还要再杀死才会死阿怪异现象会发生，请勿尝试（被害者路过）





爽送的金杯

爽送的金杯

Elevator pitch
被电梯砸到

顾名思义就是被电梯压死而Gameover，这游戏不少上上下下的电梯，先让电梯上行然後滚出电梯，再从最下层呼叫电梯後就待在电梯下面等被他压死就行。

推荐一：到DOWNTOWN的CAPS LES右边大楼就有电梯。
提示二：需要lockpicking开锁技能LV1
提示三：可以跑酷快速把所有地图都解开，之後从MAP选项可以快速移动到各个地图。

無特殊要求散杯

Cyborg
安装7个植入物

提醒一：全植入的插槽不够全部技能植入而且因为是植入脑袋所以不能拔除所以慎选技能。
提醒二：剧情进展到Get Augmented任务完成後就可以到CHINESE QUARTER找Dr. Niles做植入物手术了
