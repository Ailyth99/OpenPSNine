BNEI新作ARPG《Code Vein》2018年发售 GE制作人富泽祐介带队开发

watarasejyun 2017-04-23 11:02编辑     21评论
Bandai Namco新作迷宫探索型动作角色扮演游戏《Code Vein》今日释出情报将在2018年发售，平台未定（预订登陆主机），由《噬神者》组开发，采用虚幻4引擎，面向世界推出，与《噬神者》新作进行分线进行，富泽祐介将带队总管《噬神者》新作和本作。



《Code Vein》设定在所有都崩坏的近未来世界，玩家将扮演拥有超常能力的吸血鬼（Revenant）与伙伴们共同探索世界的真实。在这个世界内Revenant们为延续生存而被在隔绝社会的Vein中生活着，虽然成为超越的人类的生物，但是作为代价失去大部分记忆，并且缺少血的Revenant变成了被称为堕鬼（Lost）的怪物。



游戏部分在迷宫探索时，玩家仅可带一位搭档组成队伍，而为能更好的体验队伍共同挑战死地的紧张感和达成感，队伍的AI有较高的调整。

武器部分，Revenan主要使用的有大剑、单手剑近战武器外还有长枪、锤子、剃刀等；同时本作还有类似捕食功能的吸血牙装（Blood Veil）的特殊装备，不同于《噬神者》是作为武器，而是作为服装形态有着防具功能，吸血时显示专用机构，使用者的面具也会变化，吸血成功后可以发动特殊能力炼血，可以强化自身、弱化敌人或者直接攻击等等。

制作STAFF：
制作人-饭冢啓太
导演-吉村广
领队-富泽祐介
