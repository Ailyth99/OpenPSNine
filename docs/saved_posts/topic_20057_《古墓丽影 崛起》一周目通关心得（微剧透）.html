《古墓丽影 崛起》一周目通关心得（微剧透）

sam030209 2015-11-09 16:24编辑     18评论
还没 100% 不过简单讲一下心得给想入坑的参考：（难免主观，仅供参考）

+ 古墓规模、数量跟细致度皆上升（总共约十座古墓）
+ 场景变化性和区隔比前作多
+ 各细节和画面品质提升
+ 故事比前作略好
+ 战斗和系统强化（改造、制作、技能、装备、服装种类）
+ 游戏时数、探索区域范围增加
+ 有提供实况主方便的功能
+ 过场动画非常细致
+ 终于有BOSS战（前作那个可以不要承认是BOSS吗）
+ 支线的数量变多且种类变丰富

- 一样一直爆炸和被敌人包围、很少有独自探索的时间
- 超级烂尾，个人觉得对劳拉的心境描述不够深刻，结局很突然且空虚
- 模组、反锯齿和贴图等皆受限于机能而明显没完全发挥
- 虽然场景蛮多样的可是叙利亚超短有种被骗的感觉（宣传的时候说劳拉环游世界）
- 战斗难度依旧偏低（个人认为）、关卡难度非常低（虽然比前作高）
- 原创性不足、有些场景印象深刻但大多数都是老梗
- 游戏末期的敌人数量多到让人觉得厌烦和没有必要
- 本作的原声带比上一款逊色，几乎都是把上一代 Jason 编的旋律拿来小改



本作是 2013 《古墓丽影》的续作，故事接续漫画和小说，主轴是抵抗圣三一和寻找神圣之源的故事。劳拉从龙三角生还后，实力强大的神秘组织圣三一压下了邪马台事件并利用舆论和媒体打压劳拉，同时持续监视她。因为在上一款游戏裡，劳拉看到了无法用科学和常理解释的现象，使得劳拉对于事物的看法有了微妙的改变。

在游戏初期，我们可以看到一些劳拉小时后和父亲的回忆。当年父亲执着于长生不死的传说而身败名裂甚至众叛亲离，但小时候的劳拉却无法谅解他的所作所为。现在的她明白了父亲所说的话可能为真，为了完成父亲的研究并洗刷他的冤屈而前往位于叙利亚西北边境的先知古墓。过程中劳拉遭到尾随并和圣三一正面对峙，然而两方人马皆扑了空，因为先知的棺木是空的。

后来劳拉透过在古墓找到的记号当线索，连结了科特兹失落之城的传说故事和先知的神蹟，发现圣三一的目的就是爸爸当年在寻找的神器神圣之源。但是圣三一闯入卡伏特宅邸并偷走了劳拉的线索和研究资料，使她意外引导圣三一的人马到西伯利亚。为了抢先一步找到失落古城并找到神圣之源，劳拉跟前作生还的船员之一约拿亦即刻出发前往该地展开这次的冒险和竞逐......。



本作总共有三个开放式探索区域和许多较为线性的关卡，除了第一个苏联基地外，另外两个区域相信都会让玩家的眼睛为之一亮。场景举凡冰封遗迹、乡村部落、峡谷绿洲、世界顶峰等等，每个都相当漂亮且精緻，地图大小远比前作还大。额外古墓也不再採用上一代类似电影 The Descent 那种幽闭恐惧的窄穴当支线古墓入口，需要玩家经过一小段的探索才能进入古墓本体。古墓的奖励比上一代好很多，是所谓的古抄本，裡面会有特殊的技能和能力可以给予劳拉相当的强化，有可能是能在危急时解救玩家的快速包扎，也可能是失传的箭术技巧（像是不用从背后取箭就可以连续发射三发箭矢等）。

这次在支线下了相当的功夫，游戏中有 NPC 可以互动，帮助他们绝对会让你未来的路更好走。收集要素也变多了，不再像之前只有经验值跟残料。劳拉可以手工制作自己身上的装备，但是玩家要有足够的素材。在行动之际也可以自己制作弹药和各种箭矢，如果战况吃紧也可以赶快制作补品治疗劳拉，抑或是利用地上的空瓶空罐头制作炸弹和燃烧瓶等等。这次可以藉由收集古代硬币来和圣三一的成员走私武器、服装、装备等，而武器部件是真的作出另外一把新武器让玩家根据习惯和偏好自由切换而不是像上一作当中同类型武器只能拥有一把，新的改造直接就把旧的取代掉了。这次的动物敌人也变多了，他们喜欢守在地图上神秘的地点阻挡劳拉探索，然后虽然凶猛，打倒他们却能获得重要的素材。



对于动作游戏不擅长的玩家不用担心，因为这款跟前作一样，主线的关卡几乎没跳准都有蛮大的空间让你失误，战斗难度也可以自由调整，谜题也大多是火、风、水等元素的物理机关，顶多加上宽鬆的时间差跳跃和绳索装备的应用等等，几乎没有什么刁鑽的地方。觉得上款太容易的玩家可以试试新的难度，劳拉将少了逆天的自动回血，除了敌人变厚变强种类变棘手外，资源相对稀少且连使用营地都需要消耗资源。

这回在劳拉的行动上加上了不少新花样，初期除了千呼万盼的游泳外，还可以试试新的爬树机制。到了后面装备越来越多，劳拉已经不是猴子而已了，她几乎是蜘蛛人，跳跃跳不到的地方可以直接抛绳钩住、墙壁没办法用登山斧爬那就自己射宽头箭来当立足点、没路的话自己 DIY 个绳桥、拿到新装备后更是可以长时间在水下行动而不溺毙。基本上到了游戏末期已经没有什么东西阻挡的了劳拉了，小姐她想往哪走就往哪走，身上的装备和能力让她横行无阻。



如果你是喜欢前作的玩家，我相信你会更喜欢这款，因为这次有很多令人惊艳的时刻，古墓和遗迹的设计和上款不是一个层次。游戏中有很棒且很有意思的片段，像是劳拉必须在水中暗杀掉一堆冰层上的佣兵、从隧道出来的时候看到整个地热谷景致那种别有洞天的震撼、四方势力交火而劳拉挑战极限攀登到最高处等桥段等（放心，我避开很多地方没说避免捏光光）。

360 版本的画面表现非常优秀，有些时候甚至一眼难以分辨出两者的差异（如下图），游戏内容的部分和 One 版本完全一样，并没有在关卡或系统上面缩水。唯一的缺点是在战斗比较杂乱的情况下会有帧数不稳的情况发生。除外，One 版本的过场动画全部为即时运算，劳拉的衣着和装备会根据玩家的状态变化，解析度为 1440x1080；游戏中则是全程 1080p / 30fps 运行。而 360 版本的过场动画则是从 One 版本预录的。




目前只想到这些，有空再补......额外模式目前还没有试就是了，反正现在也没好友跟我互动。
