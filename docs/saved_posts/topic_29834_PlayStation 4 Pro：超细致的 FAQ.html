PlayStation 4 Pro：超细致的 FAQ

sai8808 2016-11-10 21:11编辑     26评论
PlayStation 正式推出 PS4 Pro，让主机玩家多了一个功能强大的新选择，为 PS4 系列产品增色不少。这款主机经过精心设计，能强化游戏体验，现有的高画质电视及全新 4K 电视皆可支援。

性能超越标准 PS4 的 PS4 Pro，不但堪称 PlayStation 系列的创举，在许多方面也是主机游戏的一大突破。我们很高兴能与玩家分享这样的愿景，也知道各位一定有许多疑问。因此，我们提供了以下钜细靡遗的 FAQ。




PS4 Pro：基本问题

问：什么是 PS4 Pro？

PS 4Pro 是 PS4 的高阶版本，能输出 4K 画面、支援 HDR以及提供更顺畅稳定的帧率，并可进行 4K 影片串流。正式发售后，PS4 Pro 将让支援的 PS4 游戏画质更加细緻、过程更为流畅；同时也将为开发人员提供升级版工具组，协助他们打造更丰富细腻的游戏世界。开发人员可根据个别需求及想呈现的游戏体验，自行决定如何使用这些强大的新工具。

值得注意的是，PS4 Pro 并非新一代的游戏主机，所以不会让您现有的 PS4 游戏遭到淘汰，也不会打散 PS4 玩家社群。PS4 Pro 仍是 PS4 系列的一分子。

问：是否会针对 PS4 Pro 推出独立版本或专属游戏？

我们不会发行 PS4 Pro 专属的游戏，因为 PS4 Pro 与标准 PS4 属于同系列产品，两款装置都可与过去、现在及未来的所有 游戏相容（包含 PlayStationVR）。 游戏都适用于 PS4 Pro，反之亦然。

PS4 Pro 于 11 月 10 日推出，基本上在这之后推出的 PS4 新游戏都可立即运用 PS4 Pro 的效能；在某些情况下，可能要等游戏推出一小段时间，安装随后推出的下载更新后才能有效使用。部分已推出的游戏经过更新后亦可使用 PS4 Pro 功能，包含《神秘海域4 盗贼末路》、《最后生还者 重制版》、《恶名昭彰 破晓》等等。

问：我须要付费才能更新现有 PS4 游戏，取得 PS4 Pro 的新功能吗？

虽然没有明文禁止发行商与开发商收费，但目前我们没有收到额外收费的消息。所有第一方游戏（如《恶名昭彰 破晓》及《神秘海域4》）启用 PS4 Pro 新功能的软体更新服务都是完全免费的。

问：所有的 PS4 游戏都适用于 PS4 Pro 主机吗？

是的，所有现行的 PS4 游戏皆与 PS4 Pro 主机完全相容。开发人员可自行选择透过下载更新套件，将 PS4 Pro 的强化效果加入先前发行的 PS4 游戏中。

问：SIE 会持续贩售并支援标准 PS4 吗？

会。PS4 Pro 为 PS4 平台的更新版本，将继续使用标准 PS4 系统。PS4 Pro 是为想站上游戏革新最前线的玩家所设计，无论您是想购买新主机，还是想继续使用标准 PS4 装置，您都能享有相同游戏在 PlayStation 上带来的绝佳游戏乐趣。

问：PS4 Pro 主机需要搭配 4K 电视使用吗？

不需要。若搭配 4K 电视，PS4 Pro 可以呈现比标准 PS4 更高的画质，但就算是搭配高画质电视（720p、1080i、1080p），PS4 Pro 依然能为您带来更优质的游戏体验。

问：PS4 Pro 搭配非 4K 的高画质电视时，能带来哪些优势？

即使您使用 PS4 Pro 时是搭配非 4K 的高画质电视，这款主机还是能提升游戏体验。开发人员可能以不同方式运用 PS4 Pro 强大的处理效能，让支援 PS4 Pro 的游戏呈现更高或更稳定的帧率、更细緻的环境与人物角色，以及更优质的整体视觉效果或其他相关的视觉强化。

此外，原本使用标准 PS4 主机只能显示低于原生 1080p（高画质电视的最高画质）的 PS4 游戏，将升级成在高画质电视上显示完整原生 1080p 画质。

问：PS4 Pro 支援多高的解析度？

若搭配 4K 电视，游戏可于最高 4K（3840 x 2160 画素，亦即「2160p」）解析度的环境下运行；若搭配高画质电视，游戏则能以 1080p 画质呈现。

问：PS4 Pro 能否与我现有的 PS4 下载游戏、储存资料及 PlayStation Network 好友名单相容？

可以。不论在哪个方面，PS4 Pro 与标准 PS4 都属于同一个数位生态系统；两款 PS4 主机装置共享同一个 PlayStation Network、同一个 PS4 社群。这表示您可以在 PS4 Pro 上开始游戏，透过 PS Plus（另售）将储存的资料上传到线上储存空间，在另一个房间使用标准 PS4 时仍可无缝衔接刚刚的游戏进度。这也表示所有您已储存在 PS4 的内容与资料，都可完全相容于 PS4 Pro。

问：如果我原本就拥有标准 PS4 ，有没有比较简单的方法可以将标准 PS4 的游戏、储存资料及撷取内容转移至 PS4 Pro？

有的。您可以使用标准的乙太网路线，将您的游戏、储存资料及撷取的影片和萤幕截图、系统设定与其他内容从标准 PS4 複製到 PS4 Pro。这个做法也适用于标准 PS4 至标准 PS4、PS4 Pro 至标准 PS4 及 PS4 Pro 至 PS4 Pro 的资料转移，前提是两台主机都必须安装 4.00 以上的系统软件，且两台主机都要登入相同的 PSN 帐户。

或者，您也可以在 PS4 Pro 中登入 PSN，存取游戏与内容资料库，然后重新下载先前购买的游戏与内容。当然，您储存在线上储存空间的资料也会跟着转移。

问：PS4 Pro 装置会针对 DUALSHOCK4 无线控制器推出改善项目吗？

PS4 Pro 以及新版的标准 PS4 Slim 中都会包含最新版本的无线控制器。新版无线控制器直接于触碰板上设置光条以投射光源，新的光源会投射出与光条同色的光线。除了Bluetooth通讯之外，也支援 USB 通讯。PS4 Pro 可完全支援当初与 PS4 一同发行的旧版无线控制器；新版无线控制器也可完全支援所有版本的 PS4 主机。


PS4 Pro：技术规格

问：PS4 Pro 与标准 PS4 相比之下差在哪里？

PS4 Pro 比标准 PS4 主机强大许多。PS4 Pro 的进阶图像讯号处理器拥有多种新功能（包括 AMD 最新的「Polaris」架构，以及几项全面自订的硬体创新），而且比标准 PS4 中的 GPU 更加强大。

总而言之，此次处理效能的升级将有助开发人员为 PS4 Pro 玩家带来更精彩的视觉效果，像是更顺畅或更稳定的帧率、支援 4K 显示及进阶图像功能等等。

问：PS4 Pro 支援哪些影像输出设定？

PS4 Pro 支援所有标准 PS4 所使用的高画质影像输出设定。值得一提的是，PS4 Pro 搭载 4K 输出效能，其设定适用 2160p YUV420 模式与 2160p RGB 模式（若电视支援，建议使用此模式），若搭配支援的 4K 电视与 Premium HDMI（亦即 HDMI 2.0）连接线，还可达到每秒 60 帧。

问：PS4 Pro 是否支援高动态范围成像（HDR）？

是的，PS4 Pro 与 HDR10 规格完全相容。HDR10 是一种非常普遍的 HDR 格式，许多大型娱乐公司都有支援。

问：什么是 HDR？

高动态范围成像（HDR）是一种新兴的显示标准，相较于在传统高画质电视上应用了数十年的标准动态范围成像，能大幅提升色彩深度与对比度。一般来说，若在支援 HDR 的电视上播放 HDR 内容，即可呈现细节更清楚、色彩更丰富的画面，更接近人眼的自然视觉效果。这就表示相较于传统的高画质电视技术，HDR 拥有更深而能保留更多细节的暗部准位、更明亮炫目的亮部，以及更广的色谱。

言语很难形容 HDR 技术的迷人之处，您必须使用支援 HDR 的萤幕与内容才能亲身体会。

问：PS4 Pro 是否会支援杜比视界 HDR？

目前没有支援杜比视界的计画。

问：PS4 Pro 需要搭配什么特殊的设备吗？

除非您想体验 4K 与 HDR，否则您不需要准备特殊的设备。PS4 Pro 和标准 PS4 一样，完全支援目前市面上的高画质电视（720p／1080i／1080p）。若使用 Premium HDMI 连接线（PS4 Pro 随附），PS4 Pro 也可在支援的 4K 电视上输出 4K 讯号。若要享用 HDR 功能，您必须拥有支援 HDR10 格式的高画质电视，以及支援 HDR10 的游戏或内容。

问：除了更优质的游戏体验、4K 电视支援功能以及支援 4K 影片串流外，PS4 Pro 还有其他优点吗？

有的。PS4 Pro 还配有 1TB 的标准硬盘、高速 Wi-Fi（IEEE 802.11 a／b／g／n／ac），且主机后方设有第三个 USB 3.1 Gen.1 连接埠。PS4 Pro 更强大的硬体性能也能带来更精彩的 PlayStationVR 游戏体验，进一步强化分享游玩、遥控游玩等串流功能，并透过 SHARE 按钮输出更高解析度的撷取媒体内容。

在其他方面，PS4 Pro 则大多与标准 PS4 非常类似。这也包括使用者界面与功能。

问：为什么需要使用 Premium HDMI 连接线来连结 PS4 Pro 与 4K 电视？

为了享受最顶级的画质，您必须拥有 PS4 Pro 主机、设有 Premium HDMI 输出连接埠的 4K 电视，以及 Premium HDMI 连接线。PS4 Pro 随盒附上 Premium HDMI 连接线，如果想购买更长的连接线，请务必选择符合 Premium HDMI 规格的产品。

较新版本的 Premium HDMI 格式能以每秒 60 帧（60Hz）的频率输出 4K 讯号。旧版的高速 HDMI 连接线（亦即 HDMI 1.4）基本上也能在 4K 电视上输出 4K 讯号，但效能仅限每秒 30 帧（30Hz）。

问：有关 PS4 Pro 对 4K 电视的支援功能，还有什么其他的注意事项吗？

由于 4K 显示标准与 HDR 功能仍属最新技术，建议您详阅 4K 电视的操作手册与其他参考资料，以便深入了解所有功能。

问：PS4 Pro 是否支援 PlayStation VR？

PS4 Pro 将比照标准 PS4 ，完全支援 PlayStationVR。但值得注意的是，PlayStationVR 内整合的显示解析度最高为 1920 x 1080，无法支援 4K 输出。此外，PS VR 也不支援 HDR 功能。

儘管如此，PS4 Pro 拥有更强大的处理效能，所以 VR 开发人员可以利用更细緻的图像、更顺畅或更稳定的帧率与特定支援游戏的强化视觉效果来提升游戏品质。开发人员也能选择透过下载更新套件，将 PS4 Pro 的功能加入之后发行的 PS VR 游戏。

问：哪些 PS VR 游戏会支援 PS4 Pro 功能？

我们会等到接近 PS4 Pro 发行日期，再开始确认游戏清单。

4K 娱乐

问：PS4 Pro 是否支援 Ultra 4K Blu-ray Disc ？

不支援。PS4 Pro 内建的蓝光光碟机并未支援最新的 Ultra 4K Blu-ray Disc 格式。该光碟机支援的Blu-ray Disc 格式与标准 PS4 相同。但Blu-ray Disc 播放器应用程式则有支援品质升级的 DVD 与Blu-ray Disc 的内容。

问：PS4 Pro 支援哪些 4K 娱乐选项？

我们正在开发支援 Netflix 与 YouTube 的 4K 串流应用程式。

问：PlayStationStore 会推出 4K 电影或电视节目吗？

目前即将推出的 PS4 Pro 并无此计画，但 PlayStation Network 团队已着手研究这个方向。


其他问题

问：PS4 Pro 的实际机身有多大？

PS4 Pro 比 2013 年推出的原版 PS4 主机稍大，外部尺寸约为 295×55×327 毫米（宽 × 高 × 长）。

问：与标准 PS4 的玩家进行线上游戏对战时，PS4 Pro 的玩家会有效能上的优势吗？

一般来说不会。游戏测试与平衡性测试是由各个游戏的开发人员主导，虽然 PS4 Pro 拥有更强大的硬体性能，可提供特定支援游戏更顺畅或更稳定的帧率，但开发人员可以运用许多工具及流程来避免失衡的状况发生。

问：PS4 Pro 有针对遥控游玩或分享游玩做出什么修改吗？

有的。只要网路频宽够大，PS4 Pro 经强化的处理效能就能带来更显着的串流优势。透过 PC、Mac 或 Xperia 遥控游玩的使用者可以从 PS4 Pro 主机以最高 1080p 的画质串流游戏过程；分享游玩的使用者同样也能从 PS4 Pro 主机以最高 1080p 的画质串流游戏过程。

在使用遥控游玩与分享游玩时，PS4 Pro 的适应性串流技术会尝试最佳化串流品质，以便充分利用可用的频宽。如要享有最佳体验，建议连线速度为 15Mbps 以上，且若能以乙太网络线连线，就不要使用 Wi-Fi。

问：以 SHARE 按钮撷取的 PS4 Pro 游戏内容，画质会更好吗？

在游玩支援 PS4 Pro 功能的游戏时，使用 DUALSHOCK4 无线控制器的 SHARE 按钮可撷取到比标准 PS4 解析度更高的游戏影片与萤幕截图。实际解析度因游戏而异，但部分游戏有支援完整的 1080p 影片撷取功能，以及最高 3840 x 2160（4K）像素的萤幕截图。

问：我能以更高阶的设定进行游戏直播吗？

可以。正式推出之后，您可以在 YouTube 上以 1080p 高画质及每秒 60 帧的频率直播游戏。

问：Sony 全新的 4K 电视适合搭配 PS4 Pro 使用吗？

我们目前无法针对 PS4 Pro 提供明确的电视搭配建议。

问：我应该购买多大的 4K 电视？

每位玩家的房间大小与座位距离均有所不同，因此我们无法针对电视尺寸提供统一的官方建议。4K 游戏与电影的解析度最高可达 1080p 讯号的四倍，所以如要体验 4K 一流的逼真效果，4K 电视萤幕的大小应足以让您获得舒适的观赏体验。

如果您想为 PS4 Pro 购买新的 4K 电视，强烈建议您选择与 HDR10 格式相容的电视型号，这样在执行支援 HDR10 的游戏与电影时，才能享有更生动的画面与更高的对比度。

问：如果我在 4K 电视上使用 PS4 Pro，但我玩的 PS4 游戏不支援 PS4 Pro 功能，游戏画面还会以 4K 输出吗？

会。旧版 PS4 游戏即使没有 PS4 Pro 专属功能，只要游玩时使用 PS4 Pro 与 4K 电视，并以 Premium HDMI 连接线连接，仍可享有 4K 画质。

问：PS4 Pro 正面的按钮是哪种类型？

按钮为机械式。

问：如果我同时拥有PS VR、PS4 Pro和4K电视，还有其他注意事项吗？

如果您使用 PS4 Pro 主机游玩一般非VR游戏， PS VR 的讯号处理器将会输出 4K 讯号到 4K 电视，但仅限于 YUV420 的格式。由于讯号处理器不支援传送 HDR，因此您需要直接连结PS4 Pro 和电视（不经由讯号处理器）来观看 HDR 内容。

问：可以把 PS4 Pro 做为「第二个」常用主机使用吗？

不行，您只能选择一台 PS4 主机做为常用主机。

问：可以在PS4 Pro 安装新的硬盘吗？

可以。与标准 PS4 的两个型号一样，您只需要使用十字螺丝起子便能更快速简单地替换掉PS4 Pro 上的1TB、5400 RPM 硬盘。您需选择厚度不超过 9.5 公厘的 2.5 吋（笔记本电脑用的）硬盘。

问：PS4 Pro 的硬盘界面有任何改变吗？

有。PS4 Pro 支援更快的SATA-III 规格。

问：使用 4K 电视会产生更多的输入延迟吗？

不一定。但如果您考虑购买电视来玩游戏，「输出延迟（ input latency 或input lag ）」的现象可能是一项重要的挑选基准。延迟量会因电视而异。许多电视都设计有「游戏模式」或类似机能，以尽可能减少延迟。但这类机能使用的方式可能不尽相同（例如停用 HDR 等特殊机能来达到效果）。若可能的话，建议在购买前尝试在新电视上连接 PS4 或 PS4 Pro，确认整体游戏时所呈现出的感觉。或许也可以参考深入的专业评论或使用者的心得感想。

问：可以在PS4 Pro 使用外接式 HDD 吗？

如同标准的 PS4 一样，PS4 Pro 不支援外接式 HDD 来储存游戏存档。但您可以用来做为主机备份、媒体内容或存档的储存空间使用。

问：PS4 Pro 支援完整的原生4K渲染吗？

是的。虽然PS4 Pro 支援输出原生 4K 讯号，但很多开发人员会致力于运用包含「棋盘渲染」等强大的渲染技术，以让原生 4K 讯号的高清晰度和其他丰富的绘图机能达到最佳平衡，并有顺畅稳定的呈现。
