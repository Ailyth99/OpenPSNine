白金简易流程

白金简易流程

12/05更新　　已打通无尽50层，新增攻略心得，目前只剩刷杯，本帖暂时完工


哟呵，本月会免的圣铠传说来了，对於本攻略或游戏有任何疑问可以在本帖提出，我尽力帮忙，我自己也还正在刷此作白金中，如果本攻略能帮到你，不妨按下感谢，如果你还没注册p9，欢迎加入这个美好的圈子

圣铠传说是俯瞰视角的动作游戏，本游戏注重杀敌走位打怪技巧，并没有打宝或练级的概念，可以本地多手柄或连网1～4人同玩，非常适合随时多人同乐的游戏
这个分页只简要说明奖杯流程，如果你是第一次接触圣铠传说可以点击上面分页　新手导引　了解一下游戏的基本，攻略心得頁面是是我自己通关时的技巧分享
难度4～6/10 视技术而定
白金时间40～60小时 通关时间约5～10小时 其余皆刷
目前版本1.03 几个良性bug已经被修正 无法降版本玩

开坑前须知

● 游戏中所有奖杯进度都可在主选单的Masteries中查看
● 除了无尽模式50层和主线hard通关以外，其余所有奖杯都是刷杯，其中有很多奖杯都是需要单一英雄完成，建议选好一个英雄作为主力专门玩，本游戏没有练级打宝，一开始可以先把四个英雄都玩过再来考虑喜欢的
● 死掉或退出关卡依然会纪录杀敌数等奖杯进度，但钱不会保留
● 如打坏500份食物、打坏5000件物品这些刷杯，平时游玩的时候就可以随手解，减少以後要刷的次数
● 人越多怪物越厉害，平时1～2人会打的比较快

1.02版本之前有买东西不扣钱和巫师隐形bug，其中巫师隐形对於通关无尽模式50层有很大帮助，现版本1.03皆修正无法使用

（1）
首先直接hard难度通关，解开全部hard奖杯，因为本游戏并没打宝练级，高难度只需要用技术来克服，所以可直接hard走起，而hard也没真那么难，主要先在第一层熟悉角色操作，游戏要到第三层（熔岩关）後才会开始真正难起
注意全部关卡包含了分支关卡，因为游戏中不能查看关卡通过时的难度，所以建议全部用hard打
不缺食物时可以把食物打坏，关系到单一英雄打坏500食物的奖杯，另外路上的瓶罐箱子等物件，随手经过也打破下，关系到打破5000物件奖杯，可以让後面刷杯之路轻松些
解开以下奖杯
The Undeath of Khamun
杀死地穴的BOSS
剧情流程杯，这个Boss不能直接攻击它，要去踩房间两侧的按钮，转动中间圆盘，圆盘刻度变为红色後，再踩按钮圆盘会反方向转，转到它满意了Boss的心脏就会出现，攻击心脏打倒Boss，两人以上打会轻松不少，一人引Boss一人踩按钮
要小心Boss踩到人是直接秒杀的

Cave Crusher
以困难难度完成全部山洞关卡
Morak Defeated
杀死神殿的BOSS
Hard难度下很难的Boss，非常建议找人一起打，我用女武神单打过了，也死了近一小时才过，後来试了4人网战一次就过了
1.他首先会开几个传送门叫出很多小怪，一边干扰玩家，这时能杀多少怪就杀
2.一段时间後他会分身成很多只，并且他的大剑会向你砸过来两次，注意躲好这一波大剑，插中基本秒杀
3.然後他的分身会逃跑，攻击分身会让分身消失掉，找出本尊打掉他15％左右的血，他就会回到1.
4.他的血量到一定程度下，他会开始放火圈，看见注意跑远或格档
小心他分身阶段，时间拖长了他的大剑会再次砸过来，次数会增加到三次，别被插中了

因为这个boss战就是大量小怪围殴，一人打无暇顾及boss和小怪容易被包围又找不到机会打boss，组队打分工合作会明显轻松很多，真要单打建议带隐形圣物或招唤幻影圣物，可以制造一点攻击空档

（2）
接着是无尽模式50层，这个奖杯比较难，是本游戏的难点，推荐1～2人解，人再多30层以上的怪物会太强，一些打通50层的心得我写在上面　攻略心得　分页内
We need to go Deeper - Bronze
成功在无尽模式1到10层生存下来
We need to go Deeper - Silver
成功在无尽模式1到25层生存下来
We need to go Deeper - Gold
成功在无尽模式1到50层生存下来
难度挺高，从30层左右就会开始变得非常难，在这个模式中退出游戏，金钱和传送门会保存下来
当到达19层和37层时游戏会提示传送门已打开，以後就能从1层的传送门直接传上这些层，使用传送门一样可以解50层奖杯
建议先一次打到37层，一路上的钱多买药水和复活币，到37层之後建议先休息（一路打过来至少也要一两个小时，容易疲劳，而且经过30～37这几层通常已经被虐成狗了），之後重新挑战可以从1层打到19层，在中途尽可能的赚钱买复活币，然後从19层传送到37层打完50层，是比较保险的作法
要注意奖杯每一层都要打过，不可跳过（如加入其他人开的游戏房间直接打後面楼层，奖杯不会解锁）

如果想省时间，多数楼层都不会锁门，可以不打怪直接往前走，只打会锁门的房间即可

这里简单说明装备效果，和一些个人的心得

--武器

战士：
1.旋风斩 移动不太好控制 威力中等 单刷可选
2.怒吼 敌人优先打你 变远程 多人时不错的选择
3.剑气 威力和范围都不错 单刷好选择
4.震地 cd长 不如剑气

女武神：
1.丢飞盘 飞行路线有时很奇怪 技能使用中不能格档
2.冲撞 cd非常短可用来赶路或走位 小范围攻击 变远程 多人时不错的选择
3.回旋斩 威力和范围都不错 单刷好选择
4.雷霆一击 威力差范围小...很糟糕的技能

巫师：
1.基本法术书 一共九种魔法
2.换掉火系三种魔法
3.换掉冰系三种魔法
4.换掉雷系三种魔法
因为巫师我还用不上手，关於书的部份心得不全，巫师非常脆，我感觉比较适合多人玩，前面有别人挡着，巫师就能发挥很大火力
基本法术方面，单刷的时候可以多用冰冻光线，被近身就冰锥控场，或是火焰冲刺来穿过敌人，火焰冲刺附带燃烧效果，可烧死大部分怪，若是有人作肉盾可用大火球，威力和范围都很不错
出现生怪点的时候可以到生怪点旁放闪电盾暂时挡住所有怪攻击，然後直接贴着生怪点用冰冻光线贴着射，伤害挺可观，遇上会冲锋的骷髅盾兵和大兽人可以用震退或是用冰锥冻住，但是巫师还是很难一人对付这些冲锋怪...建议找人一起玩

精灵：
1.炸弹/爆炸箭 伤害不错 挺好用的
2.绑人炸弹/捆绑箭 没伤害 能绑住敌人 不如直接有伤害的炸弹来的快 而且精灵又缺乏范围伤害 一般不选这个
3.烟雾弹/钻箭 隐身很不错 cd很短 安全性很高的技能
4.跳扔炸弹/榴弹 我感觉不太好使...

先来个残忍的剧透，一开始的邪恶黑巫师竟然就是最後的boss！怎么就没人猜中！




无尽模式50层

这个的确比较难，一定要熟悉自己的英雄，圣物先搞到三级，装备齐全再来打

这个模式几个重点一定要把握住：
● 只有部份房间是会锁门的！玩多了一看房间样子就知道会不会锁，其余不锁门的房间都可以直接跑过去，某些房间是走跳台离开的，直接跳台走起，别理怪物是上策
　总之进房间看见大群怪物别慌，很多房间能跑，跑的时候别硬顶，合理放圣物和技能开路，或者选择隐形圣物直接穿过
　部份房间捡三把钥匙就能开门，还有木门挡着的可以直接打破再跑
● 一般路上的小怪建议都打掉，它们不会重生，顺便刷奖杯，打小怪求稳健，别伤血就行，刻意不打怪反而容易被包围打伤
● 魔药保存好，平时维持四瓶～五瓶，遇到比较难打的锁门房间再用
● 这个游戏的难点就在那些生怪点，就是像墓碑或图腾柱的东西，生怪点不打掉，敌人越来越多，总会寡不敌众
　生怪点只会在玩家一进房间时才会出，可以在房间外等圣物和技能cd好了再进房
　如果场上有超过两到三个生怪点，就可以考虑丢魔药技能了
　一般房间里面都会有食物或魔药，尽量合理运用这些东西
● 赚来的钱优先买复活币，後期可以考虑买点魔药或食物
● 死神的关卡，遇到锁门房间不要急，带着死神溜几圈，冷静打完，别为了贪打或捡钱被死神抱了，不划算


（1）
第一次打无尽模式的人，建议直接从1打到37层然後退出游戏

无尽模式的楼层，每六层换一种关卡
也就是说，第1～6层是地穴关卡，一般有木乃伊和骷髅，比较简单的
第7～12层是山洞关卡，一般是蜘蛛和兽人，地图会比较黑，如果遇到蜘蛛和兽人同时包围会比较危险，通常山洞关卡能捡到比较多的魔药
第13～18层是熔岩关卡，熔岩关卡是最难的，很多会自爆的教徒和会锁定法术的邪法师，比较棘手
第19层又重新开始地穴关卡，难度会提高一个档次，并且有一传送门连接第1层，当你踏入19层，传送门就永远开着了，以後再重新玩无尽模式也能从第1层传到这
接着19～24地穴，25～30山洞，31～36熔岩，其中31～36非常非常困难，房间里面通常有大眼球怪和非常多的敌人，游戏从这里难度提升更多，建议前面多赚钱买复活币，来到30层时尽量要有20枚以上的复活币，以便撑到37层
抵达37层开了传送门就可以退游戏

（2）
接着攻略50层，建议从1打到19层，赚钱买复活，再从19层传送到37层打完後面，大约要有15个复活比较保险

首先说说理由，因为1～19层的怪物相对简单，打的谨慎点一般是不会死的，可以有效赚复活币，要注意第9层、第18层是死神关卡，别被抱了
再来传送到37层，跳过了30～36这几个非常困难的熔岩关卡，大大省事

37～42层是地穴关卡，难度当然比之前提升，而且视野会变得很黑，从这里开始记住，怪能不打就不打，没锁门的房间一律猥琐逃跑，带隐形圣物也行，锁门房间合理放魔药打通，尽量别死，一层死一到两次为限，当然能不死最好
锁门房间，第一眼先找白骨堆（会出女鬼的），第一时间打掉，然後再对付生怪点，多多利用会自爆的木乃伊可以省下很多事
42层是死神关卡

接着43～48的山洞关卡，我打的时候，路线都非常短很快能通过一层，估计这几层就是刻意设计成这么短，没锁的房间都跑酷通过，有时一层只有一房间，还没锁，直接跑过就是爽
视线非常的黑，走在路上随时看着有没有怪
48层是死神，这里我遇到锁门房间了，用了一瓶魔药，死神出来时记得溜下，别被抱，到这里我还有18次复活，非常足够


接着只要打49就行，也就是说，用这条路线我们只需要打仅此一层的困难熔岩关（前面13～18是相对简单的熔岩关），49层会有很恶心的眼球+众多恶魔+众多邪教法师一起上，这时候别省魔药，会死是一定的，到这边尽量保存5～10枚以上的复活币，就稳操胜算了，堆屍也能过

进到50层就跳杯了，不需要打通50层




女武神一些技巧
我自己用了女武神打通单机困难和无尽50层，这里说下我对这个英雄的心得

- 很重要的关键...猥琐格档...
　除了少部份不能档的攻击，像是邪法师的锁定魔法，其他都能档
　反弹魔法很好用，真的非常好用，杀飞行恶魔、木乃伊法师都非常方便，大眼球有一招火球散射，最後的几颗能挡回去伤害他

- 推荐的武器是第三把，回旋斩，范围大，冷却时间短，伤害也足，被打中的大型敌人会出现硬直，能开路制造空档，有时被包围想办法背墙格档等cd也是非常好用的战术
　使用时是无敌的，可以拿来躲招救命，这一招也是女武神无敌时间最长的技能了

- 平砍对一般小怪能造成很长时间的昏眩，至少1秒，被小怪包围时试着左右砍，顺利的话小怪全都能被你控住
　平砍第四下和第七下是强力攻击，砍生怪点的时候尽量合理运用

- 冲刺攻击能秒杀法师敌人，但是准备动作很长，如果身边有近身小怪别用，肯定被打的不成人形
　可以先平砍身边小怪一下，让它们昏眩，就能安全冲刺了
　冲刺完会有小段时间的硬直，这时很容易被打，可以用武器技能解围
　总之把握原则，身边有敌别用，也别用这招冲进敌群，除非你有回旋斩保命

- 邪法师的锁定魔法非常讨厌，尤其你身边又有许多小兵包围，一下过不去邪法师的位置，可以先用回旋斩开路再冲刺过去
　如果被小兵包围又想着要冲刺法师，你会先给小兵揍死，善用走位，绕开小兵直接到邪法师旁平砍也能中断它
　总之，邪法师尽量先杀

- 圣物推荐幻影圣物（最後一个），开路、制造空档、打生怪点都非常实用

- 魔药技能，招唤3女武神输出非常强力，但是可能会不小心打坏食物或魔药，而且敌人的AI很聪明，叫出女武神的时候，法师会立刻施法保护生怪点，女武神可能会呆呆的攻击受保护的塔而完全没输出
　无尽模式比较推荐用无敌技能，虽然我是用女武神技能打过的，虽然打的过程中女武神老是打坏我的烤香鸡...

- 连机可以考虑用第二把武器，远程攻击还不错
