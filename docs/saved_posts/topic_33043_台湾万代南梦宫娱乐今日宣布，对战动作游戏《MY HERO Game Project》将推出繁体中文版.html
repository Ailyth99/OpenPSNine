《MY HERO Game Project》游戏简介



现正于「周刊少年Jump」人气连载中、确定播映第三季电视动画的《我的英雄学院》，将以正宗动作游戏之姿上市。

《MY HERO Game Project》是一款以各角色的「个性」展开激斗的正宗对战动作游戏，

无论是在平地上写实的场景破坏、或是以高楼大厦为舞台的空中激战，将带给玩家各式各样的对战乐趣。

关于本作的详细资讯将陆续公开于官方专页，敬请期待。
