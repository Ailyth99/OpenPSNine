PS4PS3《人中之龙 0 誓言的场所》公布游戏舞台「神室町」「苍天堀」游乐设施介绍

sai8808 2014-11-04 15:10编辑     1评论
SEGA 于近日（2014年10月31日）公布PS4 ／PS3《人中之龙 0 誓言的场所》故事中舞台的东京「神室町」与大坂「苍天堀」的相关情报！

累计销售数量已突破 650 万片的《人中之龙》系列。为系列最新作品的 PlayStation 4 ／ PlayStation 3《人中之龙 0 誓言的场所》，公佈了关于本作品内的主角・桐生一马与真岛吾朗各自活跃着的两个舞台，位于东京的「神室町」与大坂的「苍天堀」的情报！

本作品是以 1988 年因前所未见的良好景气而十分热络的日本为舞台，请玩家们务必关注于至今为止没有人体验过，在系列史上最为激动且刺激的两个街道。

街道里的焦点

1988 年与现今（于《人中之龙 5》裡所登场的 2012 年的街道）相比，面貌可说是完全截然不同的神室町与苍天堀。将充斥着灯红酒绿的慾望，展露出人性本能的街道，在经过了缜密的时代考察后，将可随处观察到製作团队连细微部分都想要清楚呈现的坚持。

为街道点缀色彩的霓虹灯招牌及让人感到目眩神移的灯光。四处被丢弃的杂志及报纸等垃圾。街道与空间被侵佔十分溷杂的气氛，更为显着的展现出 1988 年时所仅有充斥着热情的街道。

仔细观察垃圾袋大多为黑色及水蓝色等色系，充斥着现今所消逝无踪的事物。




神室町




苍天堀

比起《人中之龙 5》裡的街道更为华丽，更加奇特。再藉由加上散置于各处十分怀旧的物品，将其作为将 1988 年当时的世界观变得更为刺激的要素于游戏内登场。


《人中之龙 0》神室町 剧场前广场


《人中之龙 5》神室町 剧场前广场

其他还有许多散置于各处的要素喔。

卡拉 OK 饮食店

为系列中玩家们所熟悉的人气娱乐场所「卡拉 OK」，在以饮食店的全盛时期做为舞台的本作品裡以「卡拉 OK 饮食店」的姿态登场。

保有着奇特般的伴奏手法及正统具备高质感的歌曲，配合时代设定对各种要素进行了重製。



能够高歌一曲的店家也不再是「卡拉 OK 包厢」而变化为「卡拉 OK 饮食店」。

除了能够充分体验饮食店独有的成人般氛围外，还能放声演唱具有怀旧风味的歌曲。也请玩家们注意以当时十分盛行的「LD 卡拉 OK」风格的细部演出手法喔。



由桐生及真岛所热唱的新曲也製作为充满着当时的氛围，且十分华丽的歌曲。

当歌曲让全场沸腾起来时，也能一窥变身为乐团歌手的桐生以及成为偶像的真岛的演出！十分值得玩家们欣赏！

迪斯可

可说是象徵着闪耀的 80 年代的娱乐场所、「迪斯可」。在本作品裡为全新登场可供玩家游玩的设施。当然除了能够一个人享受舞蹈的乐趣外，还能搭讪在迪斯可裡的女孩子们，一同共舞。

除了能够饮食的吧檯外，还准备了只有常客才能进入的 VIP 房间，也能体验到除舞蹈外各种不同的乐趣。



在迪斯可内所进行的游戏是将在画面上所出现的「可爱角色」，配合迪斯可内音乐的节奏输入指令使之移动，变为风格有些奇特的节奏游戏。如果能顺利输入，桐生与真岛将会施展出有如职业舞者般的舞技喔。



如能展现出流畅的舞技来提升观众们的热情指数，将会触发能够施展出特殊舞蹈的「FEVER 时间」！与观众融为一体并沸腾起来的感觉，有如实际上在迪斯可裡享受乐趣般。

游戏中心「SEGA 高科技乐园」

从抓娃娃机到大型电玩，能够游玩各种不同种类的游戏，在游戏中惯有的游乐设施「游戏中心」也配合时代设定，变更为当时 SEGA 旗下系列游戏中心的主流「SEGA 高科技乐园」。忠实重现当时游乐中心的气氛，保证能使老玩家会感到十分怀念，年轻玩家也能感受到其新鲜感喔。



能够游玩的新作游戏的阵容为，居然有四种！完全实装了于当时获得大幅迴响，风靡着整个世代，代表 SEGA 的大型电玩游戏《太空哈利》、《幻想空间》 、《超级摩托车》、《OutRun》等作品喔！能够游玩到四款游戏的大份量内容喔。


《OutRun》


《太空哈利》


《超级摩托车》


《幻想空间》

以及也准备了与身为游戏达人的宿敌们相互比拚游戏内最高得分的事件。鑽研游戏并更新最高得分，目标成为神室町、苍天堀的最强游戏达人！游戏中更有着以实际店家身份登场的「新宿运动乐园」以及其招牌。充满了有如让当时的热情复甦般的景象。
