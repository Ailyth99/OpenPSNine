PSV《圣诞之吻 加强版》一周目初玩心得

rcalibuk 2014-02-16 08:49编辑     7评论
哇哦～学姐真是太可爱了～原来也有这种的反差萌啊！



由于第一次接触这款作品，所以多少会有些美化，若玩过旧版的玩家，基本上99%都是一样的。

拼了两个晚上打完一轮，最主要的感想是：

eb之后的 恋曲写真 在游戏体验上退步很多！

系统方面基础上是相同，不过恋曲写真的随机性高出许多（不使用果音进行攻略根本不可能）

以游戏性来讲或许是进化，但是很血尿...

使用 3D 模组在 PSP 无印的读取速度也是令人摇头（不使用果音又积极的 S/L，哇哦～简直不是人了吧？）

即使后来 PSV 的 Kiss 读取，和本作相比也是差好几条街的规格，

3D 虽然可以增加表现力，但处理不好也会拖累性能。

本作除了人物表情动作之外，使用生动的演出来弥补表现力的不足，

场景的 camera work 也相当频繁。

小人玩过的恋爱模拟/养成/文字冒险类型的游戏还不够多，

所以不太清楚还有哪一款 2D 也是如此，至少在本作是第一次看到^^a

主要是演出手法个人认为很有趣 : )

缺点大概就是这款是纯移植加特典，并非重制版（虽然特典算不错...可以拿来自製PSV桌面）

画面只是把4:3放大裁切成16:9而已，有些可惜，不过实况台的观众说有变好？

还有大部分恋爱模拟类型的共通毛病：感情专一的话攻略途中会有很多垃圾时间，

多少会有些不耐烦，虽然直接跳过可以很快但又觉得浪费 : P

无触控操作，其他没有太大的问题（读取快真是超爽der）

以前只有听说是经典名作，玩过之后才知道不是骗人，

不下以前第一次玩完 DCCzero 时的感动（所以半夜才写了肺雯抒发情绪^^a）

模拟类型与纯文字冒险类型比较不同的是，玩家能做的选择会比较多

甚至可以在结局前放弃整轮的心血（？）

其实现在恋爱模拟/养成感觉越来越少了，喜欢这种类型的朋友值得一玩的作品（尤其是只有玩过恋曲写真＋KISS的玩家...）

本人也知道许多玩家看到 PSV 又要推出美少女游戏就先叹气再说，

其实大部分是看不懂、不了解，自然就会比较少接触，

虽然大都脱离不了校园故事，但看过每一个剧本之后总是会回想起当年勇（？）

或者是感觉自己的青春怎麽这么空白，这也是校园故事这么被常套用的原因：几乎大家都上过学校嘛 : )　自然代入感会比较强烈。

在今年的电玩展之后有越来越多的中文化，

只是这样类型的游戏目前还是很少有中文化的待遇（大概是连神作 S;G都卖不好...）

觉得最需要中文化的应该是这种类型，

期望以后可以让更多人认识其实这类游戏也是不乏优秀作品。
