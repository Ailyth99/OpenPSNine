PSV《俺尸2 跨越俺的尸体前进吧》中将自家分享給其他人

kcf1201 2014-09-04 21:37编辑     1评论
这次续作最重要的一个新元素就是加入远征功能，让大家可以到其他国家参观 / 探险
但到底要如何利用这个机能让大家可以分享自己的国家 / 取得其他玩家的国家的资料呢？下面就讲解一下方法了
P.S. 由于这游戏要截图很多时都只能直接上传到社交网络，还有限制短时间不能上传多张图片，所以以下图片都是使用手提电话的相机拍摄

首先在自家的内政介面 (?) 选一下设定


之后选一下ネットワーク设定


进入后，可以看到以下选项
1) SNS利用设定
就是控制使用右下角相机按掣截图后如何分享相片至社交网络

2) 自国的公开方法设定
这篇文的重点
只要将这个选项设定到"PSN"，就会自动将自家的资料上传至PSN网络，以供其他玩家取得了
当然选了这个选项后也能取得其他已经上传了资料的玩家的资料呢

3) 庆弔情报的送受信设定
应该是当自家有人出生 / 逝世时会自动发送讯息给其他玩家
这个选项如果要生效的话就一定要将自国公开才可以


只要设定好后，在出阵后选船作远征就可以看到其他玩家的国家了 (你可以看到有一个 ID 在国名下)


另外，于选择远征国家时除了可以选择系统自动取得的国家资料外，也可以手动输入远征哪一个国家
方法就是于上图选择远征国家时按一下三角形掣，就会出现下图的画面
1) キーボードで入力
就是让大家手动输入其他人的ID

2) フレンドリストから选択
就是让大家从好友列表中找寻朋友，免去手动输入ID

3) 自分のIDを入力
这个应该是让你看一下自己的ID是甚么
相信应该不会搜寻到自己的国家吧？ XD

4) 检索开始
就是使用各位已经输入的ID进行搜寻了


注意事项：
1) 断线后要再到设定中重新设定以PSN公开自国资料才可以更新其他国家资料和使用手动输入远征国家功能

以上就是将分享自己的国家 / 取得其他玩家的国家的资料的方法
希望大家能顺利用到这个功能，享受到这个游戏的乐趣
