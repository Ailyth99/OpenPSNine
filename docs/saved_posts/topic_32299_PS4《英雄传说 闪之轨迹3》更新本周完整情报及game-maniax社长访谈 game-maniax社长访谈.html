《闪之轨迹III》社长访谈重点总结：

1、光是参加战斗的人数就有25人以上，加上共同行动的角色和敌方势力，制作模型做到焦头烂额。
2、按制作《空之轨迹》时的构想，到闪之轨迹系列已完成整个故事的60%以上
3、包括冒险范围、登场角色、故事的容量在内，就舞台规模来说“闪之轨迹”系列在轨迹系列中是规模最大的。
4、这回主要描写帝国西部，也会去成为帝国领土的克洛斯贝尔，严格来说是西部+α
5、故事前半讲述：前作最后结束内乱，宰相在清理地盘的同时，又急需对新吞并的克洛斯贝尔、敌国卡尔瓦德共和国以及情势不稳的帝国内部做出相应对策。在这样的背景下，托尔兹第II分校成立，通过去各地“实习”解决各种事件。
6、为什么到目前为止没有出现罗伊德等人的名字，在闪3中去了克洛斯贝尔就会明白其理由。除了罗伊德之外还会得知艾莉和琪雅、塞尔盖科长、《风之剑圣》亚里欧斯的情报……。
7、这回故事的中心是《VII组》，主人公是黎恩，在黎恩的带领下，悠娜、库尔特、亚尔缇娜等人，以及后面加入的亚修、缪洁组成游击部队在各地活动战斗。现《VII组》因日常活动战斗辗转各地时会有旧《VII组》成员加入我方，也会有其他角色临时加入我方小队。可以看到新旧《VII组》分成两队共斗的场面哦。新旧VII组成员会共有部分核心回路。
8、故事进行模式基本上和前作一样。利用日期制“自由行动日”寻访街道。只不过前往演习地架势更大了，第II分校全员乘坐专用列车前往目的地，相当于是整个“学校”的移动。没有了平民和贵族的对立，大家团结一心进行演习，这也许是最大的变化吧。
9、会描写《空之轨迹》系列全场的约修亚及原帝国贵族卡普亚一家的过去。闪3全篇剧情密度非常高。
10、本次剧本以“部”作为单位，每一部容量都很大，第一部很长，第二部更长，第三部还要长……（一部比一部长），加上支线和羁绊事件，通关时间难以想象。
11、PS4上S技的演出加强很多，顺便一说，前作的角色在本作中参战，以前的S技会变为普通战技，这样可以表现他们的成长。
12、关于今后轨迹系列的展开：一直以来谜团重重的“卡尔瓦德共和国”渐渐浮出水面，我想差不多到了要描写共和国的时候了。在技术方面会有和帝国的“ARCUS”不同体系的导力器，我们会为玩家送上不同感觉的东西。
13、前作中出现的“黑色史书”的谜题将在本作中解开。
