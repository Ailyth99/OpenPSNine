奖杯白1 金3 银11 铜24 总39
白金时间10h+（不跳剧情）
是否有网杯有

本作奖杯看似简单，实则内里隐藏大坑（主要是联机方面），不过好在现如今几乎已无人问津，排行赛已变成了无脑刷刷刷即可解，所以也就不是什么难事。
为什么说几乎呢，因为排行赛偶尔还是能匹配到野人，不知道这群人是真的要拼排行吗

以下是推荐顺序及原因，后续的攻略也根据该顺序列出：

1. 教程
- 本作刚开头的基础讲解结束以后，就再也无“强制性”的教学，如果直接奔向故事模式，就会发现：
怎么AI放的招，我都按不出来呢？？
怎么AI能放大，我就放不出呢??
怎么按键操作就这么基础的几个，除了特殊技就没有其他出招表了呢？？
...

- 另一个必须完成才能获取到2个【卡片背景】，所以对于白金来说还是属于“强制性”的

2. 故事模式
- 最后一个【卡片背景】的获取条件很简单，通关一次协力任务并达成S级；但必须通关故事模式4-1后才会出现在【卡片背景】栏（即使已经有达成S级的协力任务也不会直接获得）

- 故事模式5-1通关后才可解锁最后一个当妈的个人/协力boss任务

3. 协力/排行赛
重点杯相关均在此

4. 单人任务
各角色200击杀称号、所有任务S相关等

5. 其他
根据实际情况补足即可，比如各角色特殊技等

排行赛杯

排行赛杯

排行赛Rank Match现在几乎无人，只需2人互刷即可，快速互刷方法：
1. 2人都选择黑子，开始后2人使用+开启所谓 Hyper Mode，血量直接变为3/10，似乎还有攻击力加成...之后要么无脑R2击杀，要么绿条快没了来一个CW；约10秒一轮，20秒一局，加上其他零碎时间，大约1分钟左右一局（可能更长）
2. 一人赢一局，2局下来净挣70+分，初始100分，总共需要8000分，总共大约需要200+局

排行赛可获取：



網路連線專家
累積3000點等級點數。

完全比賽
已達成在1回合內不被對手得到點數並獲得勝利。

完美勝利
在RANK MATCH中擊破敵方取得1回合勝利。

排名戰　初嚐勝果
在RANK MATCH中獲得1次戰鬥勝利。

排名戰　熟練者
在RANK MATCH中獲得10次戰鬥勝利


由首页 TERMINAL 切换至 REPLAY LIST 可回放，任一录像倒计时后按X退出重复5次即可
参与了排行赛才有REPLAY LIST
重播　播放
播放5次重播影片。


以A、B代表玩家，A玩家需获得此杯，都选择黑子为例：
A先惯例赢一局，第二轮开始后，B玩家不动（不开Hyper Mode），A玩家击倒B拿到点数后,A玩家开启Hyper Mode，然后不动，等待时间结束。
虽然实操的时候不是这样，但意思就是HP（黄条）少，依靠点数胜利
逆轉勝利

协力杯

协力杯

1. 两人开坑，协力杯结束前，其中一人至少通过故事模式4-1，其中一人故事模式全通（该玩家开房、选择/解锁任务）
2. 协力任务模式（借用港中翻译）：
- 标准模式
- 殊死战模式
- 街机战斗
- 挑战战斗： 最烦的是难度3、难度4和上面一个模式要打7波，7波不可怕，可怕的是打着打着就掉线...一轮下来至少半小时
- 爆裂·代码
- 毁灭模式
- 致命攻击
- 随机模式： 仅难度4存在
- BOSS战： 仅难度3存在，随剧情模式逐渐开放，一共11个（含Another、BossRush），其中打败最后一个BOSS后才解锁Another，再打败后才解锁BossRush

除BOSS战、随机模式外，其他模式乘以4个难度等于28个，加上BOSS战、随机模式一共是40个
3. 大部分关卡建议选择炮姐（美琴），R1和CW（L2+R2）伤害非常高
4. 除了下面提到的【爆裂·代码】模式，其余放到单人任务中说明

协力任务可获得：
通过4-1后解锁的背景及获取条件

单人杯

单人杯

1.. 单人任务模式（借用港中翻译）：
- 标准模式（1V1）
- 标准模式（2V2）
- 殊死战模式
- 街机战斗
- 挑战战斗
- 毁灭模式
- 致命攻击
- 随机模式，仅难度4有，同协力任务
- BOSS战，仅难度3有，同协力任务
总任务数量和协力任务相同也是40个。

- 单人+奖杯： boss任务中不进行维修（pit in，仅BOSS任务中一个随机出现的可回复HP的小区域），低难度自然获得
- 协力： BOSS伤害比较高，尽量不要被打到、提高自己的攻击命中率、能打多快就多快（主要针对Boss Rush）
BOSS戰　高分
已在任務的BOSS戰中不進行中途維修就通關。


- 单人+ 奖杯： 难度1，选择茵蒂克丝+天空猎场1，因为除了追踪的攻击，其他几乎打不到人，开局炮姐贴脸CW，再来2-3个R1就结束了
另外需要强调的是： 不需要2轮都无伤，第二轮无伤就可以了！殊死战模式同理
- 协力： 正常通关即可（需S评价）
標準戰鬥　高分
已在任務的標準戰鬥中不受損傷就通關。


- 单人 + 奖杯： 太简单以至于想扔到流程杯了
- 协力： 正常通关即可（需S评价）
挑戰戰鬥　高分
已在任務的挑戰戰鬥中一次也沒讓對手取得回合勝利就通關。


- 单人 + 奖杯： 同标准战斗，只是周围多了一圈伤害区域，还选择天空猎场的话，总容易跑到伤害区域，于是地图换成祭典会场2，方法同标准战斗，不需要2轮都无伤！
- 协力： 难度1-3选择了2只茵蒂克丝， 难度4发现开局半血打不过了也算没摸出最弱的，后来选择了神裂(也不一定是最好的选择）
殊死戰模式　高分
已在任務的殊死戰模式中不受損傷就破壞敵方。


- 单人 + 奖杯： 60s内KO即可，应该和无伤杯同理，只需要第二轮达成即可
- 协力： 正常通关即可（需S评价）
隨機鏡像戰鬥　高分
已在任務的隨機鏡像戰鬥中於倒數60秒內通關。


- 单人 + 奖杯： 任意一个难度全程选择不加成（调校？）即可
- 协力： 正常通关即可（需S评价）
街機戰鬥　高分

- 单人 + 奖杯： 正常打就行...正常打
- 协力： 正常通关即可（需S评价）
毀滅模式　高分
已在任務的毀滅模式中於30計數之內通關所有WAVE。


- 单人 + 奖杯： R2无脑秒杀但最好全程移动； 打到第10波就停，因为第10波能S，而且是新的一轮开始，场上只有一个敌人，不容易被击中（作死除外， 难度4要求一绿条都不能掉），低难度可换不同角色同时刷200击杀
- 协力： 同上
