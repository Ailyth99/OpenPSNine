《高达破坏者3》中文版将于3月3日与日版同步发售！

sai8808 2016-03-03 09:23编辑     58评论
在今年1月份举办的台北电玩展2016中BNEI宣布《高达破坏者3》中文版将于今年3月发售。现在PlayStation HK公布，《高达破坏者3》将于日版在3月3日同步上市！



《高达破坏者3》是自由组装高达并进行战斗的动作游戏「高达破坏者」系列的第三部作品，登陆于PS4及PSV双平台。本作的剧情方面一改前作战争剧，转为动画创战者似的热血青春剧。系统方面大幅进化，武器配置更自由。同时也新加入SD高达，可作为僚机战斗并可安装独有的SD模块。此外游戏还会追加了赏金猎人模式，玩家可上传自制钢普拉并成为“赏金犯”，其他玩家击败此钢普拉就能获得游戏货币等的奖励。



From.PS FB HK
