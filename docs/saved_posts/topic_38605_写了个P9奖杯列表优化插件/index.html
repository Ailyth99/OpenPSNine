kuusoumenreiki 06-20 16:26编辑     40评论
由于索尼把PS5版游戏的图标形状由矩形改为了方形，导致网站奖杯列表显示不同版本奖杯时列表中的各游戏显示高度不一致；

强迫症看着老难受了，所以在AI的帮助下，尝试写了个插件进行改善。

用mac的safari浏览器的同学，可以直接修改自定义样式表：

/* p9 */
/* 游戏列表pspc图片高度 */
body.bg tr:has(.pf_pspc) img.imgbgnb {
height: 91px !important;
}
/* pspc标签样式 */
body.bg .pf_pspc {
font-size:11px;
color:white;
background-color: black;
border-radius:2px;
padding:2px 6px;
margin-right:4px;
font-weight:300;
}
/* 游戏详情pspc图片高度 */
body.bg:has(.pf_pspc) img.imgbgnb.l {
height: 91px !important;
}

窜改猴在safari上居然是收费的～
