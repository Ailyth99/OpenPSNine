PS4PSV《NEW GAME ! –THE CHALLENGE STAGE!–》发售决定

rexxun 2016-07-21 14:23编辑     26评论
根据得能正太郎漫画原作所改编的人气动画《NEW GAME!》日前宣布游戏化，游戏名称为《NEW GAME!–THE CHALLENGE STAGE!–》，登陆PS4、PSVita平台。
制作公司为5pb. 预计将于2017年1月26日发售，游戏类型为ADV，CERO有待审核，目前开发进度50%。游戏售价7300円(不含税)/7884円(含税)，限定版价格为10300円。限定版同捆特典为青葉SD手办和游戏原声CD。早期购入特典为换装DLC。

游戏的主人公名为涼風青葉，场景设定为游戏公司，动画中所描绘的充满魅力的女性角色都将于其中悉数登场。涼風青葉为了完成“突然决定制作的RPG大作”的DLC的工作，将会跟随前辈们学习各种技能。每位角色都有自己擅长的领域，决定青叶跟随哪位前辈学习将会大幅影响游戏参数。这是游戏的养成部分。

另一部分则是冒险，故事的日常将作为ADV进行，与前辈、朋友畅谈的同时，亲密度也会根据所选的回应进行变动，选择最恰当的回应将会大幅提升亲密度。随着青叶的成长，故事的后半部分也将产生不同的角色路线，进入关系最为亲密的前辈的个人章节。
