PS4《死亡终局 轮迴试炼》上手心得文 — 从疯狂的 VRMMO 世界脱逃

sai8808 2018-04-14 10:00编辑     83评论
Compile Heart 旗下针对 “日本特定客群” 的游戏品牌「加拉巴哥（ガラパゴス）RPG」第四款游戏，PlayStation 4 专用游戏《死亡终局 轮迴试炼（Death end re;Quest）》已在 2018 年 4 月 12 日发售。



剧本找来以《尸骸派对（コープスパーティー）》系列闻名的 祁答院慎，从这就能推测出本作会是款「让女孩子遭受恐怖及苦难」，加入许多恐怖惊悚风味的 RPG。。

本作打着「Bad End 率 99％」这种惊悚口号，究竟要让人见识什么样的疯狂与绝望呢。在此将一边避免洩漏故事主要情节，一边为玩家们送上游玩心得。



左起为普通版与限定版的实体包装插图


逃出因 Bug 陷入疯狂的 VRMMORPG，穿梭游戏与现实解开谜团

集合了顶尖 VR 技术实现 “意识的完全投影”，却因总监失踪而中止开发的 VRMMORPG「奥德赛世界（ワールズ・オデッセイ）」（以下简称 WO）。过去曾参与开发的程式设计师．水梨 新（みずなし あらた），发现理应在 1 年前停止营运的 WO 如今竟然启动了。经过一年时光的 WO 世界被大量 Bug 侵蚀，而且 Bug 宛如有自我意识一样，改写了整个游戏世界。

伺服器内，还发现了理应失踪的总监．二之宫 椎菜的帐号。新与椎菜取得联络，得知她处于无法自行登出的状态。

为了成功登出因 Bug 陷入疯狂的世界，椎菜与新开始朝游戏内的 “结局” 迈进……。这便是本作的故事大纲。




游戏内的椎菜失去了部分的记忆。随着游戏进行，将逐步取回开发者时代的记忆



旅行途中，游戏内登场的 NPC 也会加入成为伙伴
因 AI 控制程式「艾莉丝代码（アリスコード）」受到 Bug 侵蚀，使 NPC 的举止感觉真的拥有智慧……


身处游戏内的椎菜等人，目标要抵达最终迷宫「奥德赛亚」解开迈入 WO 结局的条件，为此需攻略各地的任务及迷宫。但 WO 是被 Bug 侵蚀的世界。常发生街道受 Bug 影响无法通行，或是其他游戏内无法解决的问题。

这些问题就要交给身处 “现实世界” 的新解决了。在 WO 攻略迷宫的途中，可从选单画面随时移到现实世界视点。于现实世界收集情报，找出解开游戏（WO）内谜团的线索。




线索出现在现实世界时，图示会显示在地图上
若不去调查将可能使游戏无法继续进行，千万别忽略掉了


于游戏与现实世界往返相当频繁。当现实世界出现了某种新事件，选单画面内的项目就会标记上「！」，要是卡关了就检查一下吧。

为了避免洩漏剧情无法详述，不过随着故事进行，现实世界也会开始发生各式各样的异变。当 WO 的数位世界与现实的物质世界交错，玩家将品尝到令人无法言喻的诡异感触。




现实世界裡，以新隶属的游戏研发公司「艾尼格玛公司」的同事们为首，现身各式各样的伙伴协助调查


正如开头所提到，本作剧本由祁答院慎负责撰写。正如粉丝为他起的暱称 “外道院”，写作风格惨无人道，经手作品无一不让登场人物饱受摧残，本作只要选项稍有不对，就会转眼间直落惨烈的 Bad End。

但走入 Bad End 也并非全都是坏事。本作准备了「情节图表（エピソードチャート）」功能，可浏览包含 Bad End 在内于曾经经历过的事件，也会根据经历过的事件取得对应报酬。


能马上看出情节图表是否有未解锁的内容
能从前后事件推测是在哪个桥段遗漏，打算完全回收的玩家请务必好好活用


而且许多事件都能取得强力的报酬，故意走向 Bad End 或许也是一种选择…… 不过这也表示得多看几个相当残忍的场面，玩家可得有所觉悟。


若抵达 Bad End，只要读档重新开始即可，而之前选过的选项也会打勾注记。由于不知道何时会出现致命选项，在此强烈建议，只要有机会存档就不要怠慢赶紧存，存档频率最好多到有些烦躁的地步


利用地图上的 Bug 支配战场，污染度的调节将左右战略

接着将介绍旅途中发生的战斗规则。本作战斗系统採用让参战角色依序行动的 “单位回合制”，并加入 RPG 令人熟悉的指令战斗要素。


与地图上的敌人记号接触就会展开战斗。按下 ◯ 钮攻击命中敌人记号，就会以包围敌人的状态展开战斗
相反的若被敌人从背后接触则会从不利状态开始，仔细确认地图前进吧


战斗画面。左侧会显示敌我双方所有单位的行动顺序


轮到行动顺序的角色，在战斗区域上进行自由移动后，就能从拥有的技能中输入三种指令。这就是本作最大的特徵「TriAct（トライアクト）」系统。

移动范围等部分并未多加限制，但会因应选择的技能使攻击范围产生变化，因此玩家得尽可能让越多敌人进入射程的位置。


三种行动中，能够穿插回复和攻击。例如能设定第一次行动为复活、第二次行动为回复、第三次行动为攻击，相当简单方便



选择特定技能组合，还有机会 "灵光一闪" 学到新技能
活用此一系统，就能在早期阶段习得高等级技能


不过战斗区域中会产生称为「地图 Bug（フィールドバグ）」的 Bug，接触后不但会受到伤害或陷入异常状态，碰触的角色 “污染度” 还会上升。污染度一旦累积 100％ 该角色就会无法行动，移动时需多加注意。


地图 Bug 根据颜色有着不同的效果，有些也会包含增益效果
一旦接触 Bug 就会立刻消失，可视情况主动碰触 Bug


要除去地图 Bug 最有效的就是「Bug 打击（ノックバグ）」系统。有些技能能够触发此效果，把敌人击飞。地图 Bug 接触到被击飞的敌人就会消灭掉，可锁定 Bug 密集的地带击飞敌人。

另外被 Bug 打击击飞的敌人，若接触到其他敌人或障碍物，还会反弹回来造成追加伤害。被撞到的敌人也会受到伤害被弹飞，刻意让众多敌人相互碰撞也是一种战略。


若想以 Bug 打击清除地图 Bug，最好就是让复数敌人捲入其中


被击飞的敌人若撞到自己之外的我方角色，还会发动追击技「击飞打击（ノックブロウ）」。击飞打击发动时会让敌人再度加速，也能连锁数次击飞打击给予大幅度伤害。也就是将敌人当成弹珠打。


敌人角色有 “重量” 设定，越重越不容易被击飞。要让敌人相互碰撞时，建议尽可能用较重的敌人撞击较轻的敌人，比较容易打出优异的数字


一个角色虽只能对同一敌人触发 1 次击飞打击，但只要利用其他敌人，即使是同一角色也能触发数次击飞打击。如能利用牆壁包围敌人，让敌人加速无数次抡牆无数次给予伤害。这就是 Bug 打击猛压（自创名称）！


清除一定数量的地图 Bug，新就能介入战斗区域，使用支援战斗的特殊能力「战斗干预（バトルジャック）」。

战斗干预分为，变更角色基础能力或地图 bug 性质的「代码干预（コードジャック）」、呼唤过去交战的头目支援我方的「召唤」、暂时变更战斗规则的「样式安装（インストールジャンル）」，共三种类型。

不论哪种都具有强劲的效果，但使用战斗干预不仅将消耗一次角色的行动回合，还会使已经消灭的地图 Bug 复活，需要稍微留心。


使用代码干预，将使污染值或地图 Bug 的性质产生变化。得善加活用建构整体战斗环境


随着游戏进行而能使用的「召唤」，可叫出至今为止交战过的头目。召唤出的头目光是站在场上，就会发动辅助效果强化我方能力，还会对所有敌方角色展开攻击。但是每次行动都会再次产生地图 Bug，头目也会渐渐无法控制消灭掉，若是以辅助效果为召唤目的，就算轮到了也不採取行动也是一项选择


选择「样式安装」，该回合内我方的行动将会变为别种类型的游戏
满足各游戏类型所设定的 “达成条件” 就能发动强力的追击，找出自己擅长的类型吧

万一移动中不小心踏到地图 Bug ，而使污染值升到接近全满，也不需要就此放弃。事实上累积的污染值也能拿来活用。那就是被污染到一定程度就会自动发动的「Gridge Style（グリッジスタイル）」。



发动 Gridge Style 时，各角色服装会变得十分煽情
为了欣赏这种模样刻意冲向地图 Bug 好像也不错


在 Gridge Style 下除了各种基础能力会大幅提升，还能使用角色固有的 Gridge Style 专属指令。也就是相当于本作的 “必杀技”。

变为 Gridge Style 后每回合污染度都会降低，降到一定程度后将解除变身，因此一般而言不会因为污染度到达 100％ 而无法战斗。但为了维持 Gridge Style 而过度踩踏地图 Bug 可是很危险的。另外也需要注意有些敌人会使用让污染度提升的反击技能。

是要活用代码干预施展 Bug 打击，还是要不在意污染度尽情踩踏地图 Bug 化身 Gridge Style 战斗。根据情况分别利用当下环境或擅长的战略，可说是本作最为重要的关键。



专用指令拥有压倒性的强大威力
最适合用来击倒重量级 ＆ 高 HP，难以利用 Bug 打击击飞的敌人


游玩时，不断刺痛内心的场面接连袭击而来。而游戏系统上，强力行动同时具备了明确的风险，与冒险部分中的选项一样，每个行动都让人感受到很 “沉重” 的设定。

虽然是款令人身心俱疲的作品，但也确实具备了直击 “特定客群” 好球带的要素，是款切合「加拉巴哥 RPG」最新作品之名的游戏。在此推荐给品味独特的玩家。



同时摄取可爱 ＆ 恐怖风格而大脑乱成一团，但玩家也能把它当作某种奖励喔
