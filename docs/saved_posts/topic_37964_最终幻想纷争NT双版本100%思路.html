最终幻想纷争NT双版本100%思路

darklaw233 2024-05-14 11:25编辑     8评论
不是想把FF系列大部分白了的人，不建议100%这两个，比较不肝的免费版无白金，而有正式版的白金太肝

因为站内有分别有免费版和正式版两种攻略，且大部分杯不需要再赘述，请大家按需使用

先说一些因为年代久远资料不多的基础信息，因为我也是摸索出来的可能会有出入欢迎评论指正

段位：青铜→白银→黄金→白金→秘银→钻石→坚钢→水晶
每个大阶段会有E～A的小阶段

免费版每周免费角色在每周四十一点轮换
* 光之战士/凯恩/皇帝/洋葱骑士
* 加兰德/斯考尔/高贝兹/巴茨
* 弗里奥尼尔 / 吉坦/蒂娜/艾克斯迪斯
* 黑暗之云 / 库加/凯夫卡 / 梵
* 塞西尔 / 泰达 / 阿尔提米西亚 / 拉姆扎
* 克劳德/杰克特/香托托/诺克提斯
* 萨菲罗斯 / 雷霆/雅修特拉 / ace

线上段位：青铜时大致是谁升会带动所有角色升，直到青铜A停止。然后从白银开始升角色不会带动其他角色，直到白银A升到黄金E其他角色会变成白银E，以此类推
线下段位：大致同上，只是线下水晶A就到顶，而升到水晶A只会让同样职业的角色升到坚钢E，其余职业还是钻石E

注意（不一定准）：线上段位是两版通用，但战斗计数是分开

免费版存档可以给正式版用，反之不行
正式版读免费版存档是可以通过删除本地反复读取，善用会员上传sl
有mtu800大法网战杯只剩肝

免费版最难达成杯是50称号
因为正式版宝物储存上限是99，我在试玩版68级就满了99宝物，因为试玩版没有获得宝物提醒所以要自己注意
在玩之前建议规划好拿称号路线
A.保底，雨露均沾
因正式版SS杯存在同时卡段位进行大部分角色雨露均沾，所以舍弃了单职业150胜和单角色100胜
所有角色Clv10：28
水晶A：4
召唤兽：3
游玩时间：3
道具：3
金钱：3
连续游玩：3
线上：3

坚刚E～C1400，BA1600
水晶E～C1700，BA1900
从坚刚开始打6场经验大概在800-900
就站在核心旁边无脑按O也能600-700
推荐大家shoot水晶A：皇帝、高贝扎、卡夫卡、香托托、雅修特拉
前几个都是地面O或者上空O（皇帝）单发伤害高，雅修特拉出手快伤害也看得过去。
Ace我觉得纯O刮，蒂娜和阿尔提米西亚不太好用
刷shoot水晶A顺便把其他周角色挂上先拿clv10，不会本周跳但对应周打任意一场就会跳，然后可以提前带好坚钢shoot开刷

不重要：经过我的测试高贝扎水晶A后开始带克劳德刷，蒂娜和阿尔提米西亚补了一下本周clv级（大概在坚钢B-C左右）卡夫卡和香托托水晶A后克劳德段位在水晶C，想刷其他职业也可以这么刷但功利性拿称号没必要

唯二需要注意的：召唤兽称号300胜每100选一只召唤兽用到跳称号，如果想线下召唤300次刷其余召唤兽那就留着攒条快的希瓦和利维坦
连续游玩天数：7/15/30，往后还可以再拿个60
经过本人初步测试，可以通过调系统时间来完成游玩天数的称号，每打完一场出去改日期，但不会发生轮换
但这里也是断了就不能续，经过试验拿到30天称号后开始断，再通过改日期打30天并没有60天的称号
有人可以试试（如果卡段位）保留几场每天改日期打会不会跳称号

地图称号是线上100次和线下300次，我称号刷完了免费版也没刷出来，正式版160+级才刷出一个约定之地，所以随缘

B.一门心思优先拿单角色100胜与职业150胜
3+1理论上不用肝水晶A
ss杯丢给正式版：26x3x5＝390，额外再打78场不算战斗时间纯等要等390分
通过调时间额外多拿60天称号，再2（百胜）+1（职业150胜）剩下100胜分其他角色
还有段位升太高导致比较难打问题，自己取舍

C.直接购入豪华版氪金
不用多说


我是将满99宝物plv68上传存档，再删本地读300网战
正式版最难杯是开300次宝物
没什么好说的，清完故事模式+连战，每一级给一个，雅开刷核心战，没有任何操作要求就是肝，升一级15000，一轮大概10000左右经验要升到241或247
