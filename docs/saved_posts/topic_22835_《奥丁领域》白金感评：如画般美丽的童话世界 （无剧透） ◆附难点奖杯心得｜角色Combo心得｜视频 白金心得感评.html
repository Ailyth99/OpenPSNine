白金心得感评

白金心得感评


和写白金攻略的擦擦约好了要写测评，今天下午都在思考如何撰文码字，见谅我文笔不佳，用心献上本感评
很久没有玩到这样一款让我感动无比的游戏了，香草社於2007年发售的奥丁领域，前身游戏为公主冠冕，持有香草社一贯让人惊艳的2D手绘美术风格，而这次重置版一扫以往的沈重手感，融合了胧村正的打击系统，和先前的RPG系统完美结合，打造出一款手感滑顺如丝的2D横版动作游戏
虽是重制，但其实我相信很多人都并没有玩过2007年的版本，那么你完全可以将本作视为新作，对於奖杯控来说，这次并没有太多奖杯要求，除全料理奖杯外，几乎可以说是通关即白金，相当推荐给奖杯控，唯一缺点是赶着解奖杯时可能会对重复场景感到不耐烦
本作的剧情在当年我并没有了解太多，这次重温剧情，给了我极大的震撼，这部份可以说是大幅的超越了胧村正和龙冠，打完游戏後，惆怅许久，舍不得关掉结尾工作人员名单，也不想打开其他游戏
中文版於4/26日发售，剧情的部份就留给大家自行体验，不做剧透

好吃好玩的噗卡街！


本作的音乐相当精美，由崎元仁担当音乐监督，无论是战斗场景，街道，过场动画，甚至是结尾工作人员名单，处处皆有精美的音乐，也有人在游玩体验版时因为音乐而立刻考虑入手的

哟，这样的冷天气也有半裸的美女，看来已经出现幻觉了


开头书房内的爱丽丝小妹妹，其实在每个位置都可以按来进行特殊的操作，例如抱着猫上沙发，不要进入导读模式，便可以摸猫，这些细节也都是一帧帧手绘画出的，这项优点也体现到了游戏里，游戏中的许多细节都非常细致，无论是角色跑步时、站立不动时、吃热食还会呼呼地吹、擦嘴的动作也都相当细心，能看出每个角色的个性

小心剧透：终焉後爱丽丝会开始有语音
其中有“曼陀罗草不知道好不好吃吔”
或是“看着看着不知不觉已经好困了”
“巨龙肯定是很大很大的龙”
“上次看到哪了呢”之类的



我其实非常期待中文版会如何翻译这些如童话如诗般的对白，就像胧村正充满了日本古语一样，奥丁领域则是充满了舞台剧的文艺气息，无论是反派，主角，个别人物都别具特色



请原谅我是个香草社粉丝，实在无法对奥丁领域做出什么负面评价，真要鸡蛋挑骨头也就是五个角色的故事线互相交错，重复性的场景比较多，如果刷的比较勤的，有些玩家可能会感到这五遍比较枯燥，但是我自己过程中是没有产生这样的感觉，我觉得五个角色都很好玩，最後刷终焉的时候急着白金，才有跳过一些动画

小姐，好性感呐，美女半价


打完结局，这是少数让我不想跳过片尾名单的游戏，音乐动人，而剧情仍缭绕於我心，感谢香草社在2016年初给我这么精美的游戏，想说的话太多了，哽在喉头最後说出的话只有这么少


天よ...名前を返します...
私は...メルセデス
真の名前は...ユグドラシル...


同时也非常期待香草社的下一预定作品《十三机兵防卫圈》
当然如果有机会我也很希望格林魔书的重制。公主冠冕估计比较困难，因为画面是像素水准，不知道香草社有没有经费重画一次公主冠冕。

角色Combo心得｜技能心得｜视频区

角色Combo心得｜技能心得｜视频区

首先附上我的创作视频吃货公主

贝儿贝特

贝儿贝特的套路很简单
可以，最後一下攻击是转两圈，在转圈的时候摁住+偷蓄力，转完第二圈正好武器发光蓄满（贝儿贝特会往下蹲）此时放开出地面连刺，这是一套很方便的套路，击飞能力很高
地面连刺後会有一个硬直无法接上一般攻击或是+，但此时可以发动赛法技
推荐视频中的接法用DANCING CHAIN拉回敌人，之後挑空
→ （+摁住） → 放开→ DANCING CHAIN → 等敌人飞回 → +

空中就是四次攻击，第四次是连续转圈，在转到第三圈时摁住，就能顺利出钩前冲，如果早按或晚按会接不上，中间接SPIRAL DRIVE这是伤害最高又最容易接的技能，如果POW快没了就接+再继续四次攻击，可以拖出一个空档回POW
（空中）（转到第三圈） → +摁住 → 转身 → （循环最多三次）

（空中） → SPIRAL DRIVE → （循环） → 中间可插进（+）或（+）

推荐的PP技只有一个，就是消耗6PP的FLARE SHOOT，可以给敌人上火炎状态搭配被动提高伤害，其余PP技多数华而不实
推荐优先升满上面几个技能


奥斯华德

变身前在地面可多用摔技CRAZE GRIP这技能伤害极高，一般小怪都能秒，空中连段推荐四次接+前冲再用FATAL RAID昇龙脚踢上去重复接，这是未变身的无限段接法，时机把握比较困难，中二剑还是需要变身的
（空中） → + → FATAL RAID → （循环）

BOSS战时只要多利用CRAZE GRIP穿到BOSS身後，或是PHANTOM KILLER反击技，就可以做到轻松无伤大部分BOSS
变身後连段只要持续照着视频中的做就可以永久不解除变身
→ EVIL RUSH → + → （循环）

途中如果位置太低可以用+调整高度

另外在地面上只要+不要打出第四段也能不解除变身
PP技推荐跳炎SERPENT GRIEF，还有吸引炎柱RASING VOID，後期的LUNATIC狂暴化也很实用


格温德琳打老爸

格温德琳的空中连段非常多变化，基本套路和上面两位相同，都是四下後接前冲再接连刺NEEDLE STRIKE，也可以三下後接接HIGH THRUST转上去，三次前冲或是抵达顶端之後比较容易掉下来，如果真要掉下来了可以再接FALL CLASH
格温德琳的性能算是最平均的，没有特别缺陷之处，无论空中或地面都有很好的表现，另外非常推荐地面限定的CRESCENT RISING转三圈，这个技能击飞值异常异常的高，在高难度用来挑空BOSS非常有用，挑上去後接一个之後就可以开始无限连套路
注意被动技能HP30％以下时攻击带冰冻，空中敌人被冰冻会强制落地，空中连技党注意这个技能的负面效用


兔王子
借用机因 @code_arcfj 的图

空中第四下会击落敌人，看似无法空中无限连，其实可以
挑空後，空中不要出第四刀，然後再次挑起
不要第四刀，稍微等一下，再次，稍微等一下，再次，这样高度会慢慢降低，此时用SPRIAL CLAW然後按把敌人重新推上去，再继续即可达到无限连
（空中，高度要够高） → 停一下 → → 停一下 → → （循环） → 位置降低时SPRIAL CLAW+

不过地面才是兔王子的天下，基本只要靠旋风斩ROUND BLADE+三雷球PLASMA CIRCLE或雷云THUNDER CLOUD就可以打天下了


妖精女王

PP技能中比较实用的是SPREAD SHOT散弹，贴着大型BOSS射有极高的伤害，横向清场可以用最初学到的ROUND SHOT
中期学会无限射击UNLIMITED SHAFT之後就可以展开无限连了，在BOSS失横被打到空中时，马上开启无限射击，然後用SLIP SLIDE滑行（我都设定在+）铲起来，转身跳起来後摁住，连射BOSS，可以一直射到强制落下为止
不追求COMBO也没关系，多利用打带跑，妖精是输出很高的角色
一部分身体部位比较多的BOSS，可以多用+二段蓄力打
小精灵系列技能消耗PP量和输出量不成正比…不太推荐
