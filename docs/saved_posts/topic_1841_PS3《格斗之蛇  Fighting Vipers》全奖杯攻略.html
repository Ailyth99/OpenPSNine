Ranking Mode
ARCADE模式下进行排位赛（在アドバタイズ/广告中按着P+K+G的同时摁START键）
广告即是选好难度和时间后看到的SEGA标志，此时按着P+K+G的同时摁START键即可。要被电脑打败后才可得到奖杯
ckyd88 2014-04-16 23:18
Random Mode
ARCADE模式下在广告中按住↑的同时摁START键
同上一个奖杯，不过这个是开打之后即可得到，拿完退出即可
ckyd88 2014-04-16 23:19
Kids Mode
ARCADE、OFFLINE VERSUS模式的游戏设定中按住L1键+R1键的同时决定
在ARCADE选好难度和时间后按住L1+R1再按确定即可，小等一会就有奖杯
ckyd88 2014-04-16 23:21
Wall Breaker
ARCADE、OFFLINE VERSUS模式中破坏墙壁
选Mahler按住前然后用△三连击，打败对方后墙壁会自动破坏，可以和完美无伤奖杯一起拿比较快
ckyd88 2014-04-16 23:25
Armor Destroyer
ARCADE、OFFLINE VERSUS模式下破坏对方的护甲
右上角血条旁边有个人物图标，当图标上半身开始闪烁时用←P或←K、图标下半身开始闪烁时用奔跑+↓+K，两种选其一即可
ckyd88 2014-04-16 23:28
