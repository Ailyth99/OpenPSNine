PSV《Run Sackboy! Run!》快速获取泡泡的方法

hcl 2015-04-05 20:24编辑     53评论
基本上, 这款游戏如果不课金想要达到 100% 奖杯, 是很困难的事. 光是要用游戏内正常的方法来, 买足10套服装就够你头痛了. 平均一次游戏大概可以获得约700-1000个泡泡, 服装的部份最少要 35万个泡泡才可以买齐, 这还不包括两套要透过买宝箱获得特殊硬币才可以解锁的特殊服装.

废话说完了. 首先, 你必须要有PS+帐号. 然后请在游戏中获得大概 10K-20K的泡泡. 你可以先不要购买任何的道具. 先取得足够的泡泡就好. 然后请将存档上传到云端, 这也就是为何需要PS+帐号的原因. 接着, 请到商店中先购买500泡泡的小宝箱. 这个宝箱最大奖为75000个泡泡. 如果你泡泡用完了,
还没抽到大奖, 请到云端中重新下载存档, 再试一次.

当你在小宝箱抽到大奖后, 立刻再把存档上传到云端. 接下来请改买价值2500的大宝箱. 同样的方法, 抽到最大奖150000泡泡时把存档上传. 不过大宝箱中有许多种不同的道具例如两种特殊金币, 救救我爱心, 贴纸跟特殊贴纸, 有很大的机会是你拿到一堆这些特殊道具把泡泡用完了, 可是还没抽到大奖.
一样从云端下载存档. 如此一来, 就可以快速的累计泡泡.

当你有一定数量泡泡时, 先存档上传, 然后去购买其中一种的强化道具买到满, 等奖杯跳出后. 从云端下载存档, 换购买服装.

游戏中总共有 11 套服装可以购买, 其中有两套必需要用特殊金币换取, 当你在抽足够泡泡买其他服装的时后, 这两套特殊服装应该会先拿到. 然后其中一套是一次跑分中获得10000的分数. 这套应该也很好拿, 言下之意就是你只需够买另外7套服装就好. 就是说游戏中最贵的那套(价值350K)可以不需要买, 就可以拿到奖杯.

这套游戏如果要用课金的方法来快速完成游戏, 花费的金额其实挺可观的.
