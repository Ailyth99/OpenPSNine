《实况足球2018》将于今年9月14日发售，还将推出「巴塞罗那特别版」

sakura_akiho 2017-05-17 15:10编辑     74评论
KONAMI公布将发售其实况足球系列最新作《实况足球2018》（PS4/PS3）。
目前数字版预约已经在PSN商店开启。还将推出包含特殊道具的“巴塞罗那特别版”（仅PS4版本）。







预约期间（2017年5月17日～2017年9月13日）内购买将获得如下特典：

实况足球2018 普通版

・实况足球2018 原创主题
・合作俱乐部 Agent
・欧冠 Agent
・myClub 高级Agent

实况足球2018 巴塞罗那特别版

・巴塞罗那俱乐部 原创主题
・巴塞罗那俱乐部 Agent
・巴塞罗那俱乐部 传奇Agent
・1,000 myClub代币
・合作俱乐部 Agent
・欧冠 Agent
・myClub 高级Agent
