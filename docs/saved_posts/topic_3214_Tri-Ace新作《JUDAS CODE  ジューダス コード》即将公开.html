Tri-Ace新作《JUDAS CODE | ジューダス コード》即将公开

mechille 2014-07-15 22:15编辑     1评论
～本週ファミ通杂志情报～

Tri-Ace 在 2014 年 7 月 17 日出刊的週刊 Fami 通杂志中，公布了将于
PS Vita 上推出下载专用游戏《JUDAS CODE | ジューダス コード》，预定夏季
上架推出，采用基本游玩免费、道具商城制方式营运。

本作是款戏剧性战场 RPG 作品，故事描写在第 3 次世界大战期间，大地被
从宇宙飞来的神秘柱体「Longinus（ロンギヌス）」所侵蚀，而新人类们以这个
被破坏殆尽的世界为舞台展开了存亡之战。

◆ 杂志内容摘要
主要游戏内容分为两大部份：单机的枪械动作 RPG，与线上的卡片战斗
枪械动作部份採用与《生化危机4》相同的第三人称肩后视点
在世界各城市展开丰富多样的任务，进而推进故事剧情
使用完成任务所获得的报酬来抽卡，编辑牌组来强化战力
角色演出声优有：花泽香菜、堀江由衣
由知名插画家负责绘製卡片的插图
主要角色设定由「凪良氏」负责担任（作品：魔塔大陆系列）
开发进度 80％
